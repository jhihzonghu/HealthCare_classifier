吴传芳与中国石化集团中原石油勘探局医疗损害赔偿纠纷上诉案

审判时间：20110509 法院：河南省濮阳市中级人民法院 案件字号：(2011)濮中法民一终字第21号


　　上诉人（原审原告）吴传芳。
　　法定代理人崔发胜。
　　委托代理人张凤川。
　　委托代理人裴英伟。
　　上诉人（原审被告）中国石化集团中原石油勘探局。
　　法定代表人孔凡群，该局局长。
　　委托代理人吴方勇。
　　原审被告濮阳市油田总医院。。
　　上诉人吴传芳与上诉人中国石化集团中原石油勘探局、原审被告濮阳市油田总医院医疗损害赔偿纠纷一案，不服濮阳市华龙区人民法院（2010）华法民初字第2235号民事判决，分别向本院提起上诉，本院依法组成合议庭审理了本案，现已审理终结。
　　原审法院审理查明，吴传芳于2006年2月10日在中原油田总医院住院进行上颌窦囊肿切除术，术中对吴传芳实行全身麻醉后，其一直处于昏迷（植物生存）状态。吴传芳起诉至法院，起诉状要求赔偿1843592元，后增加诉讼请求，其赔偿清单共主张了六项损失，计2698133元：一、住院期间的经济损失，包括住院伙食补助费、护理费、误工费、营养费、交通费，共计164013元；二、出院后的经济损失，包括后续治疗费、护理费，共计2148570元；三、残疾赔偿金264620元；四、助残工具费用经济损失12250元；五、司法鉴定费8680元；六、精神损失费100000元。审理中，经报请濮阳市中级人民法院委托，2007年9月3日法大法庭科学技术鉴定研究所作出了法大（2007）医鉴字第490号法医学鉴定意见书，确认中原油田总医院的诊疗行为存在不当，与吴传芳的目前后果之间具有部分因果关系，参与度为D级，理论系数值50%。并经吴传芳申请，报濮阳市中级人民法院委托，2009年4月14日，安阳威校法医临床司法鉴定所作出安威校司鉴所（2009）临评字第812号司法鉴定意见书，确认吴传芳的伤情构成一级伤残。同日，安阳威校法医临床司法鉴定所作出安威校司鉴所（2009）临评字第44号司法鉴定意见书，评估意见为：吴传芳维持生命、继续治疗费每日550元左右；助残工具费2250元至2650元，3至5年更换一次。案件经过一审、二审和发还重审，2009年6月30日，经本院主持，吴传芳和濮阳市油田总医院达成调解协议，并制作了（2009）华法民初字第1104号民事调解书，调解书内容：一、濮阳市油田总医院自2008年1月22日吴传芳出院后，每五年为一个阶段向吴传芳支付后续治疗费及护理费，每次支付办法为一次性付清；二、濮阳市油田总医院向吴传芳支付其第一个阶段（期限为自2008年1月22日起至2013年1月21日止）的后续治疗费、护理费及住院期间的各项经济损失600000元，于调解书生效后30日内一次性支付完毕，该600000元包含濮阳市油田总医院于2009年1月先予支付的80000元；三、第二、第三等阶段的后续治疗费及护理费问题，到期后双方另行协商；四、吴传芳在住院期间发生的医疗费等各项费用问题，双方不再有任何争执；五、濮阳市油田总医院退回吴传芳住院期间的押金9000元，但吴传芳必须协助濮阳市油田总医院将其住院期间的医疗费209614元予以处理，吴传芳提供统筹医疗报销所需的医疗本、卡及其他相关材料，吴传芳不承担医疗报销过程中的相关费用；六、濮阳市油田总医院承担本案的鉴定费8680元和诉讼费19380元。调解书已经发生法律效力。在本案审理过程中，中国石化集团中原石油勘探局认为吴传芳经过精心护理和康复治疗，有可能苏醒，伤残等级会发生变化，目前应减轻至八级伤残或更轻，并以此为由申请对吴传芳目前的伤残等级进行重新鉴定，吴传芳认为中国石化集团中原石油勘探局未能提供相关证据，不同意鉴定。本院报请濮阳市中级人民法院后，案件被退回，未能鉴定。
　　原审法院另查明，中原油田总医院系中国石化集团中原石油勘探局举办、经河南省卫生厅批准的三级甲等医院。根据国家关于国有大中型企业主辅分离改制分流富余人员的精神和要求，中原油田总医院于2006年11月改制，改制后成为由医院全体职工和中国石化集团中原石油勘探局共同出资的非企业法人实体，纳入濮阳市行业管理，使用名称为濮阳市油田总医院。同年11月23日和24日，河南省卫生厅和民政厅对濮阳市油田总医院分别颁发了医疗机构执业许可证和民办非企业单位登记证书。2006年12月1日，中国石化集团中原石油勘探局下发中油局人资（2006）324号文件，撤销了中原油田总医院，并对中原油田总医院债权债务已全部清理，人员已妥善安置。
　　原审法院认为，中原油田总医院在吴传芳住院治疗期间，存在不当行为，和吴传芳的损害后果具有部分因果关系，对由此给吴传芳造成的损失，应当根据司法鉴定结果，承担50%的赔偿责任。由于中原油田总医院经改制已被中国石化集团中原石油勘探局撤销，根据中国石化集团中原石油勘探局下发的中油局人资（2006）324号文件，中国石化集团中原石油勘探局对中原油田总医院的全部债权债务负有清理的义务，因此，中国石化集团中原石油勘探局应对中原油田总医院给吴传芳造成的损失承担赔偿责任。濮阳市油田总医院系2006年11月新设立的非企业法人实体，并未承接中原油田总医院的债权债务，故吴传芳要求濮阳市油田总医院承担赔偿责任，没有事实根据和法律依据。中国石化集团中原石油勘探局辩称吴传芳主张的损失已经由本院调解处理，应根据一事不再理的原则，驳回吴传芳的起诉。经审查本院（2009）华法民初字第1104号民事调解书，并未涉及吴传芳本次主张的伤残赔偿金、残疾器具辅助费和精神损害抚慰金，调解书调解结果第四条写明：“原告在被告处住院期间发生的医疗费等各项费用问题，双方不再有任何争执”，也明确是对住院期间的费用不再争执。对于未能达成调解协议、也未明确表示放弃的事项，吴传芳重新起诉并不违背法律禁止性规定，中国石化集团中原石油勘探局应当赔偿吴传芳以上三项损失。关于中国石化集团中原石油勘探局的重新鉴定申请，本院认为，在原审中最后一次司法鉴定确定吴传芳为一级伤残的时间是2009年4月14日，距今时间不长，中国石化集团中原石油勘探局没有证据证明吴传芳伤情发生了变化，不能仅仅因为吴传芳不同意鉴定，就推定由吴传芳承担鉴定不能的后果，在没有新证据的情况下，仍应当按照原鉴定的一级伤残确定吴传芳伤残等级。关于吴传芳的残疾赔偿金，根据吴传芳伤残等级，按照河南省2009年度城镇居民人均可支配收入确定为143720元；吴传芳的残疾器具辅助费，根据司法鉴定情况和中原油田总医院的过错程度，暂时保护一次的费用计1325元；精神损害抚慰金根据吴传芳伤情、中原油田总医院的过错程度和司法实践掌握的标准确定为25000元。综上，根据《中华人民共和国民法通则》第一百一十九条，《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》第二十五条、第二十六条，《最高人民法院关于确定民事侵权精神损害赔偿责任若干问题的解释》第十条之规定，原审法院判决：一、中国石化集团中原石油勘探局赔偿吴传芳残疾赔偿金143720元、残疾器具辅助费1325元、精神损害抚慰金25000元，共169595元，于本判决生效后10日内履行完毕；二、驳回吴传芳的其他诉讼请求。案件受理费4230元，由中国石化集团中原石油勘探局负担。吴传芳和中国石化集团中原石油勘探局均不服原审判决，分别向本院提起上诉。
　　吴传芳上诉称，吴传芳依照法律规定主张的残疾器具辅助费和精神损害抚慰金，原审法院未全部支持不当，请求二审法院依法撤销原审判决，改判支持吴传芳的全部诉讼请求。
　　中国石化集团中原石油勘探局上诉称，原审判决认定“（2009）华法民初字第1104号民事调解书，并未涉及吴传芳本次主张的残疾赔偿金、残疾器具辅助费、精神损害抚慰金”，属认定事实错误。理由如下：1、在（2009）华法民初字第1104号一案中，吴传芳的诉讼请求已包含了上述三项损失。2、法庭调解是对吴传芳所主张的全部损失一并处理。3、调解书的内容亦明确包含了吴传芳主张的该三项损失。综上，原审判决认定事实错误，判决中国石化集团中原石油勘探局承担赔偿责任缺乏事实根据和法律依据，请求二审法院依法撤销原审判决，改判驳回吴传芳的诉讼请求，或者将本案发回重审。本案诉讼费用由吴传芳承担。
　　濮阳市油田总医院未到庭，亦未书面陈述意见。
　　本院经审理查明的事实与原审查明的事实一致。
　　本院认为，吴传芳与中原油田总医院发生医疗纠纷后，虽然吴传芳在濮阳市华龙区人民法院审理的（2009）华法民初字第1104号案件中，主张了残疾赔偿金、残疾器具辅助费、精神损害抚慰金，但该案调解书内容并不显示已对吴传芳的上述三项诉讼请求进行了处理，也不显示吴传芳放弃该三项诉讼请求。故吴传芳就残疾赔偿金、残疾器具辅助费、精神损害抚慰金重新提起诉讼，并不违反一事不再理原则，原审判决支持吴传芳的诉讼请求并无不当。关于吴传芳称原审判决未全部支持其诉讼请求错误的上诉理由，本院认为，原审法院根据鉴定意见及本案实际情况，对吴传芳的残疾器具辅助费及精神损害抚慰金具体数额作出的认定，有事实和法律依据，吴传芳要求全部支持其诉讼请求的上诉理由不成立，本院不予支持。综上，原审认定事实清楚，适用法律正确，应予维持。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费4162元，由吴传芳负担470元，由中国石化集团中原石油勘探局负担3692元。
　　本判决为终审判决。
审判长郭英涛
　　审判员刘道芹
　　审判员李瑞玲
　　二○一一年五月九日
　　书记员王世俭
