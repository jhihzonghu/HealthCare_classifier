开封市鼓楼医院与赵桂荣医疗损害纠纷上诉案

审判时间：20110525 法院：河南省开封市中级人民法院 案件字号：(2011)汴民终字第536号


　　上诉人（一审被告、反诉原告）开封市鼓楼医院。
　　法定代表人李保成，该院院长。
　　委托代理人靳祥钰、张学坤，河南三友律师事务所律师，系一般代理。
　　被上诉人（一审原告、反诉被告）赵桂荣。
　　委托代理人杨守义。系特别授权。
　　上诉人开封市鼓楼医院（以下简称鼓楼医院）因与被上诉人赵桂荣医疗损害纠纷一案，不服鼓楼区人民法院做出的（2007）鼓民初字第686号民事判决，向本院提起上诉，本院受理后，依法组成合议庭审理了本案，现该案已审理终结。
　　一审查明，原告赵桂荣的丈夫马天喜(1937年12月23日生）长期患xxx病并患有细菌性肺炎、Ⅱ型呼吸衰竭和肺性脑病。2006年3月23日，原告赵桂荣的丈夫到被告在延庆街开办的鼓楼区开封府社区卫生服务站治疗，在输液过程中出现胸闷、呼吸困难，隧打120急救电话，经开封市第一人民医院抢救无效死亡。双方就此发生纠纷，原告诉至本院，要求被告赔偿丧葬费14208元、死亡赔偿金158092元，共计172300元。被告提出反诉，要求原告返还补偿金26000元。案件审理中，被告提出申请，要求对被告的医疗行为是否有过错及马天喜的死亡与被告的医疗行为是否有因果关系进行鉴定，因马天喜尸体已经火化，鉴定在技术上失去了可操作性，鉴定部门无法鉴定；被告又提出申请要求对被告在为马天喜的诊疗行为中是否有过错进行医疗事故鉴定，因原告方对被告方所提供的病历真实性不认可，医学会中止受理该鉴定。另查明，马天喜尸体于2006年3月29日火化。
　　一审认为，根据最高人民法院《关于民事诉讼证据的若干规定》第四条第一款第（八）项针对医疗纠纷举证责任的规定，本案应由被告开封市鼓楼医院就其医疗行为与损害结果之间不存在因果关系及不存在医疗过错承担举证责任。在诉讼中，被告提出就被告的医疗行为是否有过错及马天喜的死亡与被告的医疗行为是否有因果关系的问题应当进行司法鉴定加以确认，因尸体已经火化，进行鉴定已失去了技术上的可操作性；诉讼中，被告又申请为其在马天喜的诊疗行为中是否有过错的医疗事故鉴定，开封市医学会受理后，因被告所提供门诊病历与其为原告出具处方后所写内容不一致，且该门诊病历本应由患者持有而却由医院一方保管，原告对病历真实性提出异议，医学会中止鉴定。被告未能举证证明其医疗行为与损害结果之间不存在因果关系及不存在医疗过错，其应承担举证不能的法律后果，对于马天喜的死亡结果应承担民事责任。因在马天喜死亡后原告没有及时申请尸检，即将尸体进行了火化，使鉴定失去了技术上的可操作性，原告应承担相应的责任，故对于原告要求被告赔偿丧葬费、死亡赔偿金共计172300元的主张，本院根据案情及双方过错责任酌定被告承担50%，即86150元，其余本院不予支持。对于被告要求原告退还26000元的反诉请求，因被告所提供证据不足以证明其主张，原告亦不予认可，故对于被告的反诉请求，本院不予支持。根据《中华人民共和国民法通则》第一百一十九条、《最高人民法院〈关于审理人身损害赔偿案件适用法律若干问题的解释〉》第十七条第三款、第二十七条、第二十九条、《中华人民共和国民事诉讼法》第六十四条、《最高人民法院〈关于民事诉讼证据的若干规定〉》第四条第一款第（八）项之规定，判决：一、本判决生效后十日内，被告开封市鼓楼医院赔偿原告赵桂荣丧葬费14208元（按河南省2009年度职工月平均工资标准计算6个月）、死亡赔偿金158092元（按河南省2009年度城镇居民人均可支配收入计算11年），共计172300元的50%，即86150元；二、驳回原告赵桂荣其它诉讼请求；三、驳回被告开封市鼓楼医院反诉请求。
　　一审宣判后，鼓楼医院不服，向本院上诉称：1、上诉人对马天喜的诊疗是正确的，马天喜的死亡是其自身病情造成，与上诉人没有任何关系，一审法院在没有证据的情况下，让上诉人承担责任是错误的；2、本案属于医疗纠纷，既然让上诉人承担特殊的举证责任，那么就应适用《医疗事故处理条例》，而一审法院适用《民法通则》及最高法院《关于审理人身损害赔偿案件适用法律若干问题解释》系适用法律错误。3、被上诉人方于2006年3月25日收到上诉人赔偿款26000元，应予认定，一审不予认定是错误的。综上，请求二审法院依法改判。
　　被上诉人赵桂荣答辩称：上诉人医务人员非法行医造成马天喜的死亡，一审法院仅让上诉人承担50%责任是不当的，但被上诉人为了尽快得到赔偿，才没有上诉，希望二审法院依法做出公正判决。
　　二审查明，马天喜于2007年3月23日死亡后，在鼓楼区卫生局协调下，上诉人方三人与被上诉人之子及儿媳五人共八人于3月25日在开封宾馆再次就赔偿事宜协商，上诉人向被上诉人方支付赔偿款26000元。
　　其它查明事实与一审查明的基本事实一致。
　　本院认为，因医疗行为引起的侵权诉讼，应由医疗机构就医疗行为与损害结果之间不存在因果关系及不存在医疗过错承担举证责任。本案中因上诉人鼓楼医院提供的门诊病历有瑕疵而导致鉴定不能，是其不能提供其医疗行为与损害结果之间是否有因果关系及是否有医疗过错证据的原因之一，所以一审让其承担50%的责任并无不当。对于上诉人称一审适用法律错误问题，本院认为，最高法院《证据规则》中对医疗行为引起的侵权诉讼的举证规则作出了特殊规定，该案属于医疗损害纠纷，应适用该举证规则规定，同时也应适用人身损害赔偿的相关法律规定，一审适用法律并无不当。关于上诉人称26000元已支付给被上诉方的问题，该事实在（2007）汴民终字第560号卷中，证人杨文胜、赵改娥、张文忠出庭作证的证言能够证明，对于上诉人的该上诉理由本院予以支持。关于被上诉人辩称张珠莲非法行医问题，首先，张珠莲提供了其执业证书，其次，赵桂荣未提起上诉，故对赵桂荣该辩称理由本院不予支持。一审判决认定事实部分有误，应予部分改判。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）、（三）项之规定，判决如下：
　　一、维持鼓楼区人民法院（2007）鼓民初字第686号民事判决第一、二项；
　　二、撤销鼓楼区人民法院（2007）鼓民初字第686号民事判决第三项；
　　三、赵桂荣于判决生效后十日内返还开封市鼓楼医院已支付的赔偿款26000元。
　　上述双方支付相抵后，上诉人开封市鼓楼医院应向赵桂荣支付赔偿款计60150元。
　　如未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　一、二审案件受理费各4640元，由上诉人鼓楼医院各承担1795元，由被上诉人赵桂荣各承担2845元。
　　本判决为终审判决。
审判长郭宝霞
　　代理审判员　胡云鹏　
　　代理审判员李翠莲
　　二○一一年五月二十五日
　　书记员李新广（兼）
　　书记员刘莉敏（兼）
