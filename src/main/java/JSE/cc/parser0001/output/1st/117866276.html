何文正等诉郴州市第一人民医院医疗损害赔偿纠纷案

审判时间：20110303 法院：湖南省郴州市北湖区人民法院 案件字号：(2010)郴北民二初字第1399号


　　原告何文正。
　　原告何毅力。
　　原告何智力。
　　原告何红艳。
　　被告郴州市第一人民医院。
　　法定代表人陈亚光。
　　委托代理人邓旸。
　　原告何文正、何毅力、何智力、何红艳诉被告郴州市第一人民医院医疗损害赔偿纠纷一案，本院于2010年7月19日立案受理，依法组成合议庭，公开开庭进行了审理。四原告及被告的委托代理人到庭参加诉讼。本案现已审理终结。
　　原告何文正、何毅力、何智力、何红艳诉称：受害人邱秀英于2009年3月11日因复视和咀嚼无力亲自到被告处看病并住入神经科。当日下午17时，经心电图测量，结果为ST段低压并明确告知为异常心电图，同年3月12日心肌酶检测报告：肌酸激酶（CK）升高至3081，与正常最高参考值195对比超出15.8倍，肌酸激酶同工酶（CK-MB）升高至89，与正常最高参考值25对比超出3.56倍，肌红蛋白（MB）升高至3162.8,与正常参考值85对比超出37.2倍，乳酸脱氨（DH）、谷丙转氨酶（ACL）、谷草转氨酶（AST）全都异常的升高；同日彩超诊断报告明确告知：左心室舒张功能减低。上述检查结果均是心肌梗死疾病的征兆，特别是肌酸酶同工酶（CK-MB）异常是诊断心肌梗死的标志之一。可是被告对上述检查结果听之任之，既不请心内科的专科医生前来会诊，也不对上述异常结果采取任何治疗措施。从而导致3月14日凌晨病情加重，心肌大面积坏死。这时才请心内科医生确诊为心肌梗死。因被告延误诊治导致受害人于2009年3月24日医治无效死亡。上述事实经省市两级医学会，鉴定为一级甲等医疗事故。受害人是称职的妻子、慈祥的母亲。与原告何文正和睦相处相依为命几十年。2009年3月11日受害人患病时，原告及时将其送往医院并不辞辛苦日夜陪伴以救治，不料不负责任的被告错过治疗时机而死亡，造成原告极大的悲伤，特别是原告何文正天天回想，为什么选择第一人民医院，为什么要完全相信主治医生而自责，由此叹息终日，痛不欲生。在我们要求医方依法赔偿以求安慰时，被告竟然说只赔1.5万元,这没有人性的语言.因此提起诉讼。2009年3月11-12日的检查结果已非常明确地显示受害人患有心急梗塞；可是被告竟然对受害人的加查结果听之任之，从而耽误治疗导致患者心细胞大量坏死。这是对受害人的生命健康极不负责任的态度，是对受害人生命权的侵害。原告经与被告协商无果，为维护原告的合法权益特起诉至法院，请求法院判令被告赔偿原告医疗费、住院伙食补助费、陪护费、丧葬费等各项费用共计152499.8元（各项费用40%即70299.80元，加上鉴定费2200元，精神损害抚慰金80000元，合计152499.80元），诉讼费由被告承担。
　　为支持其诉讼请求，原告向本院提交如下证据：
　　1、常住人口登记，证明原告何文正与患者是夫妻关系。
　　2、郴州市北湖区下湄桥街道同辉社区居委会证明，证明原告何毅力与患者是母子关系。
　　3、身份证，证明患者邱秀英年龄及其他自然状况。
　　4、郴州市第一人民医院住院病案首页及死亡记录，证明患者邱秀英住院时间及检查诊断治疗全过程。
　　5、心电图报告、生化检验报告单、彩超诊断报告单证明2009年3月11日至12日心梗疾病诊断指标（标准）已向被告医生明确指示。
　　6、1.5T磁共振扫描报告、影像学检查报告单证明2009年3月12日磁共振（MRI）检查患xxx病。
　　7、住院医药费收据，证明患者住院治疗所用医药费9694.5元。
　　8、收据，证明购坟地款1600元。
　　9、收据，证明办丧事所有的其他费用11693元。
　　10、车费票据、住宿费发票，证明因医疗事故鉴定所用车费及住宿费共计935元。
　　11、鉴定费发票，证明医疗事故鉴定费为2200元。
　　12、郴州医鉴[2009]34号医疗事故技术鉴定书及湘医鉴[2009]105号医疗事故技术鉴定书，证明患者邱秀英死亡是被告医疗事故造成的。
　　13、协商情况，证明2009年12月31日原告向被告提出赔偿。
　　14、来访事项告知单（复印件），证明原告请求有关部门要求被告赔偿。
　　15、病理学的有关复印资料，证明有关心细胞损伤坏死的自然规律，只是作为参考资料供法庭参考。
　　被告郴州市第一人民医院辩称：患者邱秀英因为复视及咀嚼无力来医院就诊。因邱秀英曾因膀胱癌做过手术，医生怀疑是不是癌细胞转移，故给患者做了必要的检查，检查结果出来后也请相应的科室进行了会诊。患者在治疗期间也没有说其胸闷、胸疼等心梗的典型症状。2009年3月14日凌晨病人病情发生变化以后转进心内科，经心内科医生处理后病人病情好转。3月16日病人病情再次加重，发现病人大面积心肌梗死，医院也对其进行积极的抢救，但是病人的病情一直不稳定，直至其不幸死亡。病人的死亡的原因是因为病人本身的疾病所致，医院认为在治疗过程中不存在过错，但医院愿意按医疗责任的划分承担相应责任，原告要求赔偿过高，请法院公正判决。
　　被告未向法庭提交证据。
　　经本院组织双方当事人举证、质证，被告对证1、2、3、4、5、6、7的真实性没有异议。但认为医院不应承担责任，对8、9的真实性被告不予认可，因为不是正式的发票。对证10的真实性没有异议，请求法院按法律规定在人数上进行核减。被告对证11、12的真实性没有异议。但是对于鉴定结论被告不认可，被告认为在治疗过程中不存在过错。对13、14、15被告认为与本案没有关系，另外证15不属于证据范围。
　　根据双方当事人举证、质证情况，本院认证如下：对原告所举证1-7因被告对其真实性无异议，故证据1-7本院予以采信，可以作为本案定案依据。对8、9号证据因为不是正式的发票，故不作为定案的依据，但本院会根据本案的实际情况酌情考虑其数额。对证据10、11因被告对其真实性没有异议，本院予以采信，可以作为定案的依据。证据12虽然被告方认为对于鉴定结论不认可，但是被告既未举出相关的证据予以反驳，也未在规定的时间内向法院提出重新鉴定，故对该证据本院予以采信，可以作为本案的定案依据。证据13、14、15与本案没有关联性，不作为本案定案的依据。
　　根据双方当事人举证、质证，结合本院认证，双方当事人陈述及庭审情况，本院确认如下案件事实：
　　患者邱秀英（女，72岁，系原告何文正之妻，原告何毅力、何智力、何红艳之母）因咀嚼肌无力两个月、复视一天于2009年3月11日入被告郴州市第一人民医院神经内科住院治疗。邱秀英既往有膀胱癌并手术切除病史，无心脏病史。入院体查：血压129/64mmHg，心率102次/分；神清合作，双肺呼吸音粗，心音正常；腹部检查无异常发现；右睑稍下垂，眼裂变小，双咽反射迟钝。入院诊断：咀嚼无力及复视查因。辅助检查：11日心电图检查提示：窦性心率，ST段压低；12日，血清酶检查回报：LDH742.05U/L、CK3081.0U/L、CK-MB89.0U/L、Mb3162.8ng/ml;胸部CT检查提示局灶性肺部感染；TCD检查提示：高阻型脑动脉硬化血流频谱改变；头部MRI未见明显异常。因邱秀英曾因膀胱癌做过手术，医生怀疑是癌细胞转移，故给邱秀英做了必要的检查，检查结果出来后于3月13日请相应的科室进行了会诊。2009年3月14日凌晨约零时45分，邱秀英突发胸闷、心悸，急请消化内科、眼科、泌尿外科医师会诊，考虑急性冠脉综合症，经扩冠、抗凝治疗后病情缓解。3月15日上午9时转心内科继续治疗，经心内科医生处理后病人病情好转。3月16日19时10分，邱秀英病情加重，出现呼之不应，意识丧失，血压下降，经抢救后生命体征平稳，意识恢复。3月18日约晚6时，邱秀英病情再次出现呼之不应、昏睡，心电图检查示急性广泛前、下壁心肌梗死，告病危，转入ICU病房进行抢救，经多日抢救无效，邱秀英于3月24日死亡。后经郴州市医学会、湖南省医学会鉴定，专家组认为，邱秀英的死亡原因为大面积心肌梗死，邱秀英入住神经内科时既往无心脏病史，临床症状不典型，但心电图有S-T改变及酶学异常增高，对异常结果考虑欠全面、欠及时，从而未能及时诊断心梗，在一定程度上延误治疗，医方存在过错。转心内科后诊疗恰当，处理合理。邱秀英死因为急性心梗、循环功能衰竭。其死亡与自身高龄、急性心梗合并肺部感染、既往膀胱癌手术治疗等基础疾病有主要因果关系，与医方过错也有一定关系。所以在神经内科对心电图异常、心肌酶学异常结果分析欠全面，最终邱秀英死亡与早期诊断和治疗有一定的关系。故该次医疗事件构成一级甲等医疗事故，医方负轻微责任。
　　另查明，原告在被告处共住院14天，期间共花费1、医疗费9694.5元，2、住院伙食补助费为12元/天ｘ14天=168元,3、护理费为26008÷12÷30×14=1011元,4、丧葬费为2167×6=13002元,5、住宿费317元,6、交通费518元,7、死亡赔偿金为15084元ｘ9=135756元,1+2+3+4+5+6+7=160466.5元；医疗事故技术鉴定费用2200元.上述各项费用合计162666.5元。
　　本院认为，本案属医疗损害赔偿纠纷。本案原、被告对各项赔偿的计算标准和依据均无异议，只是对应承担的责任比例有分歧。被告认为原告要求承担40%的比例及赔偿8万元精神损害抚慰金共计赔偿152499.8元的要求过高，按轻微责任计算，最高不超过5%，提高点比例按10%左右计算，也只应赔偿15000元。而原告方因痛失伴侣和母亲，则认为即使要求被告全部赔偿亦不过分，故双方酿成纠纷诉至本院。根据湖南省医学会专家组的结论，邱秀英的死亡结果与其自身高龄、急性心梗合并肺部感染、既往膀胱癌手术治疗等基础疾病有主要因果关系，但若在邱秀英入院之时，被告神经内科的医生能认真分析心电图的检查结果，及时正确的做出诊断，积极对症治疗，即使后来邱秀英的死亡结果仍无法避免发生，但至少被告无责，原告方亦不会怪罪于医方。现鉴定结论虽为医方负轻微责任，被告认为相对于主要责任、次要责任，轻微责任是很小的责任，故只愿承担赔偿金额很小的比例。被告的辩称于法有据，但于情终有所欠缺。原告何文正晚年丧偶，另三原告丧母，虽说邱秀英患有癌症且已高龄，与亲人的聚散终有时，但离别的这一天作为亲人的原告，总是希望越晚到来越好。于情于理，医院均应适当提高赔偿金额。经本院多次调解，但双方终未能达成一致意见。故本院适当调高被告应承担的比例，该比例以20%为宜（即160466.5元×20%=32093.30元）；另医疗事故鉴定费2200元，精神损害抚慰金10000元，亦由被告承担。上述款项合计44293.30元。综上所述，依照《中华人民共和国民法通则》第五条、第一百零六条第二款、第一百一十九条、第一百三十四条第一款第七项，《中华人民共和国民事诉讼法》第一百二十八条，《最高人民法院关于参照〈医疗事故处理条例〉审理医疗纠纷民事案件的通知》第三条之规定，判决如下：
　　一、被告郴州市第一人民医院赔偿原告何文正、何毅力、何智力、何红艳医疗费、住院伙食补助费、护理费、丧葬费、住宿费、交通费、死亡赔偿金以及医疗事故技术鉴定费、精神损害抚慰金等各项费用合计44293.30元，该款限被告于本判决书生效后十日内支付完毕。
　　二、驳回原告何文正、何毅力、何智力、何红艳其他诉讼请求。
　　如果被告郴州市第一人民医院未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　本案诉讼费3350元，由原告何文正、何毅力、何智力、何红艳承担1670元，由被告郴州市第一人民医院承担1680元。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于湖南省郴州市中级人民法院。
审判长石纳新
　　审判员郭蓉
　　人民陪审员段小明
　　二○一一○年三月三日
　　代理书记员邓淑娟
