朱A与复旦大学附属华山医院医疗损害赔偿纠纷案

审判时间：20110510 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第733号


　　上诉人(原审原告)朱A。
　　法定代理人朱B。
　　被上诉人(原审被告)复旦大学附属华山医院。
　　法定代表人丁某，院长。
　　委托代理人童剑云、黄承瑶，上海市康正律师事务所律师。
　　上诉人朱A因医疗损害赔偿纠纷一案，不服上海市静安区人民法院(2010)静民一(民)初字第3343号民事判决，向本院提起上诉。本院依法组成合议庭,公开开庭审理了本案。上诉人朱A及其法定代理人朱B、被上诉人复旦大学附属华山医院(以下简称华山医院)委托代理人黄承瑶到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，2003年7月11日原告因车祸致头部外伤，2004年2月23日转至华山医院住院治疗，经检查诊断为脑积水，次日在全麻下行脑室-腹腔分流术。分流管由Medtronic公司制造，两天后检查“阀门弹起有力”，同年3月4日出院。2006年10月24日原告入住中国人民解放军第一零五医院进行治疗，同年11月6日行分流管探查术，探查结果证明分流阀已坏，用新的分流阀更换，同年12月23日出院。2009年6月18日原告入住中国人民解放军一二三医院诊治，同日行“脑室-腹腔引流管造影术”。摄片提示，左侧引流管欠通畅等，同年6月22日出院。之后，原告又陆续在北京天坛医院等住院治疗。现原告起诉来院要求被告赔偿医疗费62,484.87元(人民币。下同)、医疗紧急处理费100,000元、交通费13,055元、残疾赔偿金246,480、精神损害赔偿100,000元、出院后的继续护理费219,000元。
　　本案在诉前调解过程中，经被告华山医院申请，本院委托上海市医学会对被告的诊疗行为有无过错，若有过错，该过错与原告的损害后果是否有因果关系以及事故等级进行医疗事故技术鉴定。2010年7月2日上海市医学会出具鉴定意见书，结论为1、患者因车祸致颅脑外伤后半年，脑室系统明显扩大，属外伤后脑积水，有施行脑室-腹腔分流术适应证。依据现有送鉴病历资料，医方的手术操作符合常规。2、病历资料同时显示患者在院检查术后的分流阀门“弹起有力”，说明分流装置状态正常，而医方所采用的分流管为国家药品监督管理局审查的医疗器械产品正式注册登记产品，现没有证据表明该分流装置为不合格产品。3、患者颅脑创伤的伤情严重，受伤后即始终昏迷，入华山医院时已处于植物状态。故目前的神经功能障碍是自身创伤性疾病的结果，与医方的医疗行为没有因果关系。综上所述，根据《医疗事故处理条例》第二条、第三十三条规定，朱A与华山医院医疗争议不构成医疗事故。
　　原审审理中，原告坚持认为被告的诊疗行为有过错，被告申请的医疗事故鉴定与医疗过错有差别；引流阀本身并无引流功能，存在质量问题。
　　原审法院经审理后认为，原告因车祸致头部外伤在被告处诊治，被告对患者实施的医疗行为经上海市医学会医疗事故技术鉴定，该鉴定程序合法，目前也无证据证明鉴定内容违反科学，该鉴定意见本院予以采信。根据鉴定结论，被告对患者在诊治过程中的治疗方案、手术操作等均符合医疗常规，也没有证据表明被告为患者安装的分流装置为不合格产品，该医疗行为不构成医疗事故。原告认为被告申请的是医疗事故鉴定，而原告要求的是医疗过错，本院认为，虽然上海市医学会作出的是医疗事故技术鉴定书，但根据本院委托鉴定的要求以及鉴定书中的实质内容，均对本案中的医疗行为是否存在医疗过错(过失)予以了确认，故原告的主张不予采信；原告另主张被告使用的引流阀存在质量问题，但并无证据予以佐证，原告的主张亦不予采信。民事赔偿以医疗行为是否存在过失为赔偿前提，现被告在该诊疗过程中并无过失(过错)，故原告要求被告赔偿其相关的损失，不符合民事侵权赔偿的要件，本院不予支持。
　　原审法院据此判决：原告朱A要求被告复旦大学附属华山医院赔偿损失的诉讼请求不予支持。
　　原审判决后，上诉人朱A不服，向本院提起上诉称：请求二审法院撤销原审判决，依法改判被上诉人赔偿因使用不合格医疗器材延误医疗时间造成上诉人成为植物人的人身损害损失741,019.89元。因为一审法院认定事实不清，适用法律错误。1、被上诉人当时在提供医疗服务活动中，出具书面证明表明其脑脊液分流器械使用寿命为8年，而2年后即出现分流不畅并且反方向流水的严重问题。既然被上诉人没有证据证明其所主张的脑脊液不能正确分流是由于病理原因所致的事实，一审法院支持其完全是凭主观臆断。无论是分流器不合格还是因为医疗行为操作不当，上诉人都有权利要求被上诉人承担损害赔偿责任。2、医疗事故鉴定结论既不能证明被上诉人已尽医疗服务合同义务，也不能证明其医疗行为不存在医疗差错。被上诉人无证据证明其医疗行为不存在构成医疗事故以外的医疗过错，因此，被上诉人应该承担举证不能的不利后果。综上，上诉人认为，一审法院判决事实认定不清，适用法律错误，请求二审法院依法支持上诉人的上诉请求。
　　被上诉人复旦大学附属华山医院答辩称：不同意上诉人的上诉请求，同意一审判决。
　　本院经审理查明，原审查明法律事实属实，本院予以确认。
　　本院认为，公民享有生命健康权。在医疗纠纷中，医院的医疗行为存在过失，侵害他人生命健康权的，应当承担相应责任。原审法院根据双方当事人的诉辩、提交的证据对本案事实进行了认定，并在此基础上结合上海市医学会医疗事故技术鉴定书依法作出原审判决，合法合理，理由阐述充分，本院不再赘述。二审中上诉人主张一审法院认定事实不清、适用法律错误，上海市医学会的鉴定结论既不能证明被上诉人已尽医疗服务合同义务，也不能证明其医疗行为不存在医疗差错，请求二审法院依法改判被上诉人赔偿因使用不合格医疗器材延误医疗时间造成上诉人成为植物人的人身损害损失741,019.89元。但其在二审中没有新的事实与理由，也没有新的证据佐证自己的主张。本院认可一审法院对事实的分析认定及对相关法律法规的理解与适用，上诉人的上诉请求缺乏事实及法律依据，本院难以支持。综上所述，原审认定事实清楚，判决并无不当。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币11,800元，由上诉人朱A负担，本院准予免缴。
　　本判决为终审判决。
审判长李伊红
　　代理审判员朱红卫
　　代理审判员张顼
　　二○一一年五月十日
　　书记员龚轶
