郭某某与复旦大学附属中山医院青浦分院医疗损害责任纠纷案

审判时间：20111121 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第2237号


　　上诉人(原审原告)郭某某。
　　委托代理人杜某某，陕西省勉县法律事务中心法律工作者。
　　被上诉人(原审被告)复旦大学附属中山医院青浦分院。
　　法定代表人王某某，院长。
　　委托代理人马某某，单位员工。
　　上诉人郭某某因医疗损害责任纠纷一案，不服上海市青浦区人民法院(2011)青民四(民)初字第1209号民事判决，向本院提起上诉。本院依法组成合议庭,公开开庭审理了本案。上诉人郭某某及其委托代理人杜某某、被上诉人复旦大学附属中山医院青浦分院的委托代理人马某某到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，2008年11月17日，郭某某因“车祸外伤致胸部、左下肢疼痛活动受限3小时”入住复旦大学附属中山医院附属青浦分院(以下简称中山医院青浦分院)处。当时查体：体温38℃，血压110/70mmHg，心率80次/分，神志清晰，呼吸平稳，胸廓无畸形，双肺呼吸音清。左大腿肿胀，左膝部皮肤挫伤。左足背动脉搏动可及，左大腿压痛(+)，左胸壁挤压痛(+)。左下肢短缩约5厘米。皮肤浅感觉正常。摄片示：1、左侧少量胸腔积液；2、左股骨中下段及股骨颈骨折；3、左侧第3、4肋骨骨皮质不连续，左侧肩锁关节半脱位。入院诊断：1、左股骨中下段、股骨颈骨折；2、左侧胸腔积液。入院后完善术前准备。11月27日，被告予原告在全麻下行“左股骨颈并股骨干骨折切开复位长重建钉内固定+左肩锁关节脱位切开复位锁骨钩钢板内固定术”。术中出血约600ml，输血浆2u，输少浆血2u。术后给予补液，预防感染(氨曲南)，预防应激性溃疡(奥美拉唑)，抬高患肢等治疗。12月11日，郭某某伤口拆线。后郭某某多次至中山医院青浦分院复诊。
　　原审另查明，2010年2月3日，上海市青浦区医学会作出鉴定结论：郭某某与中山医院青浦分院医疗争议不构成医疗事故；根据《医疗事故处理条例》第二条、第三十三条(等)，本病例不属于医疗事故。郭某某为此支付鉴定费人民币(以下币种均同)3,500元。2010年2月24日，上海市青浦区卫生局委托上海市医学会对上述事项进行鉴定。2010年4月9日，上海市医学会作出鉴定结论：根据《医疗事故处理条例》第二条、第四条，《医疗事故分级标准》(试行)、《医疗事故技术鉴定暂行办法》第三十六条，本案例构成三级丙等医疗事故，中山医院青浦分院承担轻微责任。2010年11月22日，华东政法大学司法鉴定中心对郭某某因医疗行为导致的后果所需的护理、休息期限出具鉴定结论：原告因医疗事故对左膝关节僵硬造成了一定的影响，现酌情给予治疗休息12个月、护理5个月。郭某某为此支付鉴定费1,000元。
　　原审法院再查明，郭某某因道路交通事故人身损害赔偿纠纷一案于2009年6月4日诉至原审法院，该案被告为马军、大众保险股份有限公司上海分公司。2009年5月13日，华东政法大学司法鉴定中心对郭某某因该起交通事故造成损害的伤残等级及休息、护理、营养期限作出鉴定：郭某某因交通事故致左股骨中下段、股骨颈骨折、左肩锁关节脱位、左侧胸腔积液，分别评定九级、十级伤残，酌情给予治疗休息8个月、营养3个月、护理3个月；二期内固定拆除术，酌情给予治疗休息4个月、营养1个月、护理1个月。2009年8月20日原审法院做出一审判决：判令该案被告大众保险股份有限公司上海分公司赔付原告郭某某109,667元(含精神损害抚慰金13,000元)；判令该案被告马军赔偿原告郭某某36,579.44元、鉴定费1,400元、查档费40元、律师代理费1,500元。2011年4月，郭某某为医疗纠纷，诉至原审法院，请求判令中山医院青浦分院赔偿医疗费人民币40,000元、误工费109,088元、护理费19,480元、交通费2,000元、住院伙食补助费600元、营养费2,700元、鉴定费5,500元、残疾赔偿金103,488元、残疾器具费2,000元、精神损害抚慰金20,000元；诉讼费、律师费由中山医院青浦分院负担。
　　原审法院经审理后认为，本案系因医疗损害引起的人身损害赔偿，本次医疗事故经鉴定为三级丙等，且医院方负有轻微责任，故医院方应在八级伤残的基础上按20%的责任赔偿郭某某的实际损失。根据相关规定并结合本案实际情况,实际损失计算如下：1、误工费，郭某某无固定收入，其误工费按照上一年度上海市职工平均工资46,757元，根据鉴定结论计算为46,757元；2、陪护费(即郭某某主张的护理费)，按照上一年度上海市职工平均工资46,757元，根据鉴定结论计算为19,482元；3、交通费，医院方无异议，确认2,000元；4、住院伙食补助费，根据郭某某的住院天数，按每天20元计算为480元；5、鉴定费，郭某某提供的证据为4,500元，法院予以支持；6、残疾生活补助费，根据郭某某的户籍性质、伤残等实际情况，按上海市上一年度农村居民人均生活消费支出10,225元计算为92,025元；7、精神损害抚慰金，根据本案的实际，酌情按每年10,225元计算3年为30,675元;8、郭某某主张的医疗费、残疾器具费、营养费、律师费均未能提供相关证据，对此不予支持。。
　　原审法院据此作出判决：一、被告复旦大学附属中山医院青浦分院应于本判决生效之日起十日内赔偿原告郭某某误工费9,351.40元；二、被告复旦大学附属中山医院青浦分院应于本判决生效之日起十日内赔偿原告郭某某陪护费3,896.40元；三、被告复旦大学附属中山医院青浦分院应于本判决生效之日起十日内赔偿原告郭某某交通费400元；四、被告复旦大学附属中山医院青浦分院应于本判决生效之日起十日内赔偿原告郭某某住院伙食补助费96元；五、被告复旦大学附属中山医院青浦分院应于本判决生效之日起十日内赔偿原告郭某某残疾生活补助费18,405元；六、被告复旦大学附属中山医院青浦分院应于本判决生效之日起十日内赔偿原告郭某某精神损害抚慰金6,135元；七、被告复旦大学附属中山医院青浦分院应于本判决生效之日起十日内赔偿原告郭某某鉴定费4,500元；八、原告郭某某的其他诉讼请求不予支持。
　　原审判决后，上诉人郭某某不服，向本院提起上诉称：中山医院青浦分院在治疗过程中未能保障患者知情权，告知其伤情不利于早期功能锻炼，未告知其锻炼膝关节，且没有让其适用膝关节功能锻炼机，延误其恢复时机，应当承担全部赔偿责任。其目前需要进行手术治疗，经咨询手术费用大概在40,000元。原审法院所采信的鉴定结论依据不足。原审法院对残疾生活补助费计算不当。故要求本院撤销原审判决支持其原审中的全部诉讼请求。
　　被上诉人复旦大学附属中山医院青浦分院答辩称：其治疗过程是否存有过错已经经过两级鉴定机构鉴定，在原审过程中对此也予以查明，故要求驳回上诉，维持原判。
　　本院经审理查明，原审查明事实属实，本院予以确认。
　　本院审理过程中，郭某某向本院提交陕西知本律师事务所、勉县法律事务中心律师费收款收据两张，共计9,000元。
　　本院认为，当事人对自己提出的诉讼请求所依据的事实或者反驳对方诉讼请求所依据的事实有责任提供证据加以证明，没有证据或者证据不足以证明当事人的事实主张的，由负有举证责任的当事人承担不利后果。鉴于医疗行为具有较强专业性，对于医疗行为的过错判断须依赖专业鉴定机构的鉴定。原审过程中，经上海市医学会鉴定，中山医院青浦分院在诊疗过程中存有不当，构成三级丙等医疗事故，医方承担轻微责任。对此郭某某虽提出异议，认为医方应当承担全部责任，但并无提供足以证明其主张的证据，故原审法院采信该鉴定报告，酌定由中山医院青浦分院对郭某某承担20%的赔偿责任并无不当，本院予以维持。关于原审中残疾生活补助费的计算，经核符合相关规定，本院予以确认。关于律师费，郭某某于二审期间提交的律师费收款收据，不属于二审新证据，其于原审中未能提交该份证据，应当承担举证不能的相应后果。关于医疗费，因未实际发生，本院难以支持，郭某某可另行主张。综上所述，原审认定事实清楚，判决并无不当。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币6,649元，由上诉人郭某某负担，本院准予免缴。
　　本判决为终审判决。
　　审判长李伊红
　　代理审判员王屹东
　　代理审判员樊形锋
　　二○一一年十一月二十一日
　　书记员卞耀辉
