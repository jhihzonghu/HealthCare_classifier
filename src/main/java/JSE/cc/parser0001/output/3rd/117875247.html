龚某某等诉怀化某某医院医疗损害赔偿纠纷案

审判时间：20110711 法院：湖南省怀化市鹤城区人民法院 案件字号：(2010)怀鹤民一初字第769号


　　原告龚某某。
　　原告胡某某。
　　原告胡某某。
　　原告胡某某的委托代理人吴某某。
　　三原告共同委托代理人龚某某。
　　被告怀化某某医院。
　　法定代表人李某。
　　委托代理人张某某。
　　委托代理人谢某某。
　　原告龚某某、胡某某、胡某某与被告怀化某某医院医疗损害赔偿纠纷一案，于2010年7月7日提起诉讼。本院于同日立案受理后，由审判员杨瑞云担任审判长与审判员银燕、审判员向和平组成合议庭，代理书记员杨小小担任法庭记录，于2011年5月18日公开开庭进行了审理，原告龚某某、胡某某、三原告的共同委托代理人龚某某以及原告胡某某的委托代理人吴日岗；被告怀化某某医院的委托代理人张某某、谢某某到庭参加诉讼，本案现已审理终结。
　　原告诉称，2009年7月13日，胡某某因冠心病入住被告怀化某某医院，在被告医院接受诊疗。7月20日9时40分，胡某某步入被告医院导管室接受被告医院行冠脉支架植入术，12时30分，被告医生通知胡某某亲属说：“胡某某在接受冠脉支架植入术中出现急性心肌梗死，经抢救无效死亡。”胡某某死亡后，原告找被告赔偿应被告医疗过错责任造成胡某某死亡的各项经济损失费，被告坚持以胡某某死于“行冠脉支架植入术中，出现急性心肌梗死，经抢救无效死亡”为由不肯赔偿原告经济损失。在原告的强烈要求下，7月21日，被告与原告龚某某、胡某某达成了“胡某某医疗纠纷协议”，由被告给原告人道主义补偿34800元。原告为了能让死者胡某某入土为安，被迫接受被告的条件。原告安葬了死者胡某某后，再次到被告单位要求被告赔偿经济损失，7月23日，被告科室人员为原告捐助人民币10200元。被告二次共给原告45000元人民币，还退不完被告收取原告向被告交纳胡某某医疗费45794元，实际上被告未给原告任何经济赔偿。原告对被告作出的胡某某死亡原因结论不服，向医学专家咨询，查阅相关行冠动脉支架植入术中发生急性心肌梗死等资料及胡某某的病历资料、医嘱、护理记录、影像资料等。得知胡某某不属于在行冠动脉支架植入术中发生急性心肌梗死，经抢救无效死亡。而是被告医院在为胡某某行冠脉支架植入，术前检查不到位，术中抢救不及时，违反诊疗常规，未尽诊疗义务，手术过失，医疗设施设备、人员资格技术水平不合格所致，被告主要过错责任有：
　　1、被告在为胡某某行冠脉支架植入术当时的设施设备和为胡某某行冠脉支架植入术人员不合格，达不到心室辅助装置应用技术管理规范（试行）要求。
　　2、病程记录严重不全，病志严重缺失，被告有重大隐匿医诊嫌疑。
　　3、被告伪造、篡改病历、医嘱、护理记录、影像资料、销毁证据、清洗导管室和患者衣物及现场，掩盖事实真相。其行为违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范常规。
　　4、胡某某在术中发生急性心肌梗死，经抢救无效死亡，与被告手术过错有直接因果关系。冠脉支架植入术是一种成功的微创手术，患xxx症至前兆，只要抢救及时得当，胡某某不会死亡在手术台上。
　　5、胡某某在接受被告手术中发生鲜血大量外喷（胡某某赤身裸体躺在手术台上，衣服丢在地上，主治医生申强身上的工作服被喷满鲜血），导致胡某某失血过多死亡，与被告医务人员手术过错有关联。
　　胡某某死亡后，原告与被告虽然达成了“胡某某医疗纠纷协议”，但该协议是在胡某某病历资料、医嘱、护理记录、影音资料被隐匿、篡改、伪造，原告对胡某某死因不明，有重大误解，原告处于万分悲痛之中达成的，且原告胡某某作为协议人之一未参加协议和签字同意，该协议缺乏主体。虽然协议达成后的第三天被告医务人员又向原告捐款10200元，但被告并没有对原告进行实质性赔偿和补偿。协议严重显失公平。由于被告的医疗过错责任，造成胡某某死亡在手术台上，给原告方造成了巨大的经济损失和精神损害，按《民法通则》119条、《医疗事故处理条例》第50条、第51条、第52条、《最高人民法院关于审理人身损害赔偿适用法律若干问题的解释》第31条、33条、34条，《最高人民法院关于参照审理医疗纠纷民事案件的通知》及相关法律规定和湖南省2008年度统计公告标准，被告应赔偿胡某某死亡补偿金270864元（15048×18年），丧葬费11541元，赔偿三原告精神损害抚慰金64968元（10828×6年）、交通费（含运尸体费）2080元、住宿费1000元，共计人民币350453元。但被告至今除退还给原告所交的医疗费外未作任何赔偿和补偿。现原告为维护自己的合法权益，根据《合同法》、《侵权责任法》、《民法通则》、《医疗事故处理条例》、《最高人民法院关于审理人身损害赔偿适用法律若干问题的解释》、《最高人民法院关于参照审理医疗纠纷民事案件的通知》等相关法律规定，特诉致人民法院依法判决：1、撤销2009年7月21日原告龚某某、胡某某与被告达成的“胡某某医疗纠纷协议”。2、被告赔偿三原告因被告医疗过错责任造成胡某某死亡的死亡赔偿金和三原告的精神损害抚慰金及各项损失费350453元。3、原告在诉讼中所需要的一切费用由被告承担。
　　被告怀化某某医院辩称：第一，患者病情复杂严重，手术中急性发作，经抢救无效死亡，医院已尽到诊疗义务。患者胡某某入院诊断：1、冠心病、陈旧性心梗、心绞痛、心功能二级；2、高血压病3级，极高危组。根据造影结果，患xxx病变，属于冠心病高危病人，有行冠脉血运重建指征，建议患者首选冠脉搭桥术，次选冠脉支架植入术，并详细向患xxx症。患者及亲属选用冠脉支架植入术。术中因严重冠状动脉病变，患者突发急性心肌梗死，经积极抢救无效死亡。在诊疗过程中医院已尽到了义务，患xxx症复杂、手术中突发疾病所导致。第二，原告起诉称手术人员不合格，病程记录不全，伪造、篡改病历等是不存在的。怀化某某医院及参加患xxx病介入诊疗技术医疗机构及医务人员资格，参与手术的医生都是从事介入诊疗的专业医生。病历记录全面并无涂改，原告所说没有事实依据。另外，原告反映手术中出血本属正常现象，抢救中医生外套上沾血亦不奇怪，医院已提交了手术抢救的录像资料，手术及抢救都是正常进行的。第三，双方达成的《胡某某医疗纠纷协议》是合法有效的，原告反悔起诉，法院不应支持。纠纷发生后，原告方自己要求在市卫生局调解下协商，达成了一致意见，签订了协议书，且当即履行完毕。此协议没有违背法律的强行性效力规定，合法有效。原告方不存在误解以及显失公平的情况。原告还提到死者女儿没有签字的理由也不能成立，医院完全有理由相信死者妻子、儿子是可以代表死者亲属的，当时也没有其他任何人提出异议，且协议已履行完毕。根据合同法的规定，应当认为签字人有代理权，未签字视为同意。事实上死者所有的近亲属签字难以做到，也没有必要。综上，患xxx病导致，双方达成的纠纷处理协议合法有效，应予支持。
　　经审理查明，龚某某系胡某某妻子，胡某某、胡某某系胡某某子女。2009年7月13日，胡某某因冠心病入住被告怀化某某医院，胡某某入院诊断：1、冠心病、陈旧性心梗、心绞痛、心功能二级；2、高血压病3级，极高危组。2009年7月16日行冠脉造影术，明确为冠状动脉多支多处严重病变。经被告与患者及其家属沟通，患者家属协商后，要求行冠脉支架植入术。7月20日9时40分，胡某某进入被告医院导管室接受被告医院行冠脉支架植入术，术中患者突发急性心肌梗死，经抢救无效死亡。7月21日，在怀化市卫生局调解下，患者胡某某家属龚某某、胡某某作为签约人代表（甲方）与被告怀化某某医院（乙方）签订《胡某某医疗纠纷协议》，双方达成协议：1、甲乙双方一次性解决本次医疗纠纷，一致认为此次事件属于医疗意外。甲方放弃对此次医疗事件进行医疗事故技术鉴定和民事诉讼权利，以及对乙方的所有相关权利要求。2、乙方考虑到甲方的实际困难，出于人道主义，一次性补偿甲方人民币34800元。3、本协议第二条约定的费用，在本协议签字生效后，于7月21日由乙方一次性支付给甲方。4、本协议一式两份，甲、乙双方各持一份，具同等效力。本协议自甲、乙双方签名并经乙方加盖公章之日起生效。被告按协议约定一次性补偿胡某某家属34800元（已由龚某某领取）。协议履行后，胡某某家属代表再次向被告提出赔偿要求，7月23日，被告心血管内科相关医护人员以自愿捐款的方式为胡某某家属捐助10200元。7月25日，龚某某、胡某某委托龚碧莲领取。之后原告龚某某、胡某某、胡某某认为协议严重显失公平，由于被告的医疗过错责任，造成胡某某死亡在手术台上，并诉至法院。
　　本案在审理过程中。原告龚某某、胡某某、胡某某于2010年8月5日向本院提出医疗过错司法鉴定申请。2011年5月9日，原告龚某某、胡某某、胡某某向本院撤销司法鉴定申请。
　　上述事实，经公开开庭审理，原、被告当庭举证、质证和辩论，本院确认，有下列证据予以证明：
　　1、原告龚某某、胡某某、胡某某身份证复印件各1份，证明原告的身份情况；
　　2、组织机构代码证复印件1份、医疗机构执业许可证复印件1份，证明被告具有合法医疗从业资质。
　　3、胡某某住院病历复印件1份56页，证明胡某某住院诊疗的病历资料。
　　4、手术摄像资料（光盘）1份，证明胡某某行冠脉支架植入手术情况。
　　5、《胡应强医疗纠纷协议》复印件1份，证明胡某某家属龚某某、胡某某作为签约人代表与被告怀化某某医院在怀化市卫生局调解下达成协议。
　　6、领条复印件2份，证明原、被告已按协议履行完毕，胡某某家属收到赔偿款45000元。
　　7、庭审笔录1份，证明本案查明的其它事实。
　　原、被告提供的其他证据，与本案处理结果无关联性，本院不予采信。
　　本院认为，《胡某某医疗纠纷协议》是由胡某某家属龚某某、胡某某作为签约人代表与被告怀化某某医院在怀化市卫生局调解下自愿达成的，虽胡某某未在协议上签字，但协议实际已经履行，根据日常生活经验法则，能推定胡某某应当知道协议内容，龚某某、胡某某在协议书上签字的行为应能代表胡某某的全体继承人，故《胡某某医疗纠纷协议》系原、被告真实意思表示，不违背法律的规定。《中华人民共和国合同法》第五十四条规定“下列合同，当事人一方有权请求人民法院或者仲裁机构变更或者撤销：（一）因重大误解订立的；（二）在订立合同时显失公平的。一方以欺诈、胁迫的手段或者乘人之危，使对方在违背真实意思的情况下订立的合同，受损害方有权请求人民法院或者仲裁机构变更或者撤销。”原告方诉请法院撤销协议，应当提供证据加以证明。原告提供的证据不足以证明双方签订的协议符合上述条款规定的撤销情形，原告诉请撤销2009年7月21日原告龚某某、胡某某与被告达成的《胡某某医疗纠纷协议》，本院不予支持。因《胡某某医疗纠纷协议》合法有效，原告诉请被告赔偿三原告因被告医疗过错责任造成胡某某死亡的死亡赔偿金和三原告的精神损害抚慰金及各项损失费350453元，本院亦不予支持。据此，根据《中华人民共和国合同法》第五十四条、《中华人民共和国民事诉讼法》第六十四条第一款、最高人民法院《关于民事诉讼证据的若干规定》第二条之规定，判决如下：
　　驳回原告龚某某、胡某某、胡某某的诉讼请求。
　　案件受理费6556.8元，依法予以免交。
　　如不服本判决，可在判决书送达之日起15日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于湖南省怀化市中级人民法院。
审判长杨瑞云
　　审判员向和平
　　审判员银燕
　　二○一一年七月十一日
　　代理书记员杨小小
