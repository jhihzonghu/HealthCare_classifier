李某与上海某医院医疗损害赔偿纠纷案

审判时间：20110415 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2011)沪一中民一(民)终字第495号


　　上诉人（原审原告）李某。
　　被上诉人（原审被告）上海某医院。
　　上诉人李某因医疗损害赔偿纠纷一案，不服上海市卢湾区人民法院（2009）卢民一(民)初字第1918号民事判决，向本院提起上诉。本院于2011年2月11日立案受理后，依法组成合议庭于同年3月10日公开开庭进行了审理。上诉人李某及其委托代理人栾晓丽、徐刚，被上诉人上海某医院（以下简称“某医院”）的委托代理人董希会、邵堂雷到庭参加诉讼。本案现已审理终结。
　　原审法院认定，2007年李某在进行职工例行身体检查时发现左侧甲状腺有肿块，2008年1月9日上海市梅陇社区卫生服务中心超声检查报告单超声提示：左叶甲状腺囊肿。同年1月14日李某因甲状腺肿块4个月至某医院就诊，B超检查示“左甲状腺结节”，为进一步诊治于同日收治入院。初步诊断为左甲占位。诊疗计划：1、完善相关检查，进一步明确诊断；2、择期手术。1月15日B超（彩超）示：双侧甲状腺多发结节，首先考虑结节性甲状腺肿。1月16日MIBI甲状腺显像报告：左侧甲状腺下极“冷”结节，双侧甲状腺未见异常放射性浓聚灶。经上级医师查房，明确诊断，择期手术治疗。1月16日术前小结、术前讨论：有手术指征，拟施双甲次全切手术。术前请五官科会诊。术前某医院告知李某手术及手术中、手术后可能发生的意外和并发症，李某在谈话记录单上签字，并在手术志愿书上签名同意。1月17日李某术前声带检查：右侧声带活动受限。某医院将此情况告知李某。同日某医院给李某行左甲状腺近全切除术＋右甲状腺近全切除术（双侧甲状腺次全切除），手术顺利。术后李某恢复可，无不适主诉。2008年1月22日冰冻石蜡病理诊断报告病理诊断为“左”结节性甲状腺肿伴囊性变、“右”结节性甲状腺肿、“甲状旁腺”纤维组织伴小片淋巴细胞。1月23日李某出院，出院诊断为结节性甲状腺肿。李某出院时情况：神清，气平，切口干洁，一般情况可。出院后用药及建议：1、注意休息及营养；2、门诊随访。在此之后，李某继续在某医院及本市其他医疗机构进行诊治。2008年11月12日某医院超声检查报告的超声意见为甲状腺术后，双侧甲状腺区及下方区域淋巴结显示。2008年12月3日复旦大学附属中山医院彩超报告单超声表现：甲状腺右叶、左叶及峡部未显示，双侧甲状旁腺未见明显肿块回声……超声印象为双侧甲状旁腺肾输尿管膀胱未见占位。根据李某提供的住院医药费专用收据及门急诊医药费专用收据，李某进行治疗花费了一定的医疗费用。
　　原审又查明，根据某医院的书面申请，法院会同李某、某医院对相关病历资料进行确认后，委托上海市静安区医学会就某医院对李某的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，某医院医疗行为与李某目前状况是否存在因果关系，明确是否属于医疗事故，事故等级及责任程度进行鉴定。上海市静安区医学会于2009年6月30日出具沪静医鉴[2009]033号医疗事故技术鉴定书，其分析意见为：李某患双侧甲状腺多发性肿块，其中部分冷结节，作双侧甲状腺近全切除术，有手术指征。目前李某发音正常，永久性喉返神经损伤和甲状旁腺功能损害证据不足，故不构成医疗事故。鉴定结论为李某与某医院的医疗争议不构成医疗事故。李某不服上述鉴定结论，申请由上海市医学会进行复核鉴定，法院则委托上海市医学会进行复核鉴定。上海市医学会受理后，李某递交中止鉴定申请书，认为某医院提交的病历资料中存在多处伪造和篡改，而且病历资料不完整，影响鉴定结果，故要求中止鉴定。上海市医学会则以此为由出具中止鉴定通知，决定中止鉴定。经法院向李某释明后，李某仍表示对上述病历资料的真实性不予认可，需重新开庭质证后，方可进行鉴定。且经法院协调，上海市医学会则坚持中止鉴定的决定。
　　2009年9月，李某诉至法院，要求某医院赔偿医疗费人民币8,131.95元、住院伙食补助费人民币200元、住院陪护费人民币1,097.33元、交通费人民币2,000元、精神损害抚慰金人民币58,194元、病历复印费人民币21元及聘请律师费用人民币10,000元。至于某医院应赔偿李某的残疾补助费、误工费、营养费及后续治疗费用则需经鉴定后方能确定。
　　某医院辩称，某医院在诊疗过程中没有任何过错及过失，也没有伪造、隐匿李某病史资料的情况，且经医疗事故技术鉴定，确定不构成医疗事故，故不同意李某的诉讼请求。
　　原审法院审理后认为，基于医疗技术的复杂性、医疗活动的专业性等特性，医疗纠纷中医疗单位有无过错以及医疗行为与损害后果之间有无因果关系均需借助专业、权威的机构进行鉴定。本案中，上海市静安区医学会的分析意见及鉴定结论是医学专家针对专门案件所作的专业意见，具有权威性，可以作为审理本案的相关依据。根据上海市静安区医学会出具的医疗事故技术鉴定书的分析意见，李某患双侧甲状腺多发性肿块，其中部分冷结节，作双侧甲状腺近全切除术，有手术指征。目前李某发音正常，永久性喉返神经损伤和甲状旁腺功能损害证据不足，故不构成医疗事故。鉴定结论为李某与某医院医疗争议不构成医疗事故。李某虽然要求由上海市医学会进行复核鉴定，法院亦委托了上海市医学会进行复核鉴定，但李某在上海市医学会受理后，又对其曾予以确认的某医院提供的病历资料提出异议，认为上述病历资料中存在多处伪造和篡改等情况，而要求中止鉴定，上海市医学会则以此为由决定中止鉴定，李某应承担无法进行医疗事故复核鉴定的法律责任。另李某虽然认为某医院提供的病历资料存在多处伪造和篡改等情况，但李某在法院委托上海市静安区医学会进行医疗事故技术鉴定之前，曾对上述病历资料予以确认，且李某亦没有提供确实的证据证明上述病历资料存在多处伪造和篡改等情况，故对李某此主张不予采纳。综上所述，李某要求某医院予以医疗损害侵权赔偿的诉讼请求，法院不予支持。据此，原审法院作出如下判决：驳回李某的诉讼请求。案件受理费人民币1,791元，由李某负担。
　　原审法院判决后，李某不服，上诉至本院称，1、在审理程序方面，上诉人始终未提出拒绝鉴定而只是多次要求在上海市医学会复核鉴定前对证据进行合法合理的认证，一审法院就此所作处理违反了有关规定；2、在事实认定方面，一审法院对被上诉人某医院伪造、隐匿、销毁病史，侵害上诉人知情权的过错等行为，均未作认定；3、一审法院在采信证据方面、适用法律方面均有不当，故请求撤销原判、发回重审。
　　被上诉人某医院则不同意上诉人的上诉请求，请求驳回上诉，维持原判。
　　经本院审理查明，原审认定事实无误，本院予以确认。
　　本院认为，本案争议的焦点在于某医院有无过错及其医疗行为与李某所受人身损害后果之间是否存在因果关系。由于医疗活动的专业性，对医疗机构是否存在过错以及医疗行为与损害后果之间因果关系的判断，通常需以医学会的医疗事故技术鉴定书为依据。本案中，上海市静安区医学会已专门就本案出具了专业的分析意见，其鉴定结论为不构成医疗事故；李某不服该鉴定结论而申请由上海市医学会进行复核鉴定，但其又以某医院伪造和篡改病历资料影响鉴定结果为由申请中止鉴定，因李某申请中止鉴定的理由缺乏事实依据，由此导致上海市医学会复核鉴定无法进行的法律后果应由李某自担。原审法院据此判决驳回李某的诉讼请求并无不当，本院予以维持。上诉人李某的上诉请求，缺乏充分的事实和法律依据，本院难予支持。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　上诉案件受理费1,791元，由上诉人李某负担。
　　本判决为终审判决。
审判长李虎
　　代理审判员沈卫兵
　　代理审判员孙卫
　　二○一一年四月十五日
　　书记员左莉莉
