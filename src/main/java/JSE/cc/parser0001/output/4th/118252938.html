王某诉某大学附属医院医疗损害责任纠纷案

审判时间：20111202 法院：上海市徐汇区人民法院 案件字号：(2011)徐民一(民)初字第4854号


　　原告王某。
　　委托代理人王某（系原告亲属），女
　　委托代理人陈某
　　被告某大学附属医院。
　　法定代表人郑某。
　　委托代理人吕某。
　　原告王某诉被告某大学附属医院（以下简称某医院）医疗损害责任纠纷一案，本院于2011年7月20日立案受理后，依法适用普通程序，组成合议庭，于2011年9月13日对本案公开开庭进行了审理。原告王某及其委托代理人王某和陈某、被告某医院的委托代理人吕某到庭参加诉讼。本案现已审理终结。
　　原告王某诉称，2003年12月12日原告因“内外痔”入住被告处欲行“内外痔外剥内扎术”，但被告未经原告同意对原告进行了“混合痔外剥内扎术、肛裂切扩术”。手术后，原告肛门部出血频繁，切口疼痛难忍，虽经被告治疗但症状仍无缓解。2004年2月17日，被告医生曹永清告知原告采取保守疗法，回家后见到硬物就去坐、去压，无需再来医院治疗，原告遂听从医生建议回家采取保守疗法。2009年11月11日，原告因肛门大量出血、排便困难请已退休的医务人员检查，发现肛门明显缩小。11月12日，原告至被告处就诊，郭修田医生利用剪刀对原告肛门部进行了处理，具体不详；同日，曹永清医生对原告进行了肛扩并放置了“药膏”。11月13日，原告肛部排出钉子和残线。2009年12月1日及之后，原告奔波于数家医院就诊，被诊断为：1、肛门狭窄；2、肛门瘢痕及束扎瘢痕；3、肛门肛口被切除后肛门流血、肛门失禁；4、PPH手术后，肛门部不适，肛周结节疼痛。对此，原告认为，原告长期以来的排便障碍及出血均系被告手术所致，被告在原告的诊疗过程中存在重大过错：1、被告违反注意义务，手术技术粗陋，与原告目前的肛管损伤有直接的因果关系；2、被告术前剥夺原告知情同意权，违反规定在术后让原告签署手术志愿书；3、被告明知和应知原告术后反复出血、排便障碍系手术操作失败、操作不当所致，本应积极采取干预措施，但被告置若罔闻，加剧了原告损害后果的发生，故被告理应对原告的损害承担赔偿责任。此外，对于区、市两级医学会的鉴定结论原告均不认可，鉴定结论显属错误。经与被告交涉未果，故原告诉至法院，要求被告赔偿原告医疗费8,790.12元、住院伙食补助费700元、营养费7,200元、误工费24,000元、护理费24,000元、交通费1,228元、后续治疗费30,000元、精神损害抚慰金69,600元、复印邮寄费283元、律师代理费30,000元。
　　被告某医院辩称：被告对原告诊疗过程中的医疗行为符合诊疗常规，不存在过错。本案业经区、市两级医学会进行医疗事故技术鉴定，被告的医疗行为均不构成医疗事故，故不同意原告的诉讼请求。
　　经审理查明，2003年12月12日原告因“肛门肿物脱出疼痛时（常发）作三月”入住某医院肛肠科。据住院病案记载：入院时专科检查：视诊：截石位7点见肛管内脱出，大约0.4×0.4m2乳白色肿块，3、7、11点齿线上粘膜隆起，伴肌缘皮赘增生；触诊：7点位齿线处肿块质硬，触痛明显，3、7、11点齿线上下肿物质软，无触痛。肛内未及菜花状硬质肿块，6点见陈旧性肛裂。中医诊断：痔（湿热下注型）。西医诊断：⑴混合痔：⑵肛乳头肥大：⑶肛裂。12月15日原告在肛周神经阻滞麻醉下行“混合痔外剥内扎术＋肛裂切扩术”。术中切除肥大的肛乳头。术后予抗感染、止血等治疗。12月17日开始伤口换药予红油膏纱条引流，痔疮宁栓钠肛。12月29日9时原告诉排便时见便血少许，色鲜红，肛门疼痛不显，无肛周瘙痒。检查：创面肉芽红活，边缘上皮爬生，创口少许分泌物。予创面常规清洁，红油膏纱条粘生肌散嵌入创面。考虑便血少许与创面未愈伤口摩擦有关。2004年1月2日原告诉排便时可见少许便纸染血，肛门处隐痛。检查：6点钟位创面肉芽红活，部分上皮爬升，结扎线已脱。继续予以换药处理。此后原告间歇性有类似症状，经治疗至2004年1月15日原告出院。出院后原告按约在某医院门诊随访治疗。2004年2月17日原告因“便血伴隐痛10天”前往某医院门诊复诊，据门诊病历记载：原告大便一日一次，软，有少量出血。检查：6点位创面顶端未愈。建议门诊换药及药物内服、外洗治疗。2009年11月12日原告因“混合痔术后4年，时有便血少许，色鲜红，间作一周左右或有发作，伴肛门疼痛，大便每日2次，成形软便，无消瘦、乏力”再次到某医院就诊。据门诊病案记载：检查：截石位近5点、7点处见表浅肛管皮肤裂开存在。建议饮食调节，润肠片及肠镜检查。2009年12月1日原告因“肛门疼痛一周”前往上海中医药大学附属曙光医院就诊。据门诊病案记载：检查：肛缘见少量赘皮样皱起，指诊于齿线上方及环状束扎疤痕，可容一指通过，触痛（＋）。诊断：肛门痛（肛管偏窄）。2009年12月7日原告因“便秘伴肛门疼痛”至华东医院就诊。据门诊病案记载：肛检：膝胸位，12点见肛裂。诊断：肛裂。予药物治疗。2009年12月8日原告用药后疼痛未能明显好转，大便略有缓解再次前往上海中医药大学附属曙光医院复诊。诊断为肛门痛。2009年12月18日原告因“有肛门疼痛，大便困难，有时有流血”前往瑞金医院就诊。据门诊病历记载：检查肛门外观正常，肛门指诊在KC位1点处有瘢痕，其他（－）。予痔疮宁栓治疗。2010年1月12日原告自诉大便变细，排便困难，肛门又有不适感再次至华东医院复诊。检查胸膝位12点有肛裂。同日原告因“肛门疼痛伴便血数年，有内痔手术史”再至复旦大学附属华山医院就诊。据门诊病历记载：原告拒作肛指检查，KC位12点见肛裂。建议肠镜检查，1周复诊。2010年1月20日原告因“右肛旁疼痛，有痔疮手术史”至华东医院就诊。据门诊病历记载：检查：肛管痉挛，食指可插入，未及新生物。诊断：混合痔。2010年1月26日及28日原告因疼痛先后至曙光医院和普陀区利群医院就诊。2010年3月9日原告因“肛门疼痛伴便血、便秘”至华东医院就诊。据门诊病案记载：肛检：左侧卧位，肛管骶尾部见肛裂。建议门诊随诊。2010年3月17日原告再次因肛门瘙痒，大便每日次数较多至曙光医院就诊。2010年8月9日原告因“痔疮手术后肛门疼痛伴排便困难6年”前往普陀区利群医院。据门诊病历记载：肛门外观无明显异常，直肠指诊食指无法伸入。诊断肛管狭窄。2010年8月24日原告因大便欠畅形细，排便次数较多，少量出血伴疼痛至上海中医药大学附属曙光医院就诊。2010年9月28日原告诉大便困难伴疼痛到华东医院门诊配药。2010年11月17日及12月8日原告两次诉肛口疼痛伴大便出血前往仁济医院就诊。2010年12月20日原告前往新华医院肛肠外科就诊。据门诊病案记载：检查“肛门外未见明显异常，肛指仅容指尖，不能耐受”。诊断考虑肛门痔术后，肛门狭窄。建议坐浴，药物口服及本院随访。此后原告在多家医疗机构随访治疗至今。
　　在诉前调解阶段，本院委托上海市静安区医学会对本次医疗争议进行鉴定。该会于2010年11月9日出具鉴定书，分析意见为：1、原告因“痔、肛裂”于2003年12月12日赴被告某医院诊治，被告在诊疗行为中，诊断明确，所行手术有手术指证，符合诊疗常规。2、原告目前诉“肛门疼痛，大便困难伴狭窄，时有脓血便”。现场检查：肛门外观正常，指检指套未见脓血，肛门括约肌收缩功能存在，无肛管狭窄。因此，建议原告作结肠纤维镜检查及肛门排粪造影检查以排除直肠上方其他疾病。3、被告对原告施行“肛裂切扩术”虽诊断明确，并有手术指证，但术前未能与原告沟通，故告知不全。4、被告手术与原告目前现状无因果关系。结论为：原告王某与被告某医院医疗争议不构成医疗事故。被告为此支付鉴定费3,500元。
　　原告对上海市静安区医学会的鉴定结论不服，申请复核鉴定。根据本院委托，上海市医学会于2011年3月4日出具鉴定书，分析意见为：1、本病例有肛门肿块脱出伴疼痛（胀、刺痛）症状，入院体检发现肛门口有皮赘，局部触痛。故其混合痔、肛乳头肥大、肛裂的诊断成立。被告施行混合痔外剥内扎、肛乳头肥大切除及肛裂手术指证明确，手术记录中无使用吻合器记载，未发现手术操作有违反诊疗常规之处。2、原告主诉出院后排便困难及便血症状，但在现有送鉴病历资料中，2004年2月17日至2009年11月11日期间未见与本疾病相关就诊的任何记录。追问病史原告术后无肛门排便失禁的症状，也否认使用过润肠通便的药物，故没有依据表明发生过吻合器痔上粘膜环切手术常见的不良后果。3、本次现场检查没有发现原告肛门内有肿块和结节，无明显肛门狭窄的表现，指套无染血，亦无脓性分泌物。原告目前症状与被告手术之间没有因果关系。但不排除有其它疾病原因（例如结肠冗长，直肠前壁膨出等）所致。4、被告在术前对肛裂手术的交待不够细致，虽然有格式化的手术志愿书签署，但在沟通上有欠缺之处；门诊随访期间检查原告后存在无病历记录的现象，这些不足都应在今后工作中引起足够重视。结论为：原告王某与被告某医院医疗争议不构成医疗事故。原告为此支付鉴定费3,500元。
　　上述事实，除原、被告一致陈述外，另有原告提供的病史资料、医疗费发票，被告提供的病史资料、医疗事故技术鉴定书等证据予以佐证。
　　审理中，原告申请本院委托中华医学会对本案医疗纠纷进行技术鉴定，本院认为本案医疗纠纷不具备疑难、复杂并在全国有重大影响的情况，故不予准许。
　　本院认为，公民享有生命健康权。原告作为患者基于医疗纠纷而提起的侵权损害赔偿，应当符合医疗损害侵权责任的四个构成要件，即医疗机构和医务人员的诊疗行为、患者的损害、诊疗行为与损害后果之间的因果关系、医务人员的过错。由于医疗行为具有较强的专业性，故对该医疗行为与损害后果间是否存有因果关系、医务人员是否存有过错的判断，除依照一般常理及当事人提供的证据材料以外，还需由权威部门即医学会出具鉴定意见。本案中，区、市两级医学会的医疗事故技术鉴定均明确了被告在施行混合痔外剥内扎、肛乳头肥大切除及肛裂切扩手术时诊断明确、具备手术指证，手术操作符合诊疗常规，不存在过错，且原告目前症状与被告手术之间亦无因果关系，因此被告的行为不符合侵权的构成要件，原告要求被告承担侵权损害赔偿责任，本院难以支持。但鉴于被告在手术前对肛裂切扩术告知方面未予原告充分解释和沟通，术后的门诊随访期间检查原告后亦存在无病历记录的情况，均有不足之处，故本院酌情判令被告给予原告一定的经济补偿以示抚慰。据此，依照《中华人民共和国民法通则》第五条之规定，判决如下：
　　一、驳回原告王某的诉讼请求；
　　二、被告某大学附属医院于本判决生效之日起十日内给付原告王某30,000元。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　案件受理费4,216元，由原告王某负担；鉴定费7,000元（原、被告各预缴3,500元），由某大学附属医院负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。

　　审判长高霄雷
　　代理审判员范萍
　　人民陪审员李雅萍
　　二〇一一年十二月二日
　　书记员廖军
