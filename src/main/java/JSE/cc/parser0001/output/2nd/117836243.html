沈某某与张某医疗损害赔偿纠纷上诉案

审判时间：20110621 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2011)沪一中民一(民)终字第1418号


　　上诉人（原审原告）沈某某
　　上诉人（原审原告）张某
　　被上诉人（原审被告）某医院
　　上诉人沈某某、张某因医疗损害赔偿纠纷一案，不服上海市浦东新区人民法院（2011）浦民一(民)初字第2353号民事判决，向本院提起上诉。本院于2011年5月24日立案受理后，依法组成合议庭于同年6月15日公开开庭进行了审理。上诉人沈某某、张某及其委托代理人黄某，被上诉人某医院（以下简称“某医院”）的委托代理人江某到庭参加了诉讼。本案现已审理终结。
　　原审认定，患儿沈某钧系沈某某、张某所生之子。2010年9月21日13时30分，张某在某医院剖宫产生下患儿。出生后，患儿哭声欠响，肤色青紫，予以清理呼吸道，吸出白粘液少许，给予持续面罩吸氧，青紫渐减轻。14时入新生儿室，予以吸氧观察。16时30分，患儿肤色转红，反应好，送回母亲身边观察。医方告知，如有病情变化，需转院治疗。另告知家长先心病不排除，待观察。9月22日晨，考虑新生儿湿肺，转入某医学中心。心脏彩超显示，完全性大动脉错位，房间隔缺损，动脉导管未闭，肺动脉流速稍增快。经诊断，患xxx病。给予相应治疗，维持内环境稳定。9月23日进一步治疗，但患儿缺氧、酸中毒明显，难以纠正，拟行手术治疗，因家属决定放弃治疗，16时15分患儿心音消失死亡。
　　2010年10月，沈某某、张某诉至法院称，由于某医院在诊疗及护理方面的严重过错，导致新生儿病情加重直至死亡，故请求判令某医院赔偿死亡赔偿金288,380元、丧葬费21,396元、精神损害赔偿金30,000元以及律师费5,000元，共计344,776元。
　　某医院辩称，某医院医务人员尽到了合理注意义务，所有的手续都符合诊疗常规。患xxx病及其发展所致，患儿死亡是因为家属放弃治疗，某医院的医疗行为并不存在过错，且与患儿死亡之间没有任何因果关系，不应该承担医疗损害的相关责任，故请求依法驳回其诉讼请求。
　　沈某某、张某就医疗行为发生争议涉讼后，经当事人申请，原审法院委托某医学会对本案医疗争议进行鉴定。鉴定分析认为：患儿出生后18小时疑新生儿湿肺转某医学中心治疗，先天性心脏病、缺氧、酸中毒诊断明确，需手术治疗，因家长放弃手术，患儿出生后50小时死亡。患xxx病本身（复杂先心大血管错位）及家长放弃手术所致。某医院在患儿诊治过程中存在以下不足：新生儿体检不全面，临床观察不仔细、不到位，仅考虑“新生儿湿肺”转院，未考虑“先天性心脏病诊断”，但以上不足与患儿死亡无因果关系。鉴定结论是本病例不构成医疗事故。
　　原审法院审理后认为，本起医疗纠纷经医疗事故技术鉴定，鉴定结论明确表述本病例不属于医疗事故，且在分析意见中载明，患儿出生后18小时即转某医学中心治疗，先天性心脏病、缺氧、酸中毒诊断明确，患xxx病本身及家长放弃手术所致，某医院在新生儿体检方面的不足与患儿死亡无因果关系。此外，某医院于2010年9月21日16时30分业已告知家长先心病不排除。沈某某、张某主张由于某医院在诊疗及护理方面的严重过错，导致新生儿病情加重直至死亡，并要求赔偿，但沈某某、张某对其主张没有依据，法院不予支持。鉴于某医院在工作中存在的不足，故医学会鉴定费用3,500元由某医院负担亦无不当。
　　据此，原审法院作出判决：驳回沈某某、张某的诉讼请求。
　　案件受理费6,471元，减半收取计3,235.50元，由沈某某、张某共同负担。某医学会鉴定费3,500元，由某医院负担。
　　原审判决后，沈某某、张某不服，上诉至本院称，原审法院仅凭医疗事故鉴定结论判上诉人败诉，不顾某医院诊治过程中存在的严重疏忽及重大护理瑕疵，某医院应就其医疗过错承担责任，故请求撤销原审判决，改判支持上诉人原审时的全部诉讼请求。
　　被上诉人某医院则不接受上诉人的上诉主张，请求驳回上诉，维持原判。
　　经本院审理查明，原审认定事实无误，本院予以确认。
　　本院认为，本案争议的焦点在于某医院对于患儿的死亡是否应该承担相应的责任，其关键则在于某医院对患儿治疗过程中是否存在过错以及医疗行为与患儿死亡是否存在因果关系。由于医疗活动的专业性，对医疗机构是否存在过错以及医疗行为与损害后果之间因果关系的判断，通常需以医学会出具的医疗事故技术鉴定书为依据。本案中，某医学会已专门就本案出具了专业的分析意见，其鉴定结论为本病例不构成医疗事故。虽医疗事故技术鉴定书明确某医院在患儿诊治过程中存在不足，但同时亦确认该等不足与患儿死亡无因果关系。故原审法院以上诉人主张没有依据而驳回其诉讼请求，并无不妥，依法可予维持。依据鉴定报告，患儿的死亡难以归咎于某医院的医疗行为，故上诉人请求撤销原审判决并改判支持其原审全部诉请，事实和法律依据不足，本院难予支持。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　上诉案件受理费6,471元，由上诉人沈某某、张某负担。
　　本判决为终审判决。
　　审判长李虎
　　代理审判员洪可喜
　　代理审判员沈卫兵
　　二○一一年六月二十一日
　　书记员左莉莉
