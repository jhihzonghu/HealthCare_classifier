许某某与某医院许某某医疗损害责任纠纷上诉案

审判时间：20110601 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2011)沪一中民一(民)终字第1102号


　　上诉人（原审原告）许某某
　　被上诉人（原审被告）某医院
　　上诉人许某某因医疗损害责任纠纷一案，不服上海市徐汇区人民法院（2011）徐民一(民)初字第148号民事判决，向本院提起上诉。本院于2011年4月18日立案受理后，依法组成合议庭进行了审理。本案现已审理终结。
　　原审法院经审理查明，2009年7月17日许某某因右肩外伤至真新地段医院就诊，摄某片表现：右肱骨外科颈骨断裂伴骨碎片分离及骨干向内侧移位。某片诊断：右肱骨外科颈粉碎性骨折。当天15时许某某入住某医院（以下简称“某医院”）骨科病房。诊断：右侧肱骨近端骨折。予对症支持治疗，择期手术。7月22日为许某某在臂麻下行右肱骨近端骨折切开复位内固定术。术中见右肱骨近端骨折粉碎，大结节上移，肱骨头塌陷，复位骨折，上抬肱骨头，断端植同种异体骨，前侧放置史塞克肱骨近端锁定钢板1套，固定后透视见骨折复位可，内固定位置良好，放置负压引流，手术顺利。予对症补液抗炎治疗。7月23日复片表现：右肱骨近端骨折术后，内固定中，对位对线可，关节在位。术后患肢肿胀较轻，患肢感觉血运良好，手术切口辅料干燥，无红肿，于7月27日下午出院。嘱隔天换药，2周拆线；定期复查，术后6周门诊随诊；石膏固定3周，患肢适当功能锻炼。8月4日许某某在真新地段医院拆线。9月9日后许某某分别至某医院、真新地段医院、瑞金医院就诊。后许某某认为某医院在对其治疗过程中存在过错，造成其身体损害。故起诉要求赔偿医疗费4,451.19元、后续治疗费27,631.67元、精神损害抚慰金60,000元，并保留向某医院主张后续费用的诉讼权利。
　　原审法院委托某医学会进行医疗事故技术鉴定，该会分析认为：1、医方诊断明确，患者系右肱骨近端粉碎性骨折移位，有手术指征，手术方式适当，符合诊疗规范。术后影像学检查，骨折对位对线良，内固定良好，术后功能恢复良，手术目的已达到。2、目前患者右肩关节功能尚有部分未完全恢复，这与其损伤本身（关节内骨折）及高龄、术后锻炼不够有关，与手术无因果关系。结论为许某某与某医院医疗争议不构成医疗事故。某医院支付鉴定费3,500元。
　　原审法院认为，判定某医院承担医疗损害责任，需要求某医院医疗违法行为与患者人身损害后果之间存在因果关系。医学会的鉴定意见是具有医学专业知识的人员根据法定程序所作出的鉴定结论，对于某医院的医疗行为是否存在过错以及其过错与患者的损害是否有因果关系，具有比较强的证明力。许某某因右肩外伤先后至真新地段医院、某医院处就诊，真新地段医院摄片表现：右肱骨外科颈骨断裂伴骨碎片分离及骨干向内侧移位，某片诊断：右肱骨外科颈粉碎性骨折，某医院诊断为：右肱骨近端骨折。某医院第二日对许某某CT检查：右肱骨近端可见骨折，对位对线可，关节在位。某医院对许某某病情的诊断正确且具有手术指征，某医院手术方式适当。许某某术后检查为骨折对位对线良，内固定良好，手术目的达到。对于许某某右肩关节功能尚有部分未完全恢复的情况，医学会指出，该情况与其损伤本身（关节内骨折）及高龄、术后锻炼不够有关，与手术无因果关系。许某某主张某医院医生在未对其“骨干向内侧移位”进行矫正的情况下直接进行固定，之后用铁锤直接敲打使关节“对线对位”，造成骨干扭曲，以及某医院医生在行内固定手术时，钉子将骨头打裂，右肘关节窝内有一块突出物，是某医院医生割断的韧带或血管经络，均无证据证明。原审法院据此判决：驳回许某某全部诉讼请求。案件受理费2,102元，鉴定费3,500元，共计5,602元（许某某已支付1,450元，某医院已支付3,500元），由许某某负担2,102元，某医院负担3,500元。
　　判决后，上诉人许某某不服，向本院提起上诉，请求撤销原判，改判支持上诉人原审的诉讼请求。许某某上诉称，其目前右肩关节活动障碍、骨折未完全愈合等情况与医疗事故技术鉴定书描述的情况不一致，鉴定结论不应被采信。某医院的不当诊疗行为导致其损害，故某医院应承担赔偿责任。
　　被上诉人某医院则不同意上诉人的上诉请求，请求二审法院驳回上诉人的上诉请求，维持原审判决。
　　本院认为，基于医疗技术的复杂性、医疗活动的专业性等特性，医疗纠纷中医疗单位有无过错以及医疗行为与损害后果之间有无因果关系均需借助专业、权威的机构进行鉴定。本案中，某医学会出具的医疗事故技术鉴定书的分析意见及鉴定结论是医学专家针对专门案件所作的专业意见，具有权威性，可以作为审理本案的相关依据。根据医疗事故技术鉴定书的分析意见，可以确定某医院对上诉人的治疗有手术指征、手术方式适当，符合诊疗规范。上诉人目前的情况与其损伤本身、高龄、术后锻炼不够有关，非某医院的错误治疗措施所致。据此，原审法院不支持上诉人要求某医院承担赔偿责任的诉讼请求，并无不当。故在上诉人二审期间不能提供充分证据证明鉴定结论存在错误及其主张的事实存在的前提下，要求二审法院支持其原审的诉讼请求，没有事实和法律依据，本院不能支持。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　上诉案件受理费2,102元，由上诉人许某某负担，本院准予其免交。
　　本判决系终审判决。
　　审判长李虎
　　代理审判员洪可喜
　　代理审判员沈卫兵
　　二○一一年六月一日
　　书记员左莉莉
