袁永林、袁晶莹诉辛光辉、中国人民武装警察部队河南省总队医院医疗损害赔偿纠纷案

审判时间：20110505 法院：河南省郑州市二七区人民法院 案件字号：(2010)二七民一初字第2220号


　　原告袁永林。
　　原告袁晶莹。
　　委托代理人张少春，江苏天之权律师事务所郑州分所律师。
　　委托代理人刘广杰，河南大豫律师事务所律师。
　　被告辛光辉。
　　委托代理人何俊，河南天坤律师事务所律师。
　　委托代理人邱占国。
　　被告中国人民武装警察部队河南省总队医院。
　　法定代表人尹继云，院长。
　　委托代理人段峰，该院工作人员。
　　委托代理人张勇强，该院工作人员。
　　原告袁永林、袁晶莹诉被告辛光辉、中国人民武装警察部队河南省总队医院（以下简称：武警医院）医疗损害赔偿纠纷一案，本院立案受理后，依法组成合议庭，公开开庭进行审理。原告袁晶莹及原告袁永林、袁晶莹的共同委托代理人张少春、刘广杰，被告辛光辉的委托代理人何俊、邱占国，被告武警医院的委托代理人张勇强到庭了参加诉讼。本案现已审理终结。
　　原告诉称：2010年4月18日下午，因段长焕感冒、嗓子疼，在女儿袁晶莹的陪同下来到郑州市“二七区辛大夫中医诊所”。该诊所仅有“护士”李园园接诊，当时袁晶莹告诉她母亲对双黄连过敏，5年前进行过肝细胞移植，让她注意点。后李园园在未做皮试情况下，就开出了注射头孢和利巴韦林等药治疗单，给段长焕输液，之后段长焕出现头晕、眼睛看不清等症状。护士过来，又给患者注射了肾上腺素和地塞米松，直接导致血压升高，出现昏迷等症状。家属拨打120急救将段长焕送至被告武警医院救治，住院四天后，救治无效死亡。后经查，该诊所雇佣非医师专业人员且未领取护士证的“护士”李园园坐诊，仅仅通过电话请示大夫，而后在未给患者做皮试的情况下，就开处方注射头孢和利巴韦林、地塞米松，致使患xxx症状后，又擅自给患者注射肾上腺素和地塞米松，致使血压升高，乃至脑出血死亡。整个过程，直到120急救车到达后，被告辛光辉才出现在现场。因此，对患者的死亡，被告负有不可推卸的责任。事发至今，被告对此事未有赔偿。故原告提起诉讼，要求法院判令：1、被告赔偿丧葬费、死亡赔偿金及精神损失费等共计374585元；2、被告承担本案诉讼费用。本院受理后，原告变更诉讼请求为：1、要求被告赔偿医疗费10857元、丧葬费13678元、停尸费6720元、火化费700元、购买墓地费21000元、鉴定费6000元、住宿费1389元、交通费500元、死亡赔偿金318605.2元、精神抚慰金30000元，以上共计409449.7元；2、本案诉讼费由被告负担。
　　原告提交的证据有：1、李园园出具的治疗单一份；2、被告武警医院急诊病历、住院病历各一份、医疗费票据四份；3、居民死亡医学证明书、遗体化证明、死亡证明、郑州铁路中心医院收费票据各一份；4、豫新乡医学院司鉴中心（2010）病鉴字第105号司法鉴定意见书一份及鉴定费票据；5、录音书面资料一组、录音光盘两张；6、《东方今报》三份（2010年4月20日、21日、23日各一份）、《大河报》一份（2010年4月23日）；7、住宿、餐饮票据一组、交通费票据一组；8、郑州市卫生局询问笔录三份、行政处罚决定书一份；9、公墓安葬证及发票各一份。
　　被告辛光辉辩称，1、原告所述事实与真实事实不符。患者到诊所治疗，向护士出具了一张血常规化验单，后护士询问后，写了一便笺，到被告处，并把患者的情况说明，被告看后出具了处方。护士以此处方用药，约几分钟，患者称其不舒服。护士给其换了瓶盐水，她还是称不舒服，护士只好按照她要求的把针拔了，给她喝了葡萄糖，并给她掐人中，且给被告打电话说明情况，被告让护士给患者打了支肾上腺素和地塞米松之后几分钟基本上已经没事了。后120到了，患者及其家属陪同去了武警医院。在被告武警医院住院治疗四天后患者死亡。2、该诊所依法成立，医生辛光辉及护士李园园都有相应的资格，并非像原告所述雇佣非医师专业人员。3、诊所的治疗行为并无过错：诊所没有给患者做皮试并无不妥，因头孢类注射并不是必须做皮试的；在患者发生过敏性休克时，诊所采取的一系列抢救措施符合诊疗规范。4、患xxx病发展导致的，与诊所的治疗行为并没有因果关系。综上所述，该诊所是依法成立的，所雇医生和护士均有资格，且在诊疗过程中是没有过错的，患xxx病的原因，与诊所无因果关系，不应由诊所承担任何赔偿责任。
　　被告辛光辉提交的证据有：第一组：1、医疗机构职业许可证一份；2、医师执业证一份；3、医师资格证一份；4、护士资格证一份。第二组：1、应用头孢菌类注射剂的用药指南和注意事项；2、《药理学》教材；3、《内科学》教材。第三组：1、被告武警医院病情观察单一份；2、武警医院住院押金凭据。第四组：《大河报》一份。第五组：郑州法医司法鉴定所、河南正诚法医临床司法鉴定所退函各一份。第六组：1、陈述一份；2、郑州市中心医院医疗费票据一份。
　　被告武警医院辩称，患者段长焕入我院治疗期间，我院诊断正确，治疗合理，抢救及时，严格按照医疗、护理操作常规做，无医疗差错及事故。对患者的死亡，考虑：病人有过敏反应刺激，对病人的心理、情绪是一个不良刺激，考虑是一个诱因；病人有白血病肝细胞移植术的病史，发生出血的几率相对于常人要大些。
　　被告武警医院未提交证据。
　　本院对原被告提交的证据认定如下：一、对原告提交的证据：1、对证据1、2、3的真实性被告辛光辉无异议，被告武警医院对证据1不予质证、对证据2、3无异议，本院对该组证据的真实性予以确认；2、对证据4中鉴定意见书被告辛光辉无异议，但对鉴定费票据提出异议，被告武警医院对该证据不发表意见，本院认为被告对其异议无证据相佐证，故对该证据真实性予以确认；3、对证据5被告辛光辉有异议，被告武警医院不发表质证意见，但结合本案卫生局的相关询问笔录，本院对该证据的真实性予以确认；4、对证据6被告辛光辉有异议，但无相应证据支持其主张，结合相关证据本院对该证据的真实性予以确认；5、对证据7被告辛光辉认为餐饮费不应支持，对其他票据的真实性有异议，本院认为被告对其主张无证据相佐证，故对该证据的真实性予以确认；6、对证据8的真实性被告无异议，本院认为该组证据客观真实，故对该证据真实性予以确认；7、对证据9被告认为应计算在丧葬费内，本院认为被告异议成立。（二）、对被告辛光辉提交的证据：对第一、二组证据的真实性原告及被告武警医院均无异议，本院对该两组证据的真实性予以确认；第三组证据中：对证据1真实性原告无异议，被告武警医院无异议，本院对该证据的真实性予以确认；对证据2原告及被告武警医院均无异议，本院对该证据依法予以采信。对第四组证据的真实性原告无异议，被告武警医院不予质证，本院认为该证据客观真实，故对其真实性予以确认。对第五组证据的真实性原告及被告武警医院均无异议，本院认为该组证据客观、合法，并具有关联性，故予以采信。对第六组证据的真实性原告有异议，被告武警医院不予质证，本院认为原告该组证据客观真实，故对其真实性予以确认。
　　根据上述有效证据及原、被告庭审陈述，本院查明以下案件事实：患者段长焕（原告袁永林之妻，袁晶莹之母)于2010年4月18日下午，因感冒、嗓子疼，至被告辛光辉所经营的二七区辛大夫中医诊所诊治。该诊所护士李园园询问过患者病情，开具治疗单，并电话请示被告辛光辉后，在未为患者做皮试情况下，给患者输液头孢等药，后患者出现“胸闷、呼吸困难、腰部双下肢疼痛、意识不清等症状”。护士紧急给患者注射了肾上腺素、地塞米松。后患者出现昏迷状况，被120急救于2010年4月18日下午15时许送至被告武警医院，于2010年4月18日20时30分入院。在该被告处住院治疗，后于2010年4月22日2时经抢救无效后死亡。在此期间，被告辛光辉已向原告支付医疗费535.5元。原、被告因赔偿问题协商未果，故原告诉至法院。
　　另查明，郑州市二七区卫生局于2010年7月5日对二七辛大夫中医诊所以“1、超出核准登记的诊疗科目开展诊疗活动；2、使用非卫生技术人员从事医疗卫生技术工作”为事实依据，作出郑二卫医罚（2010）051号《行政处罚决定书》。
　　本案审理过程中，原告申请对本案中被告方存在的过错进行司法鉴定，本院依法委托河南正诚法医临床司法鉴定所后，该机构退回该鉴定，后依法委托郑州严实法医临床司法鉴定所后，该机构又以申请人未缴纳鉴定费为由将鉴定退回我院。
　　本院认为：公民享有生命健康权，公民法人由于过错侵害他人人身的应当承担民事责任。侵害公民身体权，造成死亡的，应当赔付医疗费、丧葬费等。本案中，原告亲属段长焕因病至被告辛光辉为业主的二七辛大夫中医诊所就诊，被告辛光辉应严格依照医疗诊疗规范为段长焕诊疗，但该被告未当面为段长焕诊疗，并且使用未领取相应资质证书的护士为段长焕诊疗，也超出了其核准登记的经营范围，严重违反了《医疗机构管理条例》之规定；在未为段长焕做皮试的情况下，直接施用头孢类药物，违反了诊疗规程。后段长焕十分钟左右出现“胸闷、呼吸困难、腰部双下肢疼痛、意识不清”等症状，简易救治后，被紧急送至被告武警医院，住院治疗后死亡。该被告上述违反诊疗规范的医疗行为是否导致患者段长焕药物过敏，被告武警医院在对患者的几次诊断，均未能确定药物过敏的存在，后患者死亡后尸检亦未能确定该情形的存在，故被告辛光辉的上述过错是否是导致患者死亡的直接原因，均不能确定；如存在药物过敏，是否是导致该患者最终死亡的直接原因，患者尸检后亦未能加以确定，所以本案被告辛光辉的过错责任对患者死亡的后果因果关系并不十分明显。
　　被告武警医院收治段长焕后，在治疗过程中，始终未能确定药物过敏与否，对该患者的治疗方案、用药等可能会产生一定影响，对患者最终抢救无效而死亡的后果，有一定过错。综上所述，本案中，虽医疗纠纷未经司法鉴定机构进行司法鉴定，但二被告过错相对存在，尤其被告辛光辉的过错，且司法鉴定结论非案件认定被告是否存在过错的唯一依据，故对被告的过错本院依法予以认定。针对二被告的医疗过错行为，结合本案中证据，造成患者段长焕最终直接死亡原因经新乡医学院司法鉴定中心出具的司法鉴定意见书认定为系脑出血致循环呼吸系统功能衰竭死亡，本院认为二被告的过错比例认定为50%较适宜，结合本案具体情况酌定被告辛光辉负35%的责任、被告武警医院负15%的责任更适合。对原告诉讼请求中：医疗费10857元合法有据予以支持；丧葬费13678.5元符合法律规定，本院予以支持；停尸费方面，原告方证据不足以证明其请求，故对该请求本院依法不予支持；鉴定费过高，应以实际票据为准，即5600元；死亡赔偿金以河南省上一年度城镇居民人均可支配收入14371.56元，计算二十年，应为287431.2元；精神损害抚慰金30000元合法有据，予以支持；火化费、购买墓地费均属上述丧葬费范围，故本院依法不予支持；住宿费方面，因原告方证据不足以证明其所提交的票据同本案的关联性，故本院不予支持；交通费500元，合法有据，予以支持。被告辩称的其不存在过错的主张，因无充分证据予以佐证，故本院不予采信。综上，本案原告损失为医疗费10857元、丧葬费13678.5元、鉴定费5600元、死亡赔偿金287431.2元、交通费500元、精神损害抚慰金30000元，以上共计348066.7元。另外，被告辛光辉已垫付的535.5元，应自赔偿数额中扣除。综上所述，依照《中华人民共和国民法通则》第九十八条、第一百零六条、第一百一十九条，最高人民法院《关于审理人身损害赔偿案件适用法律若干问题的解释》第十七条第三款、第十八条、第十九条、第二十二条、第二十七条、第二十九条、第三十五条，最高人民法院《关于确定民事侵权精神损害赔偿若干问题的解释》第十条，《中华人民共和国民事诉讼法》第六十四条第一款之规定，判决如下：
　　一、被告辛光辉于判决生效之日起十日内赔偿原告袁永林、袁晶莹131787.85元（医疗费10857元、丧葬费13678.5元、鉴定费5600元、死亡赔偿金287431.2元、交通费500元，以上共计318066.7元的35%即111323.35元，加上精神损害抚慰金21000元，共计132323.35元，扣除已支付的535.5元，即131787.85元），逾期履行，加倍支付迟延履行期间的债务利息；
　　二、被告中国人民武装警察部队河南省总队医院于判决生效之日起十日内赔偿原告袁永林、袁晶莹56710元（医疗费10857元、丧葬费13678.5元、鉴定费5600元、死亡赔偿金287431.2元、交通费500元，以上共计318066.7元的15%即47710元，加上精神损害抚慰金9000元，共计56710元），逾期履行，加倍支付迟延履行期间的债务利息；
　　三、驳回原告袁永林、袁晶莹的其他诉讼请求。
　　案件受理费7442元，由原告负担3721元，由被告辛光辉负担2604.7元，由被告中国人民武装警察部队河南省总队医院负担1116.3元。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状一式十三份，上诉于河南省郑州市中级人民法院。
审判长王忠锋
　　人民陪审员李淑珍
　　人民陪审员方红
　　二○一一年五月五日
　　书记员王利霞
