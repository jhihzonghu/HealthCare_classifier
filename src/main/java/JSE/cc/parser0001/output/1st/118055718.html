邱xx诉上海市xx人民医院医疗损害赔偿纠纷案

审判时间：20110328 法院：上海市徐汇区人民法院 案件字号：(2011)徐民一(民)初字第1521号


　　原告邱xx。
　　委托代理人翁xx，上海城开律师事务所律师。
　　被告上海市xx人民医院。
　　法定代表人何xx，院长。
　　委托代理人徐x，上海市康正律师事务所律师。
　　委托代理人沈xx，上海市康正律师事务所律师。
　　原告邱xx诉被告上海市xx人民医院（以下简称x院）医疗损害赔偿纠纷一案，本院于2011年3月2日立案受理后，依法适用简易程序，于3月10日公开开庭进行了审理。原告邱xx的委托代理人翁xx，被告x院的委托代理人徐x到庭参加诉讼。本案现已审理终结。
　　原告邱xx诉称，2006年12月23日，原告在工地工作时不慎自高处摔落地面，造成右手、双下肢骨折，被送往被告x院治疗，被告处医师诊断为右手桡骨骨折，需行远端骨折复位内固定术，原告根据被告医师建议选择进口内固定材料，至2008年7月原告右手手腕区域突起，因被告医师称进口内固定材料术后无需取出，故原告未予注意，直至2009年3月，原告疼痛难忍，至医院治疗时方知内固定螺丝多处松动、脱落，只有手术再行安装。原告认为，被告未尽风险预见义务，在术前小结及术前讨论中均未能预见到内固定物可能出现螺丝松动、脱落等问题；被告未尽风险告知义务，被告医师在治疗中告知原告使用进口内固定材料无需术后取出，对于内固定植入后可能出现的螺丝松动、脱落等问题均未告知，明显违反了法律规定；被告在术后7日即要求原告出院，对原告出院后用药及建议未作告知，原告在复诊时陈述手部异常时，被告医师亦未引起充分重视，直至引起严重后果；故被告医师在诊疗过程中缺乏责任心，违反诊疗常规，造成了原告巨大的经济损失、精神痛苦，现诉至法院，要求判令被告x院赔偿原告邱xx医疗费33,102.55元、误工费21,396元、护理费10,698元、住院伙食补助费140元、交通费3,000元、住宿费2,000元、律师代理费3,000元。
　　被告x院辩称，医方诊治过程遵守诊疗常规，应用药物亦无不当，且涉案医疗纠纷经静安区医疗事故鉴定机构鉴定，结论未构成医疗事故，亦证实被告在治疗过程中不存在过错，原告现状系其自身疾病发展所致，x院对此并无过错，不应承担赔偿责任，要求法院驳回原告所有的诉讼请求。
　　经审理查明，2006年12月23日，原告邱xx因高处坠落后双足与双腕肿痛，至被告x院骨科急诊，查体示双腕部、双足肿胀伴压痛，活动受限。X片检查示双侧跟骨、双侧桡骨远端骨折，建议住院手术治疗。翌日收入院，专科检查示双足及腕部肿胀，左腕石膏外固定中，活动受限。诊断为双跟骨、双桡骨远端骨折，给予完善检查、抗炎消肿、限期手术。手术指征为双跟骨、双桡骨远端骨折。原告入院后经消肿、完善术前检查，无明显手术禁忌。术前将术中、术后可能出现的意外情况向家属及患者告知，患者表示理解，并在手术知情同意书上签名同意手术。于2007年1月3日在腰麻+臂丛阻滞下行双跟骨、右桡骨远端切复内固定术。手术顺利，安返病房。术后给予抗感染、止血治疗，患者双跟骨、右腕切口疼痛逐渐减轻，无红肿、渗出。原告邱xx于2007年1月10日出院。
　　2007年3月5日，原告至x院复诊，检查示双跟、右腕伤口愈合好，右腕活动受限。诊断为骨折愈合。
　　2009年3月2日，原告至复旦大学附属金山医院门诊治疗，其因右腕骨折术后2年2个月要求复片，了解骨折愈合情况。体检示右腕横纹上2CM可见豆状大小包块，压痛，右腕屈35°，背伸30°。诊断为右腕、双跟骨骨折术后。治疗意见：1、右腕正侧位片、双侧跟骨侧位片（X片诊断为两侧跟骨、右腕骨折内固定中）；2、建议拆内固定。
　　2010年4月23日，上海市静安区医学会就涉案医疗纠纷进行鉴定，分析意见为：1、邱xx因右手腕桡骨远端严重粉碎性骨折于2007年1月3日在x院行切开复位内固定，诊断明确，手术指征及治疗符合医疗常规；2、邱xx于术后18个月发现手腕部有“小包块”，未及时去医院复查。术后26个月发现有一枚螺钉松动脱出。术后3年零3个月施行拔钉。根据门诊记录，邱xx右手腕屈35°、伸30°，功能尚好；3、x院在指导邱xx术后随访及功能锻炼方面不够仔细，存在一定缺陷。结论为邱xx与x院医疗争议不构成医疗事故。
　　以上事实，除原、被告陈述一致外，另有原告邱xx提供的病史资料、医疗费发票、医疗事故技术鉴定书、聘请律师合同、律师代理费发票等证据证实，并经当庭质证无疑，本院予以确认。审理中，原告提供了证人证言，被告认为证人应到庭接受质询，对该证据不予认可，本院认为被告意见合理予以采纳，现证人未到庭作证，本院无法确认证人身份及其陈述的真实性，对原告提供证人证言的待证事实不作认定。
　　本院认为，医疗行为是一个具有相当专业的行为过程，对于医疗行为中医方是否存在过错及与损害后果间是否存有因果关系需要专业权威部门出具意见。本案中，上海市静安区医学会组织专家进行了医疗事故技术鉴定，明确了被告x院有明确的手术指征，治疗措施符合规范，对上海市静安区医学会据此作出涉案医疗争议不构成医疗事故的鉴定结论，本院予以认同，故原告邱xx要求被告x院承担医疗过错责任本院不予支持。但本院亦注意到，鉴定结论亦明确了被告在整个治疗过程中，存在术后随访及功能锻炼指导方面不够仔细，存在一定缺陷，上述治疗中存在的不足，反映出被告作为专业的医疗机构，未能尽高度注意之义务，其服务水准与被告资质等级、患者的期盼尚有一定差距，故本院判令被告酌情补偿原告，更有利于被告今后改进工作、安抚原告的情绪、取得原告的谅解。依照《中华人民共和国民法通则》第五条之规定，判决如下：
　　一、驳回原告邱xx所有的诉讼请求;
　　二、被告上海市xx人民医院于本判决生效之日起十日内补偿原告邱xx6,000元。
　　如果未按本判决指定的期间履行金钱给付义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　案件受理费1,634元，减半收取计817元，由被告上海市xx人民医院负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
审判员孙谧
　　二Ｏ一一年三月二十八日
　　书记员王嵘
