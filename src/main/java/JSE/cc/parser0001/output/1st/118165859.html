上海晨瀛投资管理有限公司与许昌市人民医院合作协议纠纷案

审判时间：20110106 法院：河南省许昌市中级人民法院 案件字号：(2010)许民二终字第139号


　　上诉人（原审被告）上海晨瀛投资管理有限公司。
　　法定代表人徐晨明，该公司董事长。
　　委托代理人常乐，北京市京悦律师事务所律师。
　　委托代理人王炜。
　　被上诉人（原审原告）许昌市人民医院。
　　法定代表人魏琦，该院院长。
　　委托代理人琚新。
　　委托代理人张建刚，该院法律顾问。
　　原审被告深圳市新产业医学发展有限公司。
　　法定代表人翁先定，该公司董事长。
　　上诉人上海晨瀛投资管理有限公司因与被上诉人许昌市人民医院、原审被告深圳市新产业医学发展有限公司合作协议纠纷一案，不服魏都区人民法院（2008）魏民二初字第86号民事判决，向本院提起上诉。本院受理后，依法组成合议庭，公开开庭审理了本案。上诉人上海晨瀛投资管理有限公司的委托代理人常乐、王炜、被上诉人的许昌市人民医院委托代理人琚新、张建刚到庭参加诉讼，原审被告深圳市新产业医学发展有限公司经合法传票传唤，未到庭参加诉讼。本案现已审理终结。
　　原审查明，2004年4月16日，原告许昌市人民医院作为乙方，被告上海晨瀛投资管理有限公司作为甲方，双方签定《屈光不正矫治项目合作协议书》一份。协议约定，甲方投入的设备是1、准分之激光；2、电脑验光机；3、角膜地形图；4、非接触眼压计；5、测厚仪；6、板层刀。投入的设备合同价为480万元。乙方以验收表中2、3、4、5、四项设备后二天内支付设备信誉保证金30万元。甲方在30个工作日内所有设备到位，并在5日内全部安装调试到位。合作期限，完成6000例手术后设备归乙方。合同约定甲方的责任为：1、保证设备的可靠性并承担医疗纠纷的赔偿费用，乙方协助解决；2、保证在合作第二年全额退还乙方信誉保证金人民币30万元；3、提供合作期间的各项技术费用；4、负责开业广告及新闻发布会的策划；5、出任中心主任，全面管理中心业务，布置市场宣传、财务预算、处理日常工作，每月30日向董事会上缴财务报表和工作报告；6、提供手术专家及暂定1000元/例，承担专家接送招待费用；7、承担水电暖气、机器维修、维护各种耗材及空调的配备等；8、承担乙方医师1名、护士人员2名的工资及各项福利待遇（参照现行工资标准）。9、甲方所投入设备必须按合同型号购买。保证为全新原装设备，由乙方验收合格后可投入使用。乙方的责任为：1、负责办理中心开业所需的各种许可证；2、负责提供手术所需的专用场地、即手术室、处置室、咨询室各两间，提供甲方长驻人员住房一间，并简单装修（刷白）；3、协助在当地的市场开发。4、负责安装一部热线电话和一部分机，费用由甲方负担；5、支持和配合甲方行政主任工作。合同还约定，甲方每年向乙方交纳管理费人民币10万元，每半年向乙方上缴5万元（400例以上，每超过100例增加人民币1万元）。合作期间，双方不得单方终止合作或另行选择合作伙伴，否则视为违约，违约方须向守约方支付总投资金额20‰的违约金。2004年6月16日，上海晨瀛公司和深圳新产业公司作为甲方，许昌市人民医院作为乙方，双方签定备忘录一份。双方在备忘录中约定，上海晨瀛公司和深圳新产业公司共同承担项目合作协议书中甲方的投资管理的义务和权利，以及赢余分配。与乙方合作共同完成该项目的合作，深圳新产业公司派出三名代表参加董事会，并担任董事长一职，上海晨瀛公司派出一名代表参加董事会，许昌市人民医院派出一名代表参加董事会。董事会负责协调各方关系，决策合作中心管理人员的人事安排和奖励，审批中心的管理制度和措施，审批经营计划和费用预算，制定中心的各项政策；深圳新产业公司和上海晨瀛公司共同派出代表担任项目中心行政正、副主任，负责合作项目中已定的甲方责任。深圳新产业公司作为供货方，负责提供原装进口美国laserscanlsx准分子激光治疗系统一套，并与上海晨瀛公司共同承担设备维修的甲方责任。备忘录生效后，即刻安排设备运送。设备运抵乙方场地后，未经深圳新产业公司同意，任何一方不得擅自移动设备；上海晨瀛公司同意将甲方利益分配由乙方直接转入深圳新产业公司指定帐户，再行分配。本备忘录经甲方代表的两家公司与乙方共同签字加盖公章后生效。双方在备忘录中特别约定，本备忘录作为甲乙双方《屈光不正矫治项目合作协议书》的组成部分和补充，享有与协议书相同的法律效力。合同签定后，原告许昌市人民医院提供了场地和相关的技术人员。2004年6月15日，原告许昌市人民医院汇给被告上海晨瀛公司信誉保证金20万元，2004年8月20日又汇给上海晨瀛公司信誉保证金10万元，共计30万元。被告提供的医疗设备到位后，2005年2月双方开始合作。在合作期间，被告上海晨瀛公司于2006年4月27日向原告许昌市人民医院送达郑重声明一份，要求解除合同。同日，原告许昌市人民医院与被告上海晨瀛公司进行了商谈，原告许昌市人民医院提出了合同解除前的相关费用如何解决的问题，双方商谈未果。2007年9月18日，深圳新产业公司向上海晨瀛公司发出了关于终止二公司之间合作协议的通知。关于原告许昌市人民医院给被告上海晨瀛公司的回复意见。原告许昌市人民医院用特快专递将回复意见邮寄给被告上海晨瀛公司，因该特快传递被原样退回。被告上海晨瀛公司没有收到回复意见，对原告许昌市人民医院的回复意见无法认定。综上，双方的合作于2005年2月正式履行，2006年4月27日被告上海晨瀛公司提出解除合同，双方共合作14个月，根据合同约定，每年应向许昌市人民医院缴纳的管理费为10万元，14个月共计应缴纳管理费为116666元。
　　原审法院认为，原告许昌市人民医院与被告上海晨瀛公司和被告深圳新产业公司在履行《屈光不正矫治项目合作协议书》的过程中，被告上海晨瀛公司单方终止合作，其行为已构成违约。根据双方的约定，上海晨瀛公司应向原告许昌市人民医院承担违约责任，被告上海晨瀛公司收取原告许昌市人民医院的信誉保证金30万元，应当退回；被告上海晨瀛公司和深圳新产业公司应向许昌市人民医院缴纳的管理费116666元，应当缴纳。因此，原告许昌市人民医院要求被告退还信誉保证金30万元；要求被告支付违约金960000元；要求被告支付管理费，理由正当，予以支持。二被告应向原告缴纳的管理费为116666元，对于原告多诉部分不予支持，原告并应承担多诉部分的诉讼费。被告上海晨瀛公司辩称的理由不足，不予采信。依法判决如下：一、被告上海晨瀛投资管理有限公司向原告许昌市人民医院退还信誉保证金300000元。二、被告上海晨瀛投资管理有限公司和被告深圳市新产业医学发展有限公司向原告许昌市人民医院连带支付管理费116666元。三、被告上海晨瀛投资管理有限公司向原告许昌市人民医院支付违约金960000元，上述一、二、三项于判决生效后十日内履行完毕。四、驳回原告许昌市人民医院的其他诉讼请求。
　　上诉人上海晨瀛投资管理有限公司上诉称，1、一审程序违法，没有组成合议庭审理本案；2、一审认定上诉人单方终止合作构成违约，无证据支持；3、一审认定116666元的管理费事实不清；4、被上诉人许昌市人民医院未按约定履行义务，30万元保证金不应退还。5、许昌市人民医院应当返还上诉人投入的全部设备，承担给上诉人造成的损失，并返还公司预存款104417.7元。
　　被上诉人许昌市人民医院答辩称，原审认定事实清楚，审判程序合法，应予维持。
　　上诉人二审期间提供许昌市人民医院就眼科项目合作的起诉状及眼科欠医院明细表各一份，证明许昌市人民医院欠上诉人104417.7元，被上诉人许昌市人民医院质证认为，这都是另一个案件中证据，属实，但与本案无关，不能证明上诉人的证明目的。经查明，起诉状、明细表所证明的对象与本案不属同一法律关系，与本案缺乏关联性，不予采信。
　　经二审审理查明的事实与证据和一审一致。
　　本院认为，上诉人上海晨瀛投资管理有限公司未提供证据证明一审未组成合议庭审理本案，一审笔录显示是组成合议庭审理案件，且许昌市人民医院认可是合议庭审理案件，故其“一审程序违法，没有组成合议庭审理本案”的上诉理由不能成立；上诉人上海晨瀛投资管理有限公司“2006年3月15日书面郑重声明”于2006年4月27日送达给许昌市人民医院，该郑重声明明确具备解除合同的意思表示，上诉人单方终止合同，违约事实清楚，应依照合同约定承担违约责任；上诉人上海晨瀛投资管理有限公司与被上诉人许昌市人民医院的合作于2005年2月正式履行，2006年4月27日被告上海晨瀛公司提出解除合同，双方共合作14个月，根据合同约定，每年应向许昌市人民医院缴纳的管理费为10万元，14个月共计应缴纳管理费为116666元；合同解除后，上诉人上海晨瀛投资管理有限公司应当退还收取的许昌市人民医院向其缴纳的信誉保证金30万元；上诉人上海晨瀛投资管理有限公司第2、3、4项上诉理由均不能成立；上诉人上海晨瀛投资管理有限公司的“许昌市人民医院应当返还上诉人投入的全部设备，承担给上诉人造成的损失，并返还公司预存款104417.7元”的上诉理由，因其未在一审提出反诉，二审不宜直接处理，当事人可另行起诉主张权利。
　　原审判决认定事实清楚，适用法律正确。依据《中华人民共和国民事诉讼法》第一百五十三条第一款第一项之规定，判决如下：
　　驳回上诉，维持原判。
　　上诉费19290元由上诉人上海晨瀛投资管理有限公司承担。
　　本判决为终审判决。
审判长孟晓克
　　代理审判员李红涛
　　代理审判员崔君
　　二〇一一年一月六日
　　书记员尚建辉
