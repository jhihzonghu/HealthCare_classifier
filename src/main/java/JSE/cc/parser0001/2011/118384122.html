宋某诉上海某医院医疗服务合同纠纷案

审判时间：20110622 法院：上海市卢湾区人民法院 案件字号：(2011)卢民一(民)初字第1393号


　　原告宋某。
　　委托代理人郑某。
　　被告上海某医院。
　　委托代理人刘某，上海某医院员工。
　　委托代理人董某，上海某医院员工。
　　本院于2011年6月13日立案受理了原告宋某诉被告上海某医院医疗服务合同纠纷一案，并依法适用简易程序公开进行了审理。原告以其在被告处因脑梗做手术至其全身瘫痪，无法言语，丧失行为能力，使其身心受伤害为由诉讼来院，要求依法判令：被告赔偿原告经济损失人民币150,000元。在审理中，被告表示希望双方调解解决。
　　本案在审理过程中，经本院主持调解，双方当事人自愿达成如下协议：
　　一、被告上海某医院支付原告宋某人民币150,000元，于2011年6月15日之前支付原告宋某人民币100,000元，于2011年6月24日之前支付原告宋某人民币50,000元。
　　二、原告宋某就原、被告间医疗纠纷放弃再向被告上海某医院主张任何权利。
　　三、案件受理费人民币1,650元减半收取，计人民币825元由被告上海某医院负担。
　　双方当事人一致同意本调解协议自双方在调解协议上签名或捺印后即具有法律效力。
　　上述协议，符合有关法律规定，本院予以确认。
代理审判员张煜
　　二〇一一年六月二十二日
　　书记员张文杰
