吕XX等诉上海XX医院医疗服务合同纠纷案

审判时间：20110701 法院：上海市浦东新区人民法院（司法公开示范法院） 案件字号：(2011)浦民一(民)初字第12794号


　　原告吕XX。
　　原告潘XX。
　　原告郁XX。
　　上列原告共同委托代理人潘XX。
　　被告上海XX医院。
　　法定代表人李XX，院长。
　　委托代理人童XX，上海市XX律师事务所律师。
　　委托代理人黄XX，上海市XX律师事务所律师。
　　原告吕XX、潘XX、郁XX与被告上海XX医院(以下简称XX医院)医疗服务合同纠纷一案，本院于2011年4月7日立案受理后，依法适用简易程序，于2011年5月30日公开开庭进行了审理，原告委托代理人潘XX，被告委托代理人童XX到庭参加诉讼。本案现已审理终结。
　　原告吕XX、潘XX、郁XX诉称，三原告系患者潘XX之亲属。2007年7月8日，患xxx症状，即呼叫救护车将患者送至被告急诊神经内科就诊，接诊医师安XX不懂上海方言，与患者交流存在障碍，其不但未尽到医师的谨慎注意义务而且也未寻求其他医师的帮助，未给患者做生命体征的检查，仅开具各类化验单要求患者予以检查。患者系由救护车送至医院，被告未采取任何保护措施，让患者自行做各类检查，导致患者脑部严重出血。之后被告亦未采取积极有效的抢救措施，延误了患者的病情，最终造成患者死亡的后果。原告认为，患者首诊接诊医师安红艳系外地进修医师，不具备法定注册医师资质即在被告医院独立执业行医，被告在医疗服务合同中未履行自己的义务，造成患者死亡的后果，故起诉要求被告支付原告医疗费人民币6,107.80元（币种下同）、丧葬费23,376元、死亡赔偿金636,760元、原告吕XX的抚养费93,514元、精神损害抚慰金50,000元并承担本案诉讼费。
　　被告XX医院辩称，不同意原告的诉讼请求。其医疗行为符合诊疗规范，对上海市杨浦区医学会的鉴定结论无异议。认为上海市医学会的复核鉴定结论不正确，如果被告有责任，本病例应属一级甲等医疗事故。原告主张的死亡赔偿金及吕根妹的抚养费无法律依据。
　　经审理查明：原告吕XX、潘XX、郁XX分别是患者潘XX的妻子、儿子及母亲。2007年7月8日13时26分，患者因“意识丧失15分钟伴头晕、左侧肢体无力1小时”由救护车送至被告处急诊。据急诊病历记载：（患者）视物旋转，无头痛、呕吐，睁闭眼无明显变化，无语言障碍，无肢体抽动。既往有高血压病史20余年，不服药。查体：血压170/90mmHg，言语略含糊，双侧鼻唇沟对称，伸舌居中。颈软，左侧上下肢肌力4级，双侧病理征（-）。辅助检查：1、头颅CT：CT提示右侧基底节区可见出血并破入脑室（口头报告）。2、心电图。3、血常规、肝肾功能、电解质。请神经外科会诊。同时予心电监护、吸氧。生理盐水250ml+亚宁安100g/静滴，根据血压调速。甘油果糖250ml/静滴，每日两次；20%甘露醇150ml/静滴，每日两次；生理盐水250ml+信法丁针400mg+10%氯化钾5ml/静滴，8小时一次；生理盐水250ml+醒脑静针400ml+10%氯化钾5ml/静滴，8小时一次。告病危。15时神经外科会诊记录：患者神志不清，深昏迷状态，双侧瞳孔不等大，右侧瞳孔扩大，（直径）0.5cm,光反射消失，左侧瞳孔（直径）0.3cm，光反射迟钝，四肢生理反射及病理反射均未引出。请示上级医师后认为：病情发展迅速恶化，手术风险极大，建议神经内科保守治疗。15时30分，患者昏迷，检查：左侧瞳孔（直径）0.15cm，右侧瞳孔（直径）0.4cm，对光（-）。呼吸深长，20次/分，心率110次/分，血压170/80mmHg。继续补液，再告病危。5%葡萄糖水250ml+可拉明4支+洛贝林4支/静滴。留置导尿。7月9日9时，神经外科会诊记录：深昏迷，GCS评分4分。复查头颅CT：右颞出血约50ml，破入诸脑室，铸型，脑疝。继续保守治疗。11时41分，神经外科会诊记录：目前患者病情危重，手术风险极大且不能有效缓解病情，无手术指证，继续保守治疗。2007年7月9日12时48分患者转入华山医院继续治疗，经抢救无效于7月12日凌晨1时宣告临床死亡。原告曾于2009年6月向本院起诉与被告的医疗服务合同纠纷一案，2010年6月30日本院依法裁定驳回原告的起诉并由被告承担鉴定费7,000元。
　　本案在前次诉讼中，经原告申请，本院委托上海市杨浦区医学会对本次医疗争议进行鉴定，结论为：本病例不属于医疗事故。并分析意见为：1、患者系急性脑出血，诊断明确，且出血量大，破入脑室，病情急剧恶化，病人的死亡是疾病本身的结果。2、院方在该患者接诊及治疗过程符合诊疗规范、常规，并无明显不当。3、医院首诊医师安XX的医师资质问题，不属于医疗事故鉴定的范畴，对此医学会在鉴定过程中作过告知。
　　后经原告申请，本院委托上海市医学会对本次医疗争议进行复核鉴定，结论为：本病例构成一级乙等医疗事故，XX医院承担轻微责任。并分析意见为：1、2007年7月8日，患者因“意识丧失15分钟伴头晕、左侧肢体无力1小时”至仁济医院急诊。根据送检材料，医方在接诊后给予患者必要的检查，通过头颅CT明确脑出血诊断，给予患者甘露醇降颅压等治疗措施，符合诊疗常规。2、患xxx病情急速变化，很快发展为深昏迷、脑疝，GCS评分4分，病情危重，在此情况下无手术指证，故采用保守治疗符合治疗规范。3、患者既往有高血压史，入院时血压达170/90mmHg，在监测下使用降压药不违反临床降压药使用规范。4、医方接诊后对于“脑卒中”患者的风险没有足够重视，未给予患者卧床处理，不利于病情稳定，不排除与患者病情加重可能有一定的因果关系。5、患者在离开XX医院时处于临床不能恢复的昏迷状态。
　　上述事实由原、被告陈述及原告提供的门、急诊就医记录册、上海市杨浦区医学会《医疗事故技术鉴定书》、上海市医学会《医疗事故技术鉴定书》各1份、医疗费收据若干、上海市卫生局的信访答复、居民死亡医学证明书等证据证实。
　　本院认为，患者潘XX至X医院神经内科急诊并付费挂号，被告在医药费收据上加盖了急诊现金收讫章，双方由此形成了医疗服务合同。依法成立的合同，受法律保护。双方当事人均应按约定全面履行自己的义务。现原告主张被告存在医疗过失，构成违约，要求其承担违约责任。医疗过失违约责任的承担，是以医方的医疗行为存在不符合医疗合同的约定、未履行其应尽注意义务的事实为基本条件。而本案中被告的医疗行为是否存在过失，导致了患者的不良后果，应当由医学会予以鉴定。本案医疗争议经区、市两级医学会鉴定，对于是否构成医疗事故的结论不同。根据《医疗事故处理条例》的规定，当事人对首次鉴定结论不服的，可以提出再次鉴定的申请。同时，法院对于委托医学会进行医疗纠纷鉴定的相关规定也明确：经区县医学会鉴定后，当事人仍有异议，可委托市医学会复核鉴定。故市医学会作为再次鉴定、复核鉴定机构，其出具的鉴定结论更具有权威性，应当被采纳。原告对市医学会鉴定结论中被告的责任承担有异议，但无证据证实市医学会鉴定结论存在错误，故本院对市医学会的鉴定结论予以采纳。现市医学会确认本次医疗争议属于一级乙等医疗事故，医方承担轻微责任，本院予以确认。由此可见，被告履行合同义务存有瑕疵，给患者造成损失的，损失赔偿额应当相当于因违约所造成的损失。根据上海市医学会的鉴定结论，被告应对原告的损失承担30%的赔偿责任。对被告应当承担损害赔偿的范围，本院确认如下：1、医疗费，患xxx病共花去医疗费7,090.86元，XX医院应赔偿原告医疗费2,127.26元（计算方式：7,090.86×30%＝2,127.26元）；2、丧葬费，按医疗事故发生地规定的丧葬费补助标准计算，被告应赔偿原告方7,013.55元（计算方式：23,378.50×30%=7,013.55元）。原告主张的死亡赔偿金，于法无据，本院不予支持。至于吕根妹抚养费，因吕根妹本人并未丧失劳动能力，故原告之请求，缺乏事实依据，本院不予支持。关于精神损害抚慰金，现被告自愿赔偿原告25,000元，经查并无不当，本院予以准许。依照《中华人民共和国民法通则》第九十八条、第一百零六条，《中华人民共和国合同法》第八条、第六十条、第一百十三条的规定，判决如下：
　　被告上海XX医院于本判决生效之日起十日内赔偿原告吕XX、潘XX、郁XX医疗费2,127.26元，丧葬费7,013.55元，精神损害抚慰金25,000元，合计34,140.81元。
　　负有金钱给付义务的当事人，如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　案件受理费11,761元(原告已预交)，减半收取计5,880.50元，由原告吕XX、潘XX、郁XX，被告上海XX医院各半负担；鉴定费7,000元，由被告上海XX医院负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
代理审判员陈培蓉
　　二〇一一年七月一日
　　书记员姚卓人
