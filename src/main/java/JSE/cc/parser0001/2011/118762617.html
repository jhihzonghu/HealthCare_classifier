周某某诉宁远县红十字会博爱医院医疗损害赔偿纠纷案

审判时间：20110121 法院：湖南省宁远县人民法院 案件字号：(2011)宁法民二初字第39号


　　原告周某某。
　　法定代理人周胜群。身份证号码：xxx
　　被告宁远县红十字会博爱医院
　　法定代表人蒋平，董事。
　　地址宁远县舜帝中路6号。
　　案由：医疗损害赔偿纠纷
　　原告周某某诉被告宁远县红十字会博爱医院医疗损害赔偿纠纷一案，本院于2011年1月21日受理后，在法庭审理之前，经本院主持调解：
　　原告周某某诉被告宁远县红十字会博爱医院医疗损害赔偿纠纷一案，双方当事人于2011年1月21日自愿达成如下调解协议：
　　一、由被告宁远县红十字会博爱医院一次性赔偿原告周某某医疗费、伤残补助金、后期治疗费等共计人民币26000元（不含已经给付的19500元）。双方签字后即时给付。
　　二、赔偿金给付后，原告方之后不得再主张任何权利，双方此次医疗纠纷终结。
　　案件受理费200元，减半收取100元，由被告宁远县红十字会博爱医院承担。
　　双方当事人一致同意本协议，自双方当事人在调解协议上签名或捺印后，即具有法律效力。
　　上述协议，符合有关法律规定，本院予以确认。

　　审判员欧明辉
　　审判员何胜军
　　人民陪审员骆友情
　　二0一一年一月二十一日
　　代理书记员陈鑫
