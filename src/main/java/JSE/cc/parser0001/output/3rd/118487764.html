XX医院与XX医疗服务合同纠纷案

审判时间：20110804 法院：上海市浦东新区人民法院（司法公开示范法院） 案件字号：(2010)浦民一(民)初字第29095号


　　原告XX医院。
　　法定代表人XX，院长。
　　委托代理人XX。
　　委托代理人XX，上海XX律师事务所律师。
　　被告XX。
　　法定代理人XX（系被告XX丈夫），住所地同被告XX。
　　委托代理人XX，上海市XX律师事务所律师。
　　委托代理人XX，上海市XX律师事务所律师。
　　原告XX医院（以下简称XX医院）与被告XX医疗服务合同纠纷一案，本院于2010年9月26日立案受理后，依法适用简易程序，由代理审判员徐俊独任审理，于2010年10月19日公开开庭进行了审理。原告XX医院的委托代理人XX、XX，被告XX的法定代理人XX、委托代理人XX到庭参加诉讼。同年10月28日，本案因需以另一案件的审理结果为依据，故裁定中止审理。之后本案转普通程序审理，依法组成合议庭，于2011年7月12日恢复审理后再次公开开庭进行了审理。原告XX医院的委托代理人XX、XX，被告XX的法定代理人XX、委托代理人XX到庭参加诉讼。本案现已审理终结。
　　原告XX医院诉称，2008年2月5日凌晨，被告因怀孕临产入住原告医院生产。原告于同日6时许为被告实施了剖宫产手术产下一子。7时左右手术结束被告回到病房。9时左右被告出现面色发紫、抽搐等症状，经抢救，被告生命体症恢复平稳，已经无需再在原告处住院的必要。原告为此多次与被告方沟通希望被告尽早出院到外院进行康复治疗。但被告方以医院有医疗过错为由拒绝出院，且一直拒绝支付医疗费。原告认为，被告入住原告医院接受检查、治疗服务，双方已建立医疗服务合同。原告已履行相关义务，被告住院产生的相关费用应由被告承担，故要求判令被告支付原告医疗费人民币704,584.64元（2008年2月5日起，至2010年8月27日止）。审理中，原告变更诉讼请求，要求被告支付医疗费、护理费总额899,567.10元的35%，计314,848.49元。被告已付原告的押金4,000元，可从其应付款中予以扣除。
　　被告XX辩称，被告对原告计算的医疗费总额不予认可。原告提供了费用大项统计，但无法证明这些费用都是用于被告。而且这些费用中出现了三项护理费用，其中联帮川沙卫益病人看护服务社的费用为每天80元，被告认为这80元一天并不是对被告一人护理的，而是护理几个病人的。医疗费中的床位费被告认为也是不合理的。至于其他可以证明用于被告身上的费用被告可以认可。虽然另案法院认定被告应该承担35%的责任，但我们对这个比例的划分持有异议，目前正在要求再审。被告不同意原告的诉讼请求。
　　经审理查明，2008年2月5日凌晨5时左右，被告因怀孕待产入住原告医院产科。当日6时左右，原告为被告行子宫下段剖宫产手术。9时45分，原告静脉滴注头孢哌酮即将结束时，即刻突然出现咽喉部不适、咳嗽，随后口吐白沫、抽搐，继而面色嘴唇青紫，呼吸浅、点头状等，即予以面罩吸氧等急救措施……。16时，原告转入内科ICU。19时12分，诊断为：1、剖腹产术后；2、羊水栓塞，过敏性休克，心肺复苏后。事后，被告仍留在原告医院ICU病房中治疗，但原、被告双方已产生医疗纠纷，为此，被告曾于2008年11月28日诉至本院，要求原告承担医疗损害赔偿责任。该案审理中，本院委托了司法鉴定科学技术研究所司法鉴定中心对被告的休息、护理以及营养时限进行鉴定，结论为：被鉴定人XX剖宫产术后突发呼吸困难、神志不清等，临床诊断为“羊水栓塞，脑梗塞”等，并持续住院治疗至今，其误工期为840日。目前其生活自理能力丧失，属于完全护理依赖，今后需长期护理，护理人数为每天2人以上，并长期补充营养。2010年10月19日，本院认定原告应承担被告医疗损害赔偿的全部责任，并判决原告赔偿被告的误工费、住院伙食补助费、陪护费、残疾生活补助费等九项经济损失。判决后原告不服提起上诉，上海市第一中级人民法院经审理后认为，该起医疗损害赔偿纠纷，原、被告双方均有一定责任，为混合过错，认定由原告承担65%的赔偿责任，被告自负35%的过错责任，并于2011年6月9日作了相应的改判。其中被告在住院期间的陪护费按1人次计算。以上一年度本市职工平均工资数额，结合双方的过错程度计算，由原告赔偿给被告。
　　另查明，被告住院时交付原告押金4,000元。自被告住院之日始，原告即为被告请了联帮川沙卫益病人看护服务社的护工1人次，对被告进行住院期间的护理，每天护理费为80元。截至2011年6月15日，共计护理1226.5天,合计护理费为98,120元。被告自2008年2月5日住院至2011年6月28日出院期间的医疗费为801,447.10元。以上医疗费及护理费合计为899,567.10元。
　　上述事实，有原告提交的（2010）沪一中民一（民）终字第2906号《》，原告于2011年7月12日打印的《上海市浦东新区人民医院病人费用大项统计》，被告提交的司鉴中心（2010）临鉴字第1311号鉴定意见书各1份，以及当事人的陈述等在案佐证。原、被告提交的其余证据材料属实与否，不影响本案的审理。
　　本院认为，被告入住原告处接受医疗服务，双方即建立了医疗服务合同关系。虽然原告在医疗服务中存在医疗过错，给被告造成人身损害，但原告在另案中已由法院判决承担了相应的赔偿责任。故被告应当向原告支付医疗费及护理费。现原告根据其过错程度，要求被告支付35%的医疗费及护理费，并无不当，本院应予支持。被告出具的医疗费结算凭据中的护理费及监护费项目，是原告为住院的被告提供的医疗服务，与原告主张的另请护工的为被告护理非同一种护理，不存在重复收费现象。根据鉴定结论，被告目前生活自理能力丧失，属于完全护理依赖，今后需长期护理，护理人数为每天2人以上，而另案中法院已判决原告赔偿被告住院期间1人的护理费，另1人系原告请人为被告护理，故被告也应按责任比例支付原告护理费。现原告主张的护理费以每日80元计，低于另案中原告赔偿被告的每日护理费标准，属合理范围，本院应予支持。被告的抗辩理由均不能成立，本院不予采纳。综上所述，依照《中华人民共和国合同法》第八条，第一百零九条的规定，判决如下：
　　一、被告XX应于本判决生效之日起十日内支付原告XX医院医疗费、护理费合计314,848.49元；
　　二、被告XX已付原告XX医院的押金4,000元，从上述第一项的应付款中予以扣除。
　　负有金钱给付义务的当事人，如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　案件受理费10,845元，由原告XX医院负担7,049元，被告XX负担3,796元。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
审判长冉志明
　　人民陪审员原金培
　　人民陪审员李蔓薇
　　二〇一一年八月四日
　　书记员姚卓人
