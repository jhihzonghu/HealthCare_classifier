XX医院诉颜XX医疗服务合同纠纷案

审判时间：20110210 法院：上海市杨浦区人民法院 案件字号：(2010)杨民一(民)初字第4750号


　　原告XX医院。
　　法定代表人李XX，院长。
　　委托代理人徐XX，上海市XX律师事务所律师。
　　委托代理人张XX，医院职工。
　　被告颜XX。
　　原告XX医院诉被告颜XX医疗服务合同纠纷一案，本院受理后，依法组成合议庭，公开开庭进行了审理。原告XX医院的委托代理人徐美琴、被告颜XX到庭参加诉讼。本案现已审理终结。
　　原告XX医院诉称，被告因右下腹疼痛，诊断为“双肾囊肿，左肾结晶或小结石”于2008年8月27日入原告泌尿外科治疗，术前诊断为“右肾多发囊肿”。2008年9月23日在全麻下行“右肾多发囊肿去顶减压术”，2008年9月29日在全麻下行“结肠肝区肠瘘冲洗引流＋末端回肠造瘘术”和“末端回肠造瘘口回纳术”，手术顺利，术后恢复正常。2009年4月23日原告医师即告知被告达到临床出院标准，要求其办理出院手续。但被告及家属以存在医疗纠纷为由拒不出院，并强占泌尿外科住院病床至今，期间原告虽数次催办，均未果，并拒付所欠医疗费。原告认为，被告入住XX医院治疗，双方之间形成事实之医疗服务合同关系，现原告已经履行了对被告提供医疗服务之义务，被告理应履行对价的出院及缴纳医疗费用之义务，现被告时常回家居住却以医疗纠纷为由，强占病床至今，已影响到原告泌尿外科床位的正常周转，妨碍了原告的正常医疗秩序，并给原告造成损失。请求依法被告立即办理出院手续，迁出原告病房。
　　被告颜XX辩称，被告自被原告收治入院后，进行了多次手术，造成医疗事故，并且原告至今未将被告治愈，存有肠粘连、肠痉挛等后遗症，需要继续治疗。原告必须保证被告能随时入院治疗，被告可以办理出院手续。
　　经审理查明，2008年8月27日，被告因“间歇性腰背部酸痛半月余”入住原告处。入院诊断：①左肾绞痛，结石？双肾囊肿；②高血压病；③糖尿病；④心肌缺血；⑤脑供血不足；⑥颈椎、腰椎外伤后。9月2日，肾脏及盆腔CT示：①左肾小囊肿，左肾结石；②右肾多房囊样病变。9月23日，被告在全麻下行“腹腔镜下右肾多发囊肿去顶减压术”。因组织粘连严重，腹腔镜下分离困难，故中转行开放手术。术中分离粘连组织时造成结肠肝曲破裂，用1号丝线缝合结肠肝曲破损处。术后予抗感染、补液、支持治疗。9月29日，发现被告伤口有粪汁样液体流出，考虑结肠瘘。当日在全麻下行“结肠肝曲肠瘘冲洗+腹腔引流+末端回肠造瘘术”。术中见结肠肝曲瘘口，直径1厘米，有粪汁样液体流出，周围组织粘连，局部包裹。术后予抗炎、补液、对症、支持治疗，局部冲洗引流。12月29日，被告在硬膜外麻醉下行“末端回肠造瘘口回纳术”。被告术后反复诉夜间腹部不适、疼痛，右侧腹部为主。查体未见明显肠梗阻体征，予对症、止痛治疗后缓解。2009年6月8日，第二军医大学附属长海医院普外科会诊。建议口服福松、得舒特对症治疗，停用凯纷等非甾体类止痛药，腹部理疗。6月23日，被告诉右下腹疼痛，疼痛局限于右侧手术疤痕邻近区域，并自行扪及右下腹有一“肿块”。查体：原末端回肠造瘘手术切口可见一约6某6厘米肿块，质地软，活动度佳，听诊可闻及肠鸣音，5次/分。右下腹压痛（+），无反跳痛或肌卫，移动性浊音（-）。考虑为“切口疝”，予手法复位后弹力绷带加压包扎。2009年4月23日起原告多次通知被告出院。被告认为原告对其造成医疗事故，其病情存有后遗症仍需住院进一步治疗，故拒绝办理出院手续。
　　本院认为，在医疗关系中，患者享有充分了解医疗活动所含风险的权利及获得适当、合理治疗的权利，患者有配合医师诊疗和遵守医方制定的规章制度的义务。在本案中，原告下属医师根据多年被告住院期间病情发展及多次手术后的临床状态，认为被告已符合出院指征，有权决定出具出院小结，通知患者出院。作为患者的被告应当尊重医师决定，及时办理出院手续，迁出病房。关于被告的辩称，本院认为，医疗事故被告可另行主张权利。据此，依照《中华人民共和国民法通则》第三条、第四条、第五条之规定，判决如下：
　　被告颜XX应于本判决生效之日十五日内办理出院手续，迁出被告XX医院病房。
　　本案受理费人民币50元，由被告颜XX负担。
审判长陈海峰
　　代理审判员施春梅
　　人民陪审员李佩蓉
　　二Ｏ一一年二月十日
　　书记员秦岭
