蔡九松与乐清市人民医院医疗损害赔偿纠纷案

审判时间：20110509 法院：浙江省温州市中级人民法院 案件字号：(2011)浙温民终字第376号


　　上诉人（原审原告）蔡九松。
　　委托代理人施泰顺。
　　上诉人（原审被告）乐清市人民医院。
　　法定代表人陈余钿。
　　委托代理人南晓浩。
　　委托代理人王绍础。
　　上诉人蔡九松与上诉人乐清市人民医院医疗损害赔偿纠纷一案，不服乐清市人民法院（2008）乐民初字第2094号民事判决，向本院提起上诉。本院于2011年3月24日受理后，依法组成合议庭，于2011年4月8日公开开庭审理了本案，上诉人蔡九松、委托代理人施泰顺、被上诉人乐清市人民医院的委托代理人王绍础到庭参加诉讼。本案现已审理终结。
　　原审法院认定：原告蔡九松妻余银桃（1937年6月13日出生）因身体不适，于2008年2月28日上午10时到被告乐清市人民医院门诊就诊。经诊断为“1、心律失常、房早、房速；2、冠心病；3、上呼吸道感染”并入住。住院后，被告方对患者予以血压、心电检测、补液、抗感染等治疗。2月29日上午，患者余银桃突发头晕，言语不清，左侧肢体活动障碍。经行头颅CT检查，发现可疑双侧颞叶低密度灶，被告予抗凝等治疗。3月1日0时10分予留置导尿处理。3月1日2时30分患者病情加重。被告考虑脑梗死范围扩大，将患者转入ICU治疗。3月1日下午患者出现深昏迷状态，被告方予呼吸机辅助呼吸、升压、抗休克等对症处理。经抢救无效后，患者余银桃于3月2日0时出院后死亡。期间，原告方支出医疗费5399.91元。医患纠纷发生后，受原审法院委托，温州市医学会于2009年3月23日作出鉴定结论，认定涉案病例不属于医疗事故，同时认为医方对患者病情的危重性及抢救措施选择的告知上存在不足（如未告知是否采用手术开颅减压），但这些不足与患者死亡无因果关系。原告对此不服，提起再次鉴定申请。浙江省医学会于2009年9月29日作出鉴定结论，认定涉案病例不属于医疗事故，同时认为被告在诊疗过程中存在没有及时导尿、CT报告发现可疑低密度颞叶病变后未及时行头颅MRI检查等过失，但与患者死亡最终结果没有因果关系。二级鉴定机构据以作出的鉴定结论的依据中，被告方提供的部分病历存在后补、错写的情形。审理过程中，余银桃子女已明确表示对赔偿权利予以放弃归原告享有。案经调解无效。
　　原告以被告医疗人员违反医疗规程所致，本案属于重大医疗事故为由，于2008年12月2日向原审法院提起诉讼，请求法院判决被告赔偿原告各项经济损失合计80万元，并承担本案诉讼费。
　　乐清市人民医院在原审中答辩称：本案不属医疗事故，有二级鉴定机构作出的鉴定结论为依据。且鉴定意见认为被告的诊疗行为与患者死亡之间不存在因果关系，被告方也不存在医疗过错。原告的诉讼请求没有事实和法律依据，应依法予以驳回。
　　原审法院认为：根据《医疗事故处理条例》第十一条的规定，在医疗活动中，医疗机构及其医务人员应当将患者的病情、医疗措施、医疗风险等如实告知患者，及时解答其咨询。而《执业医师法》第二十二条规定，医师实行医疗、预防、保健措施，签署有关医学证明文件，必须亲自诊查、调查，并按照规定及时填写病历资料，不得隐瞒、伪造或者销毁病历资料及有关资料。本案被告在对患者余银桃的诊疗过程中，存在告知不足及发现可疑病变危险后没有及时采取相应措施等过错，且提供鉴定的部分病历资料系后补，应依法认定其未尽医疗义务并构成一定程度的医疗过错。虽然相关鉴定机构认为被告的诊疗行为不属医疗事故，但不能以此免除其对患者的死亡承担相应赔偿责任的民事法律后果。根据《最高人民法院关于参照﹤医疗事故处理条例﹥审理医疗纠纷民事案件的通知》的相关规定，适用《民法通则》规定的过错责任赔偿处理原则，原审法院确定被告应对原告造成的经济损失承担50%的赔偿责任。原告的经济损失中，医疗费按认定数额计算；余银桃死亡时年满71周岁，其死亡赔偿金应按浙江省2009年度城镇居民人均纯收入24611元的标准计算9年；丧葬费按浙江省2008年度职工平均工资34146元的标准计算半年；余银桃的死亡确已造成其近亲属的精神痛苦，应予赔偿精神损害抚慰金，具体数额由根据审判实践和被告的过错程度予以确定。原审法院依照《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》第十七条第一款、第三款、第十八条、第十九条、第二十九条、第三十一条、第三十二条的规定，判决：被告乐清市人民医院应赔偿原告蔡九松医疗费5399.91元、死亡赔偿金221499元、丧葬费17073元、精神损害抚慰金20000元，合计经济损失263971.91元的50%，计131986元。款限判决生效后十日内交原审法院民事审判第一庭转付。如果未按判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百三十二条的规定，加倍支付延迟履行期间的债务利息。案件受理费11800元，由原告蔡九松负担6400元，被告乐清市人民医院负担5400元。
　　宣判后，双方均不服，向本院提起上诉。
　　蔡九松上诉称：一、余银桃因感身体不舒服，门诊后住院，住院二天后死亡，这完全是医方违反诊疗护理规范、常规造成的结果。二、医方为掩饰自己过失，采取了隐匿、伪造病历资料的行为。三、医疗事故技术鉴定背离了事实，失去了公平和正义。请求：1、进一步查明被告违反医疗法规、操作规程和事后隐匿、伪造有关病案证据的事实；2、按照《医疗事故处理条例》确认被告的医疗事故责任；3、重新判定被告应当承担的经济赔偿责任及案件受理费和7250元医疗事故鉴定费。4、对温州和省医学会的鉴定进行甄别质证，并要求通知两医学会派人在二审出庭作出说明。
　　乐清市人民医院答辩称：医院方在诊疗过程中基本上是符合医疗规范的，一些错误与余银桃死亡后果没有因果关系，病历有些地方书写不规范是存在的，但有些是蔡九松不知道相关规定而造成的误会。2004年至2007年余银桃曾三次住院，“体位性低血压”有可能造成脑栓脱落，从而造成脑梗塞而死亡。综上，蔡九松的诉讼请求应予驳回。
　　乐清市人民医院上诉称：一、一审事实认定错误，责任定性不准确。一审认定“被告对涉案病历确有存在后补、错写的情形。”对此，我们不予认可，在病历上出现的有关记载符合事实，如病历首页的2008年3月7日，系病历质控时间。而非患者死亡时间。其他时间及报告单记载也是符合常规的。二、该案经过温州及浙江医学会鉴定，并不构成医疗事故，且鉴定报告已经明确表明患者的死亡和医院的诊疗行为并不存在因果关系，因此根据侵权责任的有关原理，医方并不需要对其死亡后果承担责任。综上请求撤销一审判决，在查明事实真相情况下依法改判，并由被上诉人承担全部的诉讼费用，
　　上诉人乐清市人民医院在二审诉讼中未提供新证据，上诉人蔡九松重申了一审已提供的全部证据，以此佐证其主张。本院经审核原审证据并经开庭审理，对原审认定的治疗经过以及省市医学会的医疗事故技术鉴定结论予以确认。关于医方是否存在过错，以及病历是否存在后补、错写的情形，在双方争议部分予以分析认定。
　　关于双方上诉争议问题，分析如下：
　　一、上诉人蔡九松主张，其妻余银桃只是小毛病，身体并无大碍，理由是：1、2008年2月28日上午10点，门诊诊断：“体位性低血压？留现”；2、入院《体格检查表㈠》报告身体正常；3、住院第二天7时许，CT报告单影象诊断：“两侧颞叶可疑低密度影，建议MRI进一步检查”，但医师未作近一步检查。
　　乐清市人民医院认为，蔡九松的观点不成立，1、余银桃曾于2004年至2007三次在乐清市人民医院住院治疗，当时的诊断为冠心病、阵发性房颤、高血压病（3级），本次门诊又晕厥一次，一度血压很低（仅60／40mmhg）；2、从诊断角度看，不能仅凭入院《体格检查表㈠》对被检查者作出身体正常或不正常的结论；3、冠心病、阵发性房颤、高血压病等疾病都是脑梗死病的易患因素。患者发生大脑动脉栓塞进而引起右侧大面积脑梗死、脑水肿致脑疝形成，病变事先无征兆，发病突然，病情严重，死亡率极高。
　　本院认为，门诊初步检查以及入院常规体检表并不能全面证明患者身体状况。根据两级医疗事故技术鉴定结论，患者直接死亡原因系住院期间发生大脑中动脉栓塞、右侧大面积脑梗死致脑疝形成而死亡。因此，对于上诉人蔡九松的主张，不予采纳。
　　二、上诉人蔡九松主张，医务人员操作不规范。1、医方不按医嘱“留置导尿、床头监护”。29日上午医师看诊，在《长期医嘱单》上7：30分记下“床头监护”，8点钟记下“留置导尿”。但后来医师却在这两项后面注上“取消”二字。2、29日没有作MRI进一步检查，只是继续大量输液，使患者腹胀难受，以致病情剧变，身体发烧。
　　乐清市人民医院认为，1、取消留置导尿并不违反护理常规。临床上需要导尿的情况有，⑴尿潴留；⑵尿失禁；⑶意识模糊或昏迷大小便不能自理；⑷需要精确计算患者出入量，无其他有效方法代替。而余银桃的2008年3月29日护理记录中，都有小便知解和大小便知解的记载，即不存在需要导尿的情况。脑梗塞病人需要静养，而留置导尿本身就是刺激源，因此临床医生权衡利弊取消了已经开出的医嘱，护士也中止执行，这一医疗行为均属正常，并不违反任何治疗护理常规。2、未进一步MRI检查与死亡无因果关系。根据2008年3月29日上午患者的CT检查报告，可以证明当时患者病情尚未发展至脑水肿阶段，根据卫生部编辑的《临床诊疗指南》第十一章第一节“脑梗死”篇的【诊断要点】规定，临床上做CT或MRI都可以。是否要做MRI或何时做MRI，需临床医生根据病人的个体情况予以决定。当时根据患者临床表现、CT检查结果和神经科医生的定位诊断，已经明确诊断为脑梗塞以及病变部位，并立即采取了相应的治疗措施与方案，故当天未予以MRI检查根本未影响到对患者病情的判断与治疗措施的实施。3、不存在蔡九松所称的大量输液使患xxx病情剧变的情况。2008年2月29日，患者总液体输入量为1550ml，而正常人体一天需要水份约2500-3000ml。而且没有及时导尿与患者病情恶化和死亡不存在因果关系。
　　本院认为，根据省医学会的医疗事故技术鉴定书的结论，医方在诊疗过程中未及时导尿以及未及时行头颅MRI检查，该事实予以确认。但蔡九松称，未及时导尿导致了病情剧变，该主张依据不足，不予采纳。
　　三、上诉人蔡九松主张，乐清市人民医院为掩饰自己过失致余银桃死亡的责任，采取了隐匿、伪造病历资料：1、隐匿28日上午门诊开出的住院留现通知单；2、档案中抽走CT检查报告单，只留MRI报告单。上诉人手上的CT检查报告单上审核医师是赵小帼，而医方迟迟举证的CT和MRI报告单上审核医师却是另外两位医师；3、3月2日凌晨余银桃已死亡出院，而《住院病案首页》时间却是“3月7日”；4、《体格检查表㈡》中写心电图辅助检查2008年7月10日，而此时余银桃已死亡4个多月；5、余银桃入住时，《体格检查表㈠》报告正常。而医院方举证的护理记录故意用《危重患者护理记录》：6、医院方在复印件上也在作弊。医院方提供的几份入院病情登记和护理记录表复印件，其余几份故意不复印抬头名称，或落款日期。7、余银桃于3月1日10时30分已转入重症监护室（床号2）；可是医院方在原内科十6A51床号的《长期医嘱单》上仍记载3月1日13时“速尿针20mg”，这是明显伪造。
　　乐清市人民医院认为，1、门诊医生开出的住院申请单（住院留现通知单）一般夹在住院病案夹中，离院时交患者结算住院费和做出门单用。按惯例不会将其整理在病历档案中，卫生行政部门也没有这个要求，所以不存在医方隐藏的事实。2、医院方出示的患者CT报告和MRI报告各一份，内容、时间、报告医生和蔡九松提供的CT报告和MRI报告相同，只是审核医生名字不一致。原因是蔡九松手头的报告单系其通过关系从医院电脑档案的文件模板中打印出的，不是一个合法有效的报告。根据卫生部颁布的《医院工作人员职责》第五十五条第4点，第五十六条第3点规定，放射科的报告单要集体阅片后，由主任或主治医生审签诊断报告单才可对外正式提供，医院提供法院的报告单上有上级负责医生的手写签名，这才是正式有效的文件。3、住院病案首页的落款时间确实是2008年3月7日，根据《卫生部关于修订下发住院病案首页的通知》住院病案首页填写说明第三十条第5项规定：“日期：由质控医生填写”。质控人员填写病历首页时间是在医疗行为和病历整理全部完成之后，病案归档之前，进行病案质量检查后填写的。本案患者在2008年3月2日出院，质控人员填写时间为2008年3月7日，这是完全符合规定。4、《体格检查表㈡》中辅助检查的日期，电脑打印成2008.7.10确实错误的。这个错误来源于首次病程录中，值班医生将辅助检查：心电图检查时间07年10月的日期描写为（07.10）。如果此项错误在病历归档前发现，医方亦有权按照《病历书写基本规范（试行）》第六条、第七条的规定进行修改。5、关于使用“危重患者护理记录”单的问题。本案患者入院时的医嘱是内科1级护理，“危重患者护理记录”单主要是对1级护理的相关内容进行记录，故属正常工作程序。6、蔡九松指责医方复印病历时故意掩盖“危重患者护理记录”抬头或时间栏，系其盲目猜测。实际情况是已装订的材料在复印时未复印到位所致，应以原始证据为准进行认定。7、《长期医嘱单》上的医生书写医嘱位置确属不规范。2008年3月1日患者转科后，下午13时30分，神经科医生张帆前往ICU会诊后书写医瞩。本来张医生应当将医嘱写在ICU科的专科医嘱栏下，由于内科长期医嘱第二页栏下尚有空白的地方，当时又未划掉，张医生由此将医嘱写了上去，所以存在书写位置不规范情况，但非伪造。“执行护士栏”中的执行护士朱晓珍为ICU科护士，这一事实可以佐证上述情况属实。
　　本院认为，根据双方的庭审质证，蔡九松主张的医院方隐藏“住院留现通知单”、“CT检查报告单、MRI报告单”、《住院病案首页》签署时间错误、编造《危重患者护理记录》、复印资料作弊等问题，均举证不足，不予采信。但医院方的确存在以下不规范之处：1、将《体格检查表㈡》中辅助检查日期打印成2008.7.10，实际应为2007年10月。2、重症监护室的医嘱写在原内科十6A51床号的《长期医嘱单》上，但该医嘱已有重症监护室护士执行。
　　本院认为，一、关于本案法律适用问题，本案发生在侵权责任法实施（2010年7月1日）之前，根据《最高人民法院关于适用〈中华人民共和国侵权责任法〉若干问题的解释》第一条之规定，侵权责任法施行前发生的侵权行为引起的民事纠纷案件，适用当时的法律规定。因此，本案应适用《医疗事故处理条例》以及民法通则的有关规定处理。二、关于证据的认定，本案上诉人蔡九松虽然对医院方的病历资料提出许多异议，但不足以证明医院方存在伪造、篡改或者销毁病历资料，或隐匿、拒绝提供与纠纷有关的病历资料的情形。根据病历资料等证据，二级医学会作出的医疗事故技术鉴定书，内容详细，分析清楚，可以认定其证明力。三、从医疗事故的角度分析，《医疗事故处理条例》第二条规定：“本条例所称医疗事故，是指医疗机构及其医务人员在医疗活动中，违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，过失造成患者人身损害的事故。”本病例经过二级医学会的医疗事故技术鉴定，可以证明医院的诊疗行为未违规，因此本病例不属于医疗事故。四、是否构成医疗事故之外的其他医疗损害问题，民法通则第一百零六条第二款规定，公民、法人由于过错侵害国家的、集体的财产，侵害他人财产、人身的应当承担民事责任。根据法院查明的实施，医院方仍存在以下过错，1、医方对患者病情的危重性及抢救措施选择的告知上存在不足（如未告知是否采用手术开颅减压）；2、医方在诊疗过程中存在没有及时导尿、CT报告发现可疑低密度颞叶病变后未及时行头颅MRI检查等过失，3、病历记载不规范。将《体格检查表㈡》中辅助检查的日期打印成2008.7.10；重症监护室的医嘱写在原内科十6A51床号的《长期医嘱单》上。特别是以上1、2点过失，足以产生一定损害后果，根据民法通则相关规定，给予一定的赔偿仍属必要。原审法院确定医方承担50%的赔偿责任，予以维持。双方上诉人的各上诉理由，均不成立，依法不予采纳。综上，依据《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　本案二审受理费，上诉人蔡九松方上诉费3000元，由蔡九松负担；乐清市人民医院方上诉费3000元，由乐清市人民医院负担。
　　本判决为终审判决。
