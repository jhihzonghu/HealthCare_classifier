济源市人民医院与李立元医疗损害赔偿纠纷上诉案

审判时间：20110728 法院：河南省济源市中级人民法院 案件字号：(2011)济中民二终字第99号


　　上诉人（原审被告）济源市人民医院。
　　法定代理人李社会，院长。
　　委托代理人成占强，该单位工作人员。
　　委托代理人欧胜宏，河南剑光律师事务所律师。
　　被上诉人（原审原告）李立元，又名李一丰。
　　委托代理人郭淑花，系李立元妻子。
　　上诉人济源市人民医院（以下简称市人民医院）与被上诉人李立元医疗过错损害赔偿纠纷一案，李立元于2007年10月8日诉至济源市人民法院，请求判令市人民医院赔偿医疗费、误工费等各项损失共计615751.32元，济源市人民法院于2011年3月25日作出（2007）济民一初字第2796号民事判决，市人民医院不服该判决，向本院提起上诉，本院于2011年7月1日受理后，依法组成合议庭公开开庭进行了审理。上诉人市人民医院的委托代理人成占强、欧胜宏、被上诉人李立元的委托代理人郭淑花到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明：2006年11月20日20时左右，李立元突发右侧上下肢活动障碍、语言不清等症，于当日21时左右到市人民医院治疗，诊断为1、脑出血，2、高血压病Ⅲ级、高危。李立元在市人民医院内一科入院治疗，市人民医院于2006年11月21日18时后为李立元行颅内血肿微创碎吸术，2006年11月22日医生查房时李立元神清，精神差，睡眠多，术后市人民医院以脑穿针引流管为李立元引流，并告知引流袋高度不能变动。2006年11月23日，市人民医院工作人员查房时发现引流袋高度变动而与李立元家属发生争吵，李立元家属称11月22日晚一实习护士将引流袋高度降低，后李立元家属找该护士时，市人民医院工作人员称已经转走，不知道叫什么名字。23日，由于李立元神志逐渐恶化，并浅昏迷，市人民医院对李立元进行了会诊，诊断为左侧基底节出血并脑疝，遂转至外三科，行血肿清除术和去骨瓣减压术。病历记载2006年11月23日中午10点（科长期医嘱单第一页、科临时医嘱单第二页、神经科护理记录表第三页均载明11月23日中午9时）医生查房时发现李立元浅昏迷状态，呼叫不睁眼，病情加重，医嘱复查头颅CT，而神经科护理记录记载中午12时李立元外出做头颅CT，手术记录记载下午3时对李立元进行手术。李立元提供的证据可以证明李立元的病历有所改动而市人民医院工作人员解释改动病历的原因不一；市人民医院工作人员说明2006年11月23日李立元昏迷后不能再用尿激酶而神经科护理记录表记载11月23日注入尿激酶5万u；病历中科长期医嘱单第1页、科临时医嘱单第3页并非李立元在神经内科治疗时的医生成占强所写；科临时医嘱单第2页医嘱11月21日注入尿激酶5万u/×2，11月23日未医嘱注入尿激酶，而神经科护理记录表却记载11月21日注入尿激酶5万u，11月23日注入尿激酶5万u。李立元于2007年1月31日出院，出院诊断1、脑出血，2、高血压病Ⅲ级（极高危）。出院当日李立元又入住郑州大学第五附属医院，于2007年7月19日出院。李立元的伤残经鉴定：脑出血治疗后遗留不完全性运动性失语、右上下肢不全瘫痪，为三级伤残，护理依赖程度经鉴定为大部分护理依赖，护理人数为一人。庭审中双方就市人民医院是否存在医疗过错均申请鉴定，该院委托河南豫天法医临床司法鉴定所进行鉴定，2009年5月7日，该所出具了《关于退回李立元司法鉴定委托的说明》，内容为“贵单位委托的李立元鉴定一案，由于患xxx病历资料不认可，无法进行医疗纠纷鉴定，故退回李立元有关医疗纠纷司法鉴定委托，特此说明”。李立元的损失为医疗费20970.07元、住院伙食补助费3630元、营养费2420元、住院期间护理费17520.8元、误工费35509.56元、残疾赔偿金211696元、定残后的护理费158224元，共计449970.43元。李立元另要求交通费600元、精神损害抚慰金50000元。另查明：2006年11月20日中午9时45分，科长期医嘱单医嘱按一级护理标准护理李立元，但神经科护理记录记载11月22日晚上并未查看李立元体温、脉搏、呼吸、血压、神志、瞳孔。
　　经查相关资料，一级护理中有一项每15-30分钟查看患者的要求。
　　原审法院认为：因医疗行为引起的侵权诉讼，由医疗机构就医疗行为与损害结果之间不存在因果关系及不存在医疗过错承担举证责任。就市人民医院是否存在医疗过错双方均申请司法鉴定，鉴定机构以患xxx病历资料不认可，无法进行鉴定为由退回鉴定委托。但综合本案的全部证据资料，李立元提供的证据确实可以证明：1、在市人民医院工作人员交代引流袋高度不能变动的情况下引流袋高度发生了变动；2、李立元的病历有所改动且市人民医院工作人员解释改动病历的原因不一；3、市人民医院工作人员说明2006年11月23日李立元昏迷后不能再使用尿激酶，而神经科护理记录表却记载11月23日注入尿激酶5万u；4、病历中科长期医嘱单第1页、科临时医嘱单第3页并非李立元在神经内科治疗时的医生成占强所写；5、科临时医嘱单第2页医嘱11月21日注入尿激酶针5万u/×2，11月23日未医嘱注入尿激酶，而神经科护理记录表却记载11月21日注入尿激酶5万u，11月23日注入尿激酶5万u；6、病历记载2006年11月23日中午10点（科长期医嘱单第一页、科临时医嘱单第二页、神经科护理记录表第三页均记载11月23日中午9时）医生查房时发现李立元浅昏迷状态，呼叫不睁眼，病情加重，医嘱复查头颅CT，而神经科护理记录记载中午12时李立元外出做头颅CT，手术记录记载下午3时对李立元进行手术；7、2006年11月20日中午9时45分，科长期医嘱单医嘱按一级护理标准护理李立元，但神经科护理记录记载11月22日晚上并未查看李立元体温、脉搏、呼吸、血压、神志、瞳孔；8、市人民医院使用实习护士护理李立元。由于不能进行医疗技术鉴定且市人民医院存在以上情形，市人民医院也不能证明其在对李立元的医疗行为中无过错，应当推定市人民医院的医疗行为存在过错，同时市人民医院也不能证明其医疗行为与李立元的损害后果之间无因果关系，故市人民医院应赔偿李立元损失。由于李立元到市人民医院就诊时被诊断为脑出血、高血压病Ⅲ级（高危）,病情严重，故可酌情减轻市人民医院的赔偿责任，该院确定市人民医院应对李立元的损失负70％的赔偿责任。李立元要求的交通费600元，虽未提供证据，但李立元住院及前往郑州鉴定必然支出交通费，故酌定交通费为300元。李立元的损失共计450270.43元，市人民医院应当赔偿315189.3元。李立元构成三级伤残，要求精神损害抚慰金符合法律规定，但其要求50000元过高，综合本案情况，考虑到当地的生活水平，该院确定市人民医院应当赔偿李立元精神损害抚慰金10000元。综上，根据《中华人民共和国民事诉讼法》第六十四条第一款、《中华人民共和国民法通则》第一百零六条第二款、第一百一十九条、《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》第十七条、《最高人民法院关于确定民事侵权精神损害赔偿责任若干问题的解释》第十条之规定，经该院审判委员会讨论决定，判决如下：一、市人民医院于判决生效后十日内赔偿李立元315189.3元；二、市人民医院于判决生效后十日内赔偿李立元精神损害抚慰金10000元。案件受理费5258.7元，由李立元负担2481.7元，市人民医院负担2777元。
　　市人民医院不服原判，上诉称：1、李立元发生的损害后果，是因自身疾病所导致，而不是医院的过错。虽原审法院向新乡医学院三附院神经内科吴××主任进行了咨询，但咨询时并未出示李立元的CT照片，吴××并不能全面了解病情，所做陈述不能作为定案依据；2、吴××称引流袋高度降低后对颅压有影响，加快脑脊液流出，对脑内血管也有影响是指血肿破入脑室的情况，并非指单纯血肿，而李立元的脑出血属单纯血肿，低位引流是必要的治疗措施，吴××陈述的情况不适合李立元；3、其根据李立元病情应用液化剂，所用药物及剂量、方法均在规定范围内，且最后一次应用尿激酶，穿刺针在血肿中，尿激酶起液化血块的作用，并不接触血管，也不会引起再出血；4、23日晨9时患者病情平稳无特殊，CT检查只是例行检查，以了解血肿引流情况，原审认为该记录即为医生发现李立元昏迷的时间，没有事实依据。李立元于23日晨11时出现昏迷后，上诉人积极给予生命体征监护，联系患者家属，讲明病情变化、行头颅CT检查的必要性及可能出现的意外，并指派专人陪护，发现血肿扩大后进一步加强生命体征监护，经家属同意后转脑外科行手术治疗，并未延误患者病情；5、李立元有高血压病史10余年，未正规服药，致血压长期高达170-180/110-120mmHg，这是患者脑出血和再出血的根本原因。上诉人在李立元入院当天的病情告知书及脑出血颅内血肿微创清除术前均已向家属讲明，脑出血本身就存在很高的再出血风险，而颅内血肿微创清除术也存在再出血风险，李立元家属均表示理解并签字；6、原审认定2006年11月22日晚没有护理记录就不符合一级护理标准的认定，没有法律依据，护理人员已经按医嘱要求对患者进行了护理和记录；7、护理费的确定应考虑我国公民的平均寿命以及李立元的疾病状况，而不应不分案件情况，一律按20年计算。综上，请求撤销原判，改判上诉人承担次要责任。
　　李立元辩称：1、市人民医院的实习护士违规大幅度降低引流袋，引流速度过快，导致其颅内压过低造成再出血，出血后因整晚没有医生、护士查房，致使出血量不断增加；2、11月23日早上查房时其已经昏迷，主治医生说是又出血了并称病人当时的情况没有、也不敢使用尿激酶，但护理记录第三页显示11月23日11点时为病人注入了尿激酶；3、23日晨9时，医生发现引流袋移动造成再出血、昏迷后就全部撤离，没有采取任何急救措施，11点30分左右一个护士送来CT通知单，直至下午4点其才被推进手术室，病历上记载病人已经脑疝。市人民医院上诉说11时才出现昏迷，而且积极抢救，没有任何依据；4、再出血是引流袋问题引起，市人民医院发现后明知会导致再出血却不采取任何急救措施，并注入尿激酶，后家人在不知道有保守治疗的情况下，被主治医生误导，选择了穿刺手术；5、其从市人民医院出院后，又转院至郑大五附院康复中心康复治疗半年，回家后锻炼从未间断，所以身体才会有较好的预后，与市人民医院的治疗行为无关；6、22日晚上，市人民医院护士未去病房查房，导致未及时发现病人昏迷；7、其对上诉人对患者寿命的人身攻击表示强烈不满。综上，请求维持原判。
　　本院审理期间，市人民医院提供如下证据：1、2003年3月，颅内血肿微创清除技术全国研究与推广协作组编写出版的《颅内血肿微创清除技术临床应用培训教材》第23、39、66页，证明李立元的脑出血属单纯血肿，低位引流是必要的治疗措施，主治医生在治疗中根据李立元病情应用液化剂，所用药物及剂量、方法均在规定范围内；
　　2、人民卫生出版社出版的《神经病学》（第四版）第148页,证明脑出血本身就存在很高的再出血风险，颅内血肿微创清除术也存在再出血风险；
　　3、人民卫生出版社出版的全国高等学校教材《神经病学》第171页，证明李立元的疾病是由于自身疾病引起的；
　　4、济源市卫生局转发河南省卫生厅关于《医疗文书规范与管理》的补充规定第46页，证明新入院病人前三天，体温、脉搏、呼吸每日至少记录2次。15-30分钟巡视病人而不是记录。
　　李立元质证称：这是专业教材，其不懂，但何为低位，教材中也未明确。
　　本院认证如下：市人民医院提供的四份证据系理论教材，是供临床医学培训、学习所用，不能作为市人民医院在本案中是否具有过错的依据。
　　本院经审理查明的事实与原审审理查明事实相同。
　　本院认为，因市人民医院的病历有所改动，且相关人员对此解释不一，李立元对病历不予认可，现无法鉴定市人民医院的诊疗行为与李立元手术后出现的损害后果是否存在因果关系，但从市人民医院制作的病历内容看，市人民医院存在以下过错：1、关于尿激酶的使用，主治医生称李立元11月23日出现昏迷状态后，已不能再使用尿激酶且临时医嘱单上也未注明使用尿激酶，但同日的护理记录表中，却载明上午11时注入尿激酶5万u；2、11月23日上午10时，主治医生已发现李立元呈昏迷状态，呼叫不睁眼，病情加重，医嘱复查头颅CT，但直至上午12时48分才进行头颅CT检查并于下午3时施行手术，从时间跨度上讲，市人民医院拖延了对李立元的治疗；3、11月22日晚上，市人民医院护士未查看李立元的体温、呼吸等各项生命体征，导致未及时发现引流袋高度的变动及由此引起的不良反应。基于上述原因，市人民医院未能举证证明医疗行为与李立元的损害后果间不存在因果关系及不存在医疗过错，原审法院综合考虑李立元自身疾病，判令市人民医院承担70%赔偿责任并无不当。经鉴定，李立元为三级伤残，护理依赖程度为大部分护理依赖，护理人数为一人，原判按20年确定李立元的护理费，符合法律规定，市人民医院上诉称护理费按20年计算，时间过长的理由，本院亦不予采信。综上，原判认定事实清楚，适用法律正确，依法应予维持。根据《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费5259元，由济源市人民医院负担。
　　本判决为终审判决。
审判长吕振军
　　审判员孙东杰
　　代理审判员段雪芳
　　二○一一年七月二十八日
　　书记员贾娃娃
