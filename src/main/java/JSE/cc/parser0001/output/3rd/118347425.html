孙某与李某、上海李某口腔诊所身体权纠纷案

审判时间：20110923 法院：上海市普陀区人民法院 案件字号：(2011)普民一(民)初字第1506号


　　原告孙某
　　被告李某
　　被告上海李某口腔诊所
　　原告孙某与被告李某、上海李某口腔诊所身体权纠纷一案，本院受理后，依法组成合议庭，公开开庭进行了审理。原告孙某及其委托代理人宫克，被告李某及其与被告上海李某口腔诊所的共同委托代理人陆燕平、张滨到庭参加诉讼。本案现已审理终结。
　　原告孙某诉称，2005年7月左右，原告因右上第六颗牙及左下第六颗牙缺失，至被告上海市普陀区李某私立口腔诊所装牙。被告利用缺失牙相邻的第五、七颗牙齿做搭桥，为原告撞了牙套。之后，原告发现左下侧牙套没装好，一直掉，并由于牙齿装得不好，导致原告左下颌区肿大、淋巴结发炎及左下第七颗牙齿发炎。原告多次至被告处要求治疗，被告多次帮原告进行处理，直至将原告左下第七颗牙齿抽去牙神经，炎症仍未消除，后被告将该牙根刺穿，致牙根炎症更加严重，左下颌区也越肿胀。最终，原告只得至上海市第九人民医院将第七颗牙齿拔除。原告认为，被告为原告装牙套时，对左下第七颗牙齿过度磨损，使得牙套套不牢，引起原告牙周炎、根尖炎，并最终导致原告第七颗牙齿缺失及左下颌区肿胀。为此原告花费了一定的医疗费，并因左下第五颗牙已死、第六、七颗牙齿已经拔掉，故需要种左下第六、七颗牙及装第五颗牙，上述费用均应由被告承担，双方曾多次交涉，被告付给原告4000元左右，对原告尚存在的其他损失拒绝赔偿，原告遂诉至法院，要求判令：被告赔偿医疗费人民币3347.16元（以下币种均为人民币）、后续种左下第六、七颗牙齿及装左下第五颗牙齿费用45000元、律师代理费4000元、精神损害抚慰金5000元；本案诉讼费由被告承担。
　　原告为证明其诉请提供：始诊时间为2007年10月30日的门急诊就医记录册一册、复旦大学附属肿瘤医院门急诊病史一本、上海群力中医门诊部病史自管卡一册、记载于原告配偶傅万里就医记录册的就诊记录、超声报告单等病史资料；医药费单据；律师委托合同及发票等证据。
　　被告李某、上海李某口腔诊所辩称，李某系有资质的执业医师，依法取得个体工商户经营资质，获得医疗机构执业许可证。被告对原告陈述的就医过程不认可，原告系2004年4月前后在被告诊所装牙。直至2007年8月，原告因左颌下区车祸外伤来被告处咨询，被告建议其摄下颌咬合片，排除颌下腺导管结石；对左下侧的固定桥重新固定。2009年1月，至被告处因牙齿问题就诊，但原告拒绝接受被告建议其所作治疗。原告左下第七颗牙的拔髓、刺穿牙根等均系其在其他医疗机构进行，与被告无关。被告认为：原告下颌有外伤史，其拔牙记录左下第七颗牙是折裂牙；原告有龋齿但未按医嘱及时治疗导致炎症；第七颗牙拔牙神经时发生髓底穿孔，以上原因最终导致原告第七颗牙齿被拔除。而原告左下颌肿胀系淋巴结炎引起，淋巴结炎与被告的装牙行为之间亦无因果关系。综上，被告认为原告主张的损害后果与被告的诊疗行为之间无因果关系，故对原告的诉请均不同意。
　　被告为证明其辩称意见提供：原告在被告处就诊的病历卡两张、收据存根联一本、收条一份。
　　根据原、被告陈述意见及当事人提供的证据，本院对以下事实予以认定：2004年3月4日，原告孙某至上海市普陀区李某私立口腔诊所要求修复其缺失的右上及左下第六颗牙，医生建议作活动或者固定修复。同年3月29日，原告再次至该诊所，要求进行固定修复，即用缺失牙相邻的两颗牙齿做“桥桩”，做搭桥固定。当天，被告根据原告要求，为固定修复进行磨牙等处理，订做牙套及假牙。同年4月5日，原告完成右下及左下侧的牙套安装佩戴。原告为此支付1700元，被告开具收据。2007年8月30日，原告至被告诊所，病历卡记载：左颌下区车祸外伤咨询，诊断：左颌下区外伤继发左下567固定桥松动。建议：（1）、摄下颌咬牙合片；（2）、固定桥重新固定，患者拒绝。2008年2月23日，原告因左颌下区肿胀至上海市同济医院就诊，检查发现左下侧567固定桥已松动脱落，左下侧第六颗牙缺牙区可见残片……，诊断为不良修复体、左颌下淋巴结炎。建议1、拔除左下第六颗牙残片，患者要求继续观察；2、左下侧567不良修复体，建议重新治疗……。2009年1月15日，原告再至被告诊所，主诉：左下假牙脱落二周，诊断：左颌下区外伤继发左下固定桥脱落，左下第七颗牙深龋，牙髓炎可能，建议：左下第七颗牙齿作髓底治疗，患者拒绝。同年2月12日，诉：左下区牙痛三天，经诊断为左下第七颗牙急性牙髓炎，进行CP按抚，瞩三天复诊（拔髓）。3月12日，诉患牙又开始疼痛二天，诊断为牙髓炎，并记录：因患者在该牙治疗中，一直不能按医嘱及时治疗，医生对患牙预后难以控制，告知患者到外院治疗，一切后果由患者自己承担。此后，无原告在被告诊所就诊的病史记录。2009年3月28日，原告至上海宏康医院治疗，经诊断为左下侧第七颗牙原固定桥基牙，牙合面开髓，X线示左下第七颗牙根尖阴影，急性根尖脓肿，并对该牙予拔髓，清理通畅根管。同年4月2日，原告至上海市第九人民医院治疗，经检查，左下第七颗牙残冠，牙冠低矮，摄片示髓底已破损，根尖周骨稀疏明显，诊断为左下第七颗牙残冠慢性根尖周炎，建议拔牙。4月13日，原告至该院复查，经查，左下第七颗牙残冠，冠周红肿，摄片提示髓底穿，根尖周骨质疏松，建议拔牙。后原告多次至上海市普陀区长风街道白玉社区卫生服务中心、上海宏康医院进行治疗。2010年2月25日，原告再次至上海市第九人民医院，拔除左下第七颗牙齿。原告多次至上述医院就牙齿及左下颌区肿胀问题门诊。原告认为，由于被告装牙时对第七颗牙磨损过度，牙套装的不好，其装牙的医疗行为存在过错，导致原告左下颌区肿胀及左下第七颗牙发炎直至拔除。原告遂起诉来院，请求法院判决如其诉请。
　　另查明，原告曾于2005年9月9日发生交通事故，并因交通事故人身损害赔偿纠纷及其左颌下区肿胀等后续治疗费用先后向本院起诉案外人王栋，本院分别作出（2006）普民一（民）初字第4829号、（2008）普民一（民）初字第3535号民事判决，为查明案件事实，本院依法调取了两案的卷宗材料，其中一份出具时间为2008年8月27日的华政法医[2008]残鉴字第4284号司法鉴定意见书，检案摘要中有如下记载：2005年9月9日上海市第六人民医院门急诊病历记载：“……同日因左侧下颌被撞伤2小时……诊断：左下颌区皮下血肿，左TMJ损伤。2005年9月12日、10月6日门诊复查，B超示：左颈部淋巴结肿大，左颌下区未见血肿。2006年1月18日口腔科复诊，诊断：左颌下淋巴结炎。1月19日口腔科：复阅见左颌下区未见明显血肿形成，颈动脉分支水平处见多枚淋巴结块，大小约为14某7毫米，诊断同上……”。分析认为：被鉴定人孙某自2005年9月9日发生交通事故后自觉左下颌区肿大，经反复物理检查及B超探视在左颌下区仅发现肿大的淋巴结，又经动脉频谱分析，未发现淋巴结有异常。目前双侧颌下区均见肿大的淋巴结，故淋巴结炎诊断成立。
　　再查明，被告李某系上海市普陀区李某私立口腔诊所个体经营者，依法取得医疗机构执业许可证，现医疗机构名称已由上海市普陀区李某私立口腔诊所变更为上海李某口腔诊所。
　　审理过程中，根据原告的申请，本院依法委托上海市黄浦区医学会就被告上海李某口腔诊所为原告孙某装牙的医疗行为是否存在过错，医疗行为与孙某左下第七颗牙的缺失及左下颌区肿胀之间有无因果关系进行医疗纠纷司法鉴定。上海市黄浦区医学会于2011年7月12日出具不予受理鉴定通知，因缺乏后续治疗病史，不予受理。2011年9月6日，本院依法至司法鉴定科学技术研究所司法鉴定中心就涉案问题咨询专家，专家认为：从现有病史无法判断下颌肿胀与装牙行为有关。原告为双侧颌下区淋巴结炎，而淋巴结炎成因复杂，牙套若要引起双侧淋巴结炎，会表现得如牙周肿大、流脓、牙齿松动等局部反应异常明显，但从现有病史，无法看出患xxx症状。患者陈述的牙套没装好、易脱落这种情况，不大可能导致患者所述的结果，且后期牙套卸下后，症状仍未消失，要将牙套装得不好与其上述症状联系起来亦无法解释。
　　本院认为，原告对自己在被告处就诊经过记忆模糊，无法明确就诊时间，其否认被告提供的病历卡真实性，又未能提供充分证据予以反驳，而被告提供的病历卡能与收据相印证，就诊时间及内容亦能与原告在外院的就诊情况相契合，故本院采信被告有关原告在其处就诊情况的陈述。根据本案证据，原告于04年3、4月期间，在被告诊所修复缺失牙，至2005年9月9日因车祸就诊，始有颌下区肿胀之症状，至2007年8月30日，始有固定桥松动之记载，即便2009年3月28日在上海宏康医院就诊时诉左下牙痛3年，亦与其装牙相隔两年之久。现原告认为其左颌下区肿胀及左下第七颗牙齿被拔除系被告的装牙行为导致，应就此提供充分证据予以证明。然本案因缺失病史，以致无法进行鉴定。本院认为病史是重要的证据材料，亦是专家对医疗纠纷作出准确鉴定、判断医疗行为与患者疾病状况之间有无因果关系的重要依据。对此，原告认为缺失病史之责在被告，但未能提供充分证据证明被告保有并隐匿病史，亦未能提供应由其保管的2007年10月之前的医保门急诊就诊自管记录册，综上，应认定原告不能提供真实完整的病历资料，由此产生的不利后果由原告承担。另，原告讼争的相关症状的出现与被告为其装牙的医疗行为之间相隔较长时间，期间，原告又历经车祸外伤，左下侧第七颗牙龋齿、牙髓炎、多家医院检查发现固定桥松动并要求其重新固定时，未遵医嘱及时进行有关治疗，后又在外院就该牙作了髓底治疗、拔髓等，综合分析双方提供的证据、原告的病史资料、参考专家意见，本院难以认定原告第七颗牙缺失及左颌下区肿胀与被告为其装牙的医疗行为之间有因果关系，据此，对原告孙某要求被告赔偿其经济损失的所有诉讼请求不予支持。综上，依据《中华人民共和国民法通则》第九十八条、第一百零六条第二款之规定，判决如下：
　　对原告孙某要求被告李某、上海李某口腔诊所赔偿各类经济损失的所有诉讼请求不予支持。
　　本案受理费人民币1133元，由原告孙某自行负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第二中级人民法院。
审判长鲁君
　　代理审判员向颖
　　人民陪审员王荣
　　二〇一一年九月二十三日
　　书记员吴文俊
