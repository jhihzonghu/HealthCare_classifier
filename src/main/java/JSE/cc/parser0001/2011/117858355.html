高振云与驻马店市中心医院医疗损害赔偿纠纷上诉案

审判时间：20110413 法院：河南省驻马店市中级人民法院 案件字号：(2011)驻民三终字第316号


　　上诉人（原审原告）高振云。
　　委托代理人赵灿，驻马店市法律援助中心律师。
　　上诉人（原审被告）驻马店市中心医院。
　　法定代表人陈建华，该医院院长。
　　委托代理人王玉祥，河南尚成律师事务所律师。
　　上诉人高振云、驻马店市中心医院因医疗损害赔偿纠纷一案，不服驻马店市驿城区人民法院（2010）驿民初字第191号民事判决，向本院提起上诉。本院依法组成合议庭，公开开庭审理了本案。上诉人高振云及其委托代理人赵灿，上诉人驻马店市中心医院的委托代理人王玉祥到庭参加诉讼。本案现已审理终结。
　　原审法院判决认定，1996年8月29日，原告高振云到原驻马店市人民医院进行检查，检查报告单载明临床诊断为：宫颈Ⅲ°糜烂；宫颈Ca?。检查标本为：宫颈刮片。检查结果为：宫颈鳞状上皮细胞癌，呈片状浸润间质及累及腺体。间质内可见癌细胞浸润。诊断为：宫颈鳞癌(浸润性)。检验号为96605。1996年9月11日，原告高振云到驻马店市中心医院(原驻马店地区人民医院)进行检查，报告单载明的检查结果为：除片未发现癌细胞，少许皮上细胞呈增生活跃、检查标本为：刮片。检查目的为：癌细胞。次日，原告高振云到该院住院治疗。病历记载门诊诊断为：宫颈Ⅲ°糜烂；宫颈Ca?。9月14日病历记载：病人拒绝宫颈活检。9月17日，医院对高振云进行活组织检查，病理报告显示检查结果为：宫颈鳞癌(浸润性)。当日住院病历记载显示：今日活组织检查，病理报告：宫颈鳞癌(浸润性)。不适应手术治疗，建议化疗及放疗。9月18日，高振云出院，共住院6天。出院医嘱建议：注意休息，加强营养；建议化疗、放疗；定期复查。出院后，高振云于1996年9月18日至12月3日在驻马店市中心医院肿瘤科接受放射治疗，共77天。原告高振云接受放射治疗后，逐渐出现放射治疗后的一些反应症状。后到多家医院进行检查和治疗。1998年4月24日到河南医科大学第一附属医院检查，检查结果为：宫颈癌术后2年；结肠炎。1998年12月21日至31日在解放军一五九医院住院治疗10天，检查结果为：下消化道出血；结肠狭窄；重度贫血。2000年11月19日，高振云到解放军一五九医院做B超检查，检查结果为：肠管扩张；盆腔积液；膀胱壁增厚；子宫未见明显异常。2000年11月23至28日在遂平县人民医院住院治疗5天，检查结果为：小肠不全阻；肠结核?。2000年12月2日至4日，高振云又到遂平县人民医院住院治疗3天，诊断结果为：肠梗阻；消化道放疗反应；水电解质紊乱；慢行盆腔炎；恶性质病变。医院为高振云行剖腹探查术、回肠部分切除。2001年4月14日至19日，高振云到遂平县公疗医院住院治疗6天，检查结果为：粘连性肠梗阻。2009年2月17日，高振云到河南省肿瘤医院，检查结果为：宫颈癌放疗后12年。2009年3月5日，高振云又到驻马店市中心医院做宫颈癌检查，检查结果为：未发现癌细胞。以上，高振云实际住院治疗24天。2009年7月23日，高振云委托驻马店蔚康法医临床司法鉴定所对其伤残等级进行评定，鉴定机构通过鉴定后出具司法鉴定书一份，鉴定结论为：被鉴定人高振云在医院行剖腹探查术、部分小肠切除。目前下腹部、会阴部、臀部硬结瘢痕形成，且已失去弹性，面积占15％。参照GB《职工工伤与职业病致残等级分级》第h款第50项、第i款第10项和第g款第62项的规定，分别评定为七级、八级、九级三项伤残。高振云支出鉴定费630元。高振云系农村居民。诉讼期间，被告驻马店市中心医院向驻马店市驿城区法院提出鉴定申请，该院依法委托信阳市医学会进行鉴定。鉴定机构于2010年7月12日，出具中止鉴定函一份，内容为：因患者高振云在陈述状上强调自己没有宫颈癌，而驻马店市市中心医院按照癌症给予放疗。但医方在答辩状上强调医院医生是根据当时患xxx病理报告(无病理号、患者住院号为202821)，并结合驻马店市人民医院1996年8月29日高振云宫颈刮片(检验号为96605)诊断为宫颈癌，采取放疗是正确的。我医鉴办认为以上两张病理片对本鉴定患者是否患有宫颈癌至关重要，由于告知以上两张病理片医患双方均提供不出来，决定中止鉴定。后该院将中止鉴定情形告知双方当事人后，双方均无法提供病理片。之后该院将该情况通知鉴定机构后，鉴定机构终止鉴定。原告高振云从1996年10月份至2010年4月份间，先后在多家医院检查和治疗，共计支出各项费用5343.8元。高振云另提供遂平县石寨铺乡卫生院出具的证明一份，内容为：兹有高振云。其近两年以来已经在我石寨铺医院治疗用药共花费达三万二千六百余元。2009年3月6日，卫生部发布的《病理科建设与管理指南(试行)》规定：病理科应当加强对病理档案的保存和管理，其中病理切片保存期限为15年。
　　原审法院认为，原、被告之间的医疗侵权纠纷发生于侵权责任法生效实施之前，其过错归责原则应使用侵权责任法生效实施之前的相关规定。故本案因医疗行为引起的侵权诉讼，应由医疗机构就医疗行为与损害后果之间不存在因果关系及不存在医疗过错承担举证责任。原告高振云在其他医院被检查为宫颈癌后，到驻马店市中心医院住院治疗，期间仍被确诊为宫颈癌并接受放射治疗。放射治疗是对癌症患者的一种科学治疗方法，如高振云患有宫颈癌，被告驻马店中心医院对高振云所采取放射治疗的行为本身并不违反医疗常规，所造成的现有的损害后果属于放射治疗后的反应症状，不属于医疗侵权行为造成的损害结果。本案双方所争议的焦点问题为，高振云是否患有宫颈癌的问题。诉讼期间，被告驻马店市中心医院向驻马店市驿城区人民法院提出鉴定申请，要求对其医疗行为是否存在过错进行鉴定，但由于在鉴定中无法提供病理切片，导致鉴定机构终止鉴定。被告驻马店市中心医院的病历有进行活组织检查的记载，其单位作为医疗机构，应对病理切片的提供负有举证义务，因其未能完成举证义务，其单位应承担相应的责任。但由于原告高振云接受放射治疗至主张权利时已时隔十三年之久，2009年3月6日，卫生部发布的《病理科建设与管理指南(试行)》的规定才对病理切片保存期限作出应保存1
　　5年的规定。在此之前，有关病理切片的保存期限没有明确的规定，被告驻马店市中心医院无法提供病理切片完成举证责任存在一定的客观因素。同时，结合原告高振云在到驻马店市中心医院住院前已被其他医院诊断为宫颈癌，驻马店市中心医院依据原有的病理刮片和检查结论对高振云进行治疗，存在一定的治疗依据。高振云现阶段检查虽然未发现癌细胞，但现有医学实践证明，放射治疗对癌症患者存在一定的治愈率。所以，仅依据现有的检查结果也无法确认高振云1996年接受放射治疗时就确定不存在宫颈癌。本案被告驻马店市中心医院由于无法提供病理切片导致未能完成举证义务，其单位所应承担的责任是基于过错推定的原则而产生的，在2009年3月6日卫生部发布的《病理科建设与管理指南(试行)》规定实施之前，有关病理切片的保存期限没有明确的规定，被告驻马店市中心医院无法提供病理切片完成举证责任存在一定的客观因素，同时结合高振云在到驻马店市中心医院住院前已被其他医院诊断为宫颈癌及宫颈癌存在一定的治愈率等综合因素，如确定由医疗机构承担高振云因接受放射治疗而产生的全部损害后果，则会加重了医疗机构的责任，也不利于平衡医患双方的关系。根据本案的具体情况，对高振云因接受放射治疗后而产生的损失，可酌定由被告驻马店市中心医院承担60％的赔偿责任。原告高振云要求赔偿住院伙食补助费、营养费、残疾赔偿金、医疗费及交通费的请求，符合法律规定，依法应予支持。住院期间按接受放射治疗77天及之后的实际住院24天认定，共101天，住院伙食补助费按每天30元支付，计款3030元。营养费按住院101天，每天10元支付，计款1010元。残疾赔偿金按农村居民人均纯收4807元／年计算，赔偿20年，三处伤残赔偿指数为50％，计款48070元(4870元／年×20×50％)。误工费及护理费均按农村居民人均纯收入4807元／年计算，误工及护理期限为住院101天，误工费及护理费数额为1330.17元。医疗费按实际支出5957.4元认定。交通费可根据高振云长期检查治疗的实际情况酌定为2000元。以上各项物质性损失共计62727.74元，被告驻马店市中心医院负担60％，计款37636.64元。原告高振云所受损害给其造成一定的精神痛苦，应当给予一定的精神抚慰，精神抚慰金的数额可酌定为5000元。以上两项共计42636.64元，被告驻马店市中心医院应依法予以赔偿。高振云所请求的60000元的误工费缺乏证据加以证明，应不予支持。其所请求的300000元的精神抚慰金明显过高，应不予支持。高振云所请求的在遂平县石寨铺医院花费的32600元医疗费，缺乏合法有效的证据加以证明，应不予支持。据此，原审法院判决：一、限被告驻马店市中心医院于判决生效后3
　　0日内赔偿原告高振云各项损失42636.64元。二、驳回原告高振云的其他诉讼请求。如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。诉讼费5050元，原告高振云负担2020元，被告驻马店市中心医院负担3030元。
　　宣判后，高振云、驻马店市中心医院均不服，向本院提起上诉。高振云上诉称，其没有癌症，而驻马店中心医院没对其做检查为其化疗，导致其身体多处病变，应对其造成的损失承担责任。另外原审法院对其提出的60000元误工费、32600元的医疗费没有支持不符合法律规定。精神抚慰金与交通费确定的数额过低。请求撤销原判，改判驻马店中心医院赔偿其50万元。驻马店市中心医院上诉称，医院根据高振云提供的宫颈活检组织病理切片进行协诊，确诊高振云患有宫颈癌并采取放疗正确。高振云现在没有癌症，说明其是被治愈的患者。请求撤销原判，驳回高振云的诉讼请求。
　　二审审理查明的事实与原审法院判决认定的事实一致。
　　本院认为，关于原审法院对高振云损失划分的责任比例是否适当的问题。原审法院依据医疗侵权纠纷发生时的相关法律规定，确定本案的举证责任适当，即由医疗机构就医疗行为与损害后果之间不存在因果关系及不存在医疗过错承担举证责任。本案中，驻马店市中心医院就其医院对高振云的诊疗行为是否存在过错及过错与损害后果之间是否存在因果关系申请鉴定。鉴定结构以医患xxx病理片终止了鉴定，故无法确定高振云是否患有宫颈癌，也就无法认定驻马店市中心医院对高振云所采取放射治疗的行为本身是否违反医疗常规。关于保存病理片的规定，在本案医疗纠纷发生时并没有明确的规定。原审法院从平衡医患双方利益的角度出发，考虑到驻马店市中心医院无法提供病理切片完成举证责任存在一定的客观因素，同时结合高振云在到驻马店市中心医院住院前已被其他医院诊断为宫颈癌及宫颈癌存在一定的治愈率等因素，对高振云因接受放射治疗后而产生的损失，酌定驻马店市中心医院承担60％的赔偿责任适当，予以维持。关于高振云上诉称60000元误工费、32600元的医疗费符合法律规定应予支持的问题。高振云在一、二审期间均未提供符合法律规定的确实充分的证据证明自己的主张，不予支持。关于精神抚慰金与交通费确定的数额是否过低的问题。其所请求的300000元的精神抚慰金明显过高，根据高振云的伤残等级、本案中双方的过错程度、本地的生活水平以及本案的基本情况等综合因素考虑，原审法院酌定为5000元适当。原审法院在高振云未提供交通费票据的前提下，根据其长期检查治疗的实际情况酌定为2000元适当，予以维持。综上，原审法院认定事实清楚，处理结果正确，应予维持。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项的规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费5050元，高振云负担2020元（免交），驻马店市中心医院负担3030元。
　　本判决为终审判决。
审判长刘俊波
　　审判员王社军
　　代理审判员董卓亚
　　二〇一一年四月十三日
　　书记员杨新华
