马某某与上海市某医院医疗损害赔偿纠纷案

审判时间：20110603 法院：上海市宝山区人民法院 案件字号：(2009)宝民一(民)初字第5895号


　　原告马某某。
　　委托代理人郁某某。
　　委托代理人朱某某。
　　被告上海市某医院。
　　负责人杨某某。
　　委托代理人吕某。
　　委托代理人王某。
　　原告马某某与被告上海市某医院（以下简称“某医院”）医疗损害赔偿纠纷一案，本院于2009年9月9日受理后，依法组成合议庭，公开开庭进行了审理。原告马某某的委托代理人郁某某、朱某某、被告某医院的委托代理人吕某到庭参加诉讼。本案现已审理终结。
　　原告马某某诉称，原告因甲状腺占位于2008年9月17日在被告某医院行甲状腺手术。出院小结记载：因原告病理显示左侧甲状腺结节性囊肿、右侧甲状腺结节性囊肿、右侧甲状腺微小乳头状癌、峡部单纯性腺肿，故在颈丛麻醉下为原告行左甲状腺次全切除、右甲状腺CA改良根治术。2008年12月17日，原告至复旦大学附属中山医院（以下简称“中山医院”）作术后复诊，经彩超提示：右叶残留20×13mm，左叶及峡部未显示。此后，原告多次前往中山医院复诊，均提示右下极有阴影，需密切随访。根据病理报告，原告右侧甲状腺发现微小乳头状癌，被告某医院在行手术治疗时应完整切除癌变的甲状腺右叶、保留部分左叶做功能性使用，但实际手术时被告却将甲状腺左叶全部切除，保留部分甲状腺右叶。被告手术方案、类别错误，未达手术目的。由于被告错误实施手术，造成原告甲状腺功能几近丧失，必须服用超过正常术后服用量的“优甲乐”，且存在右叶残留部分癌变隐患，应构成三级戊等医疗事故标准且被告应承担全部责任。故要求被告赔偿：一、已发生费用：医疗费人民币4,393.80元、误工费8,607元、交通费1,020元、精神损害抚慰金5,000元、手术费（甲状腺改良根治术）1,750元、鉴定费3,500元、律师费10,000元；二、预计发生的费用：医疗费28,279.70元、误工费40,507.20元、交通费4,800元；三、保留20年后的诉权。
　　被告某医院辩称，不同意原告的诉讼请求。本案进行了两级医疗事故鉴定，结论清楚；本案发生于侵权责任法实施前，应按鉴定结论的责任性质和责任大小分担损失。对于原告主张的各项费用：1、对于已经发生的医疗费和手术费用，该些费用均系原告治疗疾病所需合理费用，与医疗事故无关，不同意赔偿；2、误工费，原告应提供劳动单位人事部门出具的工资收入情况和相应的缴纳“四金”的情况，方可确定误工损失；3、交通费，由于无法确认停车费发票和油费发票与本案的关联性，故不予认可；4、精神损害抚慰金，按医疗事故鉴定结论是没有精神损害抚慰金的，请法院在三级戊等以下酌定。5、鉴定费，请求法院按责任比例判定；6、律师费，对律师费发票无异议，请求法院按责任比例判定；7、后续治疗的相关费用：根据鉴定结论，原告不需要后续治疗，故对于原告主张的与后续治疗有关的费用均不予认可。另，在黄浦区医学会鉴定的鉴定费3,500元由被告垫付，要求按责分担。
　　经审理查明：
　　原告系上海市某律师事务所律师，因“发现双侧颈部肿块２月”于2008年9月16日入住被告某医院。经诊断：双侧甲状腺占位，结节性甲状腺肿；予以完善相关检查，择期手术。次日，原告在被告某医院行甲状腺手术，出院小结记载：9月17日，原告在颈丛麻醉下行左甲状腺次全切除＋右甲状腺癌改良根治术。当日术中冰冻病理报告：左甲状腺结节性腺肿，右甲状腺结节性腺肿，右甲状腺微小乳头状癌，峡部单纯性甲状腺肿。同年9月19日石蜡病理诊断报告：双侧甲状腺及峡部结节性甲状腺肿，右侧伴微小乳头状癌（直径约0.15cm）。同年年9月23日，原告出院，医嘱：口服优甲乐2粒/日，门诊随访。
　　2008年12月17日，原告至中山医院就诊，彩超显示：甲状腺右叶残留20×13mm，甲状腺左叶及峡部未显示。同年12月24日，中山医院颈部CT增强检查报告：甲状腺MT术后，左侧甲状腺缺如，右侧甲状腺下极见一直径约5mm结节影，增强后，未见明显强化；余甲状腺组织正常化强化；颈部未见肿大的淋巴结。2009年3月12日，中山医院甲状腺功能报告：T3、T4正常范围内，s-TSH0.09（参考值0.27-4.3）uIU/ml。同年4月20日，中山医院彩超报告：甲状腺右叶残留13×8×18mm。2010年7月15日，中山医院甲状腺功能报告：T3、T4、FT3、FT4正常范围内，促甲状腺激素0.02（参考值0.27-4.2）uIU/ml。
　　审理中，经被告申请，由上海市黄浦区医学会对本次医疗纠纷进行鉴定，分析意见为：“患者双侧甲状腺结节行甲状腺手术，符合诊疗常规。医方在手术操作中行左叶甲状腺次全切除过多，影响甲状腺一定功能。右侧有微小癌变的甲状腺组织，手术所保留的腺组织与‘改良根治术’不符；但微小癌的切除范围目前医学上无统一标准，保留少量腺组织并不违反治疗原则，今后产生潜在损害，目前无法评估。”鉴定结论为：“马某某与某医院医疗争议构成医疗事故：……本病例属于四级医疗事故，医方承担次要责任。”被告支付鉴定费3,500元。
　　被告对上述鉴定结论并无异议，但原告对上述鉴定结论不服，申请重新鉴定。后经上海市医学会鉴定，分析意见为：“1、患者2008年9月16日因‘发现双侧颈部肿块2月’住院，结合ECT检查提示双侧甲状腺‘凉结节’，手术指征明确。2、根据术中冰冻病理资料，患者甲状腺右叶微小乳头状癌诊断明确，结合患者术后随访资料表明医方施行了甲状腺右叶次全切除术。甲状腺微小乳头状癌行次全切除，符合当时甲状腺癌治疗原则。3、根据送鉴资料，医方对于患者良性病变的甲状腺左叶进行了近全切除，但术后外院影像学检查发现甲状腺左叶缺如。医方手术范围过大，不符合甲状腺良性肿块外科治疗原则，医方应承担相应责任。同时医方术中改变手术方案，仅有口头告知，未见书面告知记录，也存在不足之处；4、患者目前应用优甲乐治疗系甲状腺癌术后治疗所必须。目前没有资料表明患者甲状腺功能低下。”鉴定结论为：“马某某与某医院医疗争议构成医疗事故：……本病例属于四级医疗事故，医方承担主要责任。”原告支付鉴定费3,500元。
　　被告对上海市医学会的鉴定结论亦无异议，但原告对其中的分析意见不予认同，并坚持认为本次事故构成三级戊等医疗事故。
　　另查明，原告为治疗病情支付医疗费用共计4,210.40元；为就诊、鉴定、诉讼所需花费一定数额的交通费；为本次诉讼，支付律师费10,000元。
　　以上事实，有原告提供的手术记录2份、中山医院的影印报告、鉴定意见书2份、住院明细、医疗费单据、纳税凭证、停车费、油费发票、工商银行转帐凭证、律师费发票及原、被告当庭陈述等证据为证，经庭审查证属实，本院予以认定。
　　本院认为，两次鉴定结论均认定被告的医疗行为不当，直接造成原告马某某人身损害，并构成医疗事故，故被告应承担相应的赔偿责任。原告虽对两次鉴定结论均持异议，并主张构成三级戊等医疗事故，但无充分证据证明，本院难以支持。综合分析两次鉴定结论，上海市医学会的意见相对更为合理、准确，本院予以采信。故本院酌情确定被告对原告的损失赔偿80%。关于原告主张的各项费用：1、医疗费（含手术费），根据原告提供的病历资料和费用凭据，手术费系原发病医疗费用，不属于本次医疗事故所致损失范围，同时扣除原告治疗其他疾病的诊疗费用和医保附加支付费用等，本院确定医疗费共计4,210.40元；2、交通费和误工费，原告依据就医情况主张交通费1,020元和误工费8,607元，均属合理范围，本院予以确认；3、律师费，原告凭据主张10,000元于法有据，本院予以确认。上述1－3项费用共计23,837.40元，由被告按80%的责任比例赔偿原告计19,069.92元。4、精神损害抚慰金，原告因本次医疗事故，除身体受到损害外，在精神上亦遭受了一定痛苦，本院根据被告的过错程度、侵权行为方式、侵权所造成的后果等因素，酌情确定由被告赔偿原告精神损害抚慰金4,000元。综上，依照《医疗事故处理条例》第四条、第五十条之规定，判决如下：
　　一、被告上海市某医院于本判决生效之日起十日内赔偿原告马某某医疗费、交通费、误工费和律师费共计19,069.92元；
　　二、被告上海市某医院于本判决生效之日起十日内赔偿原告马某某精神损害抚慰金4,000元；
　　三、原告马某某的其余诉讼请求不予支持。
　　负有金钱给付义务的当事人如果未按照本判决指定的期间履行给付金钱义务，应当按照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　本案受理费656元，由原告马某某负担131元，被告上海市某医院负担525元。鉴定费7,000元，由被告上海市某医院负担。
　　如不服本判决，可在判决书送达之日起十五日，向本院递交上诉状及副本，上诉于上海市第二中级人民法院。
审判长金猷
　　代理审判员葛璐萍
　　人民陪审员时金兰
　　二〇一一年六月三日
　　书记员王振华
