臧家宝等诉浙江省人民医院医疗损害赔偿纠纷案

审判时间：20110301 法院：浙江省杭州市下城区人民法院 案件字号：(2009)杭下民初字第1618号


　　原告：臧家宝。
　　原告：童银香。
　　原告：白锡腾。
　　原告：白宸宇。
　　法定代理人：白锡腾，
　　四原告共同委托代理人：谷光辉、胡时习。
　　被告：浙江省人民医院。
　　法定代表人：叶再元。
　　委托代理人：王正华、尹口。
　　原告臧家宝、童银香、白锡腾、白宸宇为与被告浙江省人民医院医疗损害赔偿纠纷一案，本院于2009年9月14日受理后，依法由金宁独任审理，后因案件复杂，依法组成合议庭，于2009年10月27日、2010年5月6日、2011年2月18日公开开庭进行了审理。原告臧家宝、童银香、白锡腾、白宸宇的委托代理人谷光辉、胡时习、被告浙江省人民医院的委托代理人尹口到庭参加诉讼。本案在审理过程中，依原告臧家宝、童银香、白锡腾、白宸宇，被告浙江省人民医院申请，本院分别于2010年4月28日和2009年10月10日对诉争病例交由本院审判监督庭，依法委托浙江大学司法鉴定中心和杭州市医学会对患者臧菊琴在治疗期间医院方是否存在过错、过错与患者死亡有无因果关系以及医院的医疗是否构成医疗事故进行鉴定。本案现已审理终结。
　　原告臧家宝、童银香、白锡腾、白宸宇起诉称：2009年7月14日，臧菊琴因“突发性头痛，意识不清3小时”至萧山区第一人民医院就诊，诊断为“自发性蛛网膜下腔出血；颅内动脉瘤、AVM破裂？”，经初步处理后于当日转被告处住院治疗，2009年7月16日被告处MRA：右侧颈内动脉C2段动脉瘤；7月21日脑血管造影：右颈内动脉C2段狭窄伴动脉瘤；7月29日：患者突发抽搐，癫痫大发作，头颅平扫：蛛网膜下腔出血，脑质肿胀（桥脑水肿）；被告医生一直给予保守治疗，直至2009年7月30日：CT检查示蛛网膜下腔出血，脑肿胀，左侧脑室后角积血，右颞枕顶脑水肿。考虑有再次出血和脑血管痉挛，仍然给予保守治疗。后经家属联系于当日转入浙江大学医学院附属第二医院治疗，转院当日即予急诊全脑CTA检查，7月31日：立即进行全脑DSA检查＋介入治疗，因出现动脉瘤再次破裂出血，随后行开颅去骨瓣减压＋右侧脑室外引流术。但因患者手术治疗时间过迟，治疗时机已经丧失，已经失去抢救的机会，经浙二医院告知病情，患者已无生存希望，于8月2日自动出院当日死亡。原告认为，被告对臧菊琴的诊疗行为存在明显的过错，辅助检查不全面、观测不仔细、治疗措施不规范、手术治疗不及时、病情及治疗方案告知不详等。上述过错不但延误了患者的治疗，加重了患者的病情，也使家属丧失了及早转院治疗的机会，最终导致患者死亡的结果发生。被告的侵权行为给原告带来了巨大的物质损失和精神痛苦，依法应当承担民事赔偿责任，现诉讼至法院，要求判令被告浙江省人民医院赔偿给原告：1、医疗费、死亡赔偿金、丧葬费、精神损失费共计571882.18元；2、本案诉讼费由被告承担。
　　为证明上述事实，原告臧家宝、童银香、白锡腾、白宸宇向本院提交了如下证据：
　　1、萧山区第一人民医院出院记录一份、放射科检查报告单一张、萧山区第一人民医院病历一本，证明患xxx病后在萧山区第一人民医院诊断情况。
　　2、浙江省人民医院住院病历及相关材料（复印件）、浙江省人民医院影像片五张，证明被告的医疗行为存在过错，和患者死亡存在因果关系。
　　3、浙江大学医学院附属第二医院病历材料一本，证明患者臧菊琴在浙二医院的治疗情况。
　　4、死亡证明一份，证明臧菊琴于2009年8月2日死亡。
　　5、医疗费结算单3份，证明原告在萧山医院、省人民医院、浙二医院所支付的医药费。
　　6、身份证、户口簿及公安机关证明各1份，证明被抚养人需被抚养的事实。
　　7、鉴定费发票1份，证明原告为鉴定支付的鉴定费。
　　被告浙江省人民医院答辩称，被告在接受患者住院后相关检查完善、对病情变化观测认真仔细、根据病情及时选择了相应的治疗方案且治疗方案符合脑血管病治疗指南，及时与患者家属进行了沟通并依法履行了告知义务。患者的死亡对被告的诊疗行为没有因果关系。患者臧菊琴在被告接受治疗后转入浙二医院，后因病情严重而死亡。根据目前医学水平，从患者的诊疗过程来看，患者最终死亡的后果，既与其自身疾病的凶险性直接相关，也与其家属不配合治疗有重大的关系。原告的陈述与事实不符。原告认为被告辅助检查不全面，与事实不符。事实上被告在患者入院时就进行了CT检查，提示蛛网膜下腔出血；在7月16日进行了MRA检查，提示右颈内动脉C2段动脉瘤；在7月21日进行了DSA检查，检查提示右颈内C2段囊行动脉瘤等。事实上，被告进行了详细检查，对患者的病情做出了全面的诊断，同时被告还对其进行了其他常规检查。被告对患者的辅助检查是十分全面的，相关诊断也十分明确。医学会的司法鉴定书和浙大的鉴定书都表明我医院不存在过错，我们不应当承担相应的法律责任，原告的诉讼请求无事实和法律依据，应予以驳回。
　　为证明上述事实，被告浙江省人民医院向本院提交了如下证据：
　　封存的臧菊琴病历一组（由原告当庭拆封），内有浙江省人民医院住院病历一份、手术知情同意书一份、检查报告单一组、护理记录一组、血压脉搏呼吸体温记录一组、耗材记录单一份、治疗知情同意书一份、医嘱单一组、病危通知单一份，欲证明被告诊断正确、医疗措施得当，对臧菊琴的死亡不存在过错及因果关系。
　　原、被告提交的上述证据，经双方当事人当庭举证、质证，本院认证如下：
　　对于原告臧家宝、童银香、白锡腾、白宸宇提交的证据1，被告浙江省人民医院对该组证据三性均有异议，认为报告单上未写明患者姓名，本院对其中已写患者姓名的萧山区第一人民医院出院记录、萧山区第一人民医院病历予以确认；对证据2至6，被告浙江省人民医院对真实性没有异议，但认为无关联性。本院对真实性予以确认；对证据7，被告浙江省人民医院没有异议，本院予以确认。
　　对于被告浙江省人民医院提交的证据1，原告臧家宝、童银香、白锡腾、白宸宇对真实性没有异议，但对证明目的有异议。本院对真实性予以确认。
　　杭州市医学会所做出的杭州医鉴（2010）0016号医疗事故技术鉴定书1份，经原、被告双方当庭质证，原告对鉴定书的真实性、合法性无异议，对鉴定内容有异议，认为与原告主张无关联性；被告对鉴定书无异议，本院认为，该鉴定书与本案的处理有直接的关联性，本院予以确认。
　　浙江大学司法鉴定中心所作出的浙大司鉴中心（2010）临鉴（医）字第618号意见书1份，经原、被告双方当庭质证，本院认证如下：原告对鉴定结论有异议，认为鉴定书所列举的事实是没有依据的，与病历上反映的事实不相符，前半部分分析的问题和后半部分结论的问题完全割裂了事物之间的内在联系，该鉴定书不科学不客观，不能作为本案判决的依据；被告对鉴定结论真实性没有异议。本院对该鉴定意见书的真实性予以确认。
　　根据当庭举证、质证及双方当事人陈述的情况，本院确认案件事实如下：
　　1、患者臧菊琴。2009年7月14日，臧菊琴因“突发性头痛，意识不清3小时”至萧山区第一人民医院就诊，诊断为自发性蛛网膜下腔出血；颅内动脉瘤，AVM破裂？，经初步处理后，于当天转至被告浙江省人民医院住院治疗，入院后经相关检查，急诊MRA检查示右颈内动脉C2段动脉瘤，2009年7月21日DSA检查明确为右颈内动脉C2段狭窄，囊性动脉瘤，颅底异常血管染色。至7月29日突发抽搐，癫痫大发作，被告浙江省人民医院一直给予保守治疗，7月30日CT示蛛网膜下腔出血，脑肿胀，左侧脑室后角积血，右颞枕顶脑水肿，考虑有再次出血和脑血管痉挛，仍给予保守治疗。后经家属联系于当日转入浙江大学医学院附属第二医院治疗。2009年7月31日下午行头颅三维DSA检查＋介入疗法，术中出现动脉瘤再次破裂出血，急复查头颅CT示颅内出血增多，改行开颅去骨瓣减压＋右侧脑室外引流术。2009年8月2日患者臧菊琴死亡。原告认为，由于被告浙江省人民医院对患者臧菊琴的诊疗行为存在明显的过错，延误了患者的治疗，加重了患者的病情，也使家属丧失了及早转院治疗的机会，最终导致患者死亡结果的发生。被告浙江省人民医院的侵权行为给原告带来了巨大的物质损失和精神痛苦，依法应当承担民事赔偿责任。故诉至法院，请求法院判如所请。
　　2、本案在审理过程中，被告浙江省人民医院申请对本案诉争病例进行医学鉴定，本院依法委托杭州市医学会进行鉴定，2010年4月7日该医学会作出杭州医鉴（2010）0016号医疗事故技术鉴定书。依据该鉴定书，其分析意见为：1、根据现有资料和鉴定会上医患双方陈述、质询，患者因突发头痛伴肢体抽搐一天转入到浙江省人民医院住院治疗，医方诊断：自发性蛛网膜下腔出血，脑血管病可能。入院后医方先后予CT、MRA（磁共振）、DSA1（脑血管造影）等检查（重要的辅助检查无遗漏），并予止血、激素、控制癫痫、预防血管痉挛等检查，措施正确，入院时有病情的告知，医方处置正确，没有违反诊疗常规。2、患者MRA检查提示右侧颈内动脉C2段动脉瘤，经脑血管造影检查确认为右侧颈内动脉C2段狭窄伴动脉瘤（宽基底动脉瘤），不适宜开颅手术，医方未将这一情况，告知患方签字，存在不足。3、患者脑水肿、脑血管痉挛、再出血等症状系疾病本身发展所致，医方采取的治疗措施符合常规，并未延误、加重其病情及导致死亡。综上分析，根据《医疗事故处理条例》第二条、第三十三条（等），其鉴定结论为：本病例不属于医疗事故。
　　3、本案在审理过程中，原告臧家宝、童银香、白锡腾、白宸宇申请就被告浙江省人民医院在对患者臧菊琴治疗期间医院方是否存在医疗过错进行鉴定，本院依法委托浙江大学司法鉴定中心进行鉴定。2011年1月12日该所作出了浙大司鉴中心（2010）临鉴（医）字第618号意见书1份，依据该鉴定书，其分析意见为：1、根据提供的资料，患者臧菊琴患自发性蛛网膜下腔出血，右颈内动脉C2段动脉瘤的临床诊断成立。2、从2009年7月14日患者臧菊琴入院就诊，院方对患者的诊治过程符合一般常规诊疗原则，且检查结果一出来，两次组织全科室讨论，说明是积极、认真的。医院的医疗行为无明显过错。3、从脑MRA检查提示C2段动脉瘤，有手术指征，当时决定开颅手术夹闭，但经科室讨论，认为病情复杂，手术风险大，决定暂缓手术，进行DSA检查。这些慎重的决定，从随后出现的术中动脉瘤再次破裂出血致死的情况看，当时的讨论与决定是正确的。4、根据资料不难看出，动脉瘤处理有最佳时间窗，以出血后72小时内，或14天或24天后为佳，因为可避开血管痉挛高峰期，因此，既然在3天内难以处理，那么仔细准备，等待时机手术，是可以考虑的。所以院方科室讨论按血管痉挛治疗方案，保守治疗两周左右是正确的，符合诊疗规范。5、经查看院方住院记录，2009年7月16日MRA检查后示：右侧颈内动脉C2段动脉瘤。拟行右侧翼点开颅内动脉C2段动脉瘤夹闭术，患者家属也已在重大疑难手术审批报告上签字，要求手术，但后来讨论认为病情复杂，手术风险大，决定暂缓手术。然而后来在病程记录中没有看到及时与患者及其家属沟通的描述或记录。表明院方与患者及其家属的沟通尚不够。6、患xxx病严重，多次出血所致。根据浙江大学医学院附属第二医院神经外科病历记录，患者臧菊琴于2009年7月31日从浙江省人民医院转入时意识清，仅存在精神软，说明当时并不存在特别的情况，而是在治疗过程中再次出现动脉瘤破裂出血而引起了最后的死亡结果。综上所述，浙江省人民医院在医患沟通以及医疗文书记录上存在缺陷，但这缺陷不影响疾病的诊治，对患者颅内动脉瘤的整个诊治过程，还是积极、认真的，不存在医疗过错。其鉴定意见：浙江省人民医院对臧菊琴自发性蛛网膜下腔出血和颅内动脉瘤的诊治过程不存在医疗过错。
　　本院认为，患xxx病在被告浙江省人民医院医治，双方形成医患关系明确，患者臧菊琴作为患方，其生命健康权应受法律保护。本案在庭审中原告选择了侵权之诉要求被告承担侵权责任，而判断医疗单位是否需承担侵权损害赔偿责任，应以其医疗行为具有过错作为构成侵权的必备要件之一。本案中，被告浙江省人民医院对患xxx病进行了医治，其诊断和治疗方法是否正确和符合医疗规范，是否存在医疗事故或医疗过错，是本案双方当事人争议的焦点问题，也是衡量被告浙江省人民医院医疗行为是否具有过错的关键所在。在从本院委托杭州市医学会和浙江大学司法鉴定中心所出具的两份鉴定报告可以看出，虽然患者臧菊琴的死亡与被告浙江省人民医院先期诊断和治疗抢救无直接的因果关系，但在整个医治过程中，被告浙江省人民医院对患者臧菊琴拟行右侧翼点开颅内动脉C2段动脉瘤夹闭术，患者家属也已在重大疑难手术审批报告上签字，要求手术，但后来被告浙江省人民医院讨论认为病情复杂，手术风险大，决定暂缓手术，然而被告浙江省人民医院未将该决定与患者的家属沟通和记录。医患关系中患方享有知悉与其生命健康相关信息的权利，医疗单位应履行必要的告知和说明义务。上述的行为在一定程度上损害了原告作为患方的合法利益。并且这种行为也使原告方无端增加了心理上、精神上本可避免的额外负担，故被告浙江省人民医院应承担相应的责任。原告要求按照一般的医疗损害赔偿纠纷适用民法通则进行赔偿符合法律规定，应予以支持。对于患者臧菊琴死亡造成原告损失的1、医疗费129219.68元；2、护理费1140原本院予以确认；3、对原告主张的丧葬费13931.5元，按照2008年度浙江省全社会在岗职工年平均工资25918元的标准，计算六个月，应确定为12959元；4、原告主张的死亡赔偿金185160元，按照2008年度浙江省农村居民人均纯收入9258元／年的标准，计算20年，本院予以确认；5、误工费1202元，原告未提供相应的证据，本院按照2008年度浙江省全社会在岗职工居民服务业和其他服务业年平均工资为21816元，确定为1135元；6、住院伙食补助费285元，予以确认。7、被扶养人生活费，原告臧家宝、童银香、白锡腾并没有提供丧失劳动能力又无其他生活来源的证据，故本院不予确认。对原告白宸宇，本院按照2008年度浙江省农村居民人均生活消费支出7072元计算，计算至18周岁，确定为53040元。上述共计382938.60元。根据过错程度，本院确定被告浙江省人民医院承担10%的赔偿责任，赔偿原告38293.86元，并赔偿原告精神损害抚慰金5000元。原告诉讼请求超出部分，因缺乏事实依据，本院不予以支持。对于原告要求重新鉴定的请求，因没有新的法定依据和事实，本院不予以准许。综上，根据最高人民法院《关于参照〈医疗事故处理条例〉审理医疗纠纷民事案件的通知》的有关规定，本案系因医疗事故以外的原因引起的其他医疗赔偿纠纷，应适用《中华人民共和国民法通则》的规定。故依照《中华人民共和国民事诉讼法》第六十四条、《中华人民共和国民法通则》第九十八条、第一百零六条、第一百一十九条的规定，判决如下：
　　一、被告浙江省人民医院于本判决生效后十日内赔偿原告臧家宝、童银香、白锡腾、白宸宇医疗费、护理费、丧葬费、死亡赔偿金38293.86元。
　　二、被告浙江省人民医院于本判决生效后十日内赔偿原告臧家宝、童银香、白锡腾、白宸宇精神抚慰金5000元。
　　三、驳回原告臧家宝、童银香、白锡腾、白宸宇的其他诉讼请求。
　　案件受理费人民币3259元（原告缓交），由原告臧家宝、童银香、白锡腾、白宸宇承担2259元，被告浙江省人民医院承担1000元。
　　如不服本判决，可在判决书送达之日起十五日内向本院递交上诉状，并按对方当事人的人数提出副本，上诉于浙江省杭州市中级人民法院，并向浙江省杭州市中级人民法院预交上诉案件受理费3259元（开户银行：工商银行湖滨分理处，账号：xxx，户名：浙江省杭州市中级人民法院）。在上诉期满后七日内未交纳的，按自动撤回上诉处理。
