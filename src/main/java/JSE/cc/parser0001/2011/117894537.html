严某诉上海市某人民医院医疗损害责任纠纷案

审判时间：20110822 法院：上海市松江区人民法院 案件字号：(2011)松民一(民)初字第3235号


　　原告严某。
　　委托代理人吴某。
　　委托代理人任某，上海保诚律师事务所律师。
　　被告上海市某人民医院。
　　法定代表人刘某，院长。
　　委托代理人陈某，该院工作人员。
　　委托代理人陈某，该院工作人员。
　　原告严某诉被告上海市某人民医院医疗损害责任纠纷一案，本院于2011年5月23日立案受理后，依法由代理审判员康晓莉适用简易程序公开进行了审理。经被告上海市某人民医院申请，本院于2010年12月23日委托上海市医学会进行医疗事故技术鉴定。本案于2011年6月29日公开开庭进行了审理。原告严某及其委托代理人任某，被告上海市某人民医院的委托代理人陈某、陈某到庭参加诉讼。本案现已审理终结。
　　原告严某诉称：2010年5月18日，原告因“膀胱切开取石术后2年余，排尿不畅伴尿痛2月余”前往被告处住院接受治疗，入院诊断为：前列腺增生症、前列腺癌待排、膀胱结石、尿道狭窄、高血压病。5月25日，原告行“尿道膀胱镜检查+尿道扩张+经尿道肽激光膀胱碎石术”，但在6月2日出院时被诊断为：前列腺癌、尿道狭窄等，并要求于3周后再次入院治疗。据此，原告于6月23日再次入院，入院时被确诊为：前列腺癌一月。6月29日，原告行“经耻骨后前列腺癌根治性切除术”，被告将原告前列腺全部切除。术后，经冰冻切片报告显示，原告仅为前列腺增生，双侧精囊腺组织未见明显病变。据此，被告为原告进行的前列腺切除术时缺乏依据的。原告认为被告方没有对其采取对症下药的有效治疗措施，且在没有确诊的情况下冒然进行前列腺切除术，给原告的身体及精神造成了极大的伤害。故起诉要求被告赔偿其医疗费86,969.36元、残疾生活补助费139,200元、误工费8,000元、护理费2,000元、营养费1,800元、住院伙食补助费760元、交通费800元、精神损害抚慰金69,600元、鉴定费1,000元，共计310,129.36元；
　　被告上海市某人民医院辩称：本案经过医学会鉴定，结论为被告方承担次要责任，故我方愿意在合理范围内承担相应的赔偿责任。
　　经审理查明：2010年6月23日，原告因“确诊为前列腺癌1月”入住被告处就诊。据住院病历记载：原告入院前1月因“尿道狭窄”至被告处行尿道扩张术+前列腺穿刺活检术。术后病理示“（左中）前列腺增生，小区前列腺癌，gleason评分3+3=6”，故入院进一步诊治。6月29日，原告在全麻下行经耻骨后前列腺癌根治性切除术。7月9日，术后病理报告：前列腺增生，双侧精囊腺组织未见明显病变。7月16日，原告出院。
　　2010年12月23日，经被告向本院提出申请，要求就被告对原告的治疗过程是否存在医疗过错、是否构成医疗事故进行鉴定。本院依法委托上海市医学会，要求鉴定被告对原告的医疗行为是否违反医疗卫生管理法律，行政法规，部门规章和诊疗护理规范、常规，被告的医疗行为是否存在医疗过错，与原告目前病情有无因果关系，是否属于医疗事故，等级及责任程度。
　　2011年2月23日，在上海市医学会医疗事故技术鉴定办公室主持下，医患双方从上海市医学会医疗事故技术鉴定专家库（泌尿外科专业、病理科）中各专业组各随机抽取1名专家编号，医学会工作人员随机抽取泌尿外科专业1名专家编号，共5名专家组成专家鉴定组。医患双方再各专业各随机抽取同等数量专家编号作为鉴定专家组候补成员。上述专家编号打印出《随机抽取鉴定专家编号清单》，三方签字确认后各执一份。随机抽取前，医学会向双方告知本会专家库中上述专业组全部专家姓名、专业、技术职务、工作单位等情况。提请双方按条例规定提出其中需要回避的专家名单及理由，并签字确认。其后，医学会在双方共同在场的情况下，对上述专家随机编号并打印包括专家姓名、单位及编号等内容的《鉴定专家随机编号清单》，由原、被告双方当场在封存的信封口上签字封存，由医学会保存备查。
　　2011年3月3日，由市医学会医疗事故技术鉴定工作办公室组织召开鉴定会。2011年2月23日随机抽取的3位正式编号专家、2位候补编号专家到会，组成专家鉴定组并由专家鉴定组成员推选产生组长1名。到会的患方代表共2名，医方代表3名。经过医患双方陈述、答辩，专家提问，经表决、合议，并根据半数以上专家鉴定组成员意见形成“构成三级丁等医疗事故，医方承担次要责任”的鉴定结论。专家鉴定组成员在鉴定结论《专家合议书》上签名，并在《医疗事故技术鉴定书》原稿中说明理由，由专家鉴定组组长签发。
　　2011年3月9日，上海市医学会出具了沪医鉴[2011]022号《医疗事故技术鉴定书》，鉴定结论为：1、2010年6月29日，原告因前列腺穿刺活检诊断为“前列腺增生，小区前列腺腺癌”在被告处行“前列腺癌根治性切除术”。泌尿外科医生据此病理报告行根治性手术，不违反诊疗常规；2、鉴定专家阅读病理切片后认为诊断“前列腺癌”的依据不充分。原告前列腺穿刺标本病理表现为小腺泡的增生，伴有异型，病灶周围有不连续的基底细胞围绕，因此诊断以“前列腺非典型性小腺泡增生”为妥。该类病理变化与前列腺癌鉴别上存在一定困难，这是导致诊断失误的原因之一。综上所述，根据《医疗事故处理条例》第二条、第四条、《医疗事故分级标准（试行）》、《医疗事故技术鉴定暂行办法》第三十六条之规定，认定原告与被告之间的医疗争议构成三级丁等医疗事故，被告承担次要责任。
　　2011年4月27日，经原告申请，本院委托华东政法大学司法鉴定中心对原告的休息、护理、营养期进行法医学鉴定，该中心于2011年5月6日出具华政[2011]法医残鉴字第F-474号《司法鉴定意见书》，结论为：原告行经耻骨后前列腺切除术，酌情给予伤后休息4个月，营养2个月，护理1个月。
　　以上事实，有医疗事故技术鉴定书、司法鉴定意见书、病历记录、住院病史录、出院小结、医疗费发票、鉴定费发票及当事人的陈述等证据予以证实。
　　本院认为：医疗机构及其医务人员在医疗活动中，必须严格遵守医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，恪守医疗服务职业道德，因过失造成患者人身损害的，应当承担相应的法律责任。由于医疗行为具有较强的专业性，故法院确定医院医疗行为是否存在过错及该过错行为与损害后果之间有无因果关系的主要判断依据来自于医学专家的鉴定结论。本案中，上海市医学会作为有资质的医疗事故鉴定机构，同时该医学会在抽取专家组成专家鉴定组时的程序合法，且对于鉴定结论专家鉴定组成员均在鉴定结论《专家合议书》上签名，故本院对该医学会所出具的鉴定结论予以确认。根据鉴定结论表明，原、被告双方的医疗争议构成三级丁等医疗事故医方承担次要责任，故本院确定由被告对原告的损害后果承担40%赔偿责任。
　　关于原告可获赔偿的项目和金额问题，本院分述如下：
　　对于医疗费，应当按照医疗事故对患者造成的人身损害进行治疗所发生的医疗费用计算，但不包括原发病医疗费用。本案原告因治疗共产生医疗费86,969.36元，其中38,457.08元属原发病医疗费用，不属于可获赔偿的范围。故本院确定原告的医疗费为48,512.28元（86,969.36元－38,457.08元）。
　　对于残疾补助生活费，根据伤残等级，按照医疗事故发生地居民年平均生活费计算，自定残之月起最长赔偿30年。本案构成三级丁等医疗事故，现原告按照上一年度居民年平均生活费标准主张残疾生活补助费139,200元（23,200元/年×30年×20%），并无不当，本院予以准许。
　　对于误工费，患者有固定收入的，按照因误工减少的固定收入计算，对收入高于医疗事故发生地上一年度职工年平均工资3倍以上的，按照3倍计算；无固定收入的，按照医疗事故发生地上一年度职工年平均工资计算。原告误工期为4个月，至于收入，根据原告的劳动合同显示其月工资收入为2,000元，故本院确定原告的误工费为8,000元（2,000元/月×4月）。
　　对于护理费，原告的护理期为1个月。参照本市护工市场的平均劳务报酬每天30元的标准，确认原告的护理费为900元（900元/月×1月）。
　　对于营养费，应根据受害人的伤残情况参照医疗机构的意见确定。营养费的标准为每天20-40元。原告的营养期为2个月，本院参照每天30元的标准计算，确定原告的营养费为1,800元（900元/月×2月）。
　　对于住院伙食补助费，应根据原告的住院期间24天，并参照本市国家机关一般工作人员每天20元的出差伙食补助标准予以确定。故本院确定原告的住院伙食补助费为480元（20元/天×24天）。
　　对于交通费，应按原告实际必需的交通费用凭据支付。现原告并未提供相关的证据，故本院难以支持。
　　对于精神损害抚慰金，应按照医疗事故发生地居民年平均生活费计算。造成患者死亡的，赔偿年限最长不超过6年；造成患者伤残的，赔偿年限最长不超过3年。本院根据原告的伤残等级，并结合本案实际，酌情确定原告的精神损害抚慰金为30,000元。
　　对于鉴定费，原、被告之间的医疗纠纷构成医疗事故，故由此产生的鉴定费用应由被告全额承担。
　　综上所述，依照《中华人民共和国民法通则》第九十八条、第一百零六条第二款、《医疗事故处理条例》第四十九条、第五十条、第五十二条之规定，判决如下：
　　一、被告上海市某人民医院于本判决生效之日起十日内赔偿原告严某医疗费48,512.28元、残疾生活补助费139,200元、误工费8,000元、护理费900元、营养费1,800元、住院伙食补助费480元，共计198,892.28元的40%，计79,556.91元；
　　二、被告上海市某人民医院于本判决生效之日起十日内赔偿原告严某精神损害抚慰金30,000元；
　　三、驳回原告严某的其余诉讼请求。
　　案件受理费5,952元，减半收取2,976元，医疗事故鉴定费3,500元，司法鉴定费1,000元，合计诉讼费7,476元，由原告严某负担1,730元（已付），被告上海市某人民医院负担5,746元（已付3,500元，余款2,246元于本判决生效之日起七日内交付本院）。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
代理审判员康晓莉
　　二〇一一年八月二十二日
　　书记员谢文倩
