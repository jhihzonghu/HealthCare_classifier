李辉等诉永兴县第二人民医院医疗事故责任纠纷案

审判时间：20110901 法院：湖南省永兴县人民法院 案件字号：(2011)永民初字第407号


　　原告李辉。
　　原告黄飞。
　　委托代理人史军平，湖南银光律师事务所律师，特别代理。
　　被告永兴县第二人民医院。
　　法定代表人邓和胜。
　　委托代理人许春艳，湖南奋斗者律师事务所律师，特别代理。
　　原告李辉、黄飞与被告永兴县第二人民医院医疗事故责任纠纷一案，本院于2011年6月8日立案受理，依法由审判员李郁适用简易程序公开开庭进行了审理，原告李辉、黄飞的委托代理人史军平、被告永兴县第二人民医院的委托代理人许春艳到庭参加诉讼，本案现已审理终结。
　　原告李辉、黄飞诉称：2010年6月16日上午10时30分左右，原告之子李俊杰（2009年2月2日出生）因发高烧入被告处治疗，被告医生见状后给李俊杰输了一瓶液，李俊杰体温下降到37.7度。中午12时左右，被告催原告交医疗费并停止治疗，原告李辉回家借钱，原告黄飞则与李俊杰在被告处等待，下午16时许，原告李辉未归，被告主管医护人员便同意原告黄飞带李俊杰回家找原告李辉，17时许，原告借到钱后将李俊杰送回被告处继续治疗。因听说李俊杰中午吃了包子和牛奶，被告医生遂对李俊杰进行清痰措施，几个医护人员把年仅一岁多的李俊杰手脚按住插入胶管进行粗暴的清痰，李俊杰这时脸色泛白、全身抽搐，原告见状要求被告医生不要再这样野蛮清痰，但被告医生未予理睬，继续对李俊杰进行野蛮的清痰措施，到晚上19时许，李俊杰脸色全白、全身持续抽搐，口鼻出血，神志模糊，但被告医生仍未给予重视，既不采取施救措施，也不告知原告转院，过了一个多小时后，见李俊杰已经奄奄一息了才决定转院抢救并打郴州市第一人民医院120，然而，救护车刚上高速路口不久，晚上21时许，李俊杰已经死亡。原告依法要求被告予以赔偿，被告拒不承认自己的过错也不愿意赔偿，2010年6月17日，经永兴县卫生局组织调解，在既未依法尸检和进行医疗事故或医疗过错鉴定等确定事件性质，也未告知原告法定赔偿标准等前提下，被告仅以人道主义为名补偿原告78600元。2010年8月3日，经永兴县卫生局委托，湘南学院附属医院病理科作出尸检报告，认定李俊杰死亡原因为呼吸道感染衰竭并心衰。原告认为被告医生在最初的诊断中没有按照相关操作程序进行检查，诊断不明确，发现原告欠医药费就停止治疗，后又进行粗暴和违规的清痰措施，发现李俊杰病情恶化后未及时采取抢救措施和告知转院，参与施救医生存在没有医疗资格的情况，李俊杰的死亡完全系被告的医疗过失造成的。双方达成的《永兴县医疗纠纷调处协议书》本身不具有民事合同性质，即使具有合同性质也是无效的，该协议没有另一权利人黄飞的签字，治疗的时候黄飞在场，调处的时候黄飞未参与，李辉没有代黄飞签字，表见代理的前提不存在，退一步讲，该协议即使有效也是可撤销的，调处时，永兴县卫生局没有站在公平的角度，没有明确是赔偿还是补偿，没有告知原告赔偿的项目和计算标准，存在欺骗情形，对原告显失公平。因此，为维护原告合法权益，特向法院提起诉讼，请求确认原、被告双方于2010年6月17日签订的《永兴县医疗纠纷调处协议书》无效或撤销该协议，判令被告赔偿死亡赔偿金112440元、精神损害抚慰金50000元，减去已支付的78600元，还应赔偿83840元，本案诉讼费由被告承担。
　　被告永兴县第二人民医院辩称：第一，原告的请求不明确，协议是撤销还是无效只能选择一个；第二，协议本身具有民事合同的性质，该协议是在第三方组织下，由双方自愿达成的，双方都是具有完全民事行为能力人，是真实的意思表示，不违反法律规定，协议是真实有效的，也不存在重大误解和显失公平，应该按协议履行，我方也已经履行完毕；第三、作为医院，之前都是李辉在跟医院接触，医院没有必要请黄飞过来主张权利，李辉是完全民事行为能力人，有当然的处理权，黄飞没有在协议上签字不影响协议效力，且黄飞没有在一年内主张权利，已经超过期限了；第四、当时是李辉不同意对李俊杰进行鉴定，李辉是一个完全民事行为能力人，他对鉴定的结果也有一个判断和分析，是出于自己利益的考虑而不主张鉴定的；第五，受害人的死亡原因经鉴定为呼吸道感染并心衰，与被告的医疗行为没有因果关系，医院有没有过错不是本案要纠缠的问题，协议已经明确医院存在过失，医院是在协议有效的前提下才愿意补偿给原告李辉78600元，我们已经履行了补偿义务。
　　原告为支持其主张，在举证期限内向本院提交了以下证据：
　　1号证据：受害人李俊杰的出生证明，拟证明原告的主体资格和受害人的出生情况；
　　2号证据：永兴县医疗纠纷调处协议书，拟证明被告存在医疗过失，该协议违反了法律规定，调处时没有查明死亡原因，未明确事故性质，没有告知原告相关的法律规定和赔偿标准，协议也没有死者李俊杰母亲的签名，是非法无效的；
　　3号证据：湘南学院附属医院病理科病理解剖诊断报告，拟证明受害人李俊杰因急性支气管炎入院治疗后因呼吸道感染并心衰死亡完全是因为被告的医疗过失行为导致的；
　　4号证据，永兴县第二人民医院死亡证明书，拟证明内容同3号证据。
　　被告对原告的举证质证如下：
　　1号证据无异议；2号证据的真实性无异议，对证明方向有异议，同时，黄飞的身份至今没有证据证明，李辉有作为家长的表见代理权，协议是在完全告知了死者家属的，告知了原告方相关权利，对原告方的证明方向我们不认可；3号证据和4号证据的真实性无异议，对证明方向有异议，两份证据只是查明死因，并没有对医院的过失或事故作出鉴定，不能证明医院有过错。
　　对于原告的举证，依据证据的相关规则，本院依法作出如下认证：
　　1号证据系医院出具的出生证明，被告无异议，是真实的，本院予以采信；2号证据本身是真实、客观的，但不能证明协议本身是非法无效的；3号、4号证据的真实性被告无异议，能够证明李俊杰的死亡原因，但其不能证明被告有医疗过失行为，本院不予采信。
　　被告永兴县第二人民医院为支持其主张，在举证期限内向本院提交了以下证据：
　　1号证据调处协议书，拟证明在调处协议的签订过程是公平公正的，主体合法，主持方已经告知了李辉相关的权利义务，从字面意思上也可以看得出来，从其他事宜这条来看，双方也不再愿意提起这个事情，双方都是想将此事了解，双方签署协议时不存在误解，意思表示真实；
　　2号证据会议记录，拟证明1号证据是真实有效的，程序合法。
　　原告对被告的举证质证如下:
　　1号证据的真实性无异议，对证明方向有异议，调处协议书不能证明调处的过程，我方不予认可；2号证据没有原告方的签名，李辉没有签名，记录只是单方的记录，没有相关人员的签字，真实性不予认可，证明方向当然也不予认可。
　　对于被告永兴县第二人民医院的举证，依据证据的相关规则，本院依法作出如下认证：
　　1号证据有双方当事人签字认可，并且是在县卫生主管部门、当地政府支持调解下达成的，是真实、客观的，本院予以采信；2号证据虽然没有原告李辉的签名，但有调解协议佐证，可以证实调解过程，本院予以采信。
　　经审理查明，2010年6月16日上午10时左右，原告之子李俊杰（2009年2月2日出生）因发高烧入被告处治疗，被告医生采取治疗措施后李俊杰体温下降到37.7度。下午16时许，原告黄飞带李俊杰回家，17时许，原告又将李俊杰送回被告处继续治疗。被告主治医生听说李俊杰中午吃了包子和牛奶，便对李俊杰采取清痰措施，但李俊杰病情逐渐严重，主治医生决定转院抢救并打郴州市第一人民医院120，晚上21时许，李俊杰在救护车上不幸死亡。原告遂与被告发生纠纷，2010年6月17日，经永兴县卫生局主持调解，马田镇政府参与，双方达成一份《永兴县医疗纠纷调处协议书》，约定被告出于人道主义补偿原告78600元，于当天下午4时前一次性付清，并写明对事件不予定性，双方签订协议后不得再就此事挑起新的矛盾和争端，后原告李辉在协议上签字，被告永兴县第二人民医院在协议上盖公章，卫生局、马田镇政府参与人在协议上签字，被告永兴县第二人民医院依约交付给了原告李辉补偿款78600元。2010年8月3日湘南学院附属医院作出鉴定，认定李俊杰系死亡原因未呼吸道感染并心衰死亡。后原告李辉认为调处存在欺骗和违法情形，对其显失公平，找到被告要求赔偿，被告认为已经处理好了拒绝赔偿，原告遂向本院提起诉讼，请求法院确认原、被告双方于2010年6月17日签订的《永兴县医疗纠纷调处协议书》无效或撤销该协议，判令被告赔偿死亡赔偿金112440元、精神损害抚慰金50000元，减去已支付的78600元，还应赔偿83840元，本案诉讼费由被告承担。
　　本院认为，合同是平等主体的自然人、法人、其他组织之间设立、变更、终止民事权利义务关系的协议，依法成立的合同，对当事人具有法律约束力，当事人应当按照约定履行自己的义务，不得擅自变更或者解除合同，依法成立的合同受法律保护。本案中，双方发生纠纷后在永兴县卫生局主持下，马田镇政府的参与下，经过自主自愿的协商达成了一致意见，签订了《永兴县医疗纠纷调处协议书》，对双方的权利义务关系作出了约定，是双方真实的意思表示，具有民事合同的性质，应该本着诚实信用的原则忠实履行。该协议本身不存在违法法律、行政法规的强制性规定的情形，从庭审查明的情况来看，原告李辉作为死者李俊杰的父亲和一家之主，可以代表患方与医院协商，原告关于协议无死者母亲签字认可而致协议无效的主张本院不予采信，因此协议不存在无效的情形。根据协议内容，明确约定了事件未经医疗事故技术鉴定，不予定性、一次性补偿给患方家属78600元，原告李辉对此是明知的，是双方协商一致的结果，李辉也签字确认，协议不存在欺诈和胁迫的情形。从协议的处理结果来看，对事件不予定性、由医院一次性补偿给原告78600元是双方基于各自对事件定性结果的分析和预判以及风险和利益的综合考虑后协商达成的一致意见，虽然后来湘南学院附属医院病理科作出了死者李俊杰死亡原因未呼吸道感染并心衰的认定，但不能由此证明被告具有医疗过错、事件系医疗事故，原告关于协议显失公平的主张缺乏证据支持，本院不予采信。本案中，双方争议的《永兴县医疗纠纷调处协议书》不存在无效和可撤销的情形，且被告永兴县第二人民医院已经履行给付义务，原告已经收到补偿款78600元，因此，原告李辉、黄飞的诉讼请求本院不予支持。
　　综上所述，根据《中华人民共和国民事诉讼法》第六十四条、《中华人民共和国民法通则》第四条、第五十七条、第八十五条，判决如下：
　　驳回原告李辉、黄飞的诉讼请求。
　　案件受理费1896元，减半收取948元,由原告李辉、黄飞负担。
　　如不服本判决，可在判决书送达之日起十五日内向本院递交上诉状，并按对方当事人的人数提出副本，上诉于湖南省郴州市中级人民法院。
　　审判员李郁
　　二〇一一年九月一日
　　书记员蒋立军
