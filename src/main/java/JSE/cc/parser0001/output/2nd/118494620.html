包某某与上海交通大学医学院附属第九人民医院医疗损害赔偿纠纷案

审判时间：20110420 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第638号


　　上诉人(原审原告)包某某。
　　被上诉人(原审被告)上海交通大学医学院附属第九人民医院。
　　法定代表人张某某，院长。
　　委托代理人戴某，该院工作人员。
　　委托代理人汪某某，该院工作人员。
　　上诉人包某某因医疗损害赔偿纠纷一案，不服上海市黄浦区人民法院(2009)黄民一(民)初字第3637号民事判决，向本院提起上诉。本院受理后，依法组成合议庭，公开开庭进行了审理。上诉人包某某，被上诉人上海交通大学医学院附属第九人民医院(以下简称市九医院)的委托代理人戴某到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明：2009年1月12日包某某持案外人王某某的医保卡前往市九医院拔牙，该院为其局麻下拔除└8┌8。同年1月14日包某某至龙华医院口腔科检查，该院检查如下：左颊略水肿、左颌下淋巴结肿大、压痛、└8┌8拔牙侧内有血块、上覆假膜、龈略水肿、压痛，诊断：└8┌8拔除后。之后，自2009年4月13日起，包某某一直以其拔牙后头晕、头痛、左耳与左眼不适之病为由前往其它医院就诊。期间，包某某向市九医院主张过错责任赔偿，遭市九医院拒绝，包某某遂诉至原审法院，要求市九医院赔偿已产生的医疗费人民币(以下币种同)1万余元、今后的治疗费8万余元，共计10万元。
　　原审审理中，包某某出示了其2009年1月12日的牙齿全景X光摄片，市九医院对此予以认可，确认是由市九医院拍摄的包某某牙齿全景片。关于双方的医疗争议，市九医院申请进行医疗事故技术鉴定，原审法院委托了上海市静安区医学会鉴定，该医学会受理后组成专家鉴定组，组织召开鉴定会，由专家听取双方的陈述，对双方提供的材料和作出的陈述与答辩进行讨论、合议，最后于2010年7月28日出具沪静医鉴[2010]004号《医疗事故技术鉴定书》，载明分析意见如下：1、患者因┌8近中阻生，牙合面龋坏，└8残根就诊，符合拔牙指征。使用劈牙法拔除└8┌8，符合医疗诊疗常规。2、现场查体：患者张口无受限，咬合关系正常，双侧颌下区皮肤感觉正常，┌8创面愈合良好。神经系统查体未见阳性体征。3、患者所述头昏、头痛、视力下降、耳廓不适等症状与拔牙无因果关系。4、医方不足：建议今后加强与患者的沟通与告知。鉴定结论：包某某与市九医院的医疗争议不构成医疗事故。对此鉴定包某某有异议，要求再次鉴定。为此，原审法院委托上海市医学会进行再次鉴定。上海市医学会受理后按照规定程序经由专家鉴定，于2011年1月13日出具沪医鉴[2010]201号《医疗事故技术鉴定书》，分析意见载明“医方诊断符合，有拔牙指征。鉴定专家现场询问双方当事人，整个拔牙过程历时时间短，约5分钟，表明操作顺利。运用劈牙法拔除＋8的方式，符合此类型阻生牙的临床治疗规范。患者拔牙次日左侧面部肿胀，系拔牙后难以完全避免的常见现象，经服药一周后消退。而10天后患者逐渐出现的头痛、头晕、耳鸣、视力减退等不适症状，没有依据表明与医方医疗行为有因果关系”。鉴定结论明确双方的医疗争议不构成医疗事故。
　　原审法院审理后认为，按照我国法律关于侵权责任的归责原则，侵权损害赔偿民事责任的构成应当具备以下要件：1、行为违法；2、有损害事实的发生；3、违法行为与损害后果有因果关系；4、行为人主观上有过错。本案包某某与市九医院之间的医疗争议先后由上海市静安区医学会、上海市医学会通过医疗专家进行鉴定，两次鉴定结果均显示市九医院的医疗行为既不构成医疗事故，也不存在医疗事故以外的与包某某目前状况具有因果关系的医疗过错。因此，包某某要求市九医院承担医疗侵权损害责任，缺乏依据，法院无法支持。关于本案已发生的鉴定费，法院考虑到市九医院在与患者沟通与告知方面存在不足，核情确定首次鉴定的鉴定费由市九医院予以负担。包某某要求再次鉴定所发生的鉴定费，由于再次鉴定的结果与前次鉴定一致，故此项费用应当由包某某负担。据此，依据《中华人民共和国民法通则》第一百零六条第二款，以及《医疗事故处理条例》第二条之规定，判决：包某某要求上海交通大学医学院附属第九人民医院赔偿的诉讼请求，不予支持。
　　原审判决后，包某某不服，向本院提起上诉。
　　上诉人包某某称，被上诉人违反诊疗常规，用榔头敲下牙齿，致使上诉人术后产生严重的眼胀、耳鸣、头痛、头晕等症状。被上诉人对上诉人的上述病症负有过错，应当承担相应的责任。据此要求被上诉人赔偿已产生的医疗费及今后治疗所需的医疗费共计10万元。
　　被上诉人市九医院辩称，被上诉人针对上诉人的牙齿情况采取的拔牙方式符合诊疗常规，且经两级医学会鉴定，被上诉人不存在医疗过错。故不同意上诉请求，要求维持原判。
　　本院经审理查明，原审查明事实属实，本院予以确认。
　　本院认为，本案系一起因医疗纠纷所引起的侵权损害赔偿案件。而在侵权损害赔偿案件中，确认是否构成侵权责任的关键是因果关系，即加害行为与损害后果之间的引起与被引起关系，在本案中也就是目前患者包某某的眼胀、耳鸣、头痛、头晕等症状是否是市九医院的拔牙所引起。上海市静安区医学会、上海市医学会对本案所涉医疗行为的医疗事故技术鉴定均明确患者包某某的头痛、头晕等症状与医院拔牙没有因果关系。同时，患者包某某在一、二审案件审理中也未能提供足够、充分的证据证明上述因果关系。故患者包某某现以其目前头痛、头晕等症状系医院拔牙造成为由上诉要求医院承担侵权损害赔偿责任，显然没有法律依据，本院不予支持。原审法院根据本案的在案证据和相关事实所作的判决是正确的，本院予以维持。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币2,300元，由上诉人包某某负担。
　　本判决为终审判决。
　　审判长张铮
　　代理审判员赵静
　　代理审判员欧阳璟璐
　　二○一一年四月二十日
　　书记员何冰
