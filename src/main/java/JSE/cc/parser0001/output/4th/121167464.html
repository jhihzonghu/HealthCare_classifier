上诉人谷某某与上诉人商丘市第一人民医院、被上诉人中华医学会河南省商丘分会医疗损害责任纠纷案

审判时间：20111101 法院：河南省商丘市中级人民法院 案件字号：(2011)商民终字第1101号


　　上诉人（原审原告）谷某某。
　　法定代理人谷建中。
　　法定代理人朱冬梅。
　　委托代理人王海鹰。
　　上诉人（原审被告）商丘市第一人民医院。
　　法定代表人韩传恩，院长。
　　委托代理人汪继华，河南华豫律师事务所律师。
　　被上诉人（原审被告）中华医学会河南省商丘分会。
　　代表人高亚萍，副会长。
　　委托代理人汪继华，河南华豫律师事务所律师。
　　上诉人谷某某与上诉人商丘市第一人民医院(以下简称商丘市第一医院)、被上诉人中华医学会河南省商丘分会(以下简称医学会商丘分会)医疗损害责任纠纷一案，谷某某于2009年8月13日向商丘市睢阳区人民法院提起诉讼，要求判令被告赔偿医疗费、交通费、鉴定费、营养费(包括定残前营养费)、护理费(包括定残前护理费)、残赔偿金、精神抚慰金、后续治疗费等共计2463966.87元并承担诉讼费。该院审理后于2011年6月22日作出（2009）商睢区民初字第1900号民事判决，谷某某、商丘市第一医院均不服该判决向本院提起上诉。本院于2011年8月2日受理后依法组成合议庭，并于2011年9月1日公开开庭进行了审理。上诉人谷某某的法定代理人谷建中、委托代理人王海鹰、朱冬梅，上诉人商丘市第一医院、医学会商丘分会的委托代理人汪继华到庭参加诉讼。本案现已审理终结。
　　原审法院审理查明，2001年9月13日，原告之母朱冬梅因宫内孕40周加5天，入住被告医院产科待产，B超检查显示：宫内孕，单胎，头位，胎心好。被告医院产科7525号病历记录：10时50分查宫缩中弱，时限30秒，间隔5-6分钟，胎心118次／分，先露-3，官口开大1cm，胎膜未破，少量出血。考虑胎儿宫内缺氧、脐带短、脐带打结等异常情况，向家属说明，建议剖宫产，并采取吸氧，左侧卧位。0时30分胎心120次／分，再次向家属讲明胎儿宫内室息，缺氧。家属同意剖宫产，现做术前准备，因宫缩强度好，内诊查宫口开大4cm，胎心124次／分，家属要求观察。0时30分至2时40分宫缩中，时限35秒，间隔3-4分钟，胎心124-128次／分；l时10分先露为0，宫口开大5cm，破水，羊水清；2时30分宫口开全，各项指标均正常；2时40分，强度中弱，时限35秒，间隔2-3分，胎心124次／分，给100%GS500ml+催产素2.5μ，胎心慢，准备侧切负吸助产，尽快结束分娩，不能保证胎儿100％成活，院方尽力救治，家属理解。2时45分宫缩中弱，时限30秒，间隔4-5分钟，胎心慢，116次／分。告诉家属侧切负吸助产，尽快结束分娩。2时50分胎心110次／分，加催产素2.5μ。2时55分加快催产素滴速，静推碳酸氢钠。3时10分10％GS250ml+催产素10μ，胎头吸引助产，并增加腹压。3时30分宫缩中，时限30秒，间隔3-4分，胎心114-118次／分。3时40分娩出一活男婴(即原告谷某某)，体重2800g，评分3-5-8分，苍白窒息，脐带扭转、绕脚。给吸痰、吸氧，脐静注碳酸氢钠、地塞米松，人工呼吸，皮下注射洛贝林、可拉明各1／3支。出生时胎头无水肿、无血肿、眼正常、鼻正常、口唇红、心律齐、肺音清、肤软、脐无渗出、外阴正常、肛门正常、皮肤红润、无畸形。2001年9月14日8时转入被告医院儿科，诊断为新生儿缺血缺氧性脑病，新生儿颅内出血，羊水吸入性肺炎。2001年9月18日头颅CT显示：轻度新生儿缺血缺氧性脑病，纵裂池蛛网膜下腔出血(少量)。被告医院儿科7534号病历显示：原告有头颅水肿血肿、前额青紫、阴囊水肿等产伤。住院治疗12天出院，出院诊断与入院诊断相同。2001年12月，原告诉讼之法院。在诉讼中，应被告医院申请，被告医学会进行了医疗事故鉴定，认为原告的脑病与被告医院无因果关系，不构成医疗事故。原告撤回起诉，该院作出(2002)商睢区民初字第808号民事裁定，准许原告撤回起诉，并于2003年12月5日送达原告之父谷建忠。2002年2月1日，原告于被告医院门诊复诊，记录头向后昂，智力差。2002年4月18日至6月9日，在郑州市儿童医院住院治疗，出院诊断为脑性瘫痪(混合型)。2002年6月22日至7月12日，在郑州大学第三附属医院住院治疗。2009年3月10日，原告之母朱冬梅在商丘市睢阳区计划生育宣技站康复医院进行血巨细胞病毒检查，检查结果正常。2009年8月18日，被告医院工作人员承诺，只要经鉴定认定医院有过错，就承担赔尝责任。2009年12月30日，河南正诚司鉴所(2009)临鉴字第645号鉴定，认为被告医院在原告之母朱冬梅分娩过程所采取的医疗措施存在过错，其过错与原告脑瘫有一定因果关系。原告支付鉴定费3000元。该院依据被告医院的申请，委托北京华大方瑞鉴定中心予以重新鉴定。2010年12月2日，该鉴定中心做出(2009)临鉴字第2061号鉴定。鉴定认为：原告出生前，其母朱冬梅入院时即出现胎心慢，第二产程长时间胎心慢，均考虑有宫内窘迫，出生后证实脐带较短40cm，扭转过度，脐带脚。出生时即有新生儿窒息，经住院治疗，诊断为新生儿缺血缺氧性脑病，羊水吸入性肺炎，新生儿颅内出血。上述因素均系新生儿缺血缺氧性脑病、脑瘫发生的原因。同时存在围产期内的母体的因素，胎儿自身因素。被告医院的医疗过程中存在无使用指征下，超浓度、超量、超速违规使用催产素；胎心减慢时未及时停用催产素；不排除违规使用胎头负压吸引器助产方式及操作不当；新生儿窒息抢救措施不得力的医疗过错行为，与谷某某缺血缺氧性脑病、脑瘫的损害后果之间存在一定的因果关系，应负少部分责任，赔偿参考范围20％～40％。2010年2月8日，开封顺天司法鉴定中心(2010)号临鉴字56号鉴定，认为原告重度脑瘫致极重度智能损伤，重度运动障碍，属一级伤残，日常生活完全不能自理，一级护理(每日需1-2人)。原告为此支出治疗、检查、康复、交通、鉴定等费用共计49725.47元。
　　原审法院认为：《中华人民共和国民法通则》第一百一十九条规定，侵害公民身体造成伤害的，应当赔偿医疗费、因误工减少的收入、残废者生活补助费等费用；造成死亡的，并应当支付丧葬费、死者生前扶养的人必要的生活费等费用。第一百三十一条规定，受害人对于损害的发生也有过错的，可以减轻侵害人的民事责任。本案中原告支出的治疗、检查、康复、交通、鉴定等费用共计49725.47元；依法计算的费用有：护理费780075．65元(定残前27357元／年÷365天×3065天×1人)，定残后27357元／年×20年×1人)，住院伙食补助费8760元(292天×30元／天)，营养费103650元(定残前10元／天×3065天，定残后10元／天×365天×20年)，残疾赔偿金318605.20元(15930.26元／年×20年)，残疾辅助器具费114156.99元(尿不湿：定残前3420元／年÷365天×3065天、定残后3420元／年×20年，推车：定残前1800元／3年÷3年÷365天×3065天、定残后1800元／3年÷3年×20年)；两项共计1374973.3l元。北京华大方瑞鉴定中心作出的(2010)临鉴字第2061号鉴定认定，被告医院与原告缺血缺氧性脑病、脑瘫的损害后果之间存在一定的因果关系，且应负少部分责任，故被告医院应当赔偿原告损失1374973.31元的40％即549989元，并赔偿原告精神损害抚慰金10000元，合计559989元。原告的后续治疗费用，待实际发生后，可另行主张。原告要求被告医学会承担赔偿责任等其他诉讼请求无依据，不予支持。原告一直在主张权利，且被告医院工作人员承诺是履行职务的行为，承诺内容又符合《最高人民法院关于审理民事案件适用诉讼时效制度若干问题的规定》第二十二条“诉讼时效期间届满，当事人一方向对方当事人作出同意履行义务的意思表示或者自愿履行义务后，又以诉讼时效期间届满为由进行抗辩的，人民法院不予支持”的规定，是向原告作出同意履行义务的意思表示，故被告医院关于原告的诉讼请求已经超过诉讼时效期间的抗辩，不予支持。原审依照《中华人民共和国民法通则》第一百一十九条、第一百三十一条之规定，判决：一、被告商丘市第一人民医院于本判决生效之日起十日内赔偿原告谷某某各项损失559989元；二、驳回原告谷某某要求被告中华医学会河南省商丘分会承担赔偿责任等其他诉讼请求。一审案件受理费26500元，原告谷某某负担15500元，被告商丘市第一人民医院负担11000元。
　　上诉人谷某某不服原判上诉称，原审认定事实不清。1、导致受害人谷某某脑瘫致残的原因是由于商丘市第一医院在医疗救治过程中，医疗行为存在过错、用药不当和助产方式使用不当造成的。医学会商丘分会以谷某某的患脑瘫是由于其脐带扭转所致产程中胎儿缺血、缺氧，致产后苍白窒息，新生儿缺血、缺氧脑病造成的，作出错误鉴定。然后又以与上诉人无关的感染先天性巨细胞病毒进行误导，致使上诉人在2003年以证据不足撤诉，本该早已结束的案件拖至今。商丘市第一医院和医学会商丘分会应承担谷某某的全部损失。2、原审判令谷某某承担60%的过错责任没有任何事实依据和法律依据。商丘市第一医院提供的病历即自相矛盾又进行伪造，且该病历又否定了北京华大方瑞鉴定中心司法鉴定意见书的鉴定结论，该机构的鉴定人员不能对上诉人及商丘市第一医院的质疑进行充分合理的解释。故该鉴定意见不能作为定案依据，谷某某不应该承担责任。3、原审判令商丘市第一医院赔偿谷某某的护理人数、精神抚慰金数额明显错误，显失公平。请求二审法院，查明事实，依法改判商丘市第一医院及医学会商丘分会承担全部赔偿责任。
　　商丘市第一医院不服上诉称，1、谷某某的起诉已经超过诉讼时效。原审认定谷某某的起诉没有超过诉讼时效缺乏法律依据。2、原审判令商丘市第一医院承担40%的赔偿责任显失公平。请求二审法院，查明事实，依法驳回谷某某的诉讼请求。
　　被上诉人医学会商丘分会辩称，对患者的医疗事故技术鉴定符合规定，在鉴定过程中没有过错，不应承担赔偿责任。请求二审法院，查明事实，撤销原判，驳回谷某某的诉讼请求。
　　根据各方诉辩意见，本院确定本案争议焦点：1、受害人谷某某的起诉是否超过诉讼时效。2、本案的当事人是否应当承担赔偿责任，承担赔偿比例划分是否适当。3、原审确认的受害人护理人数、精神抚慰金数额是否适当。
　　各方当事人对上述争议焦点均无异议。
　　二审庭审中双方均没有新证据向法庭提交。
　　本院经审理查明的事实与原审查明的基本事实一致。
　　本院认为，1、关于受害人谷某某的起诉是否超过诉讼时的问题。《民法通则》第一百三十七条规定：诉讼时效期间从知道或者应当知道权利被侵害时起计算。《最高人民法院关于审理民事案件适用诉讼时效制度若干问题的规定》第二十二条规定。诉讼时效期间届满，当事一方向对方当事人作出同意履行义务的意思表示或者自愿履行义务后，又以诉讼时效期间届满为由进行抗辩的，人民法院不予支持。本案中，朱冬梅于2001年9月13日晚住进商丘市第一人民院分娩，谷某某出生后即患有脑瘫。谷某某、谷建中、朱冬梅以商丘市第一人民院有过错为由，于2001年12月以该医院为被告向睢阳区人民法院提起民事诉讼，要求该医院赔偿其损失。后谷某某、谷建中、朱冬梅以证据不足为由，于2003年12月5日撤回起诉。睢阳区人民法院作出。但在2009年8月18日，商丘市第一人民医院医务科的主要领导在接待谷建中要求解决医疗纠纷时，有：经过第三方，只要医院有过错、有多大过错，承担多大的责任的表述。该医院对此表述的事实并无异议。据此可以认定该医院做出了同意履行赔偿的意思表示。依照上述法律和解释的规定，由于商丘市第一人民医院已经同意履行赔偿义务，而又以谷建忠的请求超过诉讼时效为由的抗辩，显然与上述法律规定和解释相违背，其上诉请求本院不能支持。
　　2、关于本案的当事人是否承担赔偿责任，赔偿比例划分是否适当的问题。北京华大方瑞司法物证鉴定中心是经北京市司法局依法批准并颁发司法鉴定许可证的鉴定机构。该鉴定意见书是受睢阳区人民法院的委托，就商丘市第一人民医院对朱冬梅及其谷某某的医疗行为是否存在医疗过错，若有医疗过错行为，该行为与谷某某的损害后果之间是否存在因果关系，程度如何作出的鉴定。该鉴定机构在鉴定过程中，组织患方、医方、委托方、鉴定人员均到场的情况下，按照司法鉴定规定的程序进行。该鉴定对商丘市第一人民医院的医疗行为、用药、助产方式进行了评价，对谷某某巨型细胞毒抗体阳性及双方的因果关系进行了分析，即：（一）对商丘市第一医院的医疗行为进行评价：认为（1）医方对谷某某之母朱冬梅孕40周孕2产1的诊断正确。（2）当胎心转为正常，孕妇及家属不同意剖妇产，经观察宫缩规律，胎心在正常范围。医方允许经阴道顺产，其医疗行为符合规范。（3）医方在分娩经过记录中按时记录宫缩、胎位、胎心、先露高低、宫口、胎膜等情况。审核该记录，医方对先露高低的记录只有入院时第一次记录。第二、三均未记录谷某某先露情况，医疗行为存在过错。（二）关于医方在朱冬梅产过程中使用催产素的评价。（1）、在无使用指证的情况下使用催产素。（2）、催产素使用方法不当。（三）关于医方助产方式的分析评价为：医方采用抬头吸引器助产存在不规范之处。（四）关于谷某某巨型细胞毒毒抗体阳性分析为：不能认定谷某某为先天性宫内感染。（五）关于医方的医疗过错行为与谷某某脑瘫后果之间的因果关系及程度分析为：1、谷某某出生后患脑性瘫痪，事实存在。2、商丘市第一医院的医疗过错行为与谷某某缺血性脑病、脑瘫的损害后果之间存在一定的因果关系，医方应负少部分责任，参与度考虑c级，理论系数值25%，（医疗过失参与度对照表显示：赔偿参考范围20-40％)。本院认为，该鉴定机构按照法定的程序接受法院的委托进行的鉴定，其鉴定程序合法，在实体鉴定中，按照医院对谷某某生产、救治过程中病历记录的情况、结合受害人谷某某的现状作出的鉴定结论客观、真实、合法、有效。本院依照《最高人民法院关于民事证据的若干规定》的有关规定，对该鉴定予以认定。上诉人谷某某虽然提出本鉴定在一审庭审中对鉴定人的质询时，鉴定人员对其质疑的问题没有能充分解释和说明，但该上诉理由不足以对抗该鉴定是具有法定鉴定资质的鉴定人员按照法定程序依法做出的鉴定结论，其上诉理由无有效证据支持，上诉理由不能成立。由于该鉴定已经由本院予以确认，据此说明河南省商丘市第一人民医院应负少部分责任。而上诉人谷某某认为商丘市第一医院提供了自相矛盾的病历，且该病历又否定了北京华大方瑞司法物证鉴定中心司法鉴定意见书中的鉴定意见，谷某某的脑瘫是由于该院记录不全面、不真实，用药不当、助产器使用错误造成。本院认为，谷某某的上诉理由无有效证据证明其主张且其理由不足以否定北京华大方瑞司法物证鉴定中心司法鉴定的鉴定意见。至于谷某某认为医学会商丘分会作为鉴定单位，首先该会与商丘市第一医院二者利用自身的优势恶意患通，隐瞒实情，把上诉人所患脑瘫错定是由于新生儿脐带扭转导致产程中的胎儿缺血缺氧、产后苍白窒息，缺血缺氧脑病造成的，作出违背医学科学的鉴定书，即：医学会医鉴字【2003-008】结论为：婴儿出现脑瘫与产程中医疗行为无因果关系。其次，后来又以与上诉人无关的感染先天性巨细胞病毒进行误导，致使上诉人以证据不足撤诉，上诉人本应能减轻的病情拖至现在严重的程度，二者应承担全部责任。本院认为，医学会商丘分会是法定的医疗事故鉴定机构，根据《医疗事故处理条例》的规定，其仅仅对有关单位的委托事项进行鉴定，上诉人谷某某虽然对该会2003年4月18日作出的鉴定结论不服，但该鉴定不是导致谷某某脑瘫的原因，对于致谷某某脑瘫及后续治疗并无过错，其要求医学会商丘分会承担赔偿责任无有效证据支持。综上，谷某某请求判令商丘市第一医院和医学会商丘分会承担谷某某全部损失的请求无事实和法律依据，原审对责任的划分并无不当，上诉人谷某某的上诉请求应予驳回。
　　3、关于原审确认的受害人护理人数、精神抚慰金数额是否适当的问题。首先，受害人谷某某的伤残程度及护理人数经商丘市睢阳区法院委托，开封顺天司法鉴定中心作出(2010)号临鉴字56号鉴定。其意见为：被鉴定人重度脑瘫及重度智力损伤，重度运动障碍，日常生活完全不能自理。属一级伤残，一级护理。每日需1-2人。并根据《最高人民法院关于确认民事侵权精神损害赔偿责任若干问题的解释》第十条规定，精神损害赔偿数额根据以下因素确定：侵权人的过错程度、侵害的手段、侵权行为所造成的后果，侵权人的承担责任经济能力及受诉法院所在地的平均生活水平等。第十一条规定，受害人对损害事实和损害后果的发生有过错的，可以根据其过错程度减轻或者免除侵权人的精神损害赔偿责任等规定。综合本案的情况，考虑本案的实际，原审依照上述法律规定和本案的事实，判令谷某某的护理人员为一人，精神抚慰金为10000元的判决适合本案。
　　综上，原判认定事实清楚，证据充分，程序合法，适用法律及判决结果正确。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉、维持原判。
　　二审案件受理费26500元，由上诉人商丘市第一人民医院负担26500元。
　　本判决为终审判决。

　　审判员赵国庆
　　审判员盛立贞
