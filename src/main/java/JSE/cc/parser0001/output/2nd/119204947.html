池某慧诉金华市中医院医疗损害赔偿纠纷案

审判时间：20110620 法院：浙江省金华市婺城区人民法院 案件字号：〔2010〕金婺民初字第3102号


　　原告：池某慧。
　　法定代理人：池华进。
　　法定代理人：王彩燕。
　　委托代理人：叶晓明，浙江浙中律师事务所律师。
　　被告：金华市中医院。
　　法定代表人：田政。
　　委托代理人：方笋玉，浙江一剑律师事务所律师。
　　原告池某慧为与被告金华市中医院医疗损害赔偿纠纷一案，于2010年11月12日诉至本院，本院同日受理后，依法适用简易程序，由审判员冯志坚独任审判，于2011年5月20日公开开庭进行了审理。原告池某慧的法定代理人池华进、王彩燕及其委托代理人叶晓明，被告金华市中医院的委托代理人方笋玉到庭参加诉讼。本案现已审理终结。
　　原告池某慧的法定代理人池华进、王彩燕诉称：2008年11月14日，原告的母亲王彩燕因停经38周，腹痛3小时，前往被告金华市中医院就诊，在8时40分收住被告妇产科，初步诊断为"孕38周临产"。据产前记录，王彩燕于9时50分有不规则宫缩，14时30分宫口开全，15时25分，宫缩，针穴位，16点15分在侧切下头手复合位助娩一女婴（原告），胎肩娩出困难，评分5-5-5分/1-5-10，无呼吸，给予人工呼吸、胸外心脏按压、皮囊加压吸氧、肾上腺素，20分钟后有自主呼吸，诊断为"新生儿重度窒息"。在17点20分转往金华市中心医院儿科治疗。入院查体：T33.3度，P110次，R49次/分，有黄色胎粪污染，先锋头，右侧头顶有一约2*3厘米血肿，口角歪斜。入院后，诊断为"新生儿窒息，新生儿缺血缺氧性脑病，蛛网膜下腔出血"，于同年11月18日出院。原告在出院后，向被告进行了医疗投诉。同月26日，被告就此医疗投诉进行了答复，认为整个医疗过程符合医疗规范和常规。此后，原告再次向被告投诉要求妥善解决，在2009年1月15日，被告回复"如有异常情况且属医方有责任可再向本院提出"。原告认为被告在医疗过程中存在严重的医疗过错，且造成脑瘫的严重后果。请求法院：1.判令被告赔偿原告的医疗费6363.21元，精神损害抚慰金40000元，交通费531元，残疾赔偿金180848元，定残前护理费50820元，定残后护理费（暂按10年）255500元，营养费12943.8元，合计547006.01元，按50%计273503.01元。2.判令被告按原告实际产生的后续治疗费（包括康复费用）支付给原告。3.本案的诉讼费、鉴定费由被告承担。为证明上述事实及主张，原告池某慧的法定代理人池华进、王彩燕向本院提交的证据有：1.身份证、户口本、结婚证，证明原告的主体身份。2.金华市中医院住院病历，证明被告存在以下的医疗过错，导致原告新生儿重度窒息、缺血缺氧性脑病：1).在孕妇有胎头停滞时，没有及时进行阴道检查。据被告提供的产前记录和产程进展图，孕妇有胎头停滞的情况，但被告没有及时进行阴道检查，导致没有及时诊断复合先露。2).被告在孕妇第二产程出现宫缩弱时，没有及时寻找原因，没有首先考虑有无头盆不对称及严重胎位异常的情况，导致没有及时诊断复合先露。3).被告没有及时针对复合先露进行处理。被告没有在孕妇宫口近开全或开全时行小肢体还纳术，也没有在产程停滞时、胎儿窘迫时，及时剖宫产终止妊娠。4)被告没有严密观察产程、及时发现异常、及时处理。5).被告在胎儿宫内窘迫时，没有对孕妇采取吸氧等措施。6).被告抢救措施不力，如在患儿无自主呼吸，没有及时行气管插管等措施，导致患儿直在20分钟后才恢复自主呼吸。7).被告的病历记载存在不合理处，如金华中心医院病历记载：患儿有胎粪污染，先锋头，右侧头顶有2*3厘米血肿，双眼裂大小不一，口角歪斜等，与被告记载：婴儿未见明显异常、羊水清等。如临产记录诊为新生儿重度窒息，但评分时却改为5分等等。3.金华市中心医院门诊病历（包括儿二科出院记录），证明原告从被告处转往金华市中心医院的治疗经过。4.浙江大学医院院附属儿童医院门诊病(包括医疗诊断证明书)，证明原告在儿童医院治疗经过，原告确诊过脑瘫的事实。5.被告提供的医疗投诉的答复二份，证明原告就此医疗纠纷多次向被告投诉的事实。6.医疗费票据，证明原告治疗的费用。7.交通费票据，证明原告的交通费用。8.金华市医学会的医疗损害技术鉴定书，证明原告的脑瘫与出生时重度窒息有关，被告在接产过程中，对产程观察不严密，未进行阴道检查，对原告重度窒息的第一时间抢救措施不力，存在医疗过错，且与原告的损害后果存在因果关系，原告残疾等级为三级。9.金华广福司法鉴定所意见书，证明原告不能说话，不能独坐、不能站立，原告的护理时间暂定为8－10年，营养时间为6个月，后续治疗费（包括康复用）以实际产生的计算。10.鉴定费票据，证明原告鉴定费用。
　　被告金华市中医院辩称：1.诊治经过。产妇王彩燕。因"停经38周+5天，腹痛3小时余"于2008年11月14日到我院分娩。入院检查：T36.6℃，P77次/分，R20次/分，产科检查：宫高31cm，腹围90cm，胎位ROA，胎心140次/分，宫缩不规则，胎膜未破，宫口未开，已衔接，骨盆外测量24-26-19.5-9.5cm。入院B超提示：头位，宫内单活胎，胎儿心率监护提示：反应型。入院诊断：孕4产0孕38周+5天。12时30分开始宫缩，14时15分胎膜自破，羊水清，14时30分宫口全开，16时15分会阴侧切保护下助娩一女婴。抬头部分娩出后，因头手复合位，胎肩娩出困难，新生儿窒息，Apgar评分1分钟5分，5分钟5分。出生时无呼吸，心率大于120次/分，即予清理呼吸道、胸外按压、皮囊加压给氧，配合药物抢救。约20分钟后有自主呼吸，心率大于120次/分，给予面罩吸氧，保暖，呼叫120，17时20分由金华市中心医院120急救车转接新生儿科治疗。患儿住院4天后，家属自动放弃治疗，于11月18日17时15分抱回我院。当时患儿反应迟钝，皮肤黄，无哭声，情况非常不好。19日建议到新生儿科继续治疗，不治疗可能引起脑瘫等后遗症，之后医院也一直建议到新生儿科继续治疗，但家属一直没有去，12月15日随产妇出院。2.患xxx症。1)产妇产前各项检查提示无剖宫产手术指征，医院根据产妇的要求自然分娩符合诊疗规范，并且已将分娩过程中可能出现的各种意外情况告知产妇。2)产妇在分娩过程中因自身原因使得胎儿头手复合位难产导致胎儿窒息，与医院的诊疗行为无因果关系。3)发现胎儿头手复合位致胎肩娩出困难，为保证母婴安全给予宫底上方加压，徒手牵引抬头，娩出胎儿。胎儿娩出后无呼吸，即予清理呼吸道、胸外按压、皮囊加压给氧，配合药物抢救，建立自主呼吸。医院已经尽力挽救了产妇和患儿的生命安全，抢救是及时有效的。4)新生儿缺血缺氧性脑病的最佳治疗时机是新生儿出生后的1-2周，经过积极有效的治疗大多数患儿都能逐渐恢复，及时有效的抢救直接关系到患者的恢复情况。但本案中患儿经在金华市中心医院抢救4天，后家属放弃治疗，错过了最佳治疗时机，无法得到有效的治疗。18日患儿被抱回我院产科，家属想放弃患儿，医院多次建议到新生儿科继续治疗，并告知不治疗可能存在脑瘫后遗症，但家属一直没有去。抱回我院后对患儿也没有喂养，还是医生、护士建议后才进行母乳喂养，最终患儿坚强的生存下来了。我们认为如果当时一直在中心医院治疗及加强营养，凭患儿如此坚强的生命力恢复的可能性是非常大的，至少不至于出现目前的状况，这些不利后果应由家属自行承担。5)患xxx症，并非分娩时窒息造成，两者不存在必然的因果关系。3.原告主张的赔偿项目计算有误：1)原告系农村居民，赔偿应按农标计算。2)残疾赔偿金和后续治疗费系重复计算。患xxx病情稳定，已构成三级伤残，如继续康复治疗，患儿症状缓解，就达不到三级伤残，为此不能再计算后续治疗费。3)护理费时间过长，因正常的婴儿原本就需要父母照顾护理三年，为此该时间应扣除。护理费应扣除3年时间，因婴儿出生本身需要护理。4)营养费、精神损害抚慰金过高。综上，原告目前的症状主要是产妇分娩头手复合位难产引发窒息及出生后放弃治疗造成的，医院的责任是轻微的，请求法院依法作出公正的判决。
　　为证明上述事实及主张，被告金华市中医院向本院提交的证据：1.住院病历，证明产妇及患儿在我院分娩、抢救的情况及11月18日患儿抱回我院后，医院一直建议到新生儿科继续治疗，并告知不治疗可能出现脑瘫后遗症等，但家属一直未去。2.医学资料，证明新生儿缺血缺氧性脑病的抢救的最佳时机为出生后1-2周，如得不到有效及时的抢救的后果会造成脑瘫。窒息与脑瘫并不存在必然的因果关系。
　　在审理过程中，本院依法组织双方当事人，对上述证据进行了当庭质证，根据质证意见及证据审核认定的有关规定，本院认证如下：
　　原告向本院提交的证据：对证据4.5.7.10的真实性被告无异议，本院确认该证据的证明力。对证据1的真实性被告无异议，但认为原告系农业家庭户，赔偿标准应按农标计算，本院确认该证据的证明力；对证据2的真实性被告无异议，但认为该系不典型的头手复合位，阴道检查也是查不出，发现头手复合位后，当时胎头大部分出来，无法终止妊娠，胎儿出生后无呼吸，医院立即给予了清理呼吸道、胸外按压、皮囊加压给氧，配合药物抢救，建立自主呼吸，抢救措施是积极有效的，不存在抢救不力的情况。医院的抢救是及时有效的，为患儿建立了自主呼吸，家属放弃治疗。本院认为该证据的真实性予以确认，但对被告抗辩的理由因无证据反驳，不予采信。对证据3的真实性被告无异议，但认为病历显示患xxx病，该疾病的最佳治疗时机是新生儿出生后的1-2周，但住院4天后，在患儿病情危重的情况下家属自动放弃治疗，错过了最佳治疗时机，因没有有效及时的治疗最终造成目前的症状，这些不利后果应由家属自行承担，本院确认该证据的证明力。对证据6的真实性被告无异议，但认为其中有2008年12月22日的一张发票为出生证工本费，与本案无关联性，本院确认该证据的证明力，认为出生证工本费4元，属医疗开支，不予扣除。对证据8的真实性被告无异议，但对鉴定书的分析意见2、3及鉴定结论有异议：1）患儿出生时除了窒息，还存在家属放弃治疗的情况，患xxx症，并非分娩时窒息造成。为此鉴定认定"除窒息史外无其他引起脑瘫的客观依据"与事实部分。2）胎儿出生后无呼吸，医院立即给予了清理呼吸道、胸外按压、皮囊加压给氧，配合药物抢救，建立自主呼吸，抢救措施是积极有效的，不存在抢救不力的情况。3）医院一直严密观察产程的变化，同时由于胎儿是不典型的头手复合位，就是作了阴道检查也是查不出来的。4）由于患儿在中心医院存在放弃治疗的情况，鉴定结论认定被告承担次要责任过重，承担轻微责任较为妥当。本院确认该证据的证明力。对证据9，被告认为：1）护理时间过长，未扣除婴幼儿原本就需要照顾护理的时间。2）后续治疗费与伤残赔偿是重复计算，康复训练治疗的结果直接影响到三级伤残是否构成，不应再计算后续治疗费。本院确认该证据的证明力。
　　被告向本院提交的证据：对证据1，原告认为与我方提供的住院病历相符，质证意见与举证内容基本一致。其中被告提到原告的监护人放弃治疗的情况，原告并不认可，根据原告当时的实际情况，原告从中心医院转往被告处治疗，也是被迫无奈的选择，并不是放弃治疗一说，本院确认该证据的证明力。对证据2的真实性原告无异议，但对被告证明目的有异议，认为因本案已经过金华市医学会医疗损害的鉴定，被告用该证据不能达到被告的证明目的，原告的脑瘫是因为重度窒息引起，已鉴定认可，本院确认该证据的证明力。
　　根据本院确认证明力的上述证据及当事人法庭陈述，本院认定本案事实如下：
　　王彩燕因"停经38周+5天，腹痛3小时余"于2008年11月14日收住金华市中医院分娩。入院时检查：T36.6℃，P77次/分，R20次/分，BP135/89mmHg。体重62.5公斤，一般情况可，心肺听诊无异常。产检：骨盆测量：24-26-19.5-9.5cm，宫高31cm，腹围90cm，先露头，胎位：ROA，胎心140次/分，已衔接，宫颈未消，扩张：未开，有宫缩，不规则；胎膜未破，肛检未查。血常规：HB117g/L、WBC10.6*109、N75.2%。B超示：头位，胎心140次/分，NTS反应型，宫内单活胎。入院诊断：孕4产0孕38周+5周临产。产程记录：2008年11月14日12：30开始宫缩；14：15胎膜自破，羊水清；14：30宫口开全；16：15在会阴侧切下经阴道自娩一成熟女婴原告池某慧，胎肩娩出困难，胎儿娩出胎位：头手复合位，体重2900克，身长50cm，Apgar评分5-5-5/1－5－10无呼吸，心率＞120次/分。即予清理呼吸道，胸外按压、人工呼吸、面罩吸氧，并静脉注射肾上腺素，肌注纳洛酮等抢救。约20分钟后有自主呼吸，心率＞120次/分。17：20转金华市中心医院继续抢救。产后诊断：1.孕4产0孕38周+5周头手复合位侧切平产+活婴；2.新生儿重度窒息。于2008年11月14日新生儿（池某慧）因"生后窒息2小时"，被告考虑无新生儿专科治疗，便送往金华市中心医院就治。原告入院金华市中心医院时检查：T33.3℃，P110次/分，R49次/分，反应差，刺激后哭声微弱，右侧头顶有一约2×3cm。11月15日头颅CT检查示：蛛网膜下腔出血。11月18日原告家属要求出院，将原告抱回，由仍住在被告的妇产科的母亲王彩燕带养。于2008年12月15日王彩燕出院，诊断：1.新生儿窒息；2.新生儿缺血缺氧性脑病；3.蛛网膜下腔出血。出院时情况：反应欠佳，刺激后无哭声，无抽搐。原告于2009年7月14日在浙江大学附属儿童医院就诊，记录：竖头不稳，不会独坐，有惊跳。检查：头围41cm，心肺（一），四肢肌张力增高。脑电图检查报告：未见异常。诊断：脑发育不全，脑瘫。2010年4月21日再次门诊记录：现15个月，不能独坐，产后不哭，有颅内出血。头颅CT检查报告：未见明显异常。
　　上述，原告多次向被告投诉要求处理该起医患纠纷，但因被告未能有满意的答复，故原告向本院起诉。
　　本院认为：原、被告双方存在医患关系，双方均无异议，应予确认。金华市医学会的金华医鉴[2010]03号医疗损害技术鉴定书客观、合法，应予确认。原告出生时有重度窒息而引起脑瘫，因金华市中医院在接产过程中，对产程观察不严密，未进行阴道检查；对新生儿原告重度窒息的第一时间抢救措施不力，存在医疗过错，与新生儿重度窒息及脑瘫的后果存在因果关系，应承担过错责任，原告家属未能听取医院的嘱咐，延误治疗时间，负有一定的过错。据此，在考量事故等级属二级乙等（三级）伤残、医疗过失行为在损害后果中的责任程度、对人身损害影响等因素，对原告主张要求被告应负事故50％经济损失的诉请，应予支持。原告的护理时间确定为9年，且因完全不能自理，按完全护理标准计算；营养费标准为每天53.94元。鉴于原告今后仍需行运动、言语等康复训练治疗，其后续治疗费以实际产生的费用主张权利。原告的合理合法部分，本院予以支持。依照《中华人民共和国民法通则》第一百一十九条、《中华人民共和国侵权责任法》第十六条、第五十四条、第二十二条等规定，判决如下：
　　一、原告池某慧的经济损失为：医疗费6363.21元、护理费114975元、残疾赔偿金180848元、营养费9709.2元、精神损害抚慰金38000元、交通费531元，合计人民币350422.41元。
　　二、上述按50％计算，由被告金华市中医院赔偿原告池某慧人民币175211.2元，限于本判决生效之日起十日内付清。
　　三、驳回原告池某慧的其他诉讼请求。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　本案受理费430元（已减半收取）、鉴定费3680元，合计人民币4110元（由原告预交），由原告池某慧负担610元，由被告金华市中医院负担3500元（在履行上述款项时，一并支付给原告）。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于浙江省金华市中级人民法院。应在递交上诉状次日起七日内，预交上诉案件受理费860元，款汇至金华市财政局帐户（开户银行：中国农行银行金华市分行营业中心，帐号：xxx），或直接交款至浙江省金华市中级人民法院立案大厅收费处。逾期则按自动撤回上诉处理。
