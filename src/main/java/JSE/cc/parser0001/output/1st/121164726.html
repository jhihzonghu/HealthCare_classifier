涂玉兰与黄天勇医疗损害赔偿纠纷案

审判时间：20110326 法院：河南省南阳市中级人民法院（司法公开示范法院） 案件字号：(2012)南民二终字第182号


　　上诉人（原审原告）涂玉兰。
　　委托代理人杨德有。
　　委托代理人吉春林，河南匡世律师事务所律师。
　　被上诉人（原审原告）黄天勇。
　　委托代理人杨瑞堂，河南海涛律师事务所律师。
　　上诉人涂玉兰与被上诉人黄天勇为医疗损害赔偿纠纷一案，唐河县人民法院于2011年7月20日作出（2011）唐昝民初字第85号民事判决，涂玉兰不服，向本院提起上诉。本院于2012年2月13日立案受理后，依法组成合议庭对本案进行了审理，经阅卷、询问上诉人涂玉兰及其委托代理人杨德有、吉春林，被上诉人黄天勇及其委托代理人杨瑞堂，本案现已审理终结。
　　原审查明，2000年8月16日，原告因患痔疮在被告个体诊所就诊治疗，被告给予青霉素等输水用药治疗，并作了皮试，当天无不良情况发生。第二天，原告又到被告处继续输水治疗，输水过程中，原告突然浑身发抖，进而休克。被告急用抗过敏药物给予注射，并紧急送入唐河县人民医院抢救，在唐河县人民医院被确诊为脑梗塞发病，同时还患有痔疮、肺炎、三期高血压。在县医院治疗后，原告因脑血栓遗留下不完全性混合性失语后遗症。后原告方要求被告赔偿无果，遂长期形成争执，于2010年经南阳市医学会（2010）038号医疗事故鉴定书认定，原告现状况原因为脑梗塞后遗症引起，医方用青霉素治疗痔疮感染符合原则，突发脑梗塞与原告长期患高血压有关，与医方医疗行为无直接因果关系。结论为该病例不构成医疗事故。经咨询执医人员告知，脑梗塞首先要有血栓存在，而血栓难以在短时间内形成，形成后的血栓常附着在血管壁上，而输水时药液的冲击会导致血栓脱落，脱落后的血栓进入细血管即猝发血栓症状。现医疗诊断中，血栓需利用专门仪器进行针对性检查才能查明，一般检查中无法查出。
　　原审认为，根据原告的病历及鉴定，原告是猝发脑梗塞患病，原因是其体内存在可致病血栓进入脑细血管形成，非针对性诊疗中对血栓无法查明。该脑梗塞的形成非医疗行为造成，被告在诊疗中的医疗行为与原告患脑血栓之间亦无因果关系。根据《侵权责任法》第57条规定，“医务人员在诊疗活动中未尽到与当时的医疗水平相应的诊疗义务，造成患者损害的，医疗机构应当承担赔偿责任”。因现常规检查中的医疗水平，血栓仍无法查明并有效预防，故被告在该诊疗中并没有过错，不应承担责任。
　　原审判决：依据《中华人民共和国侵权责任法》第五十七条之规定，判决，驳回原告涂玉兰对被告黄天勇的诉讼请求。案件受理费825元由原告涂玉兰负担。
　　涂玉兰不服原审判决，向本院上诉称，被上诉人在为上诉人检查诊断时没有进行全面、认真检查，致上诉人潜在病情因缺少检查而无法预见并加以防范，造成上诉人残疾，因被上诉人具有未认真、全面检查，且未能对上诉人履行告知义务，故被上诉人有一定的过错责任。请求二审法院撤销原判，依法改判被上诉人赔偿上诉人伤残补助费、误工费、护理费等项费用40999元。
　　黄天勇答辩称，1、涂玉兰的身体伤害与我方的诊疗行为无关，且县医院的诊断证明、南阳医学会的医疗事故鉴定都认定的很清楚，涂玉兰的病情是脑血栓、高血压引起，随时可能发生。2、人身损害赔偿的诉讼时效是一年，本案的损害发生已过去12年，超诉讼时效。请求二审法院依法驳回上诉人诉请，维持原判。
　　二审中，各方均未提交新证据。
　　本院二审查明的事实与一审认定事实相一致。
　　本院认为，在医疗损害赔偿纠纷案件中，医方对患方承担赔偿责任的必要条件为医方在诊疗过程中存在过错且该过错与损害结果之间存在因果关系。侵权法上判断行为人的行为是否构成过错，系从行为人是否尽到其应尽之注意义务来考察。就医疗机构的医疗行为而言，判断其是否尽到了注意义务，一方面应从其是否尽到了法律和规章所规定的医疗机构的各项注意义务，另一方面也应考察该医疗机构的医疗行为是否达到了与其资质相应的医疗水准、是否尽到了符合其相应专业要求的注意、学识及技能。在该起医疗纠纷中，根据当时的医疗条件及检查水平，血栓在一般检查中确无法查出，黄天勇的诊疗行为并无过错，且南阳市医学会鉴定结论已明确阐明涂玉兰突发脑梗塞与医方医疗行为并无直接因果关系，故涂玉兰请求黄天勇承担赔偿责任的上诉理由不能成立，本院不予支持。综上所述，原审查明事实清楚，适用法律正确，应当予以维持。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费825元，由上诉人涂玉兰负担。
　　本判决为终审判决。

　　审判长陈鉴
　　审判员陈德林
　　审判员郭林慧
　　二〇一一年三月二十六日
　　书记员李路明
