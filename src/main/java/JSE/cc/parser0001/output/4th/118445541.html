彭甲诉上海市第一人民医院宝山分院医疗损害责任纠纷案

审判时间：20111009 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第1715号


　　上诉人(原审原告)彭甲。
　　被上诉人(原审被告)上海市第一人民医院宝山分院。
　　法定代表人杨某某，院长。
　　委托代理人张翀，上海市康正律师事务所律师。
　　原审原告程甲。
　　上诉人彭甲因医疗损害责任纠纷一案，不服上海市宝山区人民法院(2009)宝民一(民)初字第4349号民事判决，向本院提起上诉。本院依法组成合议庭,公开开庭审理了本案。上诉人彭甲、被上诉人上海市第一人民医院宝山分院的委托代理人王翀到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，一、原告彭甲、程甲系兄妹，患者程乙系两原告的母亲，程乙的丈夫即两原告的父亲彭乙已先于程乙过世。二、2008年5月6日，患者程乙因“发现高血压升高10年，伴头晕、恶心呕吐2天”入住被告处干部病房。据住院病案记载：患者有高血压病史10年，期间间断服降压药；精神分裂症30年，长期服用精神类药物治疗；2年前右股骨颈骨折，2月前左股骨粗隆间骨折，均未手术治疗，小便留置导尿管。入院后查体：体温37℃，脉搏70次/分，呼吸20次/分，血压140/90mmHg。神志清楚，对答切题，推入病房，查体合作，皮肤巩膜无黄染，浅表淋巴结未及肿大，两肺呼吸音粗，未闻及明显干湿罗音，心律齐，无杂音。腹平软，肝脾未及。左侧臀部一大小约3×2cmⅡ°褥疮，皮肤已破溃。左下肢外旋外展位，右下肢肌力正常，双下肢无肿，病理征阴性。留置导尿中。入院后诊断：①高血压病；②左股骨粗隆间骨折；③精神分裂症；④带入褥疮。入院后予降血压、活血化瘀、胃肠动力、支持、抗精神分裂等药物治疗。5月7日床旁胸片检查示：两肺纹理增多，心影增大，主动脉硬化。5月8日头颅CT示：两侧脑室旁腔隙灶；脑萎缩。5月11日补充诊断：高尿酸血症；低蛋白血症。5月13日胸部CT检查报告：两侧胸廓入口气管旁软组织占位病变；甲状腺病变可能；两肺间质增生；心影增大，心包积液，主动脉、冠状动脉硬化；附见肝脏低密度灶。5月17日患者因留置导尿，尿液渗漏致尿路感染，予拔除，同时头孢唑肟抗感染治疗。5月21日起患者最高发热38℃。5月23日患者仍有反复发热。查体：神志欠清，两肺呼吸音粗，未闻及干湿罗音。予换用阿莫西林克拉维酸钾治疗。治疗至6月4日患者仍反复发热。同日摄胸片后提示：慢支伴两肺感染；心影明显增大，主动脉硬化。6月6日起改用莫西沙星抗感染治疗。6月18日下午起患者神志欠清，精神萎，略气促。查体：颜面及颈部、左下肢浮肿明显，双上肢及右下肢尚可，双肺呼吸音粗，双下肺闻及少许湿罗音。呼吸科会诊后诊断：①高血压病；②慢支，肺心病？③慢性Ⅱ型呼衰；④冠心病？心衰？⑤上腔静脉综合征？⑥肺栓塞待排？⑦甲状腺占位。建议：加强抗感染(根据痰培养及药敏结果，调整抗生素)，呼吸机辅助通气，扩张支气管，改善心功能，加强利尿，保持内环境稳定等治疗措施。同日心内科会诊后结合血气分析考虑Ⅱ型呼吸衰竭，建议加强抗感染治疗，辅助通气。6月19日行右颈深静脉穿刺。6月21日患者诉腹痛。查体：神欠清，腹软，无压痛、反跳痛，肠鸣音不亢，双下肢轻度浮肿。检查大便隐血阳性。给予奥美拉唑抑酸治疗。6月22日超声检查提示：双侧胸腔积液；腹腔积液。6月23日患者神志欠清，全身浮肿。复查胸片提示：慢支；两肺感染；心影增大；两侧胸腔积液。同日呼吸科会诊“肺栓塞”不能除外。查凝血功能正常，建议予低分子量肝素钠抗凝治疗及胸腔穿刺抽液治疗。6月28日患者解黑色糊状大便数次。查大便隐血阳性，予止血、抗酸药物治疗。6月29日晨又有反复解黑便。检查血红蛋白低下。停用低分子量肝素钠，予输血治疗。7月5日患者神志不清，无创呼吸机辅助呼吸中，口唇不绀，双肺呼吸音低，少许湿罗音。血压120/70mmHg，心率76次/分。7月7日根据患者病情，考虑已出现多功能脏器衰竭。7月8日19时56分患者即刻出现呼之不应，血压测不出，心率为零，无呼吸，大动脉搏动未及，瞳孔散大，即予心外按压，气管插管，静脉滴注多巴胺、肾上腺素、可拉明等药物急救。21时20分心电图仍示一直线，经救治无效宣告临床死亡。患者程乙在被告处住院治疗期间，自费支付医疗费1,810.42元。另，为诉讼需要，原告支付律师费11,000元。三、2008年5月7日，患者浦小妹入住被告处，与患者程乙住在同一病房。其门诊诊断：发热待查：左下肺炎？其入院诊断为二肺肺炎，低白蛋白血症，电解质紊乱(低钾、低钠血症)，脂代谢紊乱，右侧甲状腺结节。5月9日，患者浦小妹被转往血液科病房。原告认为被告医疗行为存在过错，起诉要求被告赔偿医疗费人民币(以下均为人民币)1,810.42元、陪护费8,198元(按照上海市2010年职工平均工资46,757元/年计算64天)、住院伙食补助费1,344元(按21元/天计算64天)、交通费750元(按15元/天计算50天)、丧葬费23,378.50元(按照上海市2010年职工平均工资46,757元/年计算6个月)、精神损害抚慰金139,200元(按照2010年上海市城镇居民平均生活费支出23,200元/年计算6年)、律师代理费11,000元，上述赔偿项目金额共计185,680.92元，要求被告承担70%即129,976.64元。
　　原审审理中，法院依法委托上海市静安区医学会就被告对患者程乙的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，其医疗行为与患者的死亡之间有无因果关系，是否属于医疗事故进行鉴定，上海市静安区医学会出具医疗事故技术鉴定书，分析意见为：1、根据医患双方提供的材料、病史资料及现场询问，医方对患者的诊断明确。2、患者为长期卧床、高龄老人，且患有低蛋白血症、褥疮、尿路感染等多种基础疾病，是获得型肺炎的高危人群，在患者肺部感染后，医方给予的抗感染治疗符合诊疗常规。3、根据患者临床症状与辅助检查“肺栓塞”诊断不能排除，医方使用“低分子量肝素钠”有治疗指征。患者出现上消化道出血，为其全身疾病恶化造成的急性胃黏膜损害可能性大。4、医方存在以下不足：①医方未尽全面的书面告知义务。②病史及护理记录有欠缺。但以上不足与患者死亡无因果关系。其结论为患者程乙与被告医疗争议不构成医疗事故。鉴定费3,500元由被告支付。
　　因原告对上海市静安区医学会鉴定结论有异议，向本院申请由上海市医学会进行再次鉴定，本院依法委托上海市医学会就被告对患者程乙的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，其医疗行为与患者的死亡之间有无因果关系，是否属于医疗事故进行再次鉴定。2011年2月15日，上海市医学会医疗事故技术鉴定工作办公室组织程乙(代理人彭甲)与上海市第一人民医院宝山分院医疗争议医患双方举行随机抽取鉴定专家仪式，双方被告知“本次鉴定主要学科专业组为呼吸内科专业、消化内科专业学科，次要学科专业组为药学学科…….”，患方代表彭甲、医方代表柏琴和医学会代表均签字确认。2011年3月22日，上海市医学会作出医疗事故技术鉴定书，分析意见为：1、该老年患者有多种基础性疾病(高血压肺心病、精神分裂症、高尿酸血症、低蛋白血症、双股骨骨折等)，在治疗过程中会存在一定矛盾之处，医方履行了相关病情告知，对患者的总体处理符合医疗常规。2、患者长期卧床且反复使用多种抗生素，住院期间发生肺部感染为难以完全避免。同时邻床患者(浦小妹)搬离病房后10余天才出现发热症状，故判断本病例肺炎系邻床患者交叉感染的依据不足。3、本身具栓塞高危因素的患者住院过程中出现不明原因低氧血症，D-二聚体检测阳性，故“肺栓塞”的诊断不能完全排除，使用低分子量肝素钠抗凝治疗有指征。且抗凝治疗前检查凝血功能正常。医方在患者出现消化道出血时停用低分子量肝素钠尚不够及时，但未加剧患者病情变化。4、患者有褥疮和顽固性尿路感染，抗菌素药物使用有指征；其中头孢类药物(头孢哌酮钠舒巴坦纳)，常规治疗可以达到每日6g，因此该患者非超常规用药。5、患者的消化道出血可以由多重因素引起(发生消化道缺氧性病变及应激性溃疡概率为大)，医方使用止血药物和质子泵抑制剂治疗后消化道出血停止(胃管内无出血，也无持续性黑便)，故患者的死亡与消化道出血没有因果关系。6、患者最终的死亡根本原因为老龄慢性基础疾病下多种感染引起的多脏器功能衰竭的结果。其结论为患者程乙与被告医疗争议不构成医疗事故。对该鉴定结论，被告无异议，原告对该鉴定结论不予认可并提出质疑：1、医方在患者入院时未履行医疗风险告知义务，剥夺了患方的知情权与选择权。2、根据《老年人肺部感染125例临床分析》一文，老年人肺炎起病隐匿、症状不典型，敏感性降低，自觉症状轻微，体温调节能力低，发热不如年轻人明显；被告违反《医院工作制度》规定从5月8日至20日每天只测1次体温可能导致漏测发热；由于治疗中被告给患者注射抗生素，大大抑制了患者的发热，故鉴定机构仅凭“患者在邻床搬离病房10余天才出现发热症状”不能排除交叉感染作用的可能。3、被告主任医生从5月9日至6月17日连续40天没有查房记录、患者自5月21日起反复发热但被告直到13天后的6月3日才安排拍摄胸片、在干部病房违规同室收治浦小妹的行为均不符合医疗常规。4、在患者发热不退将近一个月经治疗无效的情况下被告却始终不进行会诊直至患者因呼吸衰竭病危，该违规行为延误了患者病情治疗。5、被告在患者出现上消化道大出血后继续注射低分子量肝素钠3支具有医疗过错，属于违反禁忌用药。鉴定机构认定停用不够及时“但未加剧患者病情恶化”的结论没有依据。6、被告对患者程乙使用的头孢哌酮钠舒巴坦钠常规治疗每日达6g且被告未对老年患者用药特殊对待，被告存在超量用药行为。7、老年患者上消化道大出血但不会导致其多器官功能衰竭，故患者因多器官功能衰竭而死亡与其上消化道大出血无关系。8、根据《医疗事故处理条例》及《医疗事故技术鉴定暂行办法》的相关规定，专家鉴定组人数为单数，涉及的主要学科的专家一般不得少于鉴定组成员的二分之一，本案专家鉴定组5名专家中，只有2名呼吸内科专业专家和2名消化内科专业专家，1名药学专业专家，不符合“其中主要学科专业的专家不得少于专家鉴定成员的二分之一”的规定。根据《医疗事故技术鉴定暂行办法》第三十九条规定应予以重新鉴定。根据原告的质疑，本院致函上海市医学会要求发表相关补充意见，2011年5月6日，上海市医学会医疗事故技术鉴定工作办公室出具了《关于程乙与市一宝山分院医疗纠纷一案鉴定补充事项的复函》，称：1、患者高龄，患低蛋白血症，双侧股骨骨折长期卧床，并有多种慢性疾病(包括循环、呼吸系统)的基础，自身就可以发生肺部感染。在邻床搬离病房后十余天后才出现发热症状，患者有肺炎典型的临床表现，不属于症状隐匿不典型类型。因此与邻床交叉感染的依据不足。2、患者浦小妹双肺炎症，不属于法定传染病，入院时血压90/60mmHg，收治入院有指征。作为干部病房在事发时并无规定不应收治普通患者。3、医方使用低分子量肝素钠治疗后，患者低氧血症病情改善。但当出现消化道出血时，医方未能立即停止该药的长期医嘱。若确实必须继续使用，也应在严密观察下以临时医嘱方式给药为妥。该药物注射后最大效应时间为3-5小时，半衰期为3-6小时，持续时间为24小时。观察1-2天内有无消化道出血明显即可判明。在此期间医方给予抗酸、止血、支持治疗后，未见消化道出血加重症状临床表现，故没有加剧患者病情变化。4、患者自身有心血管系统(高血压病、动脉粥样硬化，慢性心功能不全)、泌尿系统(尿失禁、尿路感染、肾功能不全)、呼吸系统(胸腔积液、肺部感染、肺栓塞，呼吸衰竭)、神经系统(精神分裂症、腔隙性脑梗)以及褥疮感染，双股骨骨折、电解质紊乱，低蛋白血症等诸多病症。故最终死亡为其基础上的多重感染，引发多脏器功能衰竭的结果，其死因明确。5、医学会按照国务院颁布的《医疗事故处理条例》及卫生部《医疗事故技术鉴定暂行办法》规定程序，完成鉴定的各项组织工作。本次鉴定学科专业组成，事前已经医患双方签字确认。鉴定费3,500元，由原告彭甲支付。
　　原审法院经审理后认为，当事人对自己提出的诉讼请求所依据的事实有责任提供证据加以证明；没有证据或者证据不足以证明当事人的事实主张的，由负有举证责任的当事人承担不利后果。根据医疗事故鉴定结论，被告对患者程乙进行的诊疗行为符合医疗常规，主观上无过错，其诊疗行为与患者的死亡之间无因果关系，故被告对原告诉称的损害不承担赔偿责任。原告对医疗事故鉴定结论虽持有异议，要求重新鉴定，但并未提供充足证据和充分理由，故本院对其主张不予采信。经审查，本院认为原告该主张亦缺乏事实和法律依据，故不予支持。
　　原审法院据此作出判决：原告彭甲、程甲要求被告上海市第一人民医院宝山分院赔偿医疗费、陪护费、住院伙食补助费、交通费、丧葬费、精神损害抚慰金、律师费合计129,976.64元的诉讼请求不予支持。
　　原审判决后，上诉人彭甲不服，向本院提起上诉称：请求二审法院重新委托上海市医学会进行医疗事故技术鉴定，撤销原审判决，判令被上诉人赔偿132,012.90元。因为：1、鉴定专家组成专业类别不符合规定，鉴定程序违法。被上诉人医疗行为违反医疗技术常规，违规用药、不按照规定护理及记录，暨加重病情又延误发现病情变化。2、鉴定结论明显依据不足。3、一审判决适用法律不当。医疗纠纷中举证责任倒置，医方应该就其“医疗行为与损害结果之间不存在因果关系”及其医疗行为“不存在过错”进行举证，而不是患方举证。4、一审判决查明事实不清。上诉人列举了被上诉人十个违规、违章的行为，均与程乙的死亡之间存在因果关系。被上诉人有过错，应该承担赔偿责任。
　　被上诉人上海市第一人民医院宝山分院辩称：不同意上诉人的上诉请求，同意一审判决。本起医疗事件已经经过两级医学会鉴定，结论均为被上诉人的医疗行为没有过错，符合诊疗常规，不构成医疗事故。
　　原审原告程甲未到庭。
　　本院经审理查明，原审查明事实属实，本院予以确认。
　　本院认为，公民享有生命健康权。在医疗纠纷中，医院的医疗行为存在过失，侵害他人生命健康权的，应当承担相应责任。原审法院根据双方当事人的诉辩、提交的证据对本案事实进行了认定，并在此基础上结合上海市医学会医疗事故技术鉴定书依法作出原审判决，合法合理，理由阐述充分，本院不再赘述。二审中上诉人主张一审法院认定事实不清，医学会的鉴定程序违法、结论不正确，而事实是被上诉人在医疗过程中都存在重大过错，故要求法院委托上海市医学会重新鉴定，并依法改判被上诉人赔偿132,012.90元。但其在二审中没有新的事实与理由，也没有新的证据佐证自己的主张。本案已经经过区、市两级医学会鉴定，上诉人虽对鉴定结论有异议，但未能提供证据证明上海市医学会作出的鉴定结论存在下列情形：(一)鉴定机构或者鉴定人员不具备相关的鉴定资格的；(二)鉴定程序严重违法的；(三)鉴定结论明显依据不足的；(四)经过质证认定不能作为证据使用的其他情形。故本案并不具备进行重新鉴定的条件，对上诉人要求重新鉴定的申请，本院不予准许。一审法院以上海市医学会鉴定结论为依据作出一审判决，合法合理，本院应予维持。本院认可一审法院对事实的分析认定及对相关法律法规的理解与适用，上诉人的上诉请求缺乏事实及法律依据，本院难以支持。综上所述，原审认定事实清楚，判决并无不当。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币2,899元，由上诉人彭甲负担。
　　本判决为终审判决。
