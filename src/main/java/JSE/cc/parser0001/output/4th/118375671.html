颜A等诉A医院等确认合同效力纠纷案

审判时间：20111121 法院：上海市徐汇区人民法院 案件字号：(2011)徐民一(民)初字第5227号


　　原告颜A。
　　原告颜B。
　　原告颜C。
　　以上三原告委托代理人王X，上海X师事务所律师。
　　以上三原告委托代理人涂X，上海X师事务所律师。
　　被告A医院。
　　法定代表人王X，院长。
　　委托代理人李X，A医院员工。
　　委托代理人朱X，A医院员工。
　　被告颜D。
　　委托代理人刘X，湖南XX律师事务所上海分所律师。
　　原告颜A、颜B、颜C诉被告A医院（以下简称A医院）、颜D确认合同效力纠纷一案，本院于2011年8月3日立案受理后，依法适用普通程序，于9月29日公开开庭进行了审理。原告颜A、三原告的委托代理人王X，被告A医院的委托代理人李X、朱X，被告颜D的委托代理人刘X到庭参加诉讼。本案现已审理终结。
　　原告颜A、颜B、颜C诉称，三原告系章X之子。2011年1月21日晚，章X因腰部疼痛至被告A医院就诊，由于被告A医院的医务人员在值班时擅离职守、接诊时情绪烦躁，违反诊疗常规，在未向患xxx病史和过敏史的情况下，为章X开具了“锋珠新”、“黄体酮”等药物。被告医生未按药物使用要求对患者进行皮肤试验，也未在输液时密切看护并采取预防休克的急救药品与措施，导致章X在输液过程中发生严重过敏休克。被告医生直到章X发生过敏症状近两个小时之后才发现问题。之后被告医生未及时向领导报告并接受上一级医师的医疗指导或者现场抢救，延误了抢救时机。被告医生存在伪造、篡改病史的行为。1月22日6时，章X因抢救无效死亡。1月23日章X在闵行区殡仪馆火化。1月24日，被告A医院利用被告颜D不了解医疗事故处理程序及相关的法律规定，与被告颜D签订了协议书。三原告对该协议书始终不同意。三原告认为该协议书的签订未经过三原告授权，属于无权代理；协议书内容存在违反法律、行政法规强制性规定的内容；赔偿额180,000元是被告颜D被被告A医院误导，显失公平，原告因此起诉要求认定两被告签订的关于章X因输液过敏休克死亡事故处理的协议书无效。
　　被告A医院辩称，两被告签订的协议是合法有效的，被告颜D是章X的配偶，是第一顺序继承人，被告A医院与其签订协议是合法的，且在协商过程中，三原告都在场并了解协议内容。该协议于2011年1月26日已经履行完毕，被告A医院已向被告颜D支付了180,000元。被告认为本案不足以构成医疗事故，要求驳回原告诉请。
　　被告颜D辩称，同意确认其与被告A医院签订的协议无效，愿意退还已收取的180,000元。
　　经审理查明，原告颜A、颜B、颜C系章X儿子，被告颜D系章X丈夫。章X于2011年1月21日至被告A医院就诊，章X因输液引起药物性休克，经抢救无效于1月22日死亡。两被告在2011年1月24日签署协议书，协议书载明：甲方：A医院，乙方（章X家属代表）（委托代理人颜D），协议主要内容为：一、乙方要求甲方一次性补偿人民币壹拾捌万元整，甲方表示同意；二、甲方要求乙方启封已封存的病史，乙方表示同意；三、乙方得到以上一次性补偿后本医疗事件之处理即告终结，乙方不再以任何理由或方式追究甲方任何责任；四、双方无其他争议。
　　上述事实，除原、被告当庭陈述外，另有原告提供的XX市公安局XX镇派出所出具的关系证明、居民死亡医学证明书、协议书、病史资料、收费单、药物说明、上海市XX区卫生局答复、上海市卫生局答复等证据予以证实，并经当庭质证无疑，本院予以确认。
　　本院认为，本案焦点在于被告A医院、颜D于2011年1月24日签订的协议书是否存在无效或可撤销的法定事由。涉案医疗纠纷章X因故死亡后，被告A医院如欲与章X家属达成医患间的和解协议，不论从法律规定，还是从社会习俗，均应与章X的第一顺序继承人，即本案三原告及被告颜D进行协商解决，在审理期间，被告既未能举证证实三原告曾授权被告颜D与被告A医院协商涉案医疗纠纷，亦未能证实三原告在知晓上述协议后予以追认，故本院认为两被告在未与三原告协商一致的前提下，于2011年1月24日订立的协议书侵害了三原告的合法权益，依照相关法律规定应认定为无效，被告颜D已收取的180,000元应当返还被告A医院。依照《中华人民共和国合同法》第七条、第五十二条第五项、第五十八条的规定，判决如下：
　　一、被告A医院与被告颜D于2011年1月24日签订的协议书无效；
　　二、被告颜D返还被告A医院180,000元。
　　案件受理费3,900元，由A医院、颜D各半负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
审判长高霄雷
　　代理审判员陈怡
　　人民陪审员李雅萍
　　二〇一一年十一月二十一日
　　书记员邵益萍
