原告叶柯青、张龙文、韩小花、张天翼、张某某为与被告洛宁县妇幼保健院医疗损害赔偿纠纷案

审判时间：20110823 法院：河南省洛宁县人民法院 案件字号：(2010)宁民初字第136号


　　原告：叶柯青。
　　原告：张龙文。
　　委托代理人：孙海民，特别授权。
　　原告：韩小花。
　　三原告共同委托代理人：王志勇，一般代理。
　　原告：张天翼。
　　法定代理人：叶柯青，系张天翼父亲。
　　原告：张某某。
　　法定代理人：叶柯青，系张某某父亲。
　　被告：洛宁县妇幼保健院。
　　法定代表人：吴花玲，院长。
　　委托代理人：董向明，特别授权。
　　委托代理人：曹万章。
　　原告叶柯青、张龙文、韩小花、张天翼、张某某为与被告洛宁县妇幼保健院医疗损害赔偿纠纷一案，于2010年10月28日向本院起诉，同日本院作出受理决定，并向原告送达了受理通知书、举证通知书。2010年10月29日向被告洛宁县妇幼保健院送达了起诉状副本、应诉通知书、举证通知书。2010年11月19日原告张龙文等向本院提出鉴定申请，要求对被告的医疗行为是否存在过错，该医疗行为与张某某1死亡有无因果关系进行鉴定。本院依法委托北京法源司法科学证据鉴定中心进行鉴定，2011年5月29日该鉴定中心作出（京）法源司鉴［2011］临鉴字第30号法医学鉴定意见书。本院依法组成合议庭，由审判员高东亚担任审判长并主审，审判员刘胜贤、代审判员张少波参加评议，于2011年7月6日公开开庭进行了审理。原告张龙文及其委托代理人孙海民，原告叶柯青、张龙文、韩小花的委托代理人王志勇，被告洛宁县妇幼保健院的委托代理人董向明、曹万章到庭参加了诉讼。本案现已审理终结。
　　五原告诉称：张某某12010年9月11日到被告洛宁县妇幼保健院分娩，当天剖腹生产一男一女双胞胎（即原告张某某、张天翼），分娩后一切正常。9月15日，医生牛京霞开具2支针剂给张某某1静脉注射后，张某某1出现浑身抽搐、呼吸困难、大小便失禁等不良反应，经医院抢救不见好转即送往150医院，在送往洛阳150医院途中死亡。牛京霞不具备医师资格行医，在对张某某1诊疗过程中存在过错，与张某某1死亡有一定因果关系，已被（京）法源司鉴［2011］临鉴字第30号法医学鉴定意见书认定。为此，请求被告赔偿原告丧葬费13678.50元、死亡赔偿金88379.68元（110474.60元×80%）、被扶养人生活费95737.46元［其中张龙文29457.68（36822.10×80%）元、张天翼33139.89元、张某某33139.89元］、精神抚慰金30000元、新生儿护理费40876.35元、营养费31390元、交通费2000元、鉴定费11000元、诉讼费1393元，上述共计314454.99元，扣除已赔付的80000元，被告应再赔234454.99元。
　　被告辩称：张某某1到被告洛宁县妇幼保健院住院经剖腹产一对龙凤胎，此后住院期间并发其它疾病，本院对症采取了治疗措施。2010年9月10日晚9时20分许，张某某1出现危重病情，经全力抢救未见好转，经亲属同意转上级医院救治，后在洛阳解放军150医院病逝。现在已经司法鉴定，本院尊重鉴定结论，愿承担与过错程度相应的责任，鉴定认为本院过错参与度为C级，理论系数值25%，即承担责任不超过25%。原告要求赔偿项目及数额有些于法无据，如抚养费的计算、新生儿护理费、营养费、精神抚慰金过高等，本院不予认可。总之，被告愿根据过错参与度承担相应责任，原告要求的赔偿项目及数额请求法院合理合法认定，被告超付应承担责任的部分，原告应当返还。
　　审理查明：2010年9月11日，五原告的亲属张某某1（系原告叶柯青妻子，原告张龙文、韩小花女儿）到被告洛宁县妇幼保健院待产，当天剖腹产生下一对龙凤胎，即原告张天翼、张某某，后张某某1在该院住院治疗。2011年9月15日晚9点20分许，张某某1出现浑身抽搐，呼吸困难等不良反应，经洛宁县妇幼保健院组织抢救不见好转，即转往洛阳解放军150医院抢救，后经150医院检查张某某1无生命指征已死亡。此后，原、被告为张某某1死亡原因及赔偿等问题发生激烈争执，经中间人秦小平说合，双方于2010年9月30日达成医疗纠纷处理协议书，由被告向原告垫付8万元，如果原告提起诉讼，最终赔偿数额以诉讼结果为准，多退少补。2010年10月28日，原告诉讼到法院。
　　本院审理期间，原告张龙文于2010年11月19日向本院提出鉴定申请，要求对被告的诊疗行为是否违反法律、法规、规章及相关诊疗规范规定，有无漏诊、误诊、处理不当，护理是否规范，肺栓塞结论是否准确等进行鉴定，本院根据原告申请，依法委托北京法源司法科学证据鉴定中心，就“洛宁县妇幼保健院对患者张某某1的治疗过程中是否存在过错（医疗行为是否存在过错），该医疗行为与张某某1的死亡之间是否存在因果关系”进行鉴定。2011年5月29日，该鉴定中心作出（京）法源司鉴［2011］临鉴字第30号法医学鉴定意见书，该鉴定认为：洛宁县妇幼保健院在被鉴定人入院的诊断及实施剖腹产符合诊疗常规。由于缺乏被鉴定人的产前检查材料，对于医院更好地了解其心脏病情有一定不利影响。但由于洛宁县妇幼保健院属专科医院，在心血管专科疾病的诊疗方面可能存在医疗水准的影响，在对于被鉴定人心脏疾病的诊疗方面未能及时认识、诊断和对症治疗其心肌缺血的病情，故医院的医疗行为存在过错，与其最终的死亡结果之间有一定的因果关系，法医学参与度为C级。鉴定意见是：1、洛宁县妇幼保健在对被鉴定人张某某1的诊疗过程中存在医疗过错，2、洛宁县妇幼保健院的诊疗行为与被鉴定人张某某1的死亡结果有一定的因果关系。原告支付鉴定费8000元，交通费1300元，住宿费200元。
　　另查明，河南省2010年度农村居民人均年收入为5523.73元，农村居民人均年生活消费支出为3682.21元，全省在岗职工平均工资为27357元。
　　本院认为：原告的亲属张某某1在被告洛宁县妇幼保健院剖腹产双胞胎后住院治疗期间，病情出现变化经抢救无效后死亡，经有关机构鉴定，被告洛宁县妇幼保健院对张某某1的诊疗存在医疗过错，被告的诊疗行为与张某某1的死亡结果有一定的因果关系，故原告要求被告赔偿丧葬费、死亡赔偿金（含被扶养人生活费）、精神抚慰金、交通费、住宿费等诉讼请求，本院依法予以支持。根据鉴定人意见即被告洛宁县妇幼保健院法医学参与度为C级，理论系数值25%参与度参考范围20-40%的鉴定结论，结合本案具体情况，被告对原告的赔偿责任本院确定为40%，即原告请求赔偿的丧葬费13678.50元、死亡赔偿金110474.60元、被扶养人生活费66279.78元（张天翼、张某某各为33139.89元）、交通费1300元、住宿费200元，共191932.88元，由被告承担40%，即76773.15元。原告要求赔偿精神抚慰金30000元，符合有关法律规定，本院予以支持。被告已付的80000元应予以扣除，原告张龙文要求赔偿其扶养费29457.68元，不符合相关法律规定，不予支持。原告张天翼、张某某要求赔偿护理费40876.35元、营养费31390元，没有法律依据，不予支持。根据《中华人民共和国侵权责任法》第十六条、第十八条、第二十二条、第五十四条，《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》第二十二条、第二十七条、第二十八条、第二十九条和《中华人民共和国民事诉讼法》第六十四条第一款的规定，经合议庭评议，判决如下：
　　一、被告洛宁县妇幼保健院赔偿原告叶柯青、张龙文、韩小花、张天翼、张某某丧葬费、死亡赔偿金（含被扶养人生活费）、交通费、住宿费共76773.15元。
　　二、被告洛宁县妇幼保健院赔偿原告叶柯青、张龙文、韩小花、张天翼、张某某精神损害抚慰金30000元。
　　三、驳回原告叶柯青、张龙文、韩小花、张天翼、张某某的其它诉讼请求。
　　上述一、二项扣除被告洛宁县妇幼保健院已付的80000元，应再付26773.15元，在本判决生效起十日内一次付清。
　　如未按本判决指定的期限履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条的规定，加倍支付迟延履行期间的债务利息。
　　案件受理费1672元，鉴定费8000元，共计9672元，原告叶柯青、张龙文、韩小花、张天翼、张某某负担4672元，被告洛宁县妇幼保健院负担5000元。
　　如不服本判决，可在判决书送达之日起十五日内，通过本院或直接向洛阳市中级人民法院提出上诉。书面上诉的，应提交上诉状正本一份，副本六份。

　　审判长：高东亚
　　审判员：刘胜贤
　　代审判员：张少波
　　二〇一一年八月二十三日
　　书记员：周晓瑞
