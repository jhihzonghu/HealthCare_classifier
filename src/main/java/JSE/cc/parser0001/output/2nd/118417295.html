刘甲与上海市杨浦区中心医院医疗损害责任纠纷上诉案

审判时间：20110621 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第906号


　　上诉人(原审原告)刘甲。
　　委托代理人孙乐民，北京盈科(上海)律师事务所律师。
　　委托代理人于某某。
　　被上诉人(原审被告)上海市杨浦区中心医院。
　　法定代表人李某某，院长
　　委托代理人雷某，该院职工。
　　委托代理人徐美琴，上海市康正律师事务所律师。
　　上诉人刘甲因医疗损害责任纠纷一案，不服上海市杨浦区人民法院(2010)杨民一(民)初字第6169号民事判决，向本院提起上诉。本院依法组成合议庭,公开开庭审理了本案。上诉人刘甲及其委托代理人孙乐民、于某某，被上诉人上海市杨浦区中心医院(以下简称杨浦医院)的委托代理人雷某、徐美琴到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，依据(2010)沪杨证字第179号公证书，患者刘乙的父母及配偶均已去世，其生前与配偶育有子女二人，一子为刘丙，一女即刘甲。患者因“言语不清、右侧肢体无力3天”于2009年11月19日入住杨浦医院。该患者既往史有高血压。查体：血压130/70mmHg，心率78次/分等。头颅CT提示：“两侧基底节区、半卵圆区腔隙性梗塞”。心电图检查：心房颤动。故诊断患者为脑梗死；高血压病(极高危组)；心律失常，心房颤动；慢性支气管炎急性发作？杨浦医院予以脱水降颅压、营养神经、法安明抗凝、平喘(地塞米松)等对症处理。11月22日查体：双肺呼吸音粗，可闻及少量干湿啰音。考虑肺部感染。当日至11月27日给予头孢呋辛1.5g静脉点滴，每日两次。11月25日血常规检查：白细胞8.6×109/L，中性粒细胞65.5%。11月30日患者早晨呕吐一次，非喷射状呕吐，呕吐为胃内容物。检查：神志清。双侧瞳孔等大等圆，光反射灵敏，颈软，双肺呼吸音粗，可闻及少量哮鸣音。血压120/80mmHg。12月6日患者大便干结，并解小便不畅。予以开塞露辅助通大便，并予加用特拉唑嗪对前列腺增生对症处理。12月7日10时00分患者神志清楚，精神较前略有好转。查体：轻度不完全性运动性失语，双侧瞳孔等大等圆，直径约2.5mm，光反射灵敏，双肺呼吸音粗，未闻及明显干湿啰音。杨浦医院遂予以推拿针灸电针等康复治疗。14时00分病程录记载：患者仍小便不畅，予以加用非那雄胺对症处理。17时50分患者突发大汗，脸色发白，呼之不应。查体：昏迷。心脏听诊未闻及心音，血压测不出。立即予以心肺复苏、气管插管、胸外按压，辅助通气及药物对症等处理。予以相关辅助实验室检查，血常规白细胞19.2×109/L，中性粒细胞76.4%，嗜酸性细胞0.2%。血气分析PH6.984，PCO29.50Kpa，PO26.0Kpa。葡萄糖18.5mmol/L。肌酐127umo/L，乳酸脱氢酶251iu/L，肌红蛋白112ng/ml。凝血酶原时间15.2秒，凝血时间13.2秒，纤维蛋白原6.6g/L，INR1.26。D-二聚体0.6mg/L(0.0-0.3)。经抢救无效，心电图提示室性自主心律。于12月8日9时宣告临床死亡。本病例于2009年12月8日在第二军医大学病理教研室行尸检。结论为患者死亡原因系两肺支气管肺炎伴肺出血。
　　另查，在审理过程中，经法院委托上海市黄浦区医学会医疗事故技术鉴定中心对杨浦医院与患者刘乙之间的医疗行为是否构成医疗事故等进行鉴定。分析意见为：1、患者刘乙基本诊断明确为脑梗死、高血压病、房颤、慢性支气管炎急性发作等，治疗过程未发现违反常规的过失。2、患者死亡与慢性支气管炎急性发作使用抗生素与否无关；住院期间三次血常规正常，一次胸片无渗出，尸检支气管腔内无渗出与痰液，12月7日血常规白细胞增高显示应激状态下采样。3、脑梗死的诊断与治疗，法安明的使用等方面不存在违规。法安明使用期间有关的指标也正常。法安明其半衰期短，在死亡前早已停用，与尸检中出血现象无关联。4、尸检的报告无特异性，因为经历过复苏后，病理报告中的广泛出、淤血，可视为复苏后的表现。5、12月7日17点50分出现的突发大汗、面色苍白、心跳停止等提示死亡原因可能为心源性。故鉴定结论为不构成医疗事故。刘甲对该份鉴定报告的鉴定结论提出异议，申请再次鉴定，法院遂于2010年9月26日委托上海市医学会医疗事故技术鉴定中心对杨浦医院与患者刘乙之间的医疗行为是否构成医疗事故等进行鉴定，该机构于2010年11月16日出具鉴定报告，分析意见为：1、患者刘乙因腔隙性脑梗塞，高血压病(极高危组)，房颤及疑慢性支气管炎急性发作被收住被告医院。入院后予以脱水降颅压、营养神经、法安明抗凝、平喘等处理，符合诊治规范。诊治期间患者突发意外，出现大汗淋漓，脸色发白，呼之不应。经心肺复苏救治无效死亡。根据送鉴材料及鉴定专家询问双方当事人，本案患者有慢阻肺、脑血管等疾病基础，心脏亦存在冠心病、房颤等病史。结合尸检病理学报告患xxx病共同作用所致。其发病之急骤与凶险，医方予以心肺复苏抢救，符合规范。尸检报告肺出血，系复苏情况下所致，现医学技术条件难以避免。2、患xxx病史，入院期间无发热，经观察至第三天予以抗生素治疗5天，并适时地观察血常规(白细胞及分类)无异常后停药，诊疗措施符合诊疗规范。3、患者系腔隙性脑梗塞、房颤，尸检报告多处血栓形成，法安明使用有其指征，医方在诊治期间及时观察出凝血指标，亦无出凝血时间延迟等征象，符合抗凝用药规范。综上所述，患者刘乙与杨浦医院医疗争议不构成医疗事故。
　　在审理过程中，刘甲要求上海市医学会医疗事故技术鉴定中心就该机构出具的鉴定报告中的有关问题进行书面质证，该机构于2011年1月12日出具了鉴定专家的答复，内容主要为：1、2009年12月7日患者刘乙病情突然恶化，被告予以胸外心脏按摩，气管插管辅助通气等心肺复苏措施救治，均会出现肺出血等后续改变，因此推定患者出现的肺出血系复苏后的后续改变，非疾病所致。2、患者的死因结合其既往史、临床病史、尸检报告等因素判断为心肺疾病共同所致。3、杨浦医院曾依据血常规检查给予患者抗炎治疗，但同样依据患者病情变化停用抗炎治疗措施，系处理得当，至于12月7日当日的血常规显示白细胞及中性粒细胞升高，系由机体处于应激状态产生的，并非感染所致。
　　原审法院经审理认为，公民享有生命健康权。公民、法人由于过错侵害他人生命健康权的，应当承担民事责任，但是构成侵权必须符合四个法定要件，即不法行为、损害后果、不法行为与损害后果之间存在因果关系、行为中有过错。参照上海市黄浦医学会医疗事故技术鉴定中心及上海市医学会医疗事故技术鉴定中心出具的医疗事故技术鉴定书和被告提供的病史，杨浦医院对患者的诊断明确，给予患者的治疗措施不论是检查、用药还是停药的时机均符合诊疗常规，患xxx病共同作用所致，这是根据患者病情的发展及尸检报告共同作出的结论，虽然刘甲对鉴定结论提出质疑，但未提供足够证据足以推翻鉴定机构所出具的鉴定报告，故法院对鉴定报告中的鉴定结论予以确认。因此杨浦医院的医疗行为不存在过错，不应当承担赔偿责任。综上，刘甲要求杨浦医院赔偿医疗费、交通费、尸检费、丧葬费、精神损害抚慰金、律师代理费的请求，不予支持。据此，依照《中华人民共和国民法通则》第一百零六条第二款之规定，作出判决：一、刘甲要求上海市杨浦区中心医院赔偿医疗费人民币8,879.10元的60%的请求，不予支持;二、刘甲要求上海市杨浦区中心医院赔偿交通费人民币300元的60%的请求，不予支持；三、刘甲要求上海市杨浦区中心医院赔偿精神损害抚慰金人民币125,952元的60%的请求，不予支持；四、刘甲要求上海市杨浦区中心医院赔偿丧葬费人民币21,396元的请求，不予支持；五、刘甲要求上海市杨浦区中心医院赔偿尸检费人民币3,500元的请求，不予支持；六、刘甲要求上海市杨浦区中心医院赔偿律师代理费人民币15,000元的请求，不予支持。
　　原审判决后，刘甲不服，向本院提起上诉称：一、杨浦医院在诊治过程中缺乏责任心，违反内科诊疗常规，与患者刘乙的死亡具有直接因果关系。1、在患者肺部感染日渐加重的情况下，未给予抗感染等治疗。存在延误诊断、未及时治疗的情况，最后致患者因肺部感染死亡，尸体报告单证实患者死因为两肺支气管肺炎伴肺出血，医方的过错与患者死亡有直接的因果关系。2、医方用药(法安明)使用不慎。患者于2009年11月20日尿隐血++，11月24日尿隐血+，医方迟至28日方停用法安明5000q12h皮下注射，与患者的两肺支气管肺炎伴肺出血死亡有直接因果关系。二、沪黄医鉴(2010)026号医疗事故技术鉴定书的分析意见多与事实不服。如对于血常规虽描述为正常，实际均高于正常，提示感染。如分析意见说死亡原因为心源性，而尸检报告明确为两肺支气管炎。三、对于上海市医学会沪医鉴(2010)178号鉴定书亦有异议。1、第二军医大学尸检报告单明确死亡原因为两肺支气管炎伴肺出血，而该报告的分析意见认为死亡原因系心肺疾病共同所致。上诉人认为死亡原因应以尸检鉴定为主，医学鉴定为主观分析，而法医的尸检鉴定为客观所见。2、2009年11月30日、12月7日患者血常规白细胞、中性粒细胞均明显高于正常，提示感染需抗炎治疗，但分析意见却认为白细胞及分类无异常，停用抗炎治疗正常。综上，上诉人认为一审法院认定事实错误、定性不正确，要求撤销原判，改判支持其在原审的诉讼请求。
　　被上诉人杨浦医院辩称：法安明的使用在本起案例中未引起不良后果，本案患者患有脑梗需要抗凝，而法安明的用药有半衰期，医学会的鉴定报告中也明确法安明的使用与出血现象无关联。本案经区、市二级医学会的鉴定，专家根据尸检报告结合患者的综合情况，最终认为合并原因导致患者死亡，专家亦均认为院方的用药及诊疗过程均符合诊疗常规。故要求维持原判。
　　二审经审理查明，原审法院查明的法律事实无误，本院予以确认。
　　本院认为，公民的生命健康权受法律保护。在医疗纠纷中，医院的医疗行为存在过失，侵害他人生命健康权的，应当承担相应责任。原审法院根据双方当事人的诉辩、提交的证据对本案事实进行了认定，并在此基础上结合上海市黄浦区医学会及上海市医学会的医疗事故技术鉴定书依法作出原审判决，合法合理，理由阐述充分，本院不再赘述。二审中上诉人主张上海市医学会的鉴定意见书分析意见与第二军医大学的鉴定结论存在矛盾，结论欠缺依据，不能作为定案依据，请求二审法院依法改判支持其一审诉讼请求，但其在二审中没有新的事实与理由，也没有新的证据佐证自己的主张。本院认为，原审法院委托鉴定的程序以及上海市医学会组织鉴定的过程、鉴定人员的资质符合法律规定。对于上海市医学会的鉴定结论，上诉人虽不接受，但没有提供证据来推翻该鉴定结论。故原审法院以此为依据作出原审判决，合法合理，本院予以确认。综上所述，原审认定事实清楚，判决并无不当。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币2,773元，由上诉人刘甲负担。
　　本判决为终审判决。
审判长高中伟
　　代理审判员樊形锋
　　代理审判员钱健民
　　二○一一年六月二十一日
　　书记员卞耀辉
