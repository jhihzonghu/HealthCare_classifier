僧舒婷等与卢氏县官坡中心卫生院医患纠纷案

审判时间：20110710 法院：河南省三门峡市中级人民法院 案件字号：(2011)三民四终字第63号


　　上诉人（原审原告）僧舒婷。
　　上诉人（原审原告）吴彦朝。
　　共同委托代理人周清，卢氏县法律援助中心律师。
　　共同委托代理人吴燕黎。
　　上诉人（原审被告）卢氏县官坡中心卫生院。
　　法定代表人殷守延，院长。
　　委托代理人李柯仪、尚明壮，河南天保律师事务所律师。
　　上诉人僧舒婷、吴彦朝与上诉人卢氏县官坡中心卫生院（以下简称官坡卫生院）因医患纠纷一案，均不服卢氏县人民法院（2009)卢五民初字第58号民事判决，向本院提起上诉。本院依法组成合议庭审理了本案。上诉人僧舒婷、吴彦朝的共同委托代理人周清、吴燕黎，上诉人官坡卫生院的委托代理人李柯仪、尚明壮到庭参加了诉讼。本案现已审理终结。
　　原审查明，僧舒婷丈夫吴振乾平时患有心脏病。2008年11月29日上午，突感胸痛胸闷，在家吃药休息后感觉没多大事，当天下午到地里干活，又感到胸部疼痛不适，并不断加剧，即回家用药后，先到官坡沟口卫生所检查，后又到官坡卫生院诊治，接诊大夫何书旺对其量血压，听心脏后，开了“诺迪康”和“丹参滴丸”，但没有做心电图，亦未采取其它措施，患者吴振乾回家后近三个小时死亡。经河南正诚临床司法鉴定所鉴定，结论为：官坡卫生院的“诊疗行为存在过错与患者吴振乾的死亡之间存在因果关系”。
　　另查明，吴振乾。非农业户口，生前系官坡中学退休教师。需要吴振乾扶养的人有，儿子吴彦朝，农村户口，系智力残疾人。
　　原审认为，患xxx病，与官坡卫生院构成医患关系，其本身所患疾病是导致其死亡的主要原因，但经相关机构鉴定，官坡卫生院的“诊疗行为存在过错与患者吴振乾的死亡之间存在因果关系”，故官坡卫生院应对其治疗措施不到位、抢救措施不力等过错行为承担相应的民事赔偿责任，酌定由官坡卫生院承担僧舒婷、吴彦朝全部损失的40%；依据“举证责任倒置”的诉讼原则，官坡卫生院主张其没有过错，不应承担民事责任，但在庭审辩论终结前，尚无证据证明其主张，故不予采纳。依照《中华人民共和国民法通则》第一百一十九条、《最高人民法院关于参照＜医疗事故处理条例＞审理医疗纠纷民事案件的通知》第一条第一款，并参照《医疗事故处理条例》第五十条之规定，判决如下：僧舒婷、吴彦朝应计吴振乾丧葬10500(21000元÷2）元、死亡赔偿金211696元（13231元×16年），被扶养人吴彦朝的生活费30440元(3044元×20÷2)，精神损害赔偿金10000元，共计262636元，由卢氏县官坡中心卫生院负担40％即105054.4元，限卢氏县官坡中心卫生院于本判决生效后30日内付清款项。如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付延迟履行期间的债务利息。案件受理费6450元，由卢氏县官坡中心卫生院负担2580元，由僧舒婷、吴彦朝负担3870元；鉴定费3000元由卢氏县官坡中心卫生院负担。
　　宣判后，僧舒婷不服，上诉称：1、吴振乾的死亡同官坡卫生院的诊疗行为之间存在因果关系。吴振乾在犯病时感到胸闷、胸痛严重不适，村医周宝莹诊断后，认为可能是心肌梗死，建议吴振乾到上级医院就诊，吴振乾随后到官坡卫生院就诊。官坡卫生院接诊医生严重不负责任，按照《常见急症诊治流程》，对于胸痛患者，通常心电图是必不可少的，官坡卫生院也有心电图设备，接诊医生严重不负责任，没有对吴振乾进行必要的心电图检查，没有确诊病情，也没有进行对症治疗，正是接诊医生的漫不经心和对心脏病治疗常规的无知，才致使吴振乾在近三小时后死亡。原审认定“患xxx病是导致死亡的主要原因”这一结论与目前我国对心脏病治疗效果（经治疗后生还、康复率）不相一致，判决僧舒婷、吴彦朝承担60%的过错责任是错误的。2、本案僧舒婷、吴彦朝提起的是“医疗过错赔偿诉讼”而非“医疗事故赔偿”，诉讼中官坡卫生院申请的鉴定也是司法鉴定。本案应适用《民法通则》和《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》，而不应使用《医疗事故处理条例》进行审理。请求法院依法改判官坡卫生院依法承担80%的赔偿责任即在一审判决的基础上再承担105054.4元。
　　官坡卫生院上诉称：1、患者吴振乾年龄较大，本身患冠心病多年，其自身疾病是导致其死亡的主要原因。吴振乾的死亡与与官坡卫生院的诊疗不存在直接或间接的因果关系。吴振乾与官坡卫生院是一种医患之间的服务合同关系，对于医生的建议是否被患者采纳，是患者的权利，在当天的诊断过程中，接诊医生曾建议吴振乾住院治疗或到县医院进一步检查和诊断，但遭到患者的拒绝。吴振乾死亡的直接原因是心肌梗塞，而不是误诊。2、原审依据的（2009）临鉴字第409号鉴定意见书，缺乏鉴定的实质要件，送检材料严重失实，所送鉴的鉴材“两份证明”是吴振乾家属为了办理火化手续，而让卫生院医生补开的，不能证明卫生院存在医疗行为过错；鉴定结论缺乏科学性和专业性，原审据此判决显属错误。请求二审改判官坡卫生院不承担赔偿责任或将本案发回重审。
　　本院另查明，（2009）临鉴字第409号鉴定意见书认为，“1、根据委托材料记载吴振乾死亡原因心肌梗塞，诊断成立。2、官坡卫生院存在下列过错：①开始时对患者的诊断不确切不具体，当时只诊断为冠心病，没有考虑到心肌梗死。其实冠心病包括心绞痛和心肌梗塞，但两者的临床症状，抢救措施，预后大不一样。患者吴振乾就诊时有胸痛、心慌、气短、心音、脉搏细弱，血压测量前后矛盾，这已是心肌梗死的先兆症状，但未引起接诊大夫的重视，就草率的给两种药让病人回家，三小时之后就死亡了。②存在误诊误治的情节，接诊大夫应该给患者做心电图，但未做，这是误诊的主要原因。对于这种病人应做到及早发现，及早住院并加强住院前的就地治疗。简单的治疗措施是休息吸氧、检测、控制饮食、排除疼痛。卫生院简单的治疗措施都没有采取。3、患者就诊时就有典型的心肌梗死，接诊时未给病人进行抢救措施，结果（患者）回家后三个小时后就死亡。所以我们分析认为卫生院的诊疗过错行为与吴振乾的死亡存在因果关系。”本院查明其他基本事实与原审查明事实一致。
　　本院认为，吴振乾有长期冠心病史，在出现胸痛、胸闷症状后到官坡卫生院就诊，官坡卫生院接诊时在被告知吴振乾冠心病史和胸痛、胸闷症状后，没有对吴振乾进行必要的心电图检查，未能及时确诊疾病并对症治疗，经鉴定卫生院的诊疗过错行为与吴振乾的死亡存在因果关系。根据原审查明事实，送检材料中吴振乾的死亡证明由官坡卫生院医生出具并加盖官坡卫生院公章，官坡卫生院认为鉴材严重失实、鉴定结论缺乏科学性和专业性的理由没有事实和法律依据，本院不予采纳。本案已经入司法程序，原审进行医疗过错司法鉴定并无不当。官坡卫生院主张其没有过错、不应承担民事责任，但其无证据证明自己的主张，本院不予采纳。虽然卫生院的诊疗过错行为与吴振乾的死亡存在因果关系，但患xxx病史，综合考虑吴振乾的年龄、病史、当地的医疗水平及官坡卫生院的过错程度，官坡卫生院应承担僧舒婷、吴彦朝全部损失的30%为宜。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（三）项之规定，判决如下：
　　一、变更卢氏县人民法院（2009)陕民初字第58号民事判决为：“僧舒婷、吴彦朝应计吴振乾丧葬费10500(21000元÷2）元、死亡赔偿金211696元（13231元×16年），被扶养人吴彦朝的生活费30440元(3044元×20÷2)，精神损害赔偿金10000元，共计262636元，由卢氏县官坡中心卫生院负担30％即78790.8元，限卢氏县官坡中心卫生院于本判决生效后30日内付清款项。”
　　二、驳回僧舒婷、吴彦朝的其他诉讼请求。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付延迟履行期间的债务利息。
　　一审案件受理费6450元，由卢氏县官坡中心卫生院负担1935元，由僧舒婷、吴彦朝负担4515元；鉴定费3000元由卢氏县官坡中心卫生院负担。
　　二审案件受理费4800元，由卢氏县官坡中心卫生院负担1440元，由僧舒婷、吴彦朝负担3360元；
　　本判决为终审判决。
审判长刘占军
　　代理审判员郭旭飞
　　代理审判员张攀峰
　　二〇一一年七月十日
　　书记员马志光
