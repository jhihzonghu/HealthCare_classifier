孙某等诉上海市普陀区某医院医疗损害赔偿纠纷案

审判时间：20110914 法院：上海市普陀区人民法院 案件字号：(2011)普民一(民)初字第4888号


　　原告孙某
　　原告孙某
　　原告赵某
　　被告上海市普陀区某医院
　　原告孙某、孙某、赵某与被告上海市普陀区某医院（以下简称某医院）医疗损害赔偿纠纷一案，本院受理后，依法由审判员邵莉星独任审判，公开开庭进行了审理。原告孙某及其与原告孙某的共同委托代理人顾文珉，原告赵某的委托代理人赵宝民，被告某医院的委托代理人赵建国到庭参加诉讼。本案现已审理终结。
　　原告孙某、孙某、赵某诉称，原告孙某系赵玉萍（已死亡）的丈夫，孙某系赵玉萍的儿子，赵某系赵玉萍的父亲。赵玉萍于2010年2月17日当晚工作加班，原告孙某驾驶机动车接其回家途中，于2月18日凌晨0时20分许发生交通事故，致原告孙某及赵玉萍受伤，当日凌晨1时许，孙某及其妻子被交警送至被告处治疗。由于被告医护人员救治措施不妥，导致赵玉萍于2月18日凌晨4时48分不治身亡。原告认为赵玉萍虽因交通事故受伤在先，但其伤势本不足以导致死亡，赵玉萍死亡系被告救治措施不妥所致。原告遂诉至法院，要求被告赔偿医疗费人民币1832.53元（以下币种均为人民币），丧葬费108631.50元，死亡赔偿金533500元，精神损害抚慰金116388元，被扶养人生活费99102.40元；本案诉讼费由被告承担。
　　被告某医院辩称，对患者赵玉萍在被告处就诊、死亡的经过均无异议，医疗事故技术鉴定书已明确被告对本次医疗纠纷承担轻微责任，被告同意按照法律规定对原告的合理损失进行赔偿；原告主张的死亡赔偿金、被扶养人生活费无法律依据，丧葬费、精神损害抚慰金过高。
　　本案在诉前调解过程中，根据当事人的申请，本院委托上海市黄浦区医学会就某医院对患者赵玉萍的医疗行为是否违反医疗卫生管理法律，行政法规，部门规章和诊疗护理规范、常规；其医疗行为与患者赵玉萍的死亡有无因果关系；明确是否属于医疗事故，等级及责任程度等方面进行了鉴定。上海市黄浦区医学会于2011年6月22日作出沪黄医鉴[2010]087号医疗事故技术鉴定书。该鉴定书的分析意见为：1、患者因车祸导致颅脑外伤、双肺广泛挫伤，是导致死亡的主要原因。2、医方在接诊过程中未进行血气分析等主要监测手段。3、抢救方面未进行气管插管、呼吸机机械通气等主要抢救措施，与患者死亡有一定关联。4、胸外按压与患者死亡无因果关系。鉴定结论为：赵玉萍与某医院医疗争议构成一级甲等医疗事故，医方承担轻微责任。
　　经审理查明，原告孙某、孙某、赵某分别系患者赵玉萍之丈夫、儿子、父亲。2010年2月18日1：00，患者赵玉萍因车祸即刻，120送入被告处急诊外科。检查：神不清，二眼瞳孔月0.3cm，光反应消失，抬头张口呼吸，口中不断有血柱喷出。头部未见明显伤口，二肺呼吸音差。胸部CT提示双肺广泛挫伤。头颅CT提示颅脑挫伤，左侧颞骨骨折，外伤性气颅。上腹部CT检查腹腔未见明显积液。诊断为：双肺广泛挫伤，脑挫伤。当日3：00患者入住普外ICU病房，予以外科Ⅰ级护理、禁食、床边心电监护、气垫床、告病危等。3：50患者即刻出现血压下降，氧饱和度下降。被告予以尼克刹米、洛贝林，多巴胺、阿拉明，肾上腺素、阿托品，同时保持呼吸道通畅。并予胸外按压。4：08患者即刻呼吸、心率、血压测不出，予以肾上腺素、阿托品强心治疗，并予胸外按压。4：48患者呼吸心跳停止、双侧瞳孔散大、血压测不出，心电图示一直线，4：50宣告临床死亡。死亡诊断：双肺广泛挫伤，脑挫伤。经交警部门委托，司法鉴定科学技术研究所司法鉴定中心对赵玉萍进行尸表检验、分析死因，于2010年3月1日出具沪交鉴（1）[2010]尸检字第151号鉴定意见书，分析说明为：……尸体检验发现：口、鼻腔及双侧外耳道积血，枕部正中头皮下肿胀，下唇左侧及下方挫裂创。胸骨及双侧肋骨多发性骨折。躯干部及四肢多处皮肤青紫、擦挫伤。另据尸体识别卡记载：临床诊断为“双肺挫伤”。根据尸体检验并结合案情分析：赵玉萍的死亡原因符合道路交通事故致头、胸部损伤。鉴定结论为：赵玉萍系道路交通事故致头、胸部损伤死亡。原告认为，由于被告对患者未采取积极有效的救治措施，导致患者死亡，被告的行为存在过错。原告遂起诉来院，请求法院判决如其诉请。
　　本院认为，原、被告双方对患者赵玉萍至被告处就诊、死亡的经过均无异议，本院依法予以认定。关于被告对赵玉萍实施的医疗行为是否存在过错、过错程度及因果关系，本院已委托了上海市黄浦区医学会进行了鉴定，该医学会作出的医疗事故技术鉴定书均已经明确：赵玉萍与某医院之间的医疗争议构成一级甲等医疗事故，某医院承担轻微责任。原告对鉴定结论持异议，并提出重新鉴定的申请，但未就该部分异议提供相应的证据予以佐证，本院不予采信。本院结合司法鉴定中心出具的鉴定意见书，对上海市黄浦区医学会的医疗事故技术鉴定书予以确认，并据此酌情确定被告对原告的合理损失承担20%的赔偿责任。
　　关于赔偿范围，原、被告在庭审中就医疗费1832.53元的赔偿标准取得一致意见，本院予以确认。关于丧葬费，本院根据目前司法实践中掌握的标准确定丧葬费为23378.50元。该费用由被告按照20%的比例赔偿。关于精神损害抚慰金，因患者赵玉萍的死亡，对原告的精神、生活均造成了严重的影响，本院根据被告过错程度酌情确定精神损害抚慰金为28000元。关于被扶养人生活费，原告孙某目前已满18周岁，本院不予支持。原告主张死亡赔偿金533500元，因本案所涉侵权行为与损害后果均发生在2010年2月18日，不适用《侵权责任法》相关法律规定，故本院对原告的该项诉请不予支持。综上，依据《中华人民共和国民法通则》第九十八条、第一百一十九条，《医疗事故处理条例》第五十条的规定，判决如下：
　　一、被告上海市普陀区某医院应于本判决生效之日起十日内赔偿原告孙某、孙某、赵某医疗费人民币1832.53元的20%，计人民币366.50元；
　　二、被告上海市普陀区某医院应于本判决生效之日起十日内赔偿原告孙某、孙某、赵某丧葬费人民币23378.50元的20%，计人民币4675.70元；
　　三、被告上海市普陀区某医院应于本判决生效之日起十日内赔偿原告孙某、孙某、赵某精神损害抚慰金人民币28000元；
　　四、对原告孙某、孙某、赵某的其余的诉讼请求不予支持。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　本案鉴定费人民币3500元，由被告上海市普陀区某医院负担。
　　本案受理费人民币12394元，减半收取计人民币6197元，由原告孙某、孙某、赵某负担人民币5571元，被告上海市普陀区某医院负担人民币626元。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第二中级人民法院。
审判员邵莉星
　　二〇一一年九月十四日
　　书记员张志刚
