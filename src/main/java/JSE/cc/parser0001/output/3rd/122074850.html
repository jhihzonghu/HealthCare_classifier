俞窑水受贿案

审判时间：20110712 法院：浙江省杭州市余杭区（市）人民法院 案件字号：(2011)杭余刑初字第560号


　　公诉机关浙江省杭州市余杭区人民检察院。
　　被告人俞窑水。因本案于2011年1月7日被刑事拘留，同年1月21日被逮捕。现押于杭州市余杭区看守所。
　　辩护人戴红兵。
　　杭州市余杭区人民检察院以杭余检刑诉（2011）393号起诉书指控被告人俞窑水犯受贿罪，于2011年6月7日向本院提起公诉。本院依法组成合议庭，公开开庭审理了本案。杭州市余杭区人民检察院指派代理检察员王慰出庭支持公诉，被告人俞窑水及其辩护人戴红兵到庭参加诉讼。现已审理终结。
　　经审理查明：2004年8月，被告人俞窑水被聘任为杭州市余杭区卫生局医政业务科副科长，主要负责医疗纠纷处理和征兵体检工作，并配合科长做好医疗质量监管工作。2008年1月，因医政业务科科长调任，被告人俞窑水开始全面负责管理科室工作，并于2008年10月正式担任杭州市余杭区卫生局医政业务科科长。杭州市余杭区卫生局医政业务科负责全区医疗机构的设置、执业许可审批和等级评审，负责医疗机构的医疗技术准入的审批，负责医疗机构的医疗质量、服务质量的监督管理，负责医疗纠纷协调处理等事项。
　　2006年至2010年，被告人俞窑水在担任杭州市余杭区卫生局医政业务科副科长、科长期间，利用职务之便，为他人谋取利益并收受贿赂款物共计价值人民币209660元。具体如下：
　　1、2006年至2010年期间，被告人俞窑水多次收受杭州余杭丰瑞门诊部负责人林某为搞好关系所送的购物卡，共计价值人民币16000元。
　　2、2008年至2010年期间，被告人俞窑水多次收受杭州余杭郎氏中医门诊部负责人郎某为搞好关系所送的现金及购物卡，共计价值人民币16000元。
　　3、2008年至2010年期间，被告人俞窑水多次收受杭州市余杭骨科医院负责人程某为搞好关系所送的佳能IXUS980IS照相机1部，以及价值共计人民币7000元的购物卡。经鉴定，赃物佳能IXUS980IS照相机价值人民币2500元。
　　4、2008年至2009年期间，被告人俞窑水多次收受杭州余杭宏正医疗门诊部负责人谢某为感谢其在门诊部升格事宜上给予的帮助和关照，以及为搞好关系所送的现金人民币50000元，以及六福珠宝吉祥如意重30克纯金金条1块。经鉴定，赃物价值人民币8160元。
　　5、2009年至2010年期间，被告人俞窑水多次收受杭州市余杭康德医学门诊部负责人金某为感谢其在门诊部升格等事宜上给予的关照，以及为搞好关系所送的现金及购物卡，共计价值人民币21000元。
　　6、2009年至2010年期间，被告人俞窑水多次收受杭州市余杭临平华辰中医针灸推拿门诊部负责人计某为搞好关系所送的购物卡，共计价值人民币9000元。
　　7、2009年至2010年期间，被告人俞窑水多次收受杭州市余杭区闲林健爱门诊部负责人许某为搞好关系所送的现金共计人民币8000元。
　　8、2009年至2010年期间，被告人俞窑水多次收受杭州市余杭塘栖塘济医疗门诊部负责人高某为搞好关系所送的现金共计人民币4000元。
　　9、2009年12月至2010年1月期间，被告人俞窑水多次收受杭州市余杭良博医学门诊部负责人吴某为感谢其在核发医疗机构执业许可证等事宜上给予的帮助和关照，以及为搞好关系所送的现金共计人民币58000元。
　　10、2010年春节前的一天，被告人俞窑水收受杭州市余杭福安医学门诊部负责人贾某为搞好关系所送的价值人民币4000元的购物卡。
　　11、2010年12月31日，被告人俞窑水收受杭州市余杭乔司永桥西医内科诊所负责人陈某为搞好关系所送的现金人民币6000元。
　　案发后，杭州市余杭区人民检察院从被告人俞窑水及其家属处扣押违法所得佳能IXUS980IS照相机1部、六福珠宝吉祥如意重30克纯金金条1快及人民币54100元；另，被告人俞窑水家属事后退出赃款共计人民币50000元，其中20000元扣押于杭州市余杭区人民检察院，另30000元扣押于本院。
　　被告人俞窑水因涉嫌本案第9节犯罪事实被侦查机关采取强制措施，后其如实供述了司法机关尚未掌握的其他犯罪事实。
　　上述事实，被告人俞窑水在开庭过程中无异议，并有杭州市余杭区卫生局组织机构代码证复印件、卫生局内部科室职能说明、干部履历表、考核登记表、公务员登记表、中共杭州市余杭区卫生局委员会文件、干部任免审批表等书证；证人林某、郎某、程某、谢某、金某、计某、许某、高某、吴某、贾某、陈某的证言；证人俞某的证言；医疗机构变更批准书、医疗机构申请变更登记注册书、设置医疗机构申请书、关于要求升格设置杭州宏正门诊部的申请、杭州余杭乔司吕松中西医结合诊所要求验收变更设置的报告、竣工报告、关于杭州市余杭宏正医疗门诊部建设项目环境影响报告的审批意见等书证；杭州余杭区良博医学门诊部营业执照复印件、杭州市余杭区卫生局责令停业通知书、卫生监督意见书、医疗机构申请变更登记注册书、申请书及承诺书书证；搜查笔录、扣押物品清单及照片；价格鉴定结论；扣押物品清单及往来结算票据；抓获破案经过及户籍证明等证据证实，证据间相互印证，足以认定。
　　本院认为，被告人俞窑水身为国家工作人员，利用职务上的便利，非法收受他人财物共计人民币20万余元，为他人谋取利益，其行为已构成受贿罪。公诉机关指控的罪名成立。被告人俞窑水因涉嫌受贿被采取强制措施后，如实供述了司法机关尚未掌握的其他较重受贿罪行，依法应从轻处罚。被告人俞窑水如实供述自己的罪行且当庭自愿认罪，并部分退赃，依法可以从轻处罚，对辩护人据此提出对被告人俞窑水从轻处罚的辩护意见，本院予以采纳。据此，依照《中华人民共和国刑法》第三百八十五条第一款、第三百八十六条、第三百八十三条第一款第（一）项、第二款、第六十七条第三款、第五十九条、第九十三条、第六十四条、最高人民法院《关于处理自首和立功具体应用法律若干问题的解释》第四条和《关于适用财产刑若干问题的规定》第一条之规定，判决如下：
　　一、被告人俞窑水犯受贿罪，判处有期徒刑十年，并处没收财产人民币一万元（刑期从判决执行之日起计算。判决执行以前先行羁押的，羁押一日折抵刑期一日，即自2011年1月7日起至2021年1月6日止）。
　　二、扣押在杭州市余杭区人民检察院未随案移送的被告人俞窑水的违法所得人民币七万四千一百元、佳能IXUS980IS照相机一部、六福珠宝吉祥如意重30克纯金金条一块，及扣押于本院的被告人俞窑水的违法所得人民币三万元，予以追缴，上缴国库。
　　三、追缴被告人俞窑水的其余违法所得，上缴国库。
　　如不服本判决，可在接到判决书第二日起十日内，通过本院或直接向浙江省杭州市中级人民法院提出上诉。书面上诉的，应当提交上诉状正本一份，副本二份。

　　审　判　长　　傅贤生
　　人民陪审员　　吴福征
　　人民陪审员　　高　翔
　　二〇一一年七月十二日
　　书　记　员　　周　虹
