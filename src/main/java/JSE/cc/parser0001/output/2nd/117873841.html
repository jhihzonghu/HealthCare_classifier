王克美与正阳县中医院等侵权纠纷上诉案

审判时间：20110523 法院：河南省驻马店市中级人民法院 案件字号：(2011)驻民二终字第178号


　　上诉人（原审原告）王克美。
　　委托代理人景高举，河南济世雨律师事务所律师。
　　被上诉人（原审被告）正阳县中医院。
　　法定代表人孟祥本，该院院长。
　　委托代理人张明友，河南明信律师事务所律师。
　　被上诉人（原审被告）正阳县卫生局。
　　法定代表人杨前亮，该局局长。
　　委托代理人杨建新，河南公义律师事务所律师。
　　上诉人王克美因侵权纠纷一案，不服正阳县人民法院（2010）正民初字第757号民事判决，向本院提起上诉。本院依法组成合议庭，公开开庭审理了本案。上诉人王克美的委托代理人景高举，被上诉人正阳县中医院的委托代理人张明友，被上诉人正阳县卫生局的委托代理人杨建新到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，原告王克美与受害人（死者陈新梅）系母女关系。2008年3月3日夜，受害人陈新梅在其丈夫熊新华、婆姐熊雪荣的陪同下到王秀芝（已因非法行医被判刑）开设的诊所里生二胎孩子。3月4日夜21时40分左右，受害人陈新梅出现产后出血，约23时，在该诊所止血未果的情况下，熊新华拨打了120急救电话，要求到县人民医院。120急救车于23时40分左右将受害人陈新梅拉到了正阳县中医院进行抢救，该院同时将受害人陈新梅处于重度休克、病情危重状态的病情及抢救过程中有可能发生的并发症告知了熊新华，熊新华在病情告知书上签名。正阳县中医院医生对受害人陈新梅做输血前检查、派人去县医院拿血同时采取抢救措施，但还没输上血时，受害人陈新梅因子宫下段及宫颈撕裂致急性失血性休克，于3月5日凌晨1时抢救无效死亡。受害人陈新梅死亡后，熊新华分别与正阳县中医院、王秀芝达成了谅解协议。王克美于2008年3月6日上午知道了陈新梅死亡的情况，后来其与王秀芝也达成了谅解协议。2010年6月18日，王克美提起诉讼，要求确认正阳县中医院、正阳县卫生局有过错；并通过正阳县电视台、正阳县广播电台、今日正阳等地方媒体向死者近亲属道歉（致歉内容必须尊重事实，经原告王克美认可）。原审法院认为，原告王克美所主张的诉求是因其女儿在生育就医过程中死亡所引起的确认之诉及侵权之诉的综合，不属于医疗事故纠纷的范畴，依照最高人民法院关于参照《医疗事故处理条例》审理医疗纠纷民事案件的通知的规定，本案应适用民法通则的规定处理。我国民法通则规定，向人民法院提起保护民事权利的诉讼时效期间为二年；诉讼时效期间从知道或者应当知道权利被侵害时起计算。本案中王克美对其女儿因他人非法行医致产后大出血并在正阳县中医院抢救无效死亡的情况是在2008年3月6日知道的，其在知情后也向非法行医人提起了附带民事诉讼并与之达成了谅解协议。这说明王克美在2008年3月6日就认为自身权利受到了侵犯，其主张侵权责任的期间依法应自2008年3月7日起计算两年，但至2010年6月18日向正阳县人民法院提起诉讼时止，王克美的起诉已超过二年的法定诉讼时效期间，所以，对王克美的诉讼请求不予支持；对正阳县中医院辩称本案已超过法定诉讼时效的辩述予以支持。王克美要求确认被告正阳县卫生局有过错，应当向死者近亲属赔礼道歉，但在诉讼中原告并没有举证证明正阳县卫生局有民事法律行为上的过错存在。王克美也仅是陈述受害人的丈夫熊新华向120指挥中心提出要到县人民医院治疗，并没有举证证明“120”指挥中心侵犯了其何种法定权利及“120”指挥中心因调度行为应承担民事责任的法律依据，所以，王克美因不能证明正阳县卫生局对其有侵权行为存在，应承担举证不能的法律后果，因此对王克美向正阳县卫生局所提出的诉讼请求不予支持。原审法院判决：驳回原告王克美的诉讼请求。诉讼费100元由原告王克美承担。宣判后，王克美不服，上诉来院。
　　王克美上诉称：1、陈新梅虽然是在2008年3月5日死亡，但其在2009年6月20日接到刑事判决书后才知道二被上诉人具有一定过错，因此其诉请未超过诉讼时效；2、正阳县卫生局所属的“120”指挥中心及出诊人员将陈新梅送到没有输血条件的医院，导致陈新梅死亡其应承担侵权责任。请求撤销原判，依法改判。
　　正阳县中医院答辩称：
　　1、在陈新梅死亡后，其家属与医院达成了谅解协议并已提起刑事诉讼，故其本案诉请已超诉讼时效；2、其医院对陈新梅治疗不存在医疗事故及过错，王克美也没有明确提出侵犯何种权益。请求驳回上诉，维持原判。
　　正阳县卫生局答辩称：1、陈新梅死亡与卫生局不存在法律因果关系，2、因中医院离接诊地点距离近故将陈新梅送到中医院，“120”指挥中心及出诊人员没有义务知道中医院是否具备输血条件。请求驳回上诉，维持原判。
　　二审审理查明的事实与原审法院判决认定的事实一致。
　　本院认为，上诉人王克美在其女陈新梅于2008年3月5日死亡后，已向王秀芝（非法行医人）提起了刑事附带民事诉讼并达成了谅解协议，且熊新华（陈新梅丈夫）也与正阳县中医院达成了协议，故应认定上诉人王克美从2008年3月6日即知道其权利受到了侵犯，诉讼时效应从2008年3月7日起计算。王克美于2010年6月18日向原审法院提起诉讼，已经超过法律规定的二年诉讼时效期间。王克美上诉称其诉请未超过诉讼时效的理由不足，不予支持。正阳县卫生局所属“120”指挥中心及出诊人员基于就近、就地等原则将当时处于大出血状态的陈新梅送往正阳县中医院的行为并无过错，且上诉人也没有证据证明正阳县卫生局对其有侵权行为的存在，故王克美上诉称正阳县卫生局应承担侵权责任的理由亦不足，不予支持。综上，原审判决认定事实清楚，处理结果正确，应予维持。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项的规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费100元，由上诉人王克美负担。
　　本判决为终审判决。
审判长侯治安
　　审判员张怀珍
　　审判员牛朝干
　　二O一一年五月二十三日
　　书记员高冉玉
