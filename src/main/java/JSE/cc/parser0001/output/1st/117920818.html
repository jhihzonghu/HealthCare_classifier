徐某诉上海邮电医院医疗损害赔偿纠纷案

审判时间：20110121 法院：上海市静安区人民法院 案件字号：(2010)静民三(民)初字第384号


　　原告徐某。
　　被告上海邮电医院。
　　法定代表人阙某，该院院长。
　　委托代理人俞某，该院工作人员。
　　原告徐某诉被告上海邮电医院(以下简称邮电医院)医疗损害赔偿纠纷一案，本院于2010年11月10日立案受理后，依法由代理审判员孙歆独任审判。2010年12月21日公开开庭进行了审理。原告及被告的委托代理人俞某到庭参加了诉讼，本案现已审理终结。
　　原告徐某诉称：2003年10月15日原告因外痔在被告处接受外痔切除手术，因麻醉不当，原告当即出现胸闷、呼吸困难、气急、心悸、血压下降等症状。同年10月22日出院，出院后第三天原告突发气急、呼吸不畅、心悸加重、胸闷眼发黑、头晕症状，即被送往利群医院急诊，同年12月19日原告又出现上述症状，并于当年12月25日再次进入被告处住院治疗，2004年1月10日出院，被诊断为冠心病、心律失常。原告随后多次住院急救，于2009年8月20日在上海市胸科医院接受心脏支架手术，出院诊断为不稳定心绞痛，心功能Ⅱ级，心律失常窦缓房颤。原告在2003年10月15日住院前无任何心脏不适，被告存在伪造入院记录住院病史的行为，2010年4月27日下午原、被告双方互看原始材料发现原住院记录原告心率存在修改痕迹，同时原告2003年出院时，被告亦隐瞒了原告存在心律失常的事实，造成原告严重的身体损伤。故原告诉至法院，请求判令被告赔偿原告医药费人民币44,000元、交通费人民币2,000元、营养费人民币14,000元、精神损失费人民币20,000元。
　　被告邮电医院辩称：案件已经过两次医疗事故鉴定，结论均为徐某与邮电医院医疗争议不构成医疗事故，被告的医疗行为与原告的心脏疾病之间亦不存在因果关系。故原告要求被告承担侵权责任缺乏事实和法律依据。
　　经本院审理查明，2003年10月15日，原告徐某因肛门外异物感一周伴疼痛入住邮电医院。据住院病案记载：入院时检查：胸膝位3点～5点肛缘可及约3×3cm大小质硬肿块，无触痛，无出血，活动度可。入院后诊断：血栓性外痔。当日心电图检查示：左心室高血压。10月17日在腰麻下行血栓外痔切除术。术中使用麻黄素10mg、阿托品合计0.75mg，因麻醉时出现心率明显下降至46次/分，术后予以吸氧6小时及抗生素应用。10月18日流质饮食。10月19日已排便。10月20日病理检查报告：“肛门”组织病变符合痔核。经治疗至10月22日原告出院。2003年10月25日原告因胸闷半小时前往上海市普陀区利群医院就诊。经心电图检查示：窦性心律；在正常范围内。予以丹参、生脉滴注。2003年12月19日原告因头昏半小时伴恶心、胸闷再次在上海市普陀区利群医院就诊。仍予丹参、生脉静脉滴注处理。2003年12月20日原告因胸痛12小时，觉压迫感前往邮电医院就诊。诊断：睡眠---呼吸暂停综合症。建议呼吸专科门诊。12月24日原告再次复诊。据病历记载：主诉：胸部压迫症，白天头晕，有时眼前发黑。夜间出现胸闷、气憋，急诊心电图正常，已发生数次，夜间打鼾明显。检查：心率64次/分，(律)齐，两肺未及干、湿罗音。诊断：胸闷待查：冠心病？睡眠呼吸暂停综合症？建议入院诊治。
　　2003年12月25日原告因“阵发性左前胸区闷痛2月伴头晕、乏力”再次入住邮电医院。入院后诊断：①冠心病；②睡眠呼吸暂停综合症，低氧血症？予以活血化淤、扩冠等治疗。12月30日动态心电图检查示：窦缓，窦不齐；偶见房早及短阵房速。12月31日心电图检查示：窦性心动过缓；左室高血压。同日行1mg阿托品试验，注射前心率56次/分，注射后最快心率81次/分在3分钟时出现。2004年1月6日行心脏超声检查示：静息状态下心超检查属大致正常。1月8日动态心电图检查示：偶见房早及一次短阵房速。经治疗后原告于2004年1月10日出院，出院诊断：冠心病；心律失常。出院后原告反复感胸闷、心悸而在各医疗机构就诊。2006年6月11日原告在上海市普陀区利群医院心电图检查示：心房颤动；ST-T改变。2006年12月7日原告在上海市普陀区利群医院心电图检查示：窦性心律不齐；逆钟向转位。2007年8月15日原告在上海市普陀区利群医院行动态心电图检查示：窦性心律；偶见房早，部分未下传，短阵房速1阵；偶见室早。2009年7月31日原告在上海市普陀区利群医院行动态心电图报告示：窦性心动过缓；偶见窦性停搏(于房速后)；房性早搏，部分成对，短阵房速5阵；偶见房早；下壁异常Q波；T波改变。2009年8月17日原告入住上海市胸科医院。次日行冠脉造影LAD：近段狭窄80%，中段不规则；LCV中段狭窄70%；遂于LAD植入Firebird-II3.0×29mm支架一枚，LCX行球囊扩张术。术后诊断：“冠心病、不稳定性心绞痛、心功能II级；心律失常：窦缓、短阵房速”。2009年8月21日原告出院。2009年12月12日原告在上海市普陀区利群医院行心电图报告示：窦性心律；T波改变。现原告仍在门诊随访中。
　　提出医疗纠纷鉴定申请，后本院依法委托上海市黄浦区医学会组织医疗纠纷司法鉴定。上海市黄浦区医学会于2010年5月13日作出沪黄医鉴[2010]015号医疗事故技术鉴定书，鉴定结论为徐某与邮电医院医疗争议不构成医疗事故。同年5月27日，原告提出再次鉴定申请，申请本院委托到上海市医学会鉴定，后本院委托上海市医学会组织鉴定邮电医院对患者徐某的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规；其医疗行为与患xxx病情有无因果关系；明确是否属于医疗事故，等级及责任程度。2010年9月26日上海市医学会出具沪医鉴[2010]151号医疗事故技术鉴定书，分析意见为：1、患者因血栓性外痔伴疼痛，有外科手术适应证。手术所选择的蛛网膜下腔阻滞(腰麻)麻醉方式合理，麻醉期间用药(麻黄素、阿托品、布比卡因等)剂量、方法正确。麻醉过程中出现短暂的血压降低和心率减慢征象，此为腰麻过程中难以完全避免的情况，经医方处置后迅速恢复正常。2、冠状动脉粥样硬化性心脏病(冠心病)是由于冠状动脉硬化使管腔狭窄或阻塞导致心肌缺血、缺氧而引起的心脏病，其发生机理复杂，但医方的手术和麻醉方式及用药均不是冠状动脉粥样硬化性心脏病的成因；3、依据现有送鉴病历资料，患者出现的胸闷、心悸、不稳定心绞痛、心功能2级以及心律失常等，为其自身慢性疾病—冠状动脉粥样硬化性心脏病演变的临床表现。4、经审核该患者所有与医方留存的出院小结设计格式不同，但两者的主要内容一致，没有发现医方有伪造、篡改、涂改之处，故不违反现行的《病历书写基本规范》。该医疗事故技术鉴定书结论为：综上所述，根据《医疗事故处理条例》第二条、第三十三条规定，徐某与邮电医院医疗争议不构成医疗事故。
　　上述事实有原、被告陈述，上海市黄浦区医学会医疗事故技术鉴定书、上海市医学会医疗事故技术鉴定书等为证，并经本院查证属实，依法应予确认。
　　本院认为：公民享有生命健康权，在其遭受到其他公民、组织非法侵害时，有获得相应赔偿的权利。本案中，原告徐某在被告处治疗过程中，如果医疗机构在医疗行为上存在违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，过失造成患者人身损害而构成医疗事故的，则医疗机构应承担相应的民事赔偿责任。即原告要求被告承担赔偿责任的前提在于，原告徐某目前病情与被告的医疗行为之间具有因果关系，且被告在医疗行为中存在过错。这是因为，医疗行为是一种复杂而技术含量极高的活动，病员出现的最终后果，往往与病员自身疾病的发展、医学科学和技术的局限性等事实密切相关。所以，科学合理地分析结果产生的原因是确定医疗行为是否有过错的客观要求。本案经上海市黄浦区医学会进行医疗事故鉴定，并由上海市医学会进行复检，鉴定结论均认定本起医疗争议不属于医疗事故。经本院核实，鉴定报告程序合法、对医患双方的争议焦点均作出了明确分析，其鉴定意见应予采信。
　　原告对鉴定结论存在异议，其依据的事实和主张的观点在鉴定过程中均已作为争议要点，但原告的观点并未被鉴定部门采纳，现亦不足以推翻鉴定报告所得出的结论；本院注意到上海市医学会鉴定报告指出患者所有与医方留存的出院小结设计格式不同，但该鉴定报告认定两者的主要内容一致，没有发现医方有伪造、篡改、涂改之处，故不违反现行的《病历书写基本规范》，原告亦无其他证据证明被告存在伪造、篡改、涂改出院小结的行为，故本院对此不予采信。
　　根据《中华人民共和国民法通则》第九十八条、第一百零六条第二款的规定，判决如下：
　　对于原告徐某的诉讼请求不予支持。
　　案件受理费人民币1,550元，减半收取，由原告徐某负担人民币775元；案件鉴定费共计人民币7,000元，由原告徐某承担人民币3,500元，被告上海邮电医院承担人民币3,500元。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第二中级人民法院。
代理审判员孙歆
　　二Ｏ一一年一月二十一日
　　书记员陆锋莉
