吕某、潘某、郁某因医疗服务合同纠纷案

审判时间：20110920 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2011)沪一中民一(民)终字第2032号


　　上诉人（原审原告）吕某
　　上诉人（原审原告）潘某。
　　上诉人（原审原告）郁某。
　　被上诉人（原审被告）某医院。
　　上诉人吕某、潘某、郁某因医疗服务合同纠纷一案，不服上海市浦东新区人民法院（2011）浦民一(民)初字第12794号民事判决，向本院提起上诉。本院于2011年7月28日立案受理后，依法组成合议庭，并于2011年8月31日公开开庭进行了审理。上诉人吕某、潘某、郁某的共同委托代理人潘某，被上诉人某医院的委托代理人童某到庭参加了诉讼。本案现已审理终结。
　　原审法院审理查明，吕某、潘某、郁某分别是患者潘苏荣的妻子、儿子及母亲。2007年7月8日13时26分，患者因“意识丧失15分钟伴头晕、左侧肢体无力1小时”由救护车送至某医院急诊。据急诊病历记载：（患者）视物旋转，无头痛、呕吐，睁闭眼无明显变化，无语言障碍，无肢体抽动。既往有高血压病史20余年，不服药。查体：血压170/90mmHg，言语略含糊，双侧鼻唇沟对称，伸舌居中。颈软，左侧上下肢肌力4级，双侧病理征（-）。辅助检查：1、头颅CT：CT提示右侧基底节区可见出血并破入脑室（口头报告）。2、心电图。3、血常规、肝肾功能、电解质。请神经外科会诊。同时予心电监护、吸氧。生理盐水250ml+亚宁安(应为“亚宁定”)100g/静滴，根据血压调速。甘油果糖250ml/静滴，每日两次；20%甘露醇150ml/静滴，每日两次；生理盐水250ml+信法丁针400mg+10%氯化钾5ml/静滴，8小时一次；生理盐水250ml+醒脑静针400（应为“40”）ml+10%氯化钾5ml/静滴，8小时一次。告病危。15时神经外科会诊记录：患者神志不清，深昏迷状态，双侧瞳孔不等大，右侧瞳孔扩大，（直径）0.5cm,光反射消失，左侧瞳孔（直径）0.3cm，光反射迟钝，四肢生理反射及病理反射均未引出。请示上级医师后认为：病情发展迅速恶化，手术风险极大，建议神经内科保守治疗。15时30分，患者昏迷，检查：左侧瞳孔（直径）0.15cm，右侧瞳孔（直径）0.4cm，对光（-）。呼吸深长，20次/分，心率110次/分，血压170/80mmHg。继续补液，再告病危。5%葡萄糖水250ml+可拉明4支+洛贝林4支/静滴。留置导尿。7月9日9时，神经外科会诊记录：深昏迷，GCS评分4分。复查头颅CT：右颞出血约50ml，破入诸脑室，铸型，脑疝。继续保守治疗。11时41分，神经外科会诊记录：目前患者病情危重，手术风险极大且不能有效缓解病情，无手术指征，继续保守治疗。2007年7月9日12时48分患者转入华山医院继续治疗，经抢救无效于7月12日凌晨1时宣告临床死亡。吕某、潘某、郁某曾于2009年6月向法院起诉与某医院的医疗服务合同纠纷一案，2010年6月30日法院依法裁定驳回吕某、潘某、郁某的起诉并由某医院承担鉴定费7,000元。
　　本案在前次诉讼中，经吕某、潘某、郁某申请，法院委托上海市杨浦区医学会对双方的医疗争议进行鉴定，结论为：本病例不属于医疗事故。并分析意见为：1、患者系急性脑出血，诊断明确，且出血量大，破入脑室，病情急剧恶化，病人的死亡是疾病本身的结果。2、院方在该患者接诊及治疗过程符合诊疗规范、常规，并无明显不当。3、医院首诊医师安红艳的医师资质问题，不属于医疗事故鉴定的范畴，对此医学会在鉴定过程中作过告知。吕某、潘某、郁某不服提出复核申请，法院委托上海市医学会对本次医疗争议进行复核鉴定，结论为：本病例构成一级乙等医疗事故，某医院承担轻微责任。并分析意见为：1、2007年7月8日，患者因“意识丧失15分钟伴头晕、左侧肢体无力1小时”至某医院急诊。根据送检材料，医方在接诊后给予患者必要的检查，通过头颅CT明确脑出血诊断，给予患者甘露醇降颅压等治疗措施，符合诊疗常规。2、患xxx病情急速变化，很快发展为深昏迷、脑疝，GCS评分4分，病情危重，在此情况下无手术指证，故采用保守治疗符合治疗规范。3、患者既往有高血压史，入院时血压达170/90mmHg，在监测下使用降压药不违反临床降压药使用规范。4、医方接诊后对于“脑卒中”患者的风险没有足够重视，未给予患者卧床处理，不利于病情稳定，不排除与患者病情加重可能有一定的因果关系。5、患者在离开某医院时处于临床不能恢复的昏迷状态。后吕某、潘某、郁某起诉要求某医院支付医疗费6,107.80元、丧葬费23,376元、死亡赔偿金636,760元、吕某的抚养费93,514元、精神损害抚慰金50,000元并承担本案诉讼费。
　　原审法院经审理后认为，医疗过失违约责任的承担，是以医方的医疗行为存在不符合医疗合同的约定、未履行其应尽注意义务的事实为基本条件。患者潘苏荣至某医院神经内科急诊并付费挂号，双方由此形成了医疗服务合同。依法成立的合同，受法律保护。双方当事人均应按约定全面履行自己的义务。而本案中某医院的医疗行为是否存在过失，导致了患者的不良后果，应当由医学会予以鉴定。本案医疗争议经区、市两级医学会鉴定，对于是否构成医疗事故的结论不同。根据《医疗事故处理条例》的规定，当事人对首次鉴定结论不服的，可以提出再次鉴定的申请。同时，法院对于委托医学会进行医疗纠纷鉴定的相关规定也明确：经区县医学会鉴定后，当事人仍有异议，可委托市医学会复核鉴定。故市医学会作为再次鉴定、复核鉴定机构，其出具的鉴定结论更具有权威性，应当被采纳。吕某、潘某、郁某对市医学会鉴定结论中某医院的责任承担有异议，但无证据证实市医学会鉴定结论存在错误，故法院对市医学会的鉴定结论予以采纳。由此可见，某医院履行合同义务存有瑕疵，给患者造成了损失，损失赔偿额应当相当于因违约所造成的损失。根据上海市医学会的鉴定结论，某医院应对吕某、潘某、郁某的损失承担30%的赔偿责任，某医院并表示自愿赔偿对方精神损害抚慰金25,000元。原审法院在审核了吕某、潘某、郁某主张的损失后判决：某医院于判决生效之日起十日内赔偿吕某、潘某、郁某医疗费2,127.26元，丧葬费7,013.55元，精神损害抚慰金25,000元，合计34,140.81元。负有金钱给付义务的当事人，如果未按判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。案件受理费11,761元(吕某、潘某、郁某已预交)，减半收取计5,880.50元，由吕某、潘某、郁某，某医院各半负担；鉴定费7,000元，由某医院负担。
　　原审判决后，吕某、潘某、郁某不服，向本院提起上诉，要求撤销原判，改判支持其原审诉讼请求。吕某、潘某、郁某上诉称，某医院安排进修医生独立接诊，病历出现多处错误，并且出现用错药，医学会的鉴定结论与事实不符。显然，某医院在履行医疗服务合同时存在瑕疵，某医院在对患者急救过程中存在过错，与患者的病情恶化直至死亡之间存在因果关系，故某医院应负全部责任。
　　被上诉人某医院则不同意上诉人的上诉请求，请求二审法院驳回上诉人的上诉请求，维持原审判决。
　　本院经审理查明，原审法院认定事实基本无误,本院依法予以确认。
　　本院认为，由于医疗行为是一种具有高度专业性、复杂性并同时具有一定风险性的活动过程，确定医疗机构在对患者的诊疗过程中是否存在过错及该过错与患者目前的损害后果是否存在因果关系是一个相当专业的问题，故需参照具有专业知识、经验、技能的医疗鉴定部门专家的权威意见和评判结论。本案经两级医学会鉴定，复核鉴定结论确定本病例构成一级乙等医疗事故，某医院承担轻微责任。据此，原审法院认定某医院在履行医疗服务合同过程中存有瑕疵，给患者造成了损失，并确定由某医院对吕某、潘某、郁某的损失承担30%的赔偿责任，已经充分考虑了本案的实际情况，本院予以认同。虽然吕某、潘某、郁某二审期间仍然坚持认为某医院在履行医疗服务合同时存在瑕疵，应对患者的死亡承担全部责任，但未能提供充分的证据材料证明其主张。故吕某、潘某、郁某请求二审支持其原审的诉讼请求,本院难以支持。综上，原审判决并无明显不当，本院予以维持。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　上诉案件受理费11,382元，由上诉人吕某、潘某、郁某负担。
　　本判决系终审判决。
审判长孙卫
　　代理审判员洪可喜
　　代理审判员沈卫兵
　　二○一一年九月二十日
　　书记员左莉莉
