张甲等诉乐清市××人××院医疗损害责任纠纷案

审判时间：20111228 法院：浙江省乐清市人民法院 案件字号：(2011)温乐柳民初字第408号


　　原告：张甲。
　　原告：张乙。
　　原告：何甲。
　　原告：叶某某。
　　被告：乐清市××人××院。
　　法定代表人：朱甲。
　　委托代理人：陈甲。
　　委托代理人：施某某。
　　原告张甲与被告乐清市××人××院医疗损害责任纠纷一案，本院于2011年9月26日立案受理，在诉讼过程中本院依职权追加张乙、何甲、叶某某为共同原告，依法组成合议庭于2011年12月9日公开开庭进行了审理。原告张甲、张乙、何甲、叶某某、被告乐清市××人××院的委托代理人陈甲、施某某到庭参加诉讼。本案现已审理终结。
　　原告诉称，2011年3月31日，何乙因头痛于早上6时42分到乐清市××人××院治疗，当时何乙神智清楚，但在急诊医生错误诊断下治疗而死。同年6月29日，温州市医疗事故技术鉴定委员会做出鉴定结论，认为被告的行为为医疗事故，在鉴定过程中，被告多次伪造证据意图推卸责任。故原告诉至本院请求判决被告赔偿各项经济损失合计人民币100万元（包某死亡赔偿金613000元、医疗费24500元、鉴定费2500元、交通费用10000元、丧葬费43382元、精神损失费306618元），并承担本案诉讼费用。
　　被告辩称，其同意温州市医学会所作的医疗事故技术鉴定，对鉴定结论认定被告方应负轻微责任无异议，法院应据此来确定被告应承担的赔偿责任；原告主张的赔偿项目及数额不符合法律规定，请法院予以审核再结合被告应承担的责任大小来确认赔偿数额；关于医生姓名及有否伪造病历等与本案没有必然的因果关系，即使存在违规行为也只应当由卫生行政部门给予相应行政处罚，而不能推定承担全部赔偿责任。
　　原告在本院指定的举证期限内，提供了下列证据：
　　证据1、原告身份证明，拟证明原告的诉讼主体资格。
　　证据2、被告医疗机构执业许可证，拟证明被告的诉讼主体资格。
　　证据3、死亡证明，拟证明何乙于2011年4月2日死亡的事实。
　　证据4、由被告方分别于2011年4月2日、2011年6月8日制作的陈乙各一份，拟证明两次陈乙内容前后有所删改以及给患者输入丹参酮针且未暂停输入的错误诊疗行为。
　　证据5、由被告方制作的急诊抢救护理评估表，拟证明该表系事后伪造以及与患者专用病历记载不符的事实。
　　证据6、病危通知单、值班某、光盘，拟证明原告从未接到病危通知单以及该通知单系事后伪造的事实。
　　证据7、患者专用病历，拟证明病历被改动以及前后时间倒置的事实。
　　证据8、由乐某市卫某某制作的核实报告、CT检查报告单，拟证明被告方医务人员朱乙没有医生执业资格以及事后为逃避法律责任变更姓名为朱丙的事实。
　　证据9、鉴定费发票，拟证明原告支出鉴定费2500元的事实。
　　经当庭质证，被告对上列证据的真实性均无异议，但对原告据以提出被告方伪造证据推脱法律责任的证明主张不予认可，认为证据5、7要结合使用，证据6确实是事后补填的，证据7时间前后倒置系笔误所致，证据8朱小（晓）章医生是有执业资格的，且与本案没有关联性。
　　被告在本院指定的举证期限内，提供了下列证据：
　　证据1、医学技术分析意见评价表复印件，拟证明患xxx病发展所致，被告方医疗行为确有存在不妥当之处，但与患者死亡无因果关系。
　　证据2、温州市医学会所作的温州医鉴【2011】043号医疗事故技术鉴定书，拟证明患xxx病，被告方在诊疗过程中存在一定的医疗过错，但仅应负轻微责任。
　　经当庭质证，原告对上列证据的真实性持有异议，认为鉴定机构据以作出鉴定意见的病历材料系被告提供，其中部分病历作假，且被告方医务人员朱乙没有执业资格，因此认定被告方仅负轻微责任不客观。
　　结合上列证据及原、被告的质证意见，本院对诉辩双方的争议焦点事实认证如下：
　　1、原告方提供的证据真实性均经被告确认，应予认定。被告方医务人员填写的患者专用病历确有时间前后倒置情况存在，而被告方关于上述错误所作的解释不能令人信服，也违背相关法律规定，因此应认定被告对涉案病历确有存在错写、后补的情形。庭审中被告明确表示病危通知单系事后制作，且未送交到患者家属手中，对此本院予以确认。关于被告方医务人员朱丙是否具有执业资格的问题，其姓名朱乙和朱丙读音相同，且原告并未提供证据证明该朱丙和朱乙并非同一人，也未提供证据证明此朱乙冒用彼朱丙的执业资格进行违法从业，故结合乐某市卫某某出具的核实报告，可以认定被告方医务人员朱丙具有执业资格。
　　2、被告方提供的证据1系复印件，且内容与庭后原告提交的有出入，故本院不予认定。被告方提供的证据2系法定鉴定机构对涉案病例是否属于医疗事故所作的权某某定，其鉴定结论应依法予以确认，鉴定结论中认为被告的医疗行为存在过错的事实，应予认定。原告方对鉴定结论有异议，但未在收到该鉴定书之日起15日内向有关部某某出重新鉴定的申请，现其主张该鉴定结论不客观，本院不予采纳。
　　经审理查明：原告张甲妻何乙因“颈部疼痛伴头痛、呕吐半小时”，于2011年3月31日早上6时55分到乐清市××人××院就诊。经急诊CT报告初步诊断为“颈部疼痛待查”入院，后被告方对患者何乙予以甘露醇针、地塞米松针、丹参酮针静滴及护胃等治疗。经治疗后，患者何乙头痛加剧，神志不清。被告方予以CT复查显示：脑室出血、脑肿胀，请结合临床进一步检查排除血管性病变可能。9时患者何乙被转入ICU治疗，并予甘露醇、白蛋白、止血、护脑治疗处理。10时20分经上级医院会诊后，予转上级医院继续治疗。患者于11时40分到达温州医学院附属第一医院急诊科，经体检：深昏迷，两侧瞳孔某某，光反射消失，无自主呼吸，机械通气辅助呼吸，双侧病理征未引出；急诊头颅CT显示：脑干（延髓段）出血伴脑室系统积血首先考虑，提出脑疝。经抢救无效后，患者何乙于4月2日8时50分死亡。医患纠纷发生后，原告于2011年5月3日向温州市医学会申请医疗事故技术鉴定，温州市医学会于2011年6月29日作出鉴定结论，认为“患xxx病，本次脑出血发病突然，病情进展迅速，发病部位主要位于脑干、延髓生命中枢，其死亡直接原因系自身疾病凶险，目前医疗技术难以有效治疗所致。但医方在整个医疗过程中存在以下医疗过错：1、未能充分结合患者症状与影像学早期异常改变的关系，对不典型脑干出血及其病情发展认识不足；2、初期治疗时使用丹参酮针，此药品虽不会直接导致脑血管破裂出血，但有一定抗凝、抑制血小板活性作用，理论上对脑出血有加重影响。本病例属于一级甲等医疗事故，医方应承担轻微责任。”案经调解无效。
　　另查明，死者何乙系1966年1月8日出生，生前任教于黄某某中学。原告张甲与死者何乙系夫妻关系，原告张乙系死者何乙之女，原告何甲系死者何乙父亲，原告叶某某系死者何乙母亲。除死者何乙外，原告何甲、叶某某还生育一女二子，现均已成年。
　　本院认为，本案患xxx病到被告处就诊，其和被告间形成医疗服务关系，被告作为提供医疗服务的一方，负有提供安全服务基本责任和义务，其医务人员应当对患者采取及时、有效的医疗措施，并将患者的病情、医疗措施、医疗风险等如实告知患者及其家属。根据《最高人民法院关于参照﹤医疗事故处理条例﹥审理医疗纠纷民事案件的通知》的相关规定，适用《民法通则》规定的过错责任赔偿处理原则，本院确定被告应对原告造成的经济损失承担40%的赔偿责任。庭审中原告自认医疗费已实际报销4000余元，且未提供其它医疗费票据，庭后由被告方提供的部分收费收据及处方单显示由患者自费支付的医疗费用为75.51元，对此本院予以确认。原告的其他经济损失，医疗鉴定费按实际发生的2500元计算；死者何乙死亡时年仅45周岁，其死亡赔偿金应按浙江省2010年度城镇居民人均可支配收入27359元的标准计算20年；丧葬费按浙江省2010年度职工平均工资30650元的标准计算半年；考虑到原告病情发展及治疗所需，交通费予以酌情补助2000元。何乙生前需要扶养的人为其父亲何甲和母亲叶某某，而原告何甲、叶某某共生育四个子女，故被抚养人生活费应按照何乙应承担的份额予以计算，原告何甲现年78周岁，原告叶某某即将满74周岁，该二人生活费应按照浙江省2010年度农村居民人均年生活消费支出标准分别计算5年、6年，即8390元／年×（5年＋6年）÷4=23072.5元。何乙的死亡确已造成其近亲属的精神痛苦，应予赔偿精神损害抚慰金，具体数额由本院根据审判实践和被告的过错程度予以确定。据此，依照《中华人民共和国侵权责任法》第五十四条、第五十五条、第五十八条、第六十一条、《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》第十七条第一款和第三款、第十八条、第十九条、第二十七条、第二十八条、第二十九条、第三十一条、第三十五条的规定，判决如下：
　　被告乐清市××人××院应赔偿原告张甲、张乙、何甲、叶某某医疗费75.51元、鉴定费2500元、死亡赔偿金547180元、丧葬费15325元、交通费2000元，合计经济损失567080.51元的40%，计人民币226832.20元。
　　被告乐清市××人××院应赔偿原告何甲、叶某某被抚养人生活费23072.5元的40%，计人民币9229元。
　　被告乐清市××人××院应赔偿原告张甲、张乙、何甲、叶某某精神损害抚慰金人民币50000元。
　　上述款项合计286061.20元限本判决生效之日起十日内交本院柳市人民法庭转付。
　　四、驳回原告张甲、张乙、何甲、叶某某的其他诉讼请求。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百三十二条的规定，加倍支付延迟履行期间的债务利息。
　　本案受理费5400元，由原告负担3570元，被告负担1830元。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提交副本，上诉于浙江省温州市中级人民法院。
