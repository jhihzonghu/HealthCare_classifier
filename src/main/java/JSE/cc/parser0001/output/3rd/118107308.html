牛仲吉与付现超其他损害赔偿纠纷案

审判时间：20110803 法院：河南省宝丰县人民法院 案件字号：(2010)宝民初字第1394号


　　原告牛仲吉。
　　委托代理人许永强。
　　被告付现超。
　　原告牛仲吉诉被告付现超其他损害赔偿纠纷一案，本院于2010年10月11日受理后，依法组成合议庭，于2010年11月18日、2011年7月15日公开开庭进行了审理，原告牛仲吉及委托代理人牛建德、许永强，被告付现超均到庭参加了诉讼。诉讼过程中，经原告申请，本院于2011年1月17日至3月29日、4月13日至7月8日，两次委托鉴定机构进行司法鉴定。本案现已审理终结。
　　原告诉称，原告之女牛梦佳2010年9月17日至2010年9月23日在被告诊所诊疗期间，因被告医疗原因致牛梦佳死亡，经协商被告同意一次性支付原告3万元赔偿款，并约定了相关责任，协议签订后被告支付原告2万元现金，剩余一万元拒不支付，依据协议约定，被告一次性支付原告3万元赔偿款，原、被告如一方违反本协议，则支付对方违约金5万元，而被告违反了协议约定，应承担违约责任。请求判令被告立即支付原告赔偿款10000元，并支付原告违约金50000元，以上共计60000元，诉讼费由被告承担。
　　被告辩称，被告与牛仲吉因医疗纠纷经县卫生局调解，双方达成协议，支付牛仲吉3万元。当时经济紧张给付2万元，下余1万元经牛仲吉本人同意写有欠条，口头约定回去后安置。回去后被告积极准备，并与中间人一起找牛仲吉偿还，怎么算违约呢？从打协议到起诉仅10余天时间，需要一个合理的准备时间。被告认为不构成违约。愿偿还下欠的1万元，其他诉讼请求，请求法庭依法驳回。
　　原告向本院提供的证据材料有：1、原告身份证复印件一份，证明原告的身份情况。2、协议书复印件一份，证明原、被告因医疗事故双方达成的赔偿协议。3、欠条复印件一份。4、鉴定费票据二份，证明原告因鉴定花费2300元。5、交通费票据12张，证明原告花费交通费3978元。6、加油费票据1张，证明原告在申请鉴定过程中花费燃油费300元。
　　被告向本院提供证据材料有：收到条一份。
　　经原告申请，本院委托平顶山鹰检司法会计司法鉴定所对被告付现超所提供的收到条进行字迹及指印司法鉴定，该所作出平检司鉴所[2011]文鉴字第12号司法鉴定意见书一份，鉴定意见为“收到条中字迹是牛仲吉所写”；作出平检司鉴所[2011]指鉴字第13号司法鉴定意见书一份，鉴定意见为指印无法检验。因申请人牛仲吉不服该鉴定结论，本院委托中华人民共和国公安部物证鉴定中心对该收到条中字迹及指印重新进行司法鉴定，中华人民共和国公安部物证鉴定中心经鉴定作出公物证鉴字[2011]3029号物证鉴定书一份，鉴定结论为“检材上字迹与样本上牛仲吉字迹是同一人所写”；作出公物证鉴字[2011]3028号物证鉴定书一份，检验意见为“收条三枚（手）印不具备检验鉴定条件”。
　　经庭审质证，被告对原告提供的第1、2、3号证据均无异议，对第4、5、6号证据有异议，认为鉴定结果已证明该收到条是原告书写，鉴定费用被告不应承担。原告对被告所提供收到条认为非原告本人所写,称其所出具的收条中还注明下余10000元应于10日内支付。原告对于两次司法鉴定意见均有异议，但不再申请重新鉴定。被告对于两次司法鉴定意见均无异议。
　　本院根据上述有效证据及当事人的陈述，对本案案件事实确认如下：2010年9月，原、被告因原告之女牛梦佳在被告诊所诊疗问题发生纠纷，后经双方协商，于2010年9月27日签订调解协议书一份，协议第4条约定由被告一次性支付原告3万元，同时第6条约定，双方如一方违反本协议则向对方支付违约金五万元。协议签订后，被告当场支付原告2万元现金，剩余一万元，由被告向原告出具欠条一份，内容为“欠条今欠10000元正（壹万元正）”。后原告以被告超过双方约定的支付欠款时间、构成违约为由，于2011年10月11日提起诉讼。
　　本院认为，原、被告之间签订的调解协议书，是双方当事人的真实意思表示，应为有效协议，原、被告双方均应按协议履行义务。协议中约定被告一次性向原告支付人民币3万元，但并未对支付时间作出约定，被告于签订协议后当场向原告支付了2万元，下余10000元，由被告向原告出具欠条一份，属于原告对被告下欠10000元的债务的认可，双方之间即形成了债权债务关系。债务依法应当清偿，被告应当向原告清偿欠款，故原告要求被告支付下余10000的诉讼请求，本院予以支持。关于原告所诉的违约金问题，因被告所提供的20000元收条已经过司法鉴定证实为原告本人书写，而原告所声称的双方约定10天内还款并已在收条中注明的意见，因不能提供出相关证据予以证实，本院不予采信。
　　故此原告所称起诉时已超过双方约定的还款时间的意见，本院不予支持，故原告要求被告支付违约金的请求，本院不予支持。根据《中华人民共和国民法通则》第八十四条、第一百零八条及《中华人民共和国民事诉讼法》第六十四条第一款、最高人民法院《关于民事诉讼证据的若干规定》第二条之规定，判决如下：
　　一、被告付现超于本判决生效后十日内支付原告牛仲吉赔偿款10000元。
　　二、驳回原告的其他诉讼请求。
　　案件受理费1300元，由原告负担1250元、被告负担50元。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百三十二条之规定，加倍支付迟延履行期间的债务利息。
　　如不服本判决，可在判决书送达之日起十五日内向本院递交上诉状，并按对方当事人的人数提出副本，上诉于河南省平顶山市中级人民法院。
　　审判长朱正栩
　　人民陪审员郜军利
　　人民陪审员潘恒超
　　二○一一年八月三日
　　书记员牛蓁蓁
　　本案所适用的法律依据：
　　《中华人民共和国民法通则》第八十四条债是按照合同的约定或者依照法律的规定，在当事人之间产生的特定的权利和义务关系。享有权利的人是债权人，负有义务的人是债务人。
　　债权人有权要求债务人按照合同的约定或者依照法律的规定履行义务。
　　《中华人民共和国民法通则》第一百零八条债务应当清偿。暂时无力偿还的，经债权人同意或者人民法院裁决，可以由债务人分期偿还。有能力偿还拒不偿还的，由人民法院判决强制偿还。
　　《中华人民共和国民事诉讼法》第六十四条第一款当事人对自己提出的主张，有责任提供证据。
　　最高人民法院《关于民事诉讼证据的若干规定》第二条当事人对自己提出的诉讼请求所依据的事实或者反驳对方诉讼请求所依据的事实有责任提供证据加以证明。
　　没有证据或者证据不足以证明当事人的事实主张的，由负有举证责任的当事人承担不利后果。
