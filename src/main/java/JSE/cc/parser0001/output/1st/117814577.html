韦子昂诉新疆维吾尔自治区中医医院医疗事故损害赔偿纠纷案

审判时间：20110303 法院：新疆维吾尔自治区乌鲁木齐市沙依巴克区人民法院（司法公开示范法院） 案件字号：(2011)沙民一初字第92号


　　原告：韦子昂。
　　委托代理人：李明雪，新疆西域律师事务所律师。
　　委托代理人：姚云，新疆西域律师事务所律师。
　　被告：新疆维吾尔自治区中医医院。
　　委托代理人：王铁军，新疆资本律师事务所律师。
　　委托代理人：吴迎春。
　　原告韦子昂与被告新疆维吾尔自治区中医医院（以下简称自治区中医院）医疗事故损害赔偿纠纷一案，本院于2010年12月20日受理后依法由审判员赵伟适用简易程序，公开开庭进行了审理。原告韦子昂及其委托代理人李明雪、姚云，被告自治区中医院的委托代理人王铁军、吴迎春均到庭参加了诉讼。本案现已审理终结。
　　原告诉称，原告韦子昂于2008年4月25日在乌市十二中学不慎摔伤，遂前往自治区中医院治疗，诊断为左尺骨冠状突骨折，予以石膏外固定。同年5月18日、19日复诊时发现关节脱位。原告于2008年5月20日在新疆医科大学第六附属医院（新疆建工医院）住院治疗，5月27日行左肘关节切开复位内固定术。6月3日原告出院，诊断左肘关节陈旧性脱位；左尺骨冠状突陈旧性骨折。经乌鲁木齐医学会医疗事故技术鉴定，本病例属四级医疗事故，医方承担主要责任。原告的伤经鉴定为十级伤残。原告要求被告赔偿残疾生活补助费27982.65元、医疗费8909.2元、护理费3597.54元、住院伙食补助费735元、交通费1260元、鉴定费680元、精神抚慰金27982.65元；诉讼费由被告承担。
　　被告辩称，对原告所述事实认可，对新疆医学会的责任划分无异议，但我方不同意全额赔偿，愿意在次要责任范围内承担。
　　经审理查明，2008年4月25日原告以左肘疼痛2小时为主诉在自治区中医院骨科门诊就诊，诊断，左尺骨冠状突骨折。自治区中医院予以石膏外固定处理，花费医疗费591.2元。原告于同年5月18日复诊时发现关节脱位。原告于同年5月20日在新疆医科大学第六附属医院（新疆建工医院）住院治疗，5月27日行左肘关节切开复位及内固定术。同年6月3日出院诊断，左肘关节陈旧性脱位；左尺骨冠状突陈旧性骨折。原告又于2009年7月28日至8月4日住院取内固定。二次住院原告花费医疗费8318元。之后原告与被告自治区中医院产生医疗纠纷，双方于2009年2月5日共同委托乌鲁木齐医学会进行医疗事故技术鉴定，乌鲁木齐医学会于同年3月30日出具（2009）11号鉴定报告，认定被告自治区中医院对原告的诊疗过程中不存在违法事实，但存在违规问题：1行石膏外固定后，未按规定及时复查拍片；门诊病历书写形式上告知内容不明确，导致患者未及时进一步治疗的事实；医方违规事实与患者未及时复诊，继续治疗存在因果关系；患者尺骨冠状突撕脱性骨折，易出现肘关节脱位，为治疗增加难度，与患者后续治疗存在一定因果关系。本病例属于四级医疗事故，医方承担主要责任。被告自治区中医院对该鉴定报告不服，申请再次鉴定，自治区医学会出具鉴定报告，认定医方在诊疗过程中存在违规事实，承担次要责任。原告的伤经鉴定为十级伤残，花费鉴定费680元。
　　以上事实有乌鲁木齐医鉴（2009）11号技术鉴定书、新医鉴2009-24号技术鉴定书、住院证明书、住院费用票据、交通费票据及庭审笔录予以证实。
　　本院认为，患者在诊疗活动中受到损害，医疗机构及其医务人员有过错的，由医疗机构承担赔偿责任。存在违规问题：行石膏外固定后，未按规定及时复查拍片；门诊病历书写形式上告知内容不明确，导致患者未及时进一步治疗的事实；医方违规事实与患者未及时复诊，继续治疗存在因果关系；患者尺骨冠状突撕脱性骨折，易出现肘关节后脱位，为治疗增加难度。根据被告自治区中医院的过错大小，本院认为，其应当对原告的经济损失按百分之六十承担较为合适。1.原告所主张的残疾生活补助费有证据证实，但被告应按比例承担；2.因医疗事故处理条列规定，对患者医疗费用的赔偿凭票据支付，但不包括原发病医疗费用，故对原告所主张的医疗费，本院予以部分支持；3.因原告未提供陪护证明，故对其主张的护理费，本院不予支持；4.原告所主张的住院伙食补助费735元，应当按每日25元计算，本院予以纠正；5.原告所主张的交通费1260元过高，本院予以部分支持；6.原告所主张的鉴定费属于诉讼费的承担范畴，本院在诉讼费的分担中予以分配；7.原告所主张的精神抚慰金27982.65元过高，本院予以部分支持。依照《中华人民共和国侵权责任法》第五十四条、《医疗事故处理条例》第四十九条、第五十条之规定，判决如下：
　　一、被告新疆维吾尔自治区中医医院赔偿原告韦子昂残疾赔偿金16789.59元（9327.55元×10%×30年×60%）；
　　二、被告新疆维吾尔自治区中医医院赔偿原告韦子昂医疗费4990.8元（8318元×60%）；
　　三、被告新疆维吾尔自治区中医医院赔偿原告韦子昂住院伙食补助费315元（25元×21天×60%）；
　　四、被告新疆维吾尔自治区中医医院赔偿原告韦子昂交通费500元；
　　五、被告新疆维吾尔自治区中医医院赔偿原告韦子昂精神抚慰金2000元。
　　六、驳回原告韦子昂的其他诉讼请求。
　　本案争议标的71147.04元，给付标的24595.39元，占争议标的的34.6%。案件受理费元1578.68元，本院减半收取789.34元，鉴定费630元，合计1419.34元（原告已预交），由被告负担34.6%即491.1元，由原告负担65.4%即928.24元。
　　以上款项合计25086.49元，被告新疆维吾尔自治区中医医院应于本判决生效后十日内给付原告韦子昂，逾期则加倍支付迟延履行期间的债务利息。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于新疆维吾尔自治区乌鲁木齐市中级人民法院（上诉期届满七日内不交纳上诉费，按放弃上诉处理）。

　　审判员赵伟
　　二○一一年三月三日
　　书记员马媛
