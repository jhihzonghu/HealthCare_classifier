胡顺坤等与汉寿县龙潭桥乡卫生院等医疗损害赔偿纠纷上诉案

审判时间：20110718 法院：湖南省常德市中级人民法院（司法公开示范法院） 案件字号：(2011)常民二终字第81-2号


　　上诉人(原审原告)胡顺坤。
　　上诉人(原审原告)王梅娥。
　　以上二上诉人的委托代理人罗启明，湖南楚为律师事务所律师。
　　被上诉人(原审被告)汉寿县龙潭桥乡卫生院。
　　法定代表人屠阿杰，该院院长
　　委托代理人彭灿。
　　委托代理人周建新，湖南周建新律师事务所律师。
　　被上诉人（原审被告）汉寿县人民医院。
　　法定代表人伍国典，该院院长。
　　委托代理人聂志强。
　　委托代理人周庆文，湖南龙阳律师事务所律师。
　　案由：医疗损害赔偿纠纷。
　　上诉人胡顺坤、王梅娥因与被上诉人汉寿县龙潭桥乡卫生院、汉寿县人民医院医疗损害赔偿纠纷一案，不服湖南省汉寿县人民法院(2009)汉民初字第872号民事判决，向本院提起上诉，请求依法改判。
　　经审理查明，2009年8月1日22时30分左右，胡顺坤、王梅娥之女（李国华之妻）胡雄辉因分娩入住被告龙潭桥卫生院，8月2日1时40分左右，李某某出生，但胡雄辉因产后出血出现昏迷，遂用120急救车转至被告汉寿县人民医院进行抢救，8月2日6时6分，胡雄辉呼吸、心跳停止，6时31分，胡雄辉被宣布临床死亡。事后，李国华等人与龙潭桥卫生院在汉寿县人民医院发生医患纠纷，在李国华不同意尸检的情况下，由汉寿县委政法委、汉寿县政府办、汉寿县公安局、汉寿县卫生局主持调解，胡顺坤、李国华参与调解，李国华等人与龙潭桥卫生院达成《关于胡雄辉医疗纠纷调解协议书》，约定龙潭桥卫生院赔偿李国华等人98000元，并以李国华为代表与龙潭桥卫生院法定代表人周三保在调解协议书上签名，龙潭桥卫生院向李国华支付了赔偿款98000元，后李国华向胡顺坤告知了该调解协议的具体内容，胡顺坤表示同意，两人及其他亲友一起离开了汉寿县人民医院。后胡顺坤、王梅娥认为该调解协议书上仅有李国华签名，其他近亲属并未签名，侵害了其合法权益，不具有法律效力，故诉至法院。
　　本案在审理过程中，经本院主持调解，双方当事人自愿达成如下协议：
　　汉寿县龙潭桥乡卫生院、汉寿县人民医院自愿共同支付胡顺坤、王梅娥因本案医疗事故造成的被扶养人生活费、丧葬费、死亡赔偿金、精神损害抚慰金、误工费、交通费、住宿费等各项损失35000元（上述款项于调解书签收之日支付）。
　　一审案件受理费2619元，按一审判决确定负担。二审案件受理费2619元，免交。
　　上述协议，符合有关法律规定，本院予以确认。
　　本调解书经双方当事人签收后，即具有法律效力。
审判长王国丽
　　审判员熊淑兰
　　审判员周建民
　　二O一一年七月十八日
　　书记员陈小莲
