林峰与梅建国医疗纠纷案

审判时间：20111101 法院：河南省南阳市中级人民法院（司法公开示范法院） 案件字号：(2011)南民一终字第805号


　　上诉人（原审原告）林峰。
　　委托代理人李德霞，邓州市司法局花洲法律服务所法律工作者。
　　上诉人（原审被告）梅建国。
　　委托代理人杨传瑞。
　　上诉人林峰与上诉人梅建国因医疗纠纷一案，均不服邓州市人民法院（2010）邓法民初字第1254号民事判决，向本院提起上诉。本院受理后，依法组成合议庭公开开庭审理了本案。上诉人林峰及其委托代理人李德霞，上诉人梅建国及其委托代理人杨传瑞均到庭参加诉讼。
　　本院经审查认为：原审认定事实不清，适用法律错误。依据《中华人民共和国民事诉讼法》第一百五十三条第一款第（二）、（三）项之规定，裁定如下：
　　一、撤销邓州市人民法院（2010）邓法民初字第1254号民事判决；
　　二、发回邓州市人民法院重新审理。
　　案件受理费5780元分别退还给二上诉人2890元。
审判长郑荣敏
　　审判员刘建华
　　审判员车向平
　　二Ｏ一一年十一月一日
　　书记员张帆
