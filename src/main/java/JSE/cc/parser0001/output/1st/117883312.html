王A因医疗损害赔偿纠纷案

审判时间：20110124 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第9号


　　上诉人(原审原告)王A。
　　法定代理人王B。
　　被上诉人(原审被告)上海市普陀区利群医院。
　　法定代表人陈某，院长。
　　委托代理人赵某某，该医院职工。
　　委托代理人张滨，上海市康昕律师事务所律师。
　　上诉人王A因医疗损害赔偿纠纷一案，不服上海市普陀区人民法院(2010)普民一(民)初字第2229号民事判决，向本院提起上诉。本院依法组成合议庭，公开开庭进行了审理。上诉人王A的法定代理人王B，被上诉人上海市普陀区利群医院(以下简称利群医院)的委托代理人赵某某、张滨到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明：王A既往有高血压病史30年，平时口服降压药物。2010年1月10日晨4时许，王A因出现右侧肢体乏力症状加重，并出现反应迟钝，时有小便失禁前往利群医院急诊。查头颅CT示：双侧基底节区及半卵区腔隙性脑梗死，脑萎缩；左侧枕叶软化灶。当日上午9时30分，王A因“右侧肢体乏力3天，加重半天”入住利群医院中医科病区。入院后检查：神清，气平，查体合作，言语欠清，双侧瞳仁等大，对光反应存在，左侧鼻唇沟变浅，口角略向右歪斜，伸舌略右偏。双肺呼吸音略粗，未闻及干湿罗音。右手握力差，右上肢肌力Ⅲ级，右下肢肌力Ⅳ级，左侧肢体肌力Ⅴ级，四肢肌张力无增高。入院后诊断：脑梗死，高血压3级(极高危)；冠心病，缺血性心肌病型，心功能Ⅲ级；慢性支气管炎。予以活血化瘀、营养脑细胞、控制血压、扩血管、保护胃粘膜等输液治疗。1月11日，经颅多普勒超声检查报告：左右双侧大脑前动脉、大脑中动脉、大脑后动脉、椎-基底动脉血流速度减慢，考虑供血不足；脑动脉硬化。同日，正位胸片检查示：两肺纹理增深，右肺多发性钙化灶。王A有咳嗽、咳痰，下午起低热，吞咽困难。1月12日起加用头孢唑肟抗感染。1月13日，监测王A血压偏高，加用珍菊降压片。1月15日，王A右侧肢体活动不利较前加重，稍有咳嗽，无咳痰，有低热。1月17日，王A稍有低热，予以头置冰袋降温，并维持头孢唑肟抗炎。1月20日，王A的颅脑CT平扫复查示：左侧基底节区脑软化灶，脑萎缩。1月25日，主任查房考虑为避免其感染进一步加重，予以二联使用抗生素，加用奎诺酮类抗生素，结合头孢唑肟加强抗炎疗效。予以喘定平喘、兰苏化痰。停用血栓通及银杏叶药物。复查血电解质发现血钾低，予以10%氯化钾注射液口服。1月27日，王A仍有咳嗽、咳痰不畅。复查胸片示：右下肺炎症，右肺多发性钙化灶。请抗生素使用专家小组会诊后，停用目前抗炎药物，改为四代头孢类药物头孢吡肟抗炎治疗。1月28日，呼吸科对王A会诊。检查王A“两肺呼吸音低，右肺可及湿性罗音”。建议可予以糜蛋白酶、地塞米松雾化吸入。采用雾化治疗后，王A出现气促症状，予以喘定平喘，并停用雾化吸入治疗。2月2日，王A近3日少气懒言，咳嗽、咳痰较前好转。2月3日，王A的胸部摄片复查示：两肺纹理增深，右侧叶间裂增厚。2月6日，对王A停用抗炎药物头孢吡肟。王A稍有肢体麻木，予以神经营养治疗。2月9日，复查血常规提示白细胞仍有增高，故予口服头孢克洛胶囊。2月16日，查房时发现王A尿色较为混浊。检查尿常规后考虑尿路感染，予以膀胱冲洗。2月19日下午，王A体温38.8°C。予以头置冰袋退热。2月20日下午，王A体温38.3°C，停用头孢克洛，改为静脉用来立信加强抗感染。经治疗至2月26日，王A留置导尿颜色明显变清，无明显咳嗽、咳痰，体温正常，稍有吞咽困难。医院同意王A当日出院。出院后王A仍在随访治疗中。王A于2010年4月诉至原审法院，要求利群医院赔偿医疗费人民币(以下币种均为人民币)8,776元、家属误工费1,052元、护理费1,440元、交通费89元、住院伙食补助费1,440元、残疾者生活补助费144,190元、精神损害抚慰金86,514元、打字复印费198元。
　　原审审理中，根据利群医院的申请，法院委托上海市医学会就利群医院对王A实施的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规；其医疗行为与患者王A目前状况之间有无因果关系；明确是否属于医疗事故，等级及责任程度等进行鉴定。上海市医学会于2010年8月12日作出沪医鉴[2010]113号医疗事故技术鉴定书。该鉴定书的分析意见为：1、患者1月10日入院时右侧肢体轻度瘫痪，至1月15日右侧肢体肌力0度，1月20日复查头颅CT，阅片发现左侧基底节区有新的梗死灶。纵观临床症状发展及影像学表现，符合进展性脑梗死，病情呈渐进性加重的特点。2、医方的施治方案符合诊疗常规：对高龄且脑梗死超过4.5-6小时的患者，不能使用溶栓治疗。给予尼莫地平扩血管治疗并无不当。患者既往有高血压史，住院期间医方已给予血压监测及相应降压治疗。3、患者原有慢性支气管炎史6年，每年冬季咳嗽、咳痰加重。入院第二天即有发热、咳嗽，胸片提示两肺纹理增深，符合慢性支气管炎表现，本次系慢性病变急性加重。医方给予抗生素治疗，符合医疗原则。其后病情有加重，经调整抗生素，得以控制，复查胸片提示肺炎吸收。4、因患xxx症状，可以使用二羟丙茶碱(老年人为慎用，并非禁用；常规剂量为1.5-2.0g/日)，医方予每日0.5g静脉滴注，其用法、剂量符合规范，没有不良反应发生。5、依据现有送鉴病历资料，并经鉴定会现场询问，医方使用地塞米松剂量为5mg/次雾化吸入，不违反呼吸科诊疗常规。6、患者目前存在的偏瘫、失语等临床表现系进展型脑梗死的后果，与上述药物处理没有因果关系。7、病史资料中部分住院号和床号书写有错误，地塞米松剂量书写潦草不清，不符合《病历书写基本规范》，医方应注意改进。鉴定结论为：王A与利群医院医疗争议不构成医疗事故。
　　原审法院审理后认为：王A、利群医院双方对王A至利群医院就诊的事实、住院治疗的经过均无异议，法院依法予以认定。法院就利群医院对王A实施的医疗行为是否存在过错、过错程度及因果关系委托上海市医学会进行了鉴定，该医学会作出的医疗事故技术鉴定书已经明确：王A与利群医院的医疗争议不构成医疗事故。虽然王A对鉴定结论的内容持异议，并坚持认为利群医院将其收住入中医科病区违反医疗常规，没有对症治疗；未注意防寒保暖及防止院内感染，违反了护理常规；利群医院对其使用了不该使用的药物等，但未提供充分的证据证明上述观点，亦未提供足以推翻鉴定结论的相关证据，故法院对上海市医学会的医疗事故技术鉴定书予以认定。根据鉴定书分析意见，王A的病状发展符合进展性脑梗死，病情呈渐进性加重的特点。利群医院对王A实施的治疗方案符合诊疗常规，使用药物的用法、剂量均符合规范，且无不良反应，王A的疾病亦得到了一定的控制，王A目前存在的偏瘫、失语等临床表现系其进展性脑梗死的后果，与利群医院使用的药物无因果关系。法院据此认定利群医院对王A施行的医疗行为与王A目前的状况之间不存在因果关系。至于利群医院在病史资料中存在部分住院号和床号的书写错误、药物剂量书写潦草不清等过错，虽然该过错与王A的疾病之间无因果关系，但考虑到利群医院确实存在不足之处，足以引起王A合理怀疑，继而引发本案诉讼，故利群医院应对王A作出适当补偿以衡平双方当事人的利益，具体补偿金额由法院根据本案的实际情况予以酌定。据此，原审法院判决：一、利群医院应于判决生效之日起十日内补偿王A5,000元；二、对王A要求利群医院赔偿各类经济损失的诉讼请求不予支持。
　　原审判决后，王A不服，上诉认为：利群医院违反诊疗常规，将王A安置在病房通道处，而早上开窗进行通风，忽冷忽热致上诉人发烧并得肺炎；利群医院使用冰袋给上诉人降温致并发症；利群医院在上诉人住院的三十六日内仅为上诉人测量了六次血压并将上诉人的血压调控得过低，致上诉人发生脑梗；上诉人系高血压患者，利群医院却给上诉人使用了高血压患者不能使用的地塞米松磷酸纳。利群医院的上述过错行为导致了上诉人目前的病情，并且利群医院伪造篡改病历，也不封存病历，故应由利群医院在本案中承担全部的赔偿责任，要求二审法院撤销原判，依法改判支持上诉人在原审提起的全部诉讼请求。
　　被上诉人利群医院辩称：因病床缺少，被上诉人只能先将王A安置在走廊，但王A得肺炎是由原发的脑梗引发的呛咳所致；冰袋降温系基本的降温治疗，不会因此而导致并发症；被上诉人按医疗规范对王A进行了血压测量，并且降压均在正常范围之内，不会因此引发脑梗；被上诉人是用低剂量雾化的方式对王A使用了地塞米松磷酸纳，王A的情况适合如此使用该药物，鉴定报告对此也作出了说明。综上，被上诉人对王A实施的诊疗行为均符合常规，不应在本案中承担赔偿责任。同意原审法院的判决，要求二审法院予以维持。
　　本院经审理查明，原审法院依据本案在案证据查明的法律事实无误，本院予以确认。
　　本院另查明以下事实：1、在原审审理中，王A提供原在利群医院复印的住院病史并要求如鉴定应以该病史为准。利群医院对此复印件的真实性没有异议。之后，由王A提供了CT片两张、X线摄片三张共计五张影像学片原件。王A要求直接至上海市医学会进行鉴定，利群医院对此表示同意。本节事实，有原审法院2010年4月13日的法庭审理笔录、2010年4月15日的证据交换笔录为证。2、原审法院委托上海市医学会就本案医疗纠纷进行鉴定并向上海市医学会移交了委托函原件一张、王A的病案原件一册、王A的病案复印件一册、影像学片原件五张。本节事实，有沪医鉴[2010]113号医疗事故技术鉴定书为证。3、原审法院在鉴定结论得出后，再次审理本案。审理中，法院询问王A何时因为纠纷而要求利群医院封存病史的？王A回答“就是2月10日，当时复印的就是病史，没有护理记录，被告不肯给我复印。复印件被告还是没有对病史进行封存，没有封存，没有封存就没有记录。法律也没有规定当天复印的就是原始病史”。王A还陈述“法律没有规定用我复印的病史作为鉴定的材料，应该是由被告提供病史进行鉴定”。本节事实，有原审法院2010年9月13日的法庭审理笔录为证。
　　本院认为，医疗行为是一种具有高度专业性、相当复杂性并同时具有一定的风险性的活动过程，故对于本案医疗纠纷是否构成医疗事故或存在医疗差错，尚有赖于具有专业知识、经验、技能的专家作出鉴定。本案中，原审法院根据王A、利群医院的共同意见，委托了上海市医学会对本案医疗纠纷进行鉴定。上海市医学会参考利群医院提供的病案原件、王A提供的病案复印件、王A提供的影像学片五件，听取了王A、利群医院双方的陈述、答辩，作出了鉴定结论。王A上诉认为上海市医学会用于鉴定的病史材料被伪造篡改且没有封存，本院对此认为，根据王A的陈述，其在2010年2月10日向利群医院就诊疗行为提出异议，而王A在利群医院治疗至2月26日方才出院，故若利群医院在2010年2月10日即按王A的要求封存病史，则无法再继续对王A进行诊疗并予以记录。在此情况下，利群医院采取了为王A提供病史复印件的措施。在原审法院委托上海市医学会在对本案医疗争议进行鉴定前，王A要求以其提供的病史复印件为鉴定依据，原审法院也将该病史复印件连同病史原件一并移送上海市医学会作为鉴定材料，而该复印件本身即可为原件是否被篡改作出参照。现王A认为病史材料被伪造篡改，对此并未提供任何证据予以证明。同时，纵观参与鉴定的人员、进行鉴定的过程、程序等方面，王A对鉴定人员的鉴定资质、鉴定中有无存在程序严重违法、鉴定结论是否明显依据不足等情形均未提出异议。因此，本院认为上海市医学会就本案医疗纠纷作出的鉴定结论具有证明效力，可作为认定本案事实的依据。该鉴定结论表明，利群医院对王A实施的施治方案符合诊疗常规，王A与利群医院间的医疗争议不构成医疗事故。但是，利群医院也存有在病史资料书写方面的欠缺，故原审法院鉴于此判决由利群医院酌情给予王A5,000元的补偿。原审法院依据本案的实际情况所作出的判决并无不当，本院依法予以维持。综上，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币4,955元，由上诉人王A负担。
　　本判决为终审判决。
审判长季磊

　　代理审判员郑璐
　　二○一一年一月二十四日
　　书记员王寒
