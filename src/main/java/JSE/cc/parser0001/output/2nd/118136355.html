季国栋与郑州市管城中医院医疗损害赔偿纠纷案

审判时间：20110509 法院：河南省郑州市中级人民法院 案件字号：(2011)郑民二终字第192号


　　上诉人（原审原告）季国栋。
　　被上诉人（原审被告）郑州市管城中医院。
　　法定代表人李显成，该医院院长。
　　委托代理人陶明辉。
　　上诉人季国栋与被上诉人郑州市管城中医院医疗损害赔偿纠纷一案，季国栋于2008年12月9日向郑州市管城回族区人民法院提起诉讼，请求：1、判令被告退还原告医疗费2977.4元。2、判令被告赔偿原告残疾生活补助费59075.4元；3、判令被告赔偿精神损害抚慰金11815元。4、判令被告赔偿原告后期的继续治疗费；5、原告保留由于被告造成的伤害引起并发症和后遗症所需治疗费的继续追偿权；6、判令被告承担本案诉讼费用。季国栋又于2010年3月17日将诉讼请求第二项增加为判令被告赔偿原告残疾生活补助费92617元；第三项增加为判令被告赔偿精神损害抚慰金13255.5元。郑州市管城回族区人民法院于2010年7月20日作出（2009）管民初字第492号民事判决。季国栋不服原判，于2010年10月8日提起上诉，本院于2010年12月13日受理后，依法组成合议庭，于2011年3月4日公开开庭审理了本案，上诉人季国栋、被上诉人郑州市管城中医院的委托代理人陶明辉到庭参加诉讼，本案现已审理终结。
　　原审法院查明，1999年12月29日至2000年1月5日，原告在被告处住院治疗左肾结石。原告住院病历载明：原告主诉为“发现左肾结石三年”；现病史为“三年前感左腰胀痛乏力，查B超发现左肾结石，经用消炎、解疼、利尿、排石等药物治疗无效，现要求体外排石治疗”；临床诊断为“左肾多发性结石、伴中重度左肾积水；诊疗经过为“患者入院后行分次体外碎石术，结石经冲击碎裂后分次排出”；出院诊断为“左肾多发性结石”。原告此次住院共花费2942.4元。出院后，原告于2000年3月5日经通州市人民医院放射线科检查，诊断报告书载明“左肾区只有一2×3cm的不规则致密影，与前片(99.10.26)比较无明显变化，余无异常发现。原告述称经向被告反映，被告承诺再给原告免费治疗一次。原告遂于2000年10月23日至11月3日又在被告处治疗一次。对此次治疗，原告坚持是住院治疗，被告辩称原告并未住院，且无相关的住院病历，而原、被告均予以认可的“2000年11月3日河南省医疗机构统一住院收费专用收据”显示床位费15元、总押金额500元、应退465元。其间，原告于2000年10月31日在郑州市第一人民医院的X线检查报告载明“印象：左肾及左输尿管结石”。2001年1月2日，原告经通州市人民医院B型超声检查，诊断“左肾结石伴积水、左输尿管上段结石伴扩张、右肾未见结石、右输尿管不扩张”。2004年9月28日的通州市人民医院B型超声检查报告单显示原告“左肾长7.2cm、宽3.2cm、前后径3.0cm、诊断为左肾缩小伴结石”。另查明，原告在被告处治疗前的1999年12月13日的通州市人民医院B型超声检查报告单显示原告“左肾长10.3cm、宽5.6cm、前后径4.9cm、诊断为左肾结石伴轻度积水”。2007年10月22日，原告在通州市人民医院的超声检查报告单显示“左肾大小：5.5×2.2×1.9cm，印象：左肾萎缩”。至此，原告述称对被告的治疗行为表示怀疑，遂委托代理人前往被告处复印病历，准备做医疗事故鉴定。2007年9月26日，原告委托代理人耿君与被告签订协议书一份。协议主要约定：一、乙方(被告)鉴于甲方(原告)的经济情况一次性补偿甲方医疗费、交通费、误工费等共计人民币24000元；二、双方签署本协议之日起，即产生效力，乙方给付甲方补偿款24000元，甲方出具收条；三、此协议为一次性处理终了协议，是甲、乙双方真实意志的反映，甲方收取乙方款项后，甲、乙双方的权利及义务终止，双方自愿放弃任何权利请求，包括不得提起医疗事故鉴定，不得向行政部门提起行政解决，不得向人民法院提起诉讼，不得写信向各级行政部门和媒体反映情况；四、协议一式二份，甲、乙双方各持一份。协议签订后，被告于当日向原告支付24000元人民币。后原告认为该协议明显损害原告的合法权益且系代理人超过代理权限所签，故不予认同，并向法院提起诉讼。协议签订的当日，原告委托代理人向被告出具载明两份内容的收条一张。一份载明“今收到郑州市管城区中医院支付季国栋的补偿款24000元，今收人季国栋的特别授权人耿君”。另一份载明“今收到郑州管城区中医院给的旅差费等1000元，今收人耿君”。原告认为后一份收条是其代理人耿君与被告恶意串通的结果。另，原告委托代理人与被告签订协议时的授权委托书载明“代理权限(1，2仅选一项，□内打√有效)：√□2、特别授权，代为承认、放弃或变更诉讼请求，进行和解、提起反诉或者上诉。□3、诉前调解、调阅复制相关的病历。委托期限：自接受委托之当日始至一审终结止”。该委托书中的□3后内容为手写，委托书的落款时间为2007年9月8日。诉讼中，原告于2009年2月16日向该院提出医疗事故鉴定申请，后由于被告未按规定提交有关医疗事故技术鉴定材料而被医学会终止鉴定。
　　原审法院认为，原告在被告处就诊后双方因故发生纠纷。原告委托代理人耿君在原告的合法授权下与被告在平等协商的基础上签订的协议是双方的真实意思表示，不违反法律、法规的强制性规定，合法有效，原、被告应遵照履行。原告述称该协议系其代理人超出代理权限所签，理由不足。因原告出具的授权委托书上明确写有“诉前调解、调阅复制相关的病历”的权限，且原告诉状中亦称曾委托代理人到被告处复印病历，由此可知此项授权是原告本人的真实意思表示。原告代理人耿君并未超出授权范围。由于原告有权在法律规定的范围内处分自己的权利，该协议的内容亦未损害原告的合法权益。该协议明确约定“此协议为一次性处理终了协议，是甲、乙双方真实意志的反映，甲方收取乙方款项后，甲、乙双方的权利及义务终止，双方自愿放弃任何权利请求”。故原告在收到被告支付的相应款项后又起诉要求被告赔偿，理由及证据不足，该院不予支持。
　　原审法院依照《中华人民共和国民法通则》第六十三条、第八十五条、第八十八条第一款、《中华人民共和国民事诉讼法》第六十四条第一款之规定，经该院审判委员会研究决定，判决如下：驳回原告季国栋的诉讼请求。案件受理费2477元，由季国栋负担。
　　上诉人季国栋上诉称，这起医疗纠纷夹杂着合同(协议)纠纷。要解决医疗纠纷，就要先认定协议是否合法。协议是否合法，又要靠医疗事故技术鉴定来确定协议是否显失公平。如果不属于医疗事故，协议还算公平。如系医疗事故，协议就很不公平。医疗事故技术鉴定成了解决合同纠纷和医疗纠纷的关键。一审中，上诉人申请医疗事故技术鉴定，被上诉人不予配合，拒不交出上诉人的全部病历，导致鉴定无法进行，被上诉人应当承担责任或推定被上诉人有过错。协议书不合法：上诉人没有也不会授权代理人“调解”。现在病历都不真不全，协商只能是恶意串通。至于“双方真实意思表示”，那只能是被上诉人和拿到1000元贿赂款的耿君的真实意思表示。签订协议时违背了诚实信用原则；协议书剥夺了公民的基本权利；限制了患者的救济权利，侵害患者的法定权利；合同主体不适格，医务科、弓慧萍没有医院法定代表人的授权，合同不成立。上诉人在拿到协议书时就要求耿君去解除协议，后又多次写信给南通市司法局、江苏省司法厅、南通市民政局，找耿君的领导机关处理上诉人与耿君的代理纠纷。综上所述，一审中认定事实和适用法律都非常不妥，恳请二审法院查明事实，撤销原审判决，依法改判，赔偿上诉人残疾赔偿金92617元、精神抚慰金13255.5元。
　　被上诉人郑州市管城中医院辩称，季国栋1999年、2000年两次来院治疗，现事隔多年，起诉超过诉讼时效，且季国栋所患病经多家医院治疗，2007年9月26日，上诉人特别授权耿君来诉前调解时，耿君和上诉人保持电话联系，且季国栋实际接到被上诉人24000元的补偿款，事隔一年后，上诉人又推翻该协议，是不诚信的行为，该协议是双方真实意思表示，且为一次性终了协议，应受法律保护。综上，原审认定事实清楚，适用法律正确，请求驳回上诉，维持原判。
　　二审中上诉人季国栋提供江苏省司法厅及南通市司法局信函二件、日记十四页，以证明其向有关部门反映情况及第二次住院治疗的事实。被上诉人质证认为：司法厅的函不是新证据，在一审、二审开庭时未提供；南通市司法局信函明显是复印件，均不予认可。据一审中上诉人提供的2008年6月30日证据，南通市松平法律服务所是由南通市民政局管理，该所是一个社团组织，不能证明耿君没有代理资格。日记是复印件，不能作为证据使用，该日记是上诉人所写，对其真实性有异议。本院对信函的真实性予以认可。被上诉人郑州市管城中医院未提供新的证据。
　　二审查明的事实与一审法院相同。
　　本院认为，上诉人季国栋称其左肾萎缩是被上诉人医疗损害造成的，并称2007年双方签订的协议书不合法，其代理人超越权限调解，协议签订主体不适格，协商中恶意串通，被上诉人院方未能提供第二次住院病历无法进行医疗鉴定，应推定院方有过错，被上诉人应承担赔偿责任。被上诉人郑州市管城中医院称季国栋1999年、2000年两次来院治疗，现事隔多年，起诉超过诉讼时效，且季国栋所患病经多家医院治疗，2007年双方已达成协议，季国栋已接受24000元的补偿，其起诉不应得到支持。本院认为，根据本案审理查明的事实，1999年、2000年期间季国栋先后两次到被上诉人医院治疗左肾结石，2007年9月26日双方就损害赔偿达成协议，且该协议已履行完毕。季国栋所称该协议不合法，但其提供的证据不足，理由不成立。一审认定该协议合法有效并无不当。季国栋在收到被上诉人支付的相应款项后又起诉要求赔偿，理由不足，本院不予支持。综上，本院认为，原审法院认定事实清楚，适用法律正确，判决适当。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费2477元，由上诉人季国栋承担。
　　本判决为终审判决。
审判长常爱萍
　　审判员贾建新
　　审判员黄智勇
　　二○一一年五月九日
