李XX诉上海市浦东新区XX医院医疗损害赔偿纠纷案

审判时间：20110215 法院：上海市浦东新区人民法院（司法公开示范法院） 案件字号：(2010)浦民一(民)初字第23211号


　　原告李XX。
　　委托代理人孙XX，XX律师事务所律师。
　　被告上海市浦东新区XX医院。
　　法定代表人陈XX，院长。
　　委托代理人顾XX。
　　委托代理人顾勤。
　　原告李XX诉被告上海市浦东新区XX医院(以下简称XX医院)医疗损害赔偿纠纷一案，本院于2010年8月9日立案受理后，依法适用简易程序。本案审理过程中，经XX医院申请，本院依法委托上海市黄浦区医学会（以下简称黄浦医学会）对本次医疗争议进行鉴定。本案于2011年1月26日公开开庭进行了审理，上列诉讼参加人均到庭参加诉讼。本案现已审理终结。
　　原告李XX诉称，2009年8月31日，原告遭人殴打后至XX医院就诊，XX医院根据检查诊断原告蛛网膜下腔出血、三叉神经损伤，并将原告收治入院。原告因此住院治疗11天，花去医疗费人民币1万多元（币种下同）。2009年9月17日，上海市公安局损伤伤残鉴定中心根据CT报告得出无蛛网膜下腔出血结论，后原告到上海市浦东新区公利医院、上海司法鉴定科学技术研究所司法鉴定中心均认为无蛛网膜下腔出血。2010年5月27日，原告再次到复旦大学附属华山医院作头颅MRA，得出头颅MRA未见明显异常结论。原告认为，周浦医院在有能力根据CT报告得出结论情况下，故意或者过失得出错误的诊断结论，并对原告使用了大量不必要的价格昂贵的药物，且存在用药错误情况，另外将注射用腺苷钴胺与维生素C用同一容器注射、将注射用腺苷钴胺与葡萄糖液配伍，严重违背了该药的使用配伍禁忌。故起诉要求周浦医院赔偿医疗费13,614.06元、交通费500元、误工费1,803.37元、伙食补助费220元、营养费500元、护理费1,803.37元、精神抚慰金30,000元，合计48,440.80元。
　　被告XX医院辩称，根据患者病史、体格检查（头颅CT），医院得出“颅脑外伤”诊断明确，而颅脑外伤原因考虑蛛网膜下腔出血、左侧面部三叉神经挫伤、多处软组织挫伤。医院治疗方案准确有效，完全符合诊疗常规，不存在误诊误治，故要求驳回原告的诉讼请求。
　　经审理查明：2009年8月31日，原告因被人打伤头面部，自觉头痛、头晕，有恶心至周浦医院急诊。周浦医院急查头颅CT提示蛛网膜下腔出血可能，急诊予以抗炎、止血、醒脑治疗后复查头颅CT提示蛛网膜下腔出血可能。为进一步治疗，周浦医院拟“头部外伤：蛛血”将原告收治入院。查体：神清，应答切题，左侧面部稍红肿，未见明显伤口，双瞳等大等圆，直径3mm，上唇内侧粘膜破损，伴出血，耳、鼻未见明显血迹。胸廓挤压痛阴性。辅检：头颅CT提示蛛网膜下腔出血可能。入院诊断：头部外伤：蛛网膜下腔出血。头面部软组织挫伤。诊疗计划：1、外科护理常规，二级护理；2、完善必要辅助检查；3、监测血压、神志瞳孔；4、暂予以抗炎（头孢匹胺、甲氧苄啶）、活血防止脑血管痉挛（尼莫地平）补液等对症治疗，继观病情变化及时对症治疗。2009年9月10日，原告出院，出院诊断：头部外伤：蛛网膜下腔出血、三叉神经损伤、头面部软组织挫伤。2010年5月27日，原告至复旦大学附属华山医院放射科作头颅MRA，诊断报告为：头颅MRA未见明显异常。
　　本案在审理过程中，经被告申请，本院委托黄浦医学会对本次医疗争议进行鉴定。该会出具鉴定结论为：李XX与XX医院医疗争议不构成医疗事故。并分析意见为：根据患者头部外伤史、CT影像学表现结合体检情况，以“外伤性蛛网膜下腔出血”收治入院是可以的。医方在患者的治疗过程中，对抗生素药品的使用上存在用法及用量不当的情况，奥扎格雷钠使用不当，但目前无证明显示医方在药物配伍方面存在问题，无证据表明由于医方因用药不当而对患者造成损害的后果。
　　上述事实由原、被告陈述及原告提供的病历卡、被告提供的黄浦医学会《医疗事故技术鉴定书》各一份等证据证实。
　　本院认为，公民享有生命健康权，原告至周浦医院就诊，双方由此建立了医疗法律关系。医疗损害赔偿责任的承担，是以医疗过失以及过失与损害后果之间存在因果关系为基本条件。根据《医疗事故处理条例》的规定，医疗机构及其医务人员在医疗活动中，违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规等，过失造成患者人身损害的，应当由医学会予以鉴定。本院委托黄浦医学会所作的司法鉴定，既包含了对医疗事故鉴定，也包含了对医疗过错的鉴定，医学会所作的鉴定结论具有客观性和权威性，且符合证据性质，可以作为确认医患双方医疗纠纷过错的依据。现黄浦医学会确认本次医疗争议不属于医疗事故，本院予以确认。根据上述鉴定结论，原告以被告误诊要求被告赔偿，既缺乏事实基础，又于法无据。虽周浦医院在使用抗生素药品上存在用法及用量不当，奥扎格雷钠使用不当情况，但上述不当行为并未对原告造成损害后果，故原告要求周浦医院赔偿的诉讼请求本院难以支持。本院考虑周浦医院在诊疗过程中确有不当情况，故本案鉴定费、受理费均由周浦医院负担。据此，依照《中华人民共和国民法通则》第九十八条、第一百零六条的规定，判决如下：
　　驳回原告李XX要求被告上海市浦东新区XX医院赔偿的诉讼请求。
　　案件受理费1,041元，减半收取计520.50元，鉴定费3,500元，合计4,020.50元，由被告上海市浦东新区XX医院负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
代理审判员宣志慧
　　二Ｏ一一年二月十五日
　　书记员廖浩宇
