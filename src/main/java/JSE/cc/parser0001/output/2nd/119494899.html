侯金X诉上海和X眼科医院有限公司医疗损害赔偿纠纷案

审判时间：20110412 法院：上海市虹口区人民法院 案件字号：(2010)虹民一(民)初字第2362号


　　原告侯金X。
　　被告上海和X眼科医院有限公司。
　　法定代表人苏庆X。
　　委托代理人章冬X。
　　委托代理人刘后X。
　　原告侯金X与被告上海和X眼科医院有限公司（以下简称“和X眼科医院”）医疗损害赔偿纠纷一案，本院受理后，依法组成合议庭，公开开庭进行了审理。原告侯金X、被告和X眼科医院的委托代理人章冬X、刘后X到庭参加诉讼。本案现已审理终结。
　　原告侯金X诉称：本人是高度近视患者。2007年4月至5月，先后在市北医院和市一医院配镜或就诊，当时诊断为轻微白内障。同年5月10日，因左眼视力下降到被告处门诊，诊断为双眼并发性白内障（轻微）、双眼高度近视、后巩膜葡萄肿、黄斑区视网膜萎缩、玻璃体混浊等。检查视力近视2600度，散光200度。术前被告的医生告知原告，术后可以保留左眼300度近视。5月23日行左眼PHACO+IOL术，一周后复查，视力全无。6月14日，被告来电要求我再次复诊时，本人对医生讲过左眼治疗效果不好，右眼就不开刀了，而医生却称他们自有办法。6月18日，按约定去做右眼手术，但术后效果更差，目前双眼视力已不能逆转。由于被告对原告右眼实施手术时存在“人体内植入的医用材料使用同意书”无中文标识的名称、产地、型号；“手术同意书”既无谈话医生签名、日期，亦无手术名称；再者被告在对原告右眼手术时，未尽告知义务，致原告未享知情和选择权。虽被告对原告的左眼手术尚有不当，但那手术的后果属无法预料，而右眼手术时被告不应再使用如左眼同样的人工晶体植入。因此，要求被告赔偿医疗费12,163.70元、交通费982元，并保留后续治疗的权利。
　　被告和X眼科医院辩称，原告所述的右眼手术存在不当与事实不符，因该眼手术不仅有手术指征，且亦征得原告同意，手术去除了右眼白内障，提高了其视力。手术虽与原告的心理预期有差异，但从医生或医学角度而言，已经达到了手术效果。关于6月18日原告的“人体内植入的医用材料使用同意书”无标识之说系无稽之谈。在手术前不仅向原告说明行如同左眼的手术，有原告的签名为证。至于被告未让原告在右眼手术时的“病员告知书”上签名和说明具体的手术时间与方式，系属工作的瑕疵，但瑕不掩瑜。原告双眼术后基本痊愈，目前的视力较差系其高度近视引起，与白内障术后无因果关系，故不同意其后续治疗的赔偿诉请。
　　经审理查明，原告有高度近视史，2005年4月27日至市北医院就医。门诊记载：验光，右眼-26.ODS／-2.ODC×60°→0.05，左眼-26.ODS／-2.ODC×90°→0.05。原告因左眼视力下降加重一年余，于2007年5月1l日至和X眼科医院门诊。检查：双裸眼视力0.08(小孔0.07)，晶体皮质密度增高，玻璃体混浊。高度近视眼底改变，窥不清。诊断：双眼并发性白内障、双眼高度近视。拟行左眼白内障超声乳化+人工晶体置入术(Phaco+IOL)。术前检查，晶体轻度核性混浊。眼B超：左眼玻璃体混浊，视网膜末见脱离。角膜内皮细胞计数2597；角膜厚度520um；闪光VEP基本正常。于5月23日行Phaco+IOL术，置入-5.ODIOL。术后第一天左眼裸眼视力0.15。6月14日原告到被告处复诊。查右裸眼视力0.02(戴镜0.05)，经查眼B超：玻璃体混浊，视网膜未见脱离，后巩膜葡萄肿。角膜内皮细胞计数2292；角膜厚度500um；闪光VEP基本正常。于6月18日行右眼Phaco+IOL术，置入-5.ODIOL。术后第一天裸眼视力0.1。9月20日原告再复诊。查右裸眼视力0.08，矫正视力0.1(-1.50DC×75°)；左裸眼视力0.08，矫正视力0.1(-0.750S／-1.50DC×ll0°)。近用光：右眼+2.50DS／-1.50DC×75°→0.2，左眼+2.OODS／-1.50DC×ll0→0.2°。角膜清，IOL正位，右眼后囊略混，左眼后囊清，玻璃体混浊。小瞳孔下见视网膜后极部、黄斑区萎缩。原告现以其在被告处先后行双眼白内障摘除术存在术后视力与预期差异为由，遂起诉来院要求索赔医疗费12,163.70元，交通费982元，并要求保留后续治疗的权利。审理中，原告要求交通费由法院酌情判处，其余坚持原诉讼请求。
　　另查明：
　　1、2007年5月10日，原告在上海市第一人民医院门诊治疗，医保账户支付医疗费223.50元；2007年5月11日至翌年8月5日，先后多次在被告处就诊，共支付医疗费12,191.70元（其中医保账户支付463.44元、现金支付3,839.27元、附加支付7,888.99元），两者合计12,415.20元；
　　2、原告于2007年5月22日，分别在《病员告知书》、《白内障摘除手术（和/或）人工晶体植入手术同意书》、《人体内植入物医用材料使用同意书》上签名；
　　3、原告于2007年6月18日，分别在《白内障摘除手术（和/或）人工晶体植入手术同意书》、《人体内植入物医用材料使用同意书》上签名；
　　4、2008年8月，本院委托上海市医学会对涉案争议进行医疗事故技术鉴定。沪医鉴[2008]164号《医疗事故技术鉴定书》载明：1、患者“双眼高度近视并发性白内障”诊断成立，有Phaco+IOL手术指征。术前检查及手术治疗符合白内障诊疗常规。2、依据送鉴资料及现场医学检查，患者双眼角膜透明，瞳孔圆，IOL位置正，手术情况良好。3、目前患者远近视力不佳，主要原因是高度近视、黄斑区视网膜萎缩所致。由于后巩膜葡萄肿的存在，导致目标屈光与术后实际屈光值的偏差，属于白内障手术允许范围。4、医院与患者术前谈话不充分，存在沟通不足。该结论为：根据《医疗事故处理条例》第二条第三十三条，本病例不属于医疗事故。为此，被告支付鉴定费3,500元。
　　原告对上述鉴定结论存在异议，于2008年10月20日书面要求市医学会答疑。同年11月12日，市医学会“医鉴办函2008年字(022)号复函”答复如下：l、根据送鉴资料所载，和X眼科医院为侯金X所作的双眼白内障摘除及人工晶体植入术，诊断明确，适用手术方法及术前术中术后处理过程符合诊疗常规。因此不构成医疗事故。术后屈光度的“偏差”在允许范围的提法，依据是目前临床上的共识(现尚无“偏差”的具体数值规定)。人工晶体应置入的屈光度(D)，是根据患者角膜曲率、眼球长轴等基本测量数据代入公式计算所得。目前通用的计算公式是根据大多数正常人眼测量获得的平均数值设定的，所以必然有个体差异形成的“偏差”，也无法规定一个法定的偏差值。至于眼球长轴过长(高度近视)或过短(高度远视)者，至今无准确的计算公式，误差常更大。2、“两眼都放了同型号的人工晶体”的确是事实。临床上双眼白内障手术，为了保证两眼视力的协调，双眼尽量选用同一“牌号”的人工晶体。如两眼测量结果正好需植入同样屈光度的晶体(例本患者根据术前角膜曲率、眼球长轴等检查数据，经公式计算双眼均为-0.5D)，也是常见的。3、关于：“术前谈话不充分”的表达：是指医院虽然已涵盖了病情严重程度、手术方案、可能的后果，也有患者知情同意签字。但医方对此视力低下患者术后可能出现的严重视力不良、预测与实际的偏差、术后戴老花镜等“个体特殊性”谈话强调不够。
　　上述事实，由原告提供的《上海市城镇职工基本医疗保险门急诊就医记录册》、医疗费专用收据；被告提供的原告手术记录册（含原告所有签名的告知书、同意书等）和沪医鉴[2008]164号《医疗事故技术鉴定书》、市医学会“医鉴办函2008年字(022)号复函”以及庭审笔录等佐证，并对上述证据进行质证，本院予以确认。
　　审理中，原告对被告提供的落款日期为“07.5.22.”的《白内障摘除手术（和/或）人工晶体植入手术同意书》和《人体内植入物医用材料使用同意书》及无署期的《白内障摘除手术（和/或）人工晶体植入手术同意书》的签名进行笔迹鉴定，以确定是否其亲笔所写。司法鉴定科学技术研究所司法鉴定中心于2010年11月15日出具的司鉴中心[2010]技鉴字第717号《鉴定意见书》载明：经综合评断，需检3处签名与样本签名，两者笔迹特征符合点数量多、价值高，特征总和均反映了同一人的书写习惯。结论为，需检的3处“侯金X”签名均是侯金X所写。为此，原告支付鉴定费2,000元。双方经对上述鉴定结论质证，均无异议，本院予以确认。
　　审理中，双方各执己见，调解不成。
　　本院认为，原告系双眼高度近视并发性白内障患者，有行Phaco+IOL手术指征，被告对原告的术前检查及适用手术方法等处理符合诊疗常规，术后存在的“屈光偏差”属于白内障手术允许范围。因为，医疗行为是一种具有高度专业性、相当复杂性、一定风险性的活动。对一起医疗纠纷是否构成医疗事故或存在医疗差错，有赖于具有专业知识、经验、技能的专家作出结论。本案所涉原告与被告的医疗争议已经上海市医学会技术鉴定，确认不构成医疗事故。因涉案鉴定程序合法、分析客观、判断合理，在原告目前无证据予以否认或反驳市医学会鉴定结论的情况下，本院对该结论予以确认，并作为本案裁判的考量依据。原告目前视力模糊，主要系其高度近视及黄斑区视网膜萎缩所致，况且，原告后巩膜葡萄肿的存在，导致其目标屈光与术后实际屈光值的偏差，又属于白内障手术允许范围。因此，现原告仅以术后远近视力不佳，或比原先视物更为模糊为由，要求被告赔偿医疗费、交通费及保留后续治疗，显然缺乏事实和法律依据，本院难以支持。但被告在对原告进行右眼手术前存在医患双方沟通不充分和未尽相关的告知义务等瑕疵，本院根据被告所涉行为及其由此造成原告相应后果的原因力，酌情确定赔偿款6,500元。另被告为医疗事故技术鉴定所支付的3,500元和原告为笔迹鉴定所支付的2,000元，系各自为举证而承担必要的诉讼成本，则应自行负担。依照《中华人民共和国民法通则》第一百零六条第三款及《医疗事故处理条例》第四十九条第二款之规定，判决如下：
　　一、自本判决生效之日起10日内，被告上海市和X眼科医院有限公司赔偿原告侯金X6,500元；
　　二、原告侯金X的其余诉讼请求，不予支持。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　本案受理费128.56元，由原告侯金X负担65元，被告上海市和X眼科医院有限公司负担63.56元；本案笔迹鉴定费2,000元，由原告侯金X负担。
　　如不服本判决，可在判决书送达之日起15日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第二人民法院。

　　审判长卞良
　　审判员洪庄花
　　人民陪审员毛福林
　　二Ｏ一一年四月十二日
　　书记员陈菁
