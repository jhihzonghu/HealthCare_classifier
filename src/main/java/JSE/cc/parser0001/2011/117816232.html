甘州区上秦中心卫生院诉中华联合财产保险股份有限公司张掖市甘州区支公司保险合同纠纷案

审判时间：20110519 法院：甘肃省张掖市甘州区人民法院（司法公开示范法院） 案件字号：(2011)甘民初字第1138号


　　原告：甘州区上秦中心卫生院。
　　法定代表人：代宏洲，系该院院长。
　　委托代理人：秦彩云，甘肃方联律师事务所律师。
　　委托代理人：陈莉萍，甘肃方联律师事务所律师。
　　被告：中华联合财产保险股份有限公司张掖市甘州区支公司（以下简称中华联合保险公司甘州支公司）。
　　负责人：高宏，该公司经理。
　　委托代理人：董家炳，系该公司副总经理。
　　委托代理人：唐丽娟，系该公司职工。
　　原告甘州区上秦中心卫生院与被告中华联合保险公司甘州支公司保险合同纠纷一案，本院于2011年3月21日立案受理后，依法由代理审判员于菊瑛独任审理，公开开庭进行了审理。原告委托代理人秦彩云、陈莉萍、被告委托代理人董家炳、唐丽娟到庭参加了诉讼，本案现已审理终结。
　　原告甘州区上秦中心卫生院诉称，2009年8月17日，我与被告签订了医疗责任保险合同，约定我及下属卫生所职业医生发生诊疗事故后，被告以每人赔偿限额100000元向原告支付赔偿金（每次事故绝对免陪2000元或者20%，以高者为限）。2010年8月6日，我处医生蔚明、李霞在给病人张爱花诊疗时，因张爱花对所注射的药物过敏，导致张爱花死亡。8月10日，经甘州区上秦镇人民调解委员会调解，我吗一次性给付张爱花亲属死亡赔偿金、被抚养人生活费等合计120000元。后我们多次找被告协商保险赔偿事宜，但被告只愿意赔偿30000余元，对其余损失拒绝赔偿。现诉讼要求被告支付保险赔偿款50000元。
　　被告中华联合保险公司甘州支公司未作书面答辩，庭审中辩称，原告所述的承保及发生医疗事故均属实。但我们认为原告给死者家属的赔偿标准按照人事损害的赔偿标准予以赔偿不合理，赔偿标准应当按照医疗事故处理条例的赔偿标准进行赔偿。
　　经审理查明，2009年8月16日，原、被告签订了医疗责任保险合同一份，合同约定保险期限为一年，自2010年8月18零时起至2011年8月17日二十四时止。保险对象为原告及下属的村卫生所，投保险种为床位险和门诊险，保险责任为：本保险合同保险单中列明的保险期间和追溯期及承包区域范围内，被保险人的医务人员在从事与其资格相符的诊疗护理工作中，因职业过失造成患者人生伤亡，依照中华人民共和国法律应由被保险人承担经济赔偿责任，并由患者或其代理人在保险期间内首次向被保险人提出索赔申请的，保险人按照本保险合同的约定负责赔偿。原告共计缴纳保险费14248元。其中约定，每人赔偿最高限额100000元，年度赔偿最高限额200000元。并特别约定，每次事故绝对免陪2000元或者20%，以高者为限。合同签订后，2010年8月6日，原告下属的上秦村卫生所医生尉明、李霞在给病人张爱花诊疗时，因张爱花对所注射药物过敏导致张爱花死亡。同年8月9日，经甘州区上秦镇人民调解委员会调解，尉明、李霞与死者丈夫高军达成赔偿协议，二人一次性给付张爱花亲属死亡赔偿金、被抚养人生活费等费用合计120000元。而尉明属于原告投保的参保人员。后原告多次找被告协商保险赔偿事宜，被告与2011年8月10日支付原告赔偿金31709.60元。
　　另查明，死者张爱花，原系甘州区二十里堡村七社朱天玉之女，系张掖市上秦镇高升庵村四社农民，生育两女，长女高小丽，生于1991年7月22日，现年20岁，次女高甜甜，生于1996年3月10日，现年15岁。
　　上述事实，由原、被告当庭陈述、原告提供的保险合同、甘州区党寨镇二十里堡村民委员会出具的证明一份、甘州区公安局二十里堡派出所出具的户籍证明一份、户口本一份、赔偿项目清单一份、被告提交的计算清单一份在卷佐证，并经庭审质证属实，足以认定。
　　本院认为，原、被告签订的保险合同，是双方当事人真实意思表示，应受法律保护。本案争议的焦点是原告要求被告支付赔偿金50000元的理由能否成立。庭审中，原告提出，按照双方签订的保险合同约定，保险人按照《医疗责任保险条款》的约定承担保险责任。我们按照法律规定，给予了死者家属12万元的赔偿，被告应当按照最高赔偿限额8万元予以赔偿。而被告抗辩称：双方签订的保险合同，名称名为医疗责任保险合同，实际就是医疗事故保险合同。发生合同约定的事故后，赔偿标准就是按照《医疗事故处理条例》的标准给予赔偿。双方签订的保险条例中虽然对赔偿标准没有明确的约定，而该保险条款的条款有明示条款和暗示条款，而按照医疗事故处理条例规定赔偿，就是暗示条款，是暗含在保险条款中的，我们在与原告签订保险合同时，也已经给原告明示了。本院认为，原、被告双方签订的保险条款第四条明确约定：在本保险合同保险单中列明的保险期间和追溯期及承包区域范围内，被保险人的医务人员在从事与其资格相符的诊疗护理工作中，因职业过失造成患者人生伤亡，依照中华人民共和国法律应由被保险人承担经济赔偿责任，并由患者或其代理人在保险期间内首次向被保险人提出索赔申请的，保险人按照本保险合同的约定负责赔偿。该保险条款第二十五条约定，发生保险范围责任内的损失，依法应由被保险人承担的经济赔偿责任，保险人按以下方式计算赔偿：（一）对每人人生伤亡赔偿金额（含医疗费用）不得超过本保险合同保险单载明的每人人身伤亡限额。而双方签订的合同明确约定，每人赔偿最高限额100000元，绝对免陪20%，即最高赔偿限额额应为80000元。本次事故发生后，原告给付死者家属赔付12万元，赔偿标准是按照《关于审理人身损害赔偿案件适用法律若干问题的解释》中所规定的各项标准计算的费用。是在法律规定的范围内给予的赔偿。双方签订的合同，对于赔偿标准并没有在合同中明确约定，但保险条款约定，保险人承担的赔偿责任是被告保险人在法律规定的范围内承担的经济赔偿责任。被告主张的依据《医疗事故处理条例》承担赔偿责任，但该条例并非属于法律，原告按照最高人民法院《关于审理人身损害赔偿案件适用法律若干问题的解释》中所规定的标准给予死者家属赔偿，是符合法律规定的，故原告主张对该次事故，要求被告在最高赔偿限额8万元的予以赔偿的理由成立，本院予以支持。保险事故发生后，原告已经给死者于2010年8月10日，赔偿各项损害12万元。原告向被告索赔时，被告已支付原告赔偿金31709.60元，下欠48290.40元，被告应承担给付义务。同时，被告提出，原告的该次事故就是医疗事故，但对该主张，被告无证据予以证明。同时结合最高人民法院《关于参照审理医疗纠纷民事案件的通知》精神，人民法院再处理因医疗事故引起的医疗赔偿纠纷时应当以《医疗事故处理条例》为依据；对于不构成医疗事故的其他医疗侵权纠纷，应当按照民法通则第106条及119条规定处理，而赔偿标准应适用最高人民法院《关于审理人身损害赔偿案件适用法律若干问题的解释》中所规定的标准。对于本案，被告提出按照医疗事故予以赔偿，但不能提供该方面的证据，被告并不能提供原告发生的该次事故时医疗事故，原告给受害人按照人身损害赔偿标准予以赔偿不违反法律和双方签订的保险合同的规定，对原告主张，被告应按照合同约定的最高限额8万元支付赔偿金的理由成立，本院予以支持。保险事故发生后，原告已经给死者于2010年8月10日，赔偿各项损害12万元。原告向被告索赔时，被告已支付原告赔偿金31709.60元，下欠48290.40元，被告应承担给付义务。
　　故依据《中华人民共和国保险法》第六十条、第六十二条一款（四）项、第一百三十五条、第一百三十六条之规定，判决如下：
　　被告中华联合财产保险股份有限公司张掖市甘州区支公司支付原告甘州区上秦中心卫生院保险赔偿款48290.40元，于本判决书生效后10日内履完毕。
　　案件受理费1050元，减半收取525元，由被告负担。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于甘肃省张掖市中级人民法院。
代理审判员于菊瑛
　　二0一一年五月十九日
　　书记员李娟
