陈某某等与义乌××医院医疗事故损害赔偿纠纷上诉案

审判时间：20110325 法院：浙江省金华市中级人民法院 案件字号：(2011)浙金民终字第156号


　　上诉人（原审原告）陈某某。
　　上诉人（原审原告）王甲。
　　两上诉人共同委托代理人（特别甲。
　　上诉人（原审原告）王乙。
　　上诉人（原审原告）王丙。
　　上诉人（原审被告）义乌××医院。
　　法定代表人丁某某。
　　委托代理人（特别乙。
　　上诉人陈某某、王甲、王乙、王丙、义乌××医院因医疗事故损害赔偿纠纷一案，不服浙江省义乌市人民法院（2009）金义民初字第3638号民事判决，向本院提起上诉。本院受理后，依法组成合议庭审理了本案。现已审理终结。
　　原审原告陈某某、王甲、王乙、王丙诉称，患者王丁系陈某某之夫，王甲、王乙、王丙之父。2008年3月13日王丁因“鼻塞、流脓涕6年”入住义乌××医院治疗，当日进行CT检查，提示：慢性副鼻窦炎、鼻息肉，3月15日由该院医生在内窥镜下行鼻息肉摘除术、鼻窦炎开放术，术后病理检查诊断：（左）侧鼻息肉、（右侧）粘膜急慢性炎症伴大量淋巴细胞增生，3月21日诊断鼻息肉、鼻窦某某床治愈出院。后因病情不断加重，于2008年6月11日到义乌市第二人民医院检查，CT提示：右侧上颌窦癌症侵及右侧鼻腔、筛窦伴右侧颈部淋巴转移。2008年6月13日入住宁波市医疗中心李某某医院，确诊为：（右鼻腔）非霍某金淋巴瘤。虽经宁波市李某某医院和浙医一院等全力救治，终因义乌××医院的诊治错误，救治无效，王丁于2008年10月20日在义乌市中医院病逝。期间仅住院治疗花去医药费153231.11元，其中自负9200.02元。原告认为，义乌××医院在对患者术前检查中，CT片Im8层面明确显示“右侧上颌窦外侧壁、前壁骨质破坏”的情况下，没有按常规进行针对性的全面检查，单凭一张片子判定，造成误（漏）诊。术前未做组织活检，术后组织取样不符合规范，在有大量摘除物的情况下，分别只做了一个病理切片，病理检查申请单上诊断结果有多处涂改，事后家属到医院复印病历时，案卷中竟没有病理报告，后到病理科补印一份送检日期为2008年3月14日（实际手术日期为3月15日）的病理报告。而临床医生对患者右鼻腔错误地选择了手术方式，导致肿瘤加速扩散。另外在三次以上出院随访过程中，当患者出现右侧鼻腔渗血水、面部红肿、颈部淋巴结肿大、眼睛睁开困难等明显症状时，主诊医生极端不负责任，错误地认定为术后正常反应，既未在病历上记录、也未采取任何治疗措施。2008年5月25日接诊医生也竟按结膜炎治疗。由于被告的多次诊断治疗错误，导致患者病情迅速恶化，终造成不可挽回结果，给原告带来了巨大精神伤害。被告在对患者执行诊断及治疗过程中存在一系列失误，被告对此主观上有过错，应某担不可推卸的责任，为此，原告诉请：要求被告赔偿医疗费9200元、陪护费22674元、遗属扶养费39600零、住院伙食补助费3900元、丧葬费12959元、交通费3000元、住宿某19500元、参加医疗事故处理和参加丧葬活动的患者直系亲属所需交通费、误工费与住宿某各5000元、精神损害抚慰金200000元，上述共计320833元。基于被告篡改病历、伪造标本，原告要求被告承担上述损失的全部责任。
　　原审被告义乌××医院辩称，原告所述与事实不符，尤其是针对被告过错的陈述。患者在2008年3月14日入往我院治疗，术前CT显示两侧上颌窦、双侧颌窦、筛窦内见大片软组织异常密度充填，鼻中隔稍偏曲，部分骨质吸收，临床医师结合患者77岁的年纪某常期慢性的鼻炎症，判断诊断为鼻窦炎和鼻息肉。并非原告所称的右侧上颌窦外侧壁、前壁骨质破坏的情况，原告所称的上述情况是在2008年6月在二院的CT显示出来的，我院是在3月13日做的CT片。临床医师对患者做出的诊断及措施都是符合医疗规范的，也是与病理相符的，院方没有过错。患者并未按医嘱随诊，2008年5月25日患者是就诊而不是随诊，是患xxx病挂号眼科就诊。被告认为，对患者的整个诊治过程无过错，原告所诉称的篡改病历、伪造标本的事实不存在。原告的诉请不能成立，没有事实和法律依据。
　　原判认定，原告陈某某系死者王丁的妻子，原告王乙、王丙、王甲系死者王丁的儿子。王丁于1932年2月16日出生，系退休教师。2008年3月13日其因“鼻塞、流脓涕6年”入住义乌××医院治疗，当日进行CT检查，检查所见：付鼻窦冠状位CT平扫显示，两侧上颌窦、双侧额窦、筛窦内见大片软组织异常密度影充填；鼻中隔稍偏曲，部分骨质吸收。印象：慢性副鼻窦炎，鼻息肉。2008年3月15日由该院医生在内窥镜下行鼻息肉摘除术＋鼻窦开放术，术后进行病理检查。病理检查申请单中“镜下检查与病理诊断”一栏内书写内容有三处涂改。2008年3月18日，病理诊断报告单载明病理诊断：1、（左侧）鼻息肉。2、（右鼻）粘膜急慢性炎伴大量淋巴细胞增生。2008年3月21日诊断鼻息肉、鼻窦炎治愈出院。2008年5月25日，王丁挂号义乌××医院眼科门诊就诊，该院诊断为结膜炎。后因病情加重，王丁于2008年6月11日到义乌市第二人民医院检查，CT诊断：1、考虑右侧上颌窦Ca侵及右侧鼻腔、筛窦伴右侧颈部淋巴结转移，建议进一步检查。2、左侧上颌窦、筛窦炎症。2008年6月13日入住宁波市医疗中心李某某医院，确诊为（右鼻腔）非霍某金淋巴瘤。后王丁先后在宁波市李某某医院、义乌市第二人民医院、浙医一院和义乌市中医院住院治疗，于2008年10月20日在义乌市中医院病逝。自2008年3月13日以来共计住院130天，其中在义乌住院26天，在宁波市医疗中心李某某医院住院99天，在杭州浙医一院住院5天，花去医疗费153231.11元，扣除已报销的费用后自负9200.02元。2008年12月19日、12月25日，原告王乙和王丙分别向被告借阅蜡块2块和病理切片2片，后由王丙一并归还被告。本案审理过程中，经被告申请，法院委托金华市医学会对被告为王丁的医疗行为有无医疗过错、是否构成医疗事故进行了技术鉴定，金华市医学会认为：1、患者王丁因“双侧鼻息肉，慢性鼻窦炎”于2008年3月13日收住义乌××医院。3月15日在内窥镜下行双侧鼻息肉摘除、鼻窦开放术。6月13日在宁波市李某某医院诊断为：（右）鼻腔非霍某金淋巴瘤（T细胞性）伴颈部、后腹膜淋巴转移。10月20日在义乌市中医院病逝。分析认为：患者王丁（右）鼻腔非霍某金淋巴瘤最终诊断明确。因淋巴瘤恶化是死亡的根本原因。2、义乌××医院术前CT显示（CT号257991）：鼻腔、筛、额、蝶及上颌窦均可见弥漫性软组织肿块充填，上颌窦腔扩大伴多处窦壁骨质破坏。医方在影像学表现不支持单纯性付鼻窦炎及鼻息肉诊断，未考虑恶性肿瘤的可能性；术中对病变观察欠仔细，术后复诊不作病历记录；病理诊断可疑未建议或进行免疫组化检查，诊疗过程中存在过失，造成误诊误治，对患者病程及预后存在一定的影响，应负一定医疗责任。鉴定结论为：本病例属于四级医疗事故，医方承担次要责任。被告交纳鉴定费用2500元。
　　原审法院认为，根据金华市医学会的技术鉴定，被告义乌××医院对患者王丁实施的诊疗行为构成四级医疗事故，因此被告依法应某担相应的民事赔偿责任。一、关于本案的法律适用问题。虽然原告起诉时以医疗过错为诉由向被告主张损害赔偿，但根据《最高人民法院关于参照＜医疗事故处理条例＞审理医疗纠纷民事案件的通知》的规定，本案系因医疗事故引起的医疗赔偿纠纷，应参照国务院《医疗事故处理条例》的有关规定办理。二、关于被告应某担的具体赔偿责任问题。医疗事故损害的赔偿责任，应结合医疗事故的等级、医疗过失行为在医疗事故损害后果中的责任程度、医疗事故损害后果与患xxx病状况之间的关系等因素综合确定，本案王丁死亡的根本原因是淋巴瘤恶化的自身疾病导致，义乌××医院在诊疗过程中存在的过失包括术前CT未考虑恶性肿瘤的可能性，术中对病变观察欠仔细，术后复诊不作病历记录，病理诊断可疑未建议或进行免疫组化检查，造成了误诊误治，对患者病程及预后存在一定的影响，应某担次要的医疗责任。结合被告在病理检查申请单中有涂改行为，违反了《医疗事故处理条例》严禁涂改病历资料的相关规定，故分析以上情形，确认被告承担45％的赔偿责任甲。原告主张被告存在伪造病理标本的事实，依据不足，不予采信。三、关于赔偿项目和标准问题。对照《医疗事故处理条例》，原告的诉请中符合条例规定的赔偿项目及数额有：1、医疗费凭据支付，依原告诉请为9200元（已扣除医保报销费用）；2、住院伙食补助费为26天×20元／天＋104天×30元／天＝3640元；3、陪护费考虑王丁的病情严重程度及治疗后死亡等因素，确定为两人陪护，数额参照医疗事故发生地上一年度职工年平均工资计算为19452元÷12月÷21.75天×130天×2人＝19377元；4、住宿某依据原告提供的宁波市海曙区晶泰旅店的发票，结合患者王丁在宁波医院的住院天数，按照医疗事故发生地国家机关一般工作人员出差住宿补助标准按一人计算，数额为150元／天×99天＝14850元，予以支持；5、参加医疗事故处理的患者直系亲属所需的交通费、误工费与住宿某，计算费用的人数不超过2人，确认原告主张的5000元尚属合理，予以认定。以上五项合计为52067元，按照确定的责任比例，被告应某担的赔偿数额为23430.15元。原告主张的交通费3000元，依据不足，不予支持。根据《医疗事故分级标准（试行）》规定，一级医疗事故系指造成患者死亡、重度残疾、四级医疗事故系指造成患者明某某身损害的其他后果。而本案病例属于四级医疗事故，被告的诊疗行为只是对患者病程及预后存在一定的影响，由此，原告主张的丧葬费、遗属扶养费、参加丧葬活动的患者直系亲属所需交通费、误工费与住宿某、精神损害抚慰金，于法无据，不予支持。依照《中华人民共和国民法通则》第一百零六条第二款、国务院《医疗事故处理条例》第三十四条、第四十九条、第五十条、第五十一条的规定，判决：一、被告义乌××医院赔偿原告陈某某、王乙、王丙、王甲医疗费、住院伙食补助费、陪护费、住宿某、参加医疗事故处理的直系亲属所需的交通费、误工费与住宿某共计52067元的45％计人民币23430.15元。于本判决生效后十日内履行完毕。二、驳回原告陈某某、王乙、王丙、王甲的其他诉讼请求。如果未按本判决指定的期间履行给付金钱义务的，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。案件受理费6112元，由原告陈某某、王乙、王丙、王甲负担5666元，由被告义乌××医院负担446元。医疗事故鉴定费2500元，由被告义乌××医院负担。
　　宣判后，原审原告陈某某、王乙、王丙、王甲，原审被告义乌××医院均不服，向本院提起上诉。
　　陈某某、王乙、王丙、王甲上诉称，一、程序错误。本案一审法院于2009年6月4日正式受理立案，到2010年11月5日通知领取判决书，历时17个月，扣除司法鉴定时间约二个月，已严重超出民诉法规定的一审审理期限，违反法定程序。二、法律适用错误。我方当时是以医疗过错为由诉义乌××医院主张损害赔偿的，而一审法院仅凭医疗事故技术鉴定结论认定赔偿责任乙错误。本案本属因医疗过错引起的医疗损害赔偿纠纷，然而在案件审理过程中义乌××医院未如实提交医疗事故技术鉴定材料，涂改病历（一审法院已认定）、伪造标本，造成医疗事故技术鉴定结论不能真实体现医疗损害的后果，因此，本案不能适用《医疗事故处理条例》。而应按《中华人民共和国侵权责任法》和《民法通则》的规定进行赔偿。三、一审法院认定义乌××医院承担45％的赔偿责任乙缺乏依据。义乌××医院作为一家三级乙等综合性医院，拥有良好的医疗条件和设施，由于其在诊疗过程中责任心缺失，误诊误治导致患者死亡，理应对此承担主要责任。况且义乌××医院还存在蓄意涂改病历和伪造标本的情况，因此本案义乌××医院应某担完全民事赔偿责任。四、一审法院认定的赔偿项目避重就轻，于法无据。由于义乌××医院的严重过错直接导致受害人死亡，理应按照法律规定赔偿丧葬费、遗属抚养费、死亡补偿金、精神损害抚慰金等费用，但一审法院对主要赔偿项目未支付。五、对义乌××医院篡改病历、伪造病理标本的行为应追究相应的法律责任。请求二审法院查某事实，依法改判。
　　义乌××医院答辩称，陈某某、王乙、王丙、王甲的上诉理由不能成立，原审判决适用法律正确，主要赔偿项目清楚，陈某某、王新某某的诉请与事实不符。请求二审法院驳回陈某某、王乙、王丙、王甲的上诉请求。
　　义乌××医院上诉称，一、原判认定陪护费等赔偿项目和标准错误，与法无据，与事实不符。陪护费计算错误，原审判决多算了5329.2元。住宿某的计算标准于法律和事实不符。住宿某的赔偿本意是为了患者在外地医院治疗所产生的直接费用支出。本案中，患者本人在宁波住院接受治疗，患者的长子在宁某某作生活，有固定的住所，没有必要产生住宿某。原审在陈某某方并没有提供任何证据的情况下认定陈某某方参加医疗事故处理的直系亲属所需交通费、误工费及住宿某5000元实属错误。二、原审判决义乌××医院承担45％的责任比例过高。本案经金华市医学会医疗事故鉴定认为，本起事故构成四级医疗事故，院方承担次要责任，而原判以义乌××医院在病理检查申请单中有涂改行为，违反了《医疗事故处理条例》严禁涂改病历资料的相关规定，从而认定义乌××医院承担45％的责任某误。病理检查申请单并非病历资料，病理检查申请单只是医院内部各部门之间用于联系明确检查项目的载体，并非正式的病历资料，原审法院依此确认我方承担45％的责任不当。综上，请求二审法院查某事实，依法改判。
　　陈某某、王乙、王丙、王甲答辩称，一审判决护理费计算正确。关于住宿某，因陪护人都是从义乌过去的，并不可能因为患者的一个儿子在宁某某作生活就不产生住宿某。关于参加医疗事故处理所需的交通费、误工费、住宿某，是有法律规定的，在事实清楚的情况下法院可以认定上述费用，我方来往宁波、义乌、金华多次，必然会产生这些费用。义乌××医院认为病理检查申请单并非病历资料，但根据有关规定，病理申请单是病历资料，有涂改病历应某担不利后果，所以义乌××医院认为45％的比例过大于法无据。综上，义乌××医院的上诉理由不能成立。请求二审法院予以驳回。
　　在二审中，陈某某、王乙、王丙、王甲提供病理会诊咨询意见书一份，证明义乌××医院的病理诊断报告结果和其提供的医疗事故鉴定的病理标本不符合。经质证，义乌××医院认为该证据的形成时间是2008年，如果证据确实存在，对方应在一审中提供，故不属于新证据，且该意见书没有公章，也没有其他证据能证明该证据的真实性，而且该病理切片是不是从我方得到也不能证明，故对该证据的三性均有异议。本院认为仅凭该意见书不足以证明待证事实，对该证据的证明力本院不予确认。
　　义乌××医院在二审中未提供新证据。
　　经审理查某，原审认定的事实清楚，证据确实充分，本院予以确认。
　　本院认为，根据金华市医学会作出的鉴定结论，义乌××医院为患者王丁所作的医疗行为构成四级医疗事故，医方承担次要责任。义乌××医院理应依据法律规定承担相应的赔偿责任。患者在2008年3月到义乌××医院就诊，在同年10月死亡，死亡结果在较短的时间内发生，义乌××医院的诊疗行为与患者的死亡有一定的因果关系，故对丧葬费和精神损害抚慰金应酌情予以考虑。丧葬费为12959元×45％＝5831.55元，精神损害抚慰金酌定为5000元为宜。对于陈某某、王乙、王丙、王甲主张的遗属扶养费，因陈某某有三个子女，而王丁去逝时已年近八十周岁，故不予支持。陈某某、王乙、王丙、王甲认为义乌××医院伪造病理标本，未提供充分证据证明。且一审已充分考虑了本案的实际情况及义乌××医院在此次医疗事故中的过错程度，而判令义乌××医院承担45％的赔偿责任，现陈某某、王乙、王丙、王甲要求义乌××医院承担全部赔偿责任，缺乏法律依据。陈某某、王乙、王丙、王甲的上诉理由合理部分，本院予以支持。陈某某、王乙、王丙、王甲为照顾在宁波治疗的患者王丁而产生住宿某符合客观情况，而参加医疗事故处理的亲属所产生的费用也属必然会发生的合理费用。义乌××医院认为其承担45％责任过高，部分费用不合理的上诉理由不能成立，本院不予支持。据此，根据《中华人民共和国民事诉讼法》第一百五十三条第一款第（三）项，《中华人民共和国民法通则》第一百零六条第二款、国务院《医疗事故处理条例》第三十四条、第四十九条、第五十条、第五十一条之规定，判决如下：
　　一、撤销浙江省义乌市人民法院（2009）金义民初字第3638号民事判决。
　　二、上诉人义乌××医院于本判决生效后十日内赔偿上诉人陈某某、王乙、王丙、王甲医疗费、住院伙食补助费、陪护费、住宿某、丧葬费、参加医疗事故处理的直系亲属所需的交通费、误工费与住宿某共计65026元的45％计人民币29261.7元，精神损害抚慰金5000元，共计34261.7元。
　　三、驳回上诉人陈某某、王乙、王丙、王甲的其他诉讼请求。
　　一审案件受理费6112元，由上诉人陈某某、王乙、王丙、王甲负担3362元，由上诉人义乌××医院负担2750元，医疗事故鉴定费2500元，由上诉人义乌××医院负担；二审案件受理费6112元，由上诉人陈某某、王乙、王丙、王甲负担3362元，由上诉人义乌××医院负担2750元。
　　本判决为终审判决。
