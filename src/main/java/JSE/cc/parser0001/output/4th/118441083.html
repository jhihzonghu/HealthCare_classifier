薛XX、薛X、薛XX、薛XX、薛XX与上海市浦东新区XX医院医疗损害赔偿纠纷案

审判时间：20111115 法院：上海市浦东新区人民法院（司法公开示范法院） 案件字号：(2010)浦民一(民)初字第32167号


　　原告薛XX。
　　原告薛X。
　　原告薛XX。
　　原告薛XX。
　　原告薛XX。
　　上列五原告的共同委托代理人栾XX，上海市XX律师事务所律师。
　　上列五原告的共同委托代理人徐X，上海市XX律师事务所律师。
　　被告上海市浦东新区XX医院。
　　法定代表人黄XX，院长。
　　委托代理人黄XX。
　　原告薛XX、薛X、薛XX、薛XX、薛XX与被告上海市浦东新区XX医院(以下至判决主文前简称XX医院)医疗损害赔偿纠纷一案，本院于2010年10月11日立案受理后，依法适用简易程序。本案在审理过程中，经XX医院申请，本院依法委托上海市黄浦区医学会（以下简称黄浦医学会）对本次医疗争议进行医疗事故鉴定。后五原告不服黄浦医学会鉴定，经五原告申请，本院依法委托上海市医学会对本次医疗争议再次进行医疗事故鉴定。本案于2011年11月9日公开开庭进行了审理，原告薛X、薛XX，五原告的共同委托代理人栾XX，被告XX医院的委托代理人黄XX到庭参加诉讼。本案现已审理终结。
　　原告薛XX、薛X、薛XX、薛XX、薛XX诉称，患者曾XX与原告薛XX系夫妻关系，原告薛X、薛XX、薛XX、薛XX系两人之子女。2009年12月10日，曾XX因咳嗽、胃口不好、脚肿至XX医院治疗，医院以“慢性支气管炎急性发作、肺气肿、肺冠心病”将曾XX收治入院。由于医院未给患者进行必要的检查和随访，在明知患xxx症及电解质紊乱的情况下，未采取针对性治疗，影响了对病情的判断和治疗。医院违规使用抗生素，剂量严重超标，在明知患者为慢性支气管炎急性发作，且近期多次住院治疗，在患xxx病情未有好转时未及时更换抗真菌药物，导致患者肺部感染未能控制，病情恶化。医院在患者二氧化碳潴留时，未积极采取改善通气的治疗措施。由于医院的上述诊疗过程，12月15日，曾XX在出院后死亡，故起诉要求XX医院赔偿五原告医疗费人民币1,024.03元（币种下同）、住院伙食补助费120元、住院陪护费713.20元、交通费1,000元、丧葬费21,396元、精神损害抚慰金125,952元、律师费5,000元，合计155,205.23元。
　　被告XX医院辩称，医院对患者的诊治符合医疗常规，不存在过错，且本次医疗争议经两级医学会作了医疗事故鉴定，结论均不构成医疗事故，故要求驳回五原告的诉讼请求。
　　经审理查明：2009年12月10日，患者曾XX因“咳、痰、喘三十余年，再发加重2周”入住南汇中心医院，诊断为：慢性支气管炎急性发作，慢性阻塞性肺气肿冠心病，心率失常（房室早搏）、心功能IV级。当日医院予以生化检查。予以吸氧、拜阿司匹林、地高辛、头孢替安等药物治疗。12月11日予以生化检查，患者仍存在∏型呼衰，二氧化碳潴留明显，提示呼吸性酸中毒。予以可拉明、洛贝林等处理。12月12日医院又给予患者血常规等常规检查，并在当日进行甲状腺功能检查，予以可拉明、洛贝林等治疗。12月13日给予患者痰培养，结果：真菌生长，草绿色链球菌和卡他奈瑟菌生长。12月14日，患者出现呼吸微弱，神志丧失状况，予以可拉明、洛贝林，麻醉科行气管插管。医院告知患者家属多脏器功能不全，随时有生命危险，经家属同意转ICU治疗。12月15日11时，患者血压测不出，双侧瞳孔直径5mm，神志不清，氧饱和度90%。家属签字后于11：30自动出院。出院诊断：1、肺性脑病，∏型呼衰；2、严重脓毒症，支气管肺炎；3、急性肾功能衰竭；4、慢性支气管炎急性发作慢性阻塞性肺气肿、肺冠心病、心率失常（房性早搏）、心功能IV级；5、原发性甲减。出院后当日患者曾XX死亡。
　　本案在审理过程中，经南汇中心医院申请，本院委托黄浦医学会对本次医疗争议进行医疗事故鉴定，结论为：本病例不属于医疗事故。并分析意见为：经专家组一致讨论认为患者曾XX因长期慢性咳嗽、咳痰三十余年，近日来纳差、尿少、气急就诊而收治入院，医院按慢性阻塞性肺病、肺心病、呼衰、心衰、肾衰，给予相应治疗处理，并与患者家属沟通，告病危。在患xxx病房，患者在诊疗过程中医院所采取药物及诊疗措施符合诊疗规范要求。患者家属放弃诊治自动出院，故患者死亡与医院治疗过程没有联系。医院在患者治疗过程中有需要改进的地方，没有进一步做胸部Ｘ线检查、心超检查等，但这些不足与患者病情演变没有必然联系。诊断中虽没写支气管扩张，但对治疗方案没有影响。XX医院为此支付鉴定费3,500元。
　　嗣后，五原告不服黄浦医学会鉴定结论，经五原告申请，本院依法委托上海市医学会对本次医疗争议再次作医疗事故鉴定，结论为：本次医疗争议不构成医疗事故。并分析意见为：1、据送检资料及鉴定会场询问双方当事人，患者为老年女性，既往有肺结核咳血史，慢性肺部疾病多年，多次住院治疗，平日多素食，多次检查血白蛋白水平低，双下肢浮肿二年。鉴定会场专家阅读2009年9月肺部ＣＴ片，提示双肺间质改变及纤维化、双肺有片状浸润性病灶、左下肺支气管扩张、双侧胸膜明显增厚、纵隔左偏。结合患者病史、体格检查、辅助检查，该患xxx病严重，肺、心功能差。此次在严重疾病基础上出现重症肺部感染（血白细胞明显增高）、呼吸衰竭，入院时伴有低蛋白血症、电解质紊乱、甲状腺功能减退。患xxx病情迅速恶化、最终死亡的根本原因。２、患者住院后医方给予抗感染、化痰、解痉、氧疗、呼吸兴奋剂、利尿等治疗符合诊疗常规。针对呼吸衰竭患者的二氧化碳潴留首选呼吸兴奋剂治疗符合医疗常规，在病情加重时再予机械通气，医方处理不违规。３、患者急性肾功能不全可能系多种因素作用的结果：自身病情进展、缺氧、二氧化碳潴留、药物应用等综合因素。４、患xxx症与其长期营养不良有关，医方未予及时纠正存在欠缺，但与患者死亡无因果关系。５、医方痰培养一天出具真菌生长报告不规范，单凭此报告不能作为使用抗真菌治疗的依据。６、患者住院期间医方未行胸片检查，病史中“肺冠心病”诊断不准确。以上不足应在今后工作中予以改进。五原告为此花去鉴定费3,500元。
　　另查明，原告薛XX系患者曾XX之丈夫，原告薛X、薛XX、薛XX、薛XX系曾XX与薛XX之子女。
　　上述事实由原、被告陈述及被告提供的黄浦医学会《医疗事故技术鉴定书》、上海市医学会《医疗事故技术鉴定书》各1份等证据证实。
　　本院认为，公民享有生命健康权，患者曾XX至XX医院就诊，双方由此建立了医疗法律关系。医疗损害赔偿责任的承担，是以医疗过失以及过失与损害后果之间存在因果关系为基本条件。为查明XX医院医疗活动中是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规等，过失造成患者死亡，本院依法委托黄浦医学会、上海市医学会做了两次医疗事故鉴定，医学会所作的鉴定结论具有客观性和权威性，且符合证据性质，可以作为确认医患双方医疗纠纷过错的依据。现黄浦医学会及上海市医学会均确认本次医疗争议不属于医疗事故，本院予以确认。根据上述鉴定结论，患者曾XX的死亡与其自身疾病有关，XX医院的诊疗行为无过错，故五原告要求XX医院承担赔偿责任，既缺乏事实基础，又于法无据，本院难以支持。本院考虑南汇中心医院诊疗过程中有不足之处，故本案鉴定费、受理费均由南汇中心医院负担。依照《中华人民共和国民法通则》第九十八条、第一百零六条的规定，判决如下：
　　驳回原告薛XX、薛X、薛XX、薛XX、薛XX要求被告上海市浦东新区XX医院赔偿的诉讼请求。
　　案件受理费3,404元（已由原告预交），减半收取计1,702元，鉴定费7,000元（原、被告各预交3,500元），合计8,702元，由被告上海市浦东新区XX医院负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
代理审判员宣志慧
　　二〇一一年十一月十五日
　　书记员廖浩宇
