傅某等与上海某医院医疗损害赔偿纠纷案

审判时间：20110317 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2011)沪一中民一(民)终字第494号


　　上诉人（原审原告）傅某。
　　上诉人（原审原告）周某。
　　上诉人（原审原告）周某1。
　　上诉人（原审原告）周某2。
　　被上诉人（原审被告）上海某医院。
　　上诉人傅某、周某、周某1、周某2因医疗损害赔偿纠纷一案，不服上海市卢湾区人民法院（2009）卢民一(民)初字第471号民事判决，向本院提起上诉。本院于2011年2月11日立案受理后，依法组成合议庭于同年2月23日公开开庭进行了审理。上诉人周某（同时系周某1、周某2共同委托代理人）及四上诉人共同委托代理人王晓霞，被上诉人上海某医院（以下简称“某医院”）委托代理人董希会、陆兴生到庭参加诉讼。本案现已审理终结。
　　原审法院认定，傅某原系患者周某某（已故）的妻子，周某、周某1、周某2系傅某和周某某的子女。2008年8月5日患者周某某因“发现左上腹肿块1年余”入住某医院治疗。入院查体：左侧腹股沟淋巴结无压痛、质韧、活动度差。腹软，无明显压痛及反跳痛，肝脾肋下未及。左上腹可及一肿块，直径10cm，质韧，活动可，边界清。初步诊断：腹部肿块。8月8日腹部CT：肠系膜根部占位；胰周、腹主动脉旁、肠系膜根部多发淋巴结肿大；左侧心膈角处类圆形软组织密度影，肿大淋巴结可能。胆囊壁厚。少量腹水。同日腹部CTA示：腹部动脉起始处钙化伴狭窄；肝左叶小片状低密度灶；左后腹膜占位，与肠系膜上动、静脉关系密切；胰周、腹主动脉旁、肠系膜根部多发淋巴结肿大；左侧心膈角处类圆形软组织密度影，肿大淋巴结可能；胆囊壁厚。8月14日周某某在全麻下行单纯进腹探查术（腹腔巨大肿块切除），手术记录记载：见腹腔内乳白色液体约500ml，较混浊，无臭味。盆腔内有积液，性质同腹腔。小肠系膜、横结肠系膜根部巨大占位性病灶，直径约15cm大小，表面光滑，质中等偏硬，固定。术中切取部分组织送冰冻切片，报告为“小圆细胞恶性肿瘤”，淋巴瘤可能大。经讨论，决定不做进一步手术。关腹，未置引流管。术后周某某转入外科ICU病房治疗。术后予抗感染、补液、对症等治疗。8月16日，周某某转入外科普通病房。8月20日，周某某意识清晰，呼吸困难，给予气管插管，接呼吸机机械通气。DIC全套提示周某某有DIC倾向，告周某某家属病危。8月25日，病理诊断报告示：非霍奇金淋巴瘤，滤泡型3级。免疫表型：B细胞。8月26日17时12分宣告周某某临床死亡。某医院于同日出具的居民死亡医学证明书记载的周某某死亡原因为DIC、非霍奇金淋巴瘤。为治疗周某某上述疾病，花费了医疗费（以下币种均同）6,989.73元（包括现金及帐户支付）。傅某、周某、周某1、周某2(周某2)就某医院的诊疗行为提出异议，对相关病历资料予以复印，并与某医院共同对周某某的病历资料予以封存。傅某、周某、周某1、周某2(周某2)花费病历复印费72元。2008年11月2日傅某、周某支付了聘请律师费用15,000元。
　　原审另查明，在原审审理中，傅某提出某医院提交的病历资料实物封存单上“傅某2008年9、5日”并非其本人所签，并书面申请对上述实物封存单上“傅某2008年9、5日”的字迹进行鉴定，法院委托华东政法大学司法鉴定中心进行笔迹鉴定，华东政法大学司法鉴定中心于2009年5月26日出具文检鉴定意见书（华政司鉴中心[2009]文鉴字第A－181号），鉴定意见为送检《实物封存单》上的“傅某2008年9、5日”字迹与样本材料上的傅某样本字迹系同一人所写。另傅某、周某、周某1、周某2(周某2)认为周某某病历资料中2008年8月14日《病家收信回证》中傅某的签字并非其本人所签，且2008年8月20日至8月25日病程录中上级医师签名处的“季晓频”的签字及日期为同一时间连续书写形成的，并为此向法院提交书面鉴定申请，法院委托华东政法大学司法鉴定中心进行笔迹及综合鉴定，2009年12月14日华东政法大学司法鉴定中心出具文检鉴定意见书（华政司鉴中心[2009]文鉴字第D－181-1号），鉴定意见为:1、送检《病家收信回证》上的“傅某”签名字迹与傅某样本字迹系同一人所写；2、日期为“2008年8月20日－2008年8月25日”、上级医师签名处的“季晓频”签字字迹是否系一次性书写形成，难以结论。傅某、周某、周某1、周某2(周某2)为此支付了鉴定费共计10,000元。傅某、周某、周某1、周某2(周某2)对上述两份文检鉴定意见书的鉴定结论不予认可，某医院则表示均没有异议。
　　在原审审理过程中，某医院提交书面申请要求进行医疗事故技术鉴定，法院于2010年1月15日委托上海市黄浦区医学会就某医院对周某某的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，某医院医疗行为与周某某的死亡是否存在因果关系，明确是否属于医疗事故，事故等级及责任程度进行鉴定。上海市黄浦区医学会于2010年3月31日出具沪黄医鉴[2010]003号医疗事故技术鉴定书,结论为周某某与上海某医院医疗争议不构成医疗事故,该病例不属于医疗事故。傅某、周某、周某1、周某2(周某2)对上述鉴定结论提出异议,提交书面申请要求进行复核鉴定。法院于2010年6月3日委托上海市医学会进行复核鉴定，上海市医学会于2010年8月27日出具沪医鉴[2010]129号医疗事故技术鉴定书，其分析意见为：1、2008年8月5日，患者周某某因“发现左上腹肿块”入住某医院。经影像学检查发现腹部肿块，为明确诊断，进而确定治疗方案，某医院予“剖腹探查术”，手术指征明确。术中取部分组织送冰冻切片，提示淋巴瘤可能，故终止手术，关腹。某医院的诊疗行为不违反诊疗常规。8月25日最终病理报告明确为非霍奇金淋巴瘤；2、根据病理报告中大体标本的描述：术中所取标本大小分别为1cm、0.8cm、0.6－1cm、2cm，故患方认为某医院行“巨大肿块切除”依据不足；3、患者术后发生白细胞增高，出现大量幼稚细胞，血小板减少等表现与淋巴瘤疾病有关，某医院给予了对应的治疗措施。2008年8月20日，患者发生呼吸困难，某医院给予气管插管，呼吸机机械通气等抢救措施，终因疾病迅速恶化，于8月26日抢救无效死亡。疾病本身的严重性是导致患者死亡的根本原因；4、根据送鉴病史资料，入院记录中描述的“左侧腹股沟淋巴结无压痛、质韧、活动度差”未得到上级医师的认同，上级医师多次查房和病史记录均表明未发现肿大浅表淋巴结，故认为患者腹股沟淋巴结肿大依据不足；5、某医院存在病史记录和手术名称欠规范等不足，但与患者死亡无因果关系。结论为患者周某某与某医院的医疗争议不构成医疗事故。傅某、周某、周某1、周某2(周某2)为此支付了鉴定费用3,500元。傅某、周某、周某1、周某2(周某2)对上述鉴定结论提出异议，并要求中华医学会再次进行医疗事故技术鉴定，原审法院通过上海市高级人民法院委托中华医学会进行医疗事故技术鉴定，中华医学会医疗事故技术鉴定办公室于2010年11月29日出具《关于不受理周某某医疗事故技术鉴定的复函》，决定不予受理。傅某、周某、周某1、周某2(周某2)表示上述医疗事故技术鉴定结论不客观、不公正，故不予认可。某医院则对上述医疗事故技术鉴定结论无异议。
　　2009年3月，傅某、周某、周某1、周某2(周某2)诉至法院，要求某医院为其医疗过错行为赔偿医疗费6,989.73元、住院伙食补助费440元、陪护费2,615.07元、交通费1,000元、丧葬费21,396元、死亡赔偿金144,190元，共计176,630.80元的50％，计88,315.40元，并偿付精神损害抚慰金50,000元、律师代理费用15,000元及病史复印费用72元。
　　上海某医院辩称，其在周某某入院后术前准备充分，手术指征明确，术中处置合理，术后监测及处理积极，对周某某的诊疗过程完全符合外科诊疗常规。周某某病情变化及最终死亡，系其本身疾病病情恶化所致，某医院不存在任何医疗过错，也没有隐匿或销毁病史资料的情况，且经两次医疗事故技术鉴定，均确定不构成医疗事故，故表示不同意傅某、周某、周某1、周某2(周某2)的诉讼请求。
　　原审法院审理后认为，基于医疗技术的复杂性、医疗活动的专业性等特性，医疗纠纷中医疗单位有无过错以及医疗行为与损害后果之间有无因果关系均需借助专业、权威的机构进行鉴定。本案中，上海市黄浦区医学会和上海市医学会的分析意见及鉴定结论是医学专家针对专门案件所作的专业意见，具有权威性，可以作为审理本案的相关依据。根据上海市黄浦区医学会和上海市医学会出具的医疗事故技术鉴定书的分析意见，周某某因“发现左上腹肿块”入住某医院。经影像学检查发现腹部肿块，为明确诊断，进而确定治疗方案，某医院予“剖腹探查术”，手术指征明确。术中取部分组织送冰冻切片，提示淋巴瘤可能，故终止手术，关腹。某医院的诊疗行为不违反诊疗常规。8月25日最终病理报告明确为非霍奇金淋巴瘤。周某某术后发生白细胞增高，出现大量幼稚细胞，血小板减少等表现与淋巴瘤疾病有关，某医院给予了对应的治疗措施。2008年8月20日，周某某发生呼吸困难，某医院给予气管插管，呼吸机机械通气等抢救措施，终因疾病迅速恶化，于8月26日抢救无效死亡。疾病本身的严重性是导致周某某死亡的根本原因。鉴定结论为周某某与某医院医疗争议不构成医疗事故。上海市医学会医疗事故技术鉴定书分析意见中虽然有“某医院存在病史记录和手术名称欠规范等不足”的文字记载，但确认与周某某的死亡无因果关系，故对傅某、周某、周某1、周某2(周某2)要求某医院予以医疗损害侵权赔偿的诉讼请求，不予支持。至于傅某、周某、周某1、周某2(周某2)称某医院当时没有发出病危通知书，刻意掩盖真实情况，侵犯了患者及其亲属的知情权，但根据病史资料中的《病家收信回证》，某医院已经告知了周某某的亲属，当时周某某病危，傅某虽然表示该《病家收信回证》上的签字并非其本人所签，但依据华东政法大学司法鉴定中心2009年12月14日出具的文检鉴定意见书（华政司鉴中心[2009]文鉴字第D－181-1号）的鉴定结论，送检的《病家收信回证》上的“傅某”签名字迹与傅某样本字迹系同一人所写，故对傅某、周某、周某1、周某2(周某2)上述主张，不予采纳。据此，原审法院作出判决：驳回傅某、周某、周某1、周某2(周某2)的诉讼请求。案件受理费3,217元，鉴定费用13,500元，共计16,717元，由傅某、周某、周某1、周某2(周某2)负担。
　　原审法院判决后，傅某、周某、周某1、周某2(周某2)不服，上诉至本院称，一审法院未能查明本案事实，某医院采取了不适当的手术方式，存在将部分报告隐匿或者销毁的情况；上海市黄浦区医学会和上海市医学会的两份鉴定书在形式上和内容上均存在重大问题，形式上欠缺鉴定人姓名，内容上不尊重事实；故请求本院撤销原判，依法改判。
　　被上诉人上海某医院则不同意上诉人的上诉请求，请求本院驳回上诉，维持原判。
　　经本院审理查明，原审认定事实无误，本院予以确认。
　　本院认为，由于医疗行为具有较强的专业性，故法院确定医院医疗行为是否存在过错及该过错行为与损害后果之间有无因果关系的主要判断依据来自于医学专家的鉴定结论。本案中，经上海市黄浦区医学会出具医疗事故技术鉴定书，结论为周某某与某医院的医疗争议不构成医疗事故；并且，经上海市医学会复核鉴定，认为某医院的诊疗行为不违反诊疗常规，疾病本身的严重性是导致患者死亡的根本原因，某医院存在的病史记录和手术名称欠规范等不足与患者死亡无因果关系，结论仍为患者周某某与某医院的医疗争议不构成医疗事故。故原审法院以上述鉴定结论作为本案定案依据，对傅某、周某、周某1、周某2(周某2)提出的赔偿诉讼请求不予支持并无不当，所作判决应予维持。上诉人请求本院撤销原判、依法改判，但并未提供充分的事实和法律依据来否定上述鉴定结论，故对其上诉请求，本院难予支持。鉴于某医院在病史记录等方面存在某些不足，故本案医疗事故技术鉴定的相关费用可由某医院承担。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　一审案件受理费3,217元，笔迹鉴定费用10,000元，共计13,217元，由傅某、周某、周某1、周某2(周某2)负担；医疗事故技术鉴定费用3,500元，由上海某医院负担。上诉案件受理费3,217元，由上诉人傅某、周某、周某1、周某2(周某2)负担。
　　本判决为终审判决。
审判长李虎
　　代理审判员沈卫兵
　　代理审判员孙卫
　　二○一一年三月十七日
　　书记员左莉莉
