李宏海、李光辉、程友堂、王菊英与湖州市第三人民医院医疗损害赔偿纠纷案

审判时间：20111020 法院：浙江省湖州市中级人民法院 案件字号：(2011)浙湖民终字第398号


　　上诉人（原审被告）李宏海。
　　上诉人（原审原告）李光辉，诠岱山县衢山镇太平村。
　　法定代理人：李宏海。
　　上诉人（原审原告）程友堂。
　　上诉人（原审原告）王菊英。
　　四上诉人的委托代理人：李堂能（系李宏海父亲）。
　　委托代理人林长寿。
　　被上诉人（原审被告）湖州市第三人民医院。
　　法定代表人钱敏才，该院院长。
　　委托代理人沈月娣，浙江正同律师事务所律师。
　　委托代理人宁丽娟。
　　上诉人李宏海、李光辉、程友堂、王菊英与被上诉人湖州市第三人民医院（以下简称湖州三院）医疗损害赔偿纠纷一案，湖州市吴兴区人民法院于2011年8月22日作出（2011）湖吴民初字第552号民事判决，上诉人李宏海、李光辉、程友堂、王菊英不服该判决而向本院提起上诉，本院受理后依法组成合议庭，于2011年10月13日进行开庭审理，上诉人的委托代理人李堂能、林长寿，被上诉人的委托代理人沈月娣、宁丽娟到庭参加诉讼，本案现已审理终结。
　　原审法院经审理查明：2010年5月24日傍晚18时许，李宏海、李光辉、程友堂、王菊英的亲属程海琴因精神分裂症入住湖州三院进行治疗，同月28日凌晨1时30分许程海琴突发病危，湖州三院立即予以抢救，经抢救无效于凌晨5时15分宣告死亡，死因为猝死，并及时通过其所在镇、村等组织告知李宏海、李光辉、程友堂、王菊英关于程海琴死亡情况。李宏海、李光辉、程友堂、王菊英得知后携众亲友于当日下午3时20分许赶到湖州三院，对医院提出的猝死持有异议，并发生争吵。因争吵行为严重影响到医院正常的医疗秩序，湖州三院向公安机关报警，吴兴区公安分局派警员赶至现场维护秩序。当日下午4时许，湖州市医患纠纷人民调解委员会指派调解员介入纠纷协调，经与李宏海、李光辉、程友堂、王菊英所在地衢山镇人民政府司法所所长齐令男，衢山镇太平村村主任徐坤能（齐令男、徐坤能两同志均系当地调解委员会的人民调解员）召集双方当事人，经过四天的共同协调而达成协议，并于2010年6月1日签订湖州市医患纠纷调解协议书，调解协议内容：“1.医院方一次性补偿患者方99999元。2.双方承诺本协议为最终解决方法，本协议生效后，该医患纠纷终结。3.本协议生效后，患者方今后不再以任何理由另行向医院方主张任何权利。4.如患者方无论以何种形式提出重新处理，患者方同意先将医院方已支付的补偿款99999元返还给医院方，以上内容为双方真实意思的表示，符合法律规定，双方自签字后生效。”签字当日，经李宏海要求并申请，由其向湖州三院领取现金10000元，以办理程海琴火化事宜之用，余款89999元以电汇方式直接汇入岱山县衢山镇村级会计服务中心。2010年12月24日李宏海以协议违反法律、显失公平为由，诉至法院，又于2011年4月18日申请撤诉。2011年5月25日李宏海、李光辉、程友堂、王菊英又以湖州三院的医治行存在严重过错，协议违反法律规定为由，起诉至法院。
　　另查明：死者程海琴系李宏海之妻，李光辉之母，程友堂、王菊英夫妇之女。
　　李宏海、李光辉、程友堂、王菊英的一审诉讼请求为：撤销调解协议书，由湖州三院赔偿死亡赔偿金、抚养费、精神损害抚慰金、丧葬费、交通费、住宿费、误工损失费等共计1324827元，扣除已付的99999元，尚应赔偿1224828元。
　　湖州三院一审答辩意见为：湖州三院对程海琴的诊疗符合诊疗常规，医院不具有过错。双方签订的调解协议合法有效且已履行。李宏海、李光辉、程友堂、王菊英请求撤销与事实和法律规定不符，请求驳回李宏海、李光辉、程友堂、王菊英的诉讼请求。
　　原审法院认为：1.经人民调解委员会调解达成的、有民事权利义务内容，并由双方签字或者盖章的调解协议，具有民事合同性质。当事人应当按照约定履行自己的义务，不得擅自变更或者解除调解协议。本案中，双方当事人因医疗行为而发生争议产生纠纷，该纠纷业经湖州市医患纠纷人民调解委员会调解，双方已达成了具有民事权利义务内容，并经双方当事人签名的调解协议，该调解协议合法有效，受法律保护。2.法律规定，允许当事人一方起诉请求变更或者撤销调解协议，但有责任对自己的诉讼请求所依据的事实提供证据予以证明。本案原告除了提供调解协议外，未提供请求撤销调解协议的相应证据。3.纠纷发生至签订调解协议，期间长达四天，李宏海、李光辉、程友堂、王菊英方完全有时间了解相关法律、法规对医疗纠纷处理的规定后，可拒绝人民调解委员会的调解，直接提起诉讼或进行医疗损害鉴定，但李宏海、李光辉、程友堂、王菊英方未拒绝人民调解委员会的调解，况且双方在达成协议前，李宏海、李光辉、程友堂、王菊英方住所地的镇村两级调解组织的人民调解员也参与协调，如调解协议存在重大误解或显失公平，作为处理纠纷的专业人员调解员，不可能促成双方达成此类调解协议。4.李宏海领取补偿款的前提是确认调解协议有效，虽调解协议第4条约定，允许李宏海、李光辉、程友堂、王菊英方对本纠纷提出重新处理，李宏海、李光辉、程友堂、王菊英方应先将领取的99999元返还医院，但李宏海、李光辉、程友堂、王菊英自提起诉讼要求撤销调解协议至今未将该款返还。5.双方达成的调解协议，不存在具备可撤销的情形，故对李宏海、李光辉、程友堂、王菊英的赔偿请求和赔偿项目不作审查与认定。综上所述，李宏海、李光辉、程友堂、王菊英要求撤销补偿调解协议的诉讼请求，缺乏事实和法律依据，该院不予支持。依照《中华人民共和国民事诉讼法》第六十四条第一款，最高人民法院《关于民事诉讼证据的若干规定》第二条的规定，判决如下：驳回李宏海、李光辉、程友堂、王菊英的诉讼请求。一审案件受理费15849元，减半收取7925元，由李宏海、李光辉、程友堂、王菊英负担。
　　上诉人李宏海、李光辉、程友堂、王菊英不服原审法院上述判决，向本院提起上诉称：原审认定一次性补偿99999元的调解协议有效，不具有合法性。该协议的签订主体明显有误，虽然有死者的丈夫签字，但没有死者的父母在协议上签名确认。湖州三院在医疗活动中，伪造病因，掩盖事实真相，致使患方当事人作出了错误的意思表示。该协议上确定赔偿数额不超守10万元，确定为99999元，与医疗侵权案赔偿费用的依据和计算标准相差甚远，存在显失公平的情形。调解协议约定患方只有先予返还99999元才能反悔的约定并不是前置性约定，而是反悔条款。至于李宏海、李光辉、程友堂、王菊英当地的司法所所长、村长参与调解，因他们属于案外人，他们的行为与本案不存在因果关系，故不能由此影响调解协议的效力。二、原审中湖州三院提供的证明，上诉人均提出了异议，但原审却作为有效证据予以采信。三、本案适用简易程序不妥。李宏海在先前的起诉中，原审法院适用的是普通程序，但在本案初期适用普通程序，后却改变适用简易程序，属程序违法。故请求二审法院撤销原判，依法改判或者发回重审，确认李宏海与湖州三院的调解协议无效。上诉人对诉讼请求中的精神抚慰金从60万元调整至20万元。
　　被上诉人湖州三院辩称：一、一审法院认定事实正确，调解协议合法有效且已实际履行完毕。在签订协议时，上诉人李宏海、程友堂、王菊英均在场，由李宏海作为代表在调解协议上签名。该调解协议经四天才予达成，期间上诉人方有律师参与，也有当地司法所、村调解人员到场，故调解协议是上诉人方的真实意思表示，故该调解协议并不存在无效或可撤销情形。二、湖州三院对于死者程海琴的死亡没有任何过错，程海琴的死亡与被上诉人的医疗行为之间不存在因果关系。程海琴的死因为心脏性猝死，属于意外死亡，被上诉人也进行了及时全力的抢救，故不存在医疗过错行为。三、一审程序合法，本案一审从立案到开庭审理时一直采用简易程序，并没有擅自变更审判程序，上诉人的上诉与一审适用的审判程序不符。故请求驳回上诉人的上诉，维持原判。
　　上诉人在二审期间提供了：1.租赁合同一份，证明死者的身份是城市居民，应以城镇居民标准予以赔偿。2.村长说明一份，用以说明签订协议的过程以及村长不具有代理人身份。被上诉人质证认为庭上提供的租赁合同系复印件，而且上述两证据属原来就已存在的证据，故不应作为二审中的新证据。村长的说明本身说明其参与了调解，而且被上诉人本身并没有伪造证据，故调解协议应属有效。本院认为，本案纠纷的实质在于双方间达成的协议是否有效，村长的说明尚不能证明该协议无效的事实。
　　本院查明的事实与原审查明的事实一致。
　　本院认为：双方当事人对于因程海琴死亡，于2010年6月1日经湖州市医患纠纷人民调解委员会主持调解，作为上诉人之一的李宏海与被上诉人达成了调解协议书一份，并已实际履行该调解协议的事实均无异议。现程海琴的亲属也即上诉人要求撤销上述调解协议的请求是否成立，在于该协议达成时是否存在可撤销的情形。根据《中华人民共和国合同法》第五十四条的规定，订立合同时存在重大误解、显失公平情形或者一方以欺诈胁迫的手段蒙昧无知乘人之危，使对方违背真实意思的情况下订立的合同，受损害方有权撤销合同。
　　本案中，程海琴死亡后，双方在湖州市医患纠纷人民调解委员会的主持下，进行了四天的协调谈判，最终签署了涉案调解协议。从该调解协议达成的时间来看，双方商谈的时间长达四天，具有充分的时间，并不存在情况紧急的情形；从参与的上诉人方人员来看，既包括程海琴丈夫在内的亲属，也有具有一定法律知识水平的上诉人所在地的镇村两级调解组织派出的调解员到场参与处理；从协议的形式来看，该协议属于经人民调解委员会参与协调而形成的医患纠纷调解协议书；从履行情况来看，被上诉人已将调解协议所涉的99999元全部按照上诉人方的要求予以支付，上诉人也并没有在收到款项立即反悔并退回相应款项，协议得到切实有效的执行。故该协议应是双方当事人的真实意思表示的体现，并不存在上诉人在被上诉人或者案外第三人胁迫、欺诈情形下在调解协议上签名的事实。
　　从本案查明的事实来看，程海琴病危突发后，医院对其进行了积极的抢救并和其亲属及时取得联系，从湖州三院提供的病历资料来看，并不存在医疗过错的情形。上诉人认为医院伪造病历缺乏相应的证据和事实，所以不存在被上诉人向上诉人作出医疗赔偿的事实和前提。被上诉人在没有过错的情况下对上诉人方作出适当的补偿，是其自愿处分行为。由于本案并不是医方因其过错而需向患方进行赔偿的纠纷，故并不存在赔偿数额不合理而造成显失公平的情形。
　　此外，本案一审适用的是简易程序，从立案至宣判，一直按简易程序审理，担任该案的审判组织也是审判员独任审判。至于庭审笔录中记载“本案经合议庭评议后择日作出判决”，系笔录制作过程中的笔误，对此二审已提请一审注意庭审笔录制作的规范，防止低级错误的发生，但此瑕疵并没有影响到当事人权利的受限和实体结果的处理，不构成程序违法。
　　综上，上诉人认为医院具有过错而应承担赔偿责任没有事实依据，本院不予采信。双方间的调解协议合法有效，对双方均具有法律约束力，上诉人要求撤销该协议，事实和理由均不能成立，本院不予支持。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项的规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费15849元，由上诉人负担。
　　本判决为终审判决。

　　审　判　长　　杨林法
　　审　判　员　　周　勇
　　代理审判员　　朱　莹
　　二〇一一年十月二十日
