李某诉某医院医疗损害责任纠纷案

审判时间：20110823 法院：上海市卢湾区人民法院 案件字号：(2011)卢民一(民)初字第999号


　　原告李某。
　　委托代理人蒋某，某律师集团上海事务所律师。
　　被告某医院。
　　委托代理人董某、丁某，某医院工作人员。
　　原告李某诉被告某医院医疗损害责任纠纷一案，本院受理后，依法组成合议庭，公开开庭进行了审理。原告李某之委托代理人蒋某、被告某医院法定代表人之委托代理人董某、丁某到庭参加诉讼。本案现已审理终结。
　　原告李某诉称，原告李某系患者严某的丈夫，严某有糖尿病、甲亢病史。2009年2月21日9：50时，严某因身体不适，至被告处就诊，被送至被告的急症病室，当时患者神智清晰，被告诊断后为患者进行输血治疗。当天15：20时的病情诊断为：头部外伤、甲亢、糖尿病。被告为患者进行保守治疗，医嘱为：注意病情变化，有恶心呕吐时、头痛、有意识障碍、有肢体活动障碍及时联系，必要时查头颅CT等。2009年2月22日16：05时，被告测出患者血糖为24.54，遂为患者注射胰岛素，期间未进行其他治疗和诊断。2009年2月23日6：30时，原告发现患者昏迷，被告为患者测量血糖未能测出，当日7：15时，再测血糖为1.18，期间被告未对患者进行有效的抢救，直至当天8：20时，白班医生上班后，患者才进入抢救室进行抢救。当日15时左右，患者因抢救无效宣告死亡。被告出具的《居民死亡医学证明书》载明患者直接死亡原因为颅内出血，但经原告申请尸检的死亡原因为：急性气管、支气管及两肺气管肺炎。鉴于患者入院时神智清晰，但经被告诊治后，患者血糖急剧升高，被告对其使用胰岛素降血糖，未对患者用药情况进行观察，因被告医务人员对患者治疗不当、抢救不及时，导致患者病情恶化直至死亡，且尸检报告与被告出具的《居民死亡医学证明书》死亡原因不一致，说明被告对患者存在严重误诊，故要求被告赔偿原告医疗费人民币4,284元(以下币种均为人民币)、死亡赔偿金636,760元、丧葬费21,394元、住院伙食补助费60元、营养费120元、住宿费180元、家属误工费980元、家属交通费172元、鉴定费3,000元、精神损害抚慰金50,000元、共计716,950元中的30％，计215,085元，并支付原告律师费10,000元。
　　原告李某为证明其上述主张向本院提供了1、严某门急诊就医记录册；2、病危通知单；3、检验报告；4、输血治疗同意书；5、药品明细；6、居民死亡医学证明；7、严某死亡通知书；8、尸检报告单；9、起诉金额清单；10、医药费专用收据；11、严某临时居住证、社会保障卡；12、结婚证、李哲姝户籍资料；13、工资证明、医疗费发票、尸检发票、聘请律师合同。
　　被告某医院辩称，患者严某至被告医院治疗时，其眼眶有淤血，且有甲亢、糖尿病史，血小板计数很低，系危重病人，故其在短时间内死亡，被告认为其死亡原因为颅内出血。经尸检鉴定严某的死亡原因为急性气管、支气管炎及两肺支气管炎，但上述疾病不会在短时间内使患者死亡，被告对鉴定结论持有异议。患者当时入住被告医院的急诊室，院方已告知需家属对患者进行陪伴，医生对患者进行胰岛素注射时，根据患者的情况改变了注射方式，但患者家属发现患者血糖低时未及时通知被告，诊治医生在发现患者血糖低时已注射了葡萄糖并将其送往抢救室，并非等白班医生上班后才对患者进行抢救。原告李某在上海有住房，故被告不应支付其住宿费，丧葬费应按2010年职工平均工资标准的2个月计算，因患xxx病所致，故精神损害抚慰金应按5,000元左右的标准计算。死亡赔偿金因不符合相关的法律规定，故不同意支付。经鉴定确认原、被告之间的医疗争议构成医疗事故，被告承担次要责任，现表示同意支付原告医疗费4,284.63元、住院伙食补助费60元、营养费120元、家属交通费172元、鉴定费3,000元、精神损害抚慰金5,000元及丧葬费等的20％。
　　被告某医院向本院提供了病历资料及化验单据等证据。
　　在本院庭审质证过程中，被告对原告提供证据12不予认可，表示无法证明其与患者之间的关系，医疗费应以原件为准，工资证明的形式不合法，对其余证据均予以认可。原告对被告提供的证据的真实性予以确认。
　　经审理查明，原告李某与患者严某原系夫妻关系。2009年2月21日上午9时，患者严某因乏力、全身浮肿、纳差，昨夜摔倒前往瑞金医院急诊。据急诊病历记载：检查：神清，全身浮肿，左眼淤血，血压90/60mmHG，两肺罗音粗，心率110次/分、律齐。予以检查血常规、肝肾功能、血电解质、血糖14.7mmol/L(参考范围3.90－6.10mmol/L),少浆血、人体白蛋白输注，同时告病重，留观处理。当日下午1时15分患者觉全身不适，检查：血压90/60mmHg,神清，颈软，心率110次/分，律齐，无杂音。请示二值班，指示密切观察。已予以吸氧。下午3时20分神经外科会诊。查体：左眼睑挫伤，眼虹（结）膜充血且水肿显著，神志清，精神软，GCS15分，双侧瞳孔等大等圆，对光反射（＋），直径0.25cm,肌体活动可，肌力V级。诊断：头部外伤；甲亢；糖尿病。建议予以保守治疗，必要时查头颅CT；请眼科会诊治疗虹（结）膜水肿充血。当晚血压100/70mmHg.2月22日晨7时26分检查患者：神清，左眼睑水肿，充血、出血，两肺呼吸音粗，心律齐，双下肢稍水肿。继续予以支持治疗。下午3时43分复测血糖24.54mmol/L。4时5分予胰岛素50单位＋50ml生理盐水中用静脉泵维持10小时。5时20分将胰岛素调整为2ml/小时。患者自觉症状有所缓解。2月23日晨6时25分家人发现患者呼之不应。检查：中度昏迷，双瞳孔直径2.5mm,光反射存在，心率85次/分，律齐，双侧病理征阳性，血压130/70mmHg。先予10％葡萄糖静脉滴注。急测血糖1.18mmol/L，即予50％葡萄糖80m静脉推注。8时20分患者转入抢救室。检查：昏迷，两眼左侧凝视，左眼血肿，两肺门（呼吸音）粗，可闻及湿罗音，心率120次/分，心音弱，血压100/60mmHg。9时30分神经内科会诊示若条件允许，建议行头颅CT检查。继续予以支持治疗，维持生命体征。当日下午2时30分患者突然呼吸减至0，心跳减至50次/分。行胸外心脏按压，心二联、呼二联及肾上腺素2mg静脉推注，呼吸囊加压辅助通气。2时40分气管插管接呼吸机后，予以阿托品2支、肾上腺素2mg静脉推注。但心率仍示为0，再予胸外按压。查大动脉无搏动，瞳孔散大，心电图示一直线，于2009年2月23日下午3时宣告临床死亡。2009年2月23日，被告出具的居民死亡医学证明书载明：严某的死亡原因为颅内出血。
　　2009年2月25日，死者严某被送往第二军医大学病理教研室进行尸体剖验。同年4月13日第二军医大学病理教研室出具尸体解剖报告：病理诊断：1、急性气管、支气管炎及两肺支气管肺炎；2、急性肺淤血，肺水肿；急性肝淤血；腹腔积液400毫升，左、右侧胸腔积液600毫升及450毫升；3、增生性骨髓象：红系、粒系均增生，以晚幼红、中幼性增生为主，偶见巨核细胞；心肌、心内膜多灶散在性出血，胸前部及上腹部皮肤淤斑；4、双侧毒性结节性甲状腺肿；5、右肾单纯性囊肿；6、主动脉、冠状动脉轻度粥样硬化。死亡原因：急性气管、支气管及两肺支气管炎。
　　严某为治疗在被告某医院花费医疗费4,284.63元（现金支付）。严某亲属为处理严某医疗及丧葬事宜亦花费了相应的交通费172元、尸检费3,500元等费用。2009年5月，原告与上海诚达永华律师事务所签订聘请律师合同，并于2011年7月5日支付了律师费10,000元。
　　在本院审理过程中，根据原告的申请，2010年10月27日，本院委托上海市医学会进行医疗事故鉴定，2011年2月25日上海市医学会出具沪医鉴[2011]X号医疗事故技术鉴定书，其分析意见为：1、医方对于危重留观患者病情严重性预判不足，观察不到位（在应用胰岛素静脉泵期间，缺乏必要的定期监测血糖措施），违反现行医疗常规，未能及时发现病情变化以调整治疗措施，该医疗过失与患者的死亡发生有一定的相关性。2、依据送鉴病历及实体解剖资料，患xxx病理性基础（糖尿病、甲状腺功能亢进、重度贫血、血小板减少、严重营养不良、继发肺部感染），严重低血糖发生是诱发患者病情进一步加重的因素，而自身病情的严重性，则是导致不良后果的主要原因。3、患xxx症状与体征，白细胞检查尚属于正常范围，医方采用支持为主的临床处理措施，不违反医疗常规。结论为：严某与某医院医疗争议构成医疗事故，本病例属于一级甲等医疗事故，医方承担次要责任。
　　在本院审理中，原告对上述医疗事故技术鉴定书予以认可，被告则认为上海市医学会的鉴定结论过重，表示被告只应承担轻微责任。2011年6月28日，原告撤回了要求被告支付被抚养人生活费的诉讼请求。
　　以上事实由原、被告提供门急诊就医记录册、病危通知单、检验报告、居民死亡医学证明、死亡通知书、尸检报告单、医药费专用收据、结婚证、医疗费发票、尸检发票、聘请律师合同及发票、病历资料、化验单据、上海市医学会（沪医鉴[2011]006号）医疗事故技术鉴定书及本院的询问笔录、庭审笔录等予以证实。
　　本院认为，公民的生命健康权受法律保护。被告作为专业的医疗机构，对病人的病情发展应当作出正确的诊断，并施以正确的治疗方案及措施。根据上海市医学会于2011年2月25日出具的（沪医鉴[2011]006号）医疗事故技术鉴定书的分析意见，医方对于危重留观患者病情严重性预判不足，观察不到位（在应用胰岛素静脉泵期间，缺乏必要的定期监测血糖措施），违反现行医疗常规，未能及时发现病情变化以调整治疗措施，该医疗过失与患者的死亡发生有一定的相关性。患xxx病理性基础（糖尿病、甲状腺功能亢进、重度贫血、血小板减少、严重营养不良、继发肺部感染），严重低血糖发生是诱发患者病情进一步加重的因素，而自身病情的严重性，则是导致不良后果的主要原因。患xxx症状与体征，白细胞检查尚属于正常范围，医方采用支持为主的临床处理措施，不违反医疗常规。结论为：严某与某医院医疗争议构成医疗事故，本病例属于一级甲等医疗事故，医方承担次要责任。为此被告应承担相应的民事责任，根据医疗事故处理条例的有关规定赔偿原告医疗费、丧葬费、交通费、住院伙食费、营养费、交通费及精神损害抚慰金等。至于被告的具体赔偿数额则根据上述上海市医学会医疗事故技术鉴定书确定的事故等级、责任程度及医疗事故处理条例的有关规定合法、合理予以确定。关于医疗费的赔偿，以原告提供的医疗费用收据来计算；丧葬费、住院伙食补助费、营养费按照医疗事故处理条例及相关标准计算；交通费以原告提供的相关单据计算；根据原告提供的发票，尸检费为3,500元，但原告在本案中主张的鉴定费为3,000元，故本院依法按照3,000元予以处理；在本院审理中，原告虽未能提供确实的证据证明其误工损失，但原告办理严某的丧葬及医疗纠纷事宜事实上花费了一定的时间，产生一定的误工损失，本院依法酌情予以处理。由于原告的医疗过失行为，致使原告精神上受到了相应的伤害，被告应当给予补偿和抚慰。另原告要求被告偿付死亡赔偿金的诉讼请求，因不符合医疗事故处理条例的有关规定，本院难以支持。原告要求被告支付住宿费180元，因原告未能提供相应的证据，本院不予支持。据此，依照《中华人民共和国民法通则》第九十八条、第一百零六条第二款、第一百一十九条之规定，判决如下：
　　一、某医院在本判决生效之日起十日内赔偿李某医疗费人民币1,285.20元；
　　二、某医院在本判决生效之日起十日内赔偿李某交通费人民币51.60元；
　　三、某医院在本判决生效之日起十日内赔偿李某丧葬费人民币6,418.20元；
　　四、某医院在本判决生效之日起十日内赔偿李某住院伙食补助费人民币18元；
　　五、某医院在本判决生效之日起十日内赔偿李某营养费人民币36元；
　　六、某医院在本判决生效之日起十日内赔偿李某误工费人民币294元；
　　七、某医院在本判决生效之日起十日内赔偿李某精神抚慰金人民币50,000元；
　　八、某医院在本判决生效之日起十日内赔偿李某鉴定费人民币3,000元；
　　九、某医院在本判决生效之日起十日内赔偿李某聘请律师费用人民币3,000元；
　　十、驳回李某的其他诉讼请求。
　　如果未按本判决指定的期间履行给付金钱义务，应当按照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　案件受理费人民币4,676元，由李某负担人民币3,281元，某医院负担人民币1,395元。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
审判长王嵘
　　审判员周红林
　　代理审判员张煜
　　二〇一一年八月二十三日
　　书记员崔妍
