段建国等诉燕昭昭等侵犯生命权纠纷案

审判时间：20110328 法院：甘肃省徽县人民法院 案件字号：(2011)徽江民初字第08号


　　原告段建国。
　　原告张菊花。
　　被告燕昭昭。
　　被告成县人民医院。
　　法定代表人梁秉均。
　　原告段建国、张菊花诉被告燕昭昭、成县人民医院侵犯生命权纠纷一案，本院依法进行审理后认为，本案存在交通肇事和医疗纠纷两个法律关系，实为两个案件，其中医疗纠纷不属于本院管辖范围，故本案应先驳回二原告的起诉。就交通肇事案件二原告可向本院另行起诉，就医疗纠纷案件二原告向成县人民法院提起诉讼。依照《中华人民共和国民事诉讼法》第一百零八条之规定，裁定如下：
　　驳回原告段建国、张菊花的起诉。
　　如不服本裁定，可在之日十日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于陇南市中级人民法院。

　　审判长：徐文化
　　代理审判员：罗爱军
　　人民陪审员：李奎
　　二○一一年三月二十八日
　　书记员：李炜
