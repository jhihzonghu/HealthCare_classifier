吴某与上海市某医院医疗损害赔偿纠纷上诉案

审判时间：20110314 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2011)沪一中民一(民)终字第68号


　　上诉人（原审原告）吴某。
　　被上诉人（原审被告）上海市某医院。
　　上诉人吴某因医疗损害赔偿纠纷一案，不服上海市徐汇区人民法院（2010）徐民一(民)初字第6082号民事判决，向本院提起上诉。本院于2010年12月27日立案受理后，依法组成合议庭，并于2011年1月5日公开开庭进行了审理。上诉人吴某及其委托代理人戴建铭，被上诉人上海市某医院（以下简称“某医院”）的委托代理人童某到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，吴某患阵发性房颤10余年，因胸闷、心悸加重1月伴胸痛，于2006年8月21日入住某医院心内科，择期EPS+CPVA术，术前听心率80次/分，心律齐，心音正常，瓣膜区未及杂音、附加音。心超检查结论为静息状态下超声心动图未见明显异常。食道超声检查结论为左房及左心耳内未见血栓形成征象。8月22日局麻下行心内电生理检查+环肺静脉消融术（CARTO指导下）。术中两次穿刺房间隔，在CARTO引导下进行左房三维重建及环肺静脉消融达到电隔离。复查心电图示窦性心律，病情稳定于8月28日出院。据吴某陈述，出院后病情未有改善，胸闷、心悸仍不断发作而至同济医院、上海市第一人民医院、上海市第十人民医院、某医院等医院就诊。2006年9月6日，铁路医院心电图示窦性心律，电轴不偏，T波低平，Ⅲ～Ⅴ4avf倒置，呈缺血性改变。2007年5月26日同济医院心电图示异位心律-房颤伴快速心室率（平均心室率141次/分），心电轴不偏，下壁+前侧壁心肌缺血。2007年9月17日某医院心电图示心室率81次/分，窦性心律，ST段与T波异常。2008年4月10日瑞金医院心超检查提示先心-房间隔缺损（继发孔型），缺口大小约0.4cm，CDF2示左向右分流。2008年4月17日某医院心超检查结论先心，继发孔型房间隔缺损,房水平左向右分流。2008年5月14日某医院心超检查结论房水平左向右分流（房间隔缺失约4.5mm-9.7mm）。
　　2010年4月19日，上海市卢湾区医学会就涉案医疗纠纷进行鉴定，分析意见为：1、吴某阵发性房颤10余年，药物效果欠佳，有行射频消融治疗指征。某医院在诊治过程中，术前检查及术中操作、术后随访等均符合诊疗常规，无过错行为；2、吴某术后超声心动图发现房间隔缺损，根据临床资料不能排除与本次手术房间隔穿刺有关，而房缺是房间隔穿刺术的并发症之一，多数会自行愈合。吴某未能愈合系少见现象，但缺损小（4.5～11mm），对血流动力学及心功能无明显影响，故对吴某健康影响甚微，可不予特殊处理。目前吴某心悸、胸闷等不适症状系房性心律失常（复发）所致，与房缺无关；3、建议医方在今后诊疗过程中，应告知患xxx症。结论为吴某与某医院医疗争议不构成医疗事故。某医院为此支付鉴定费3,500元。
　　因吴某对上述鉴定结论持有异议，申请重新鉴定，2010年8月12日，上海市医学会就涉案医疗纠纷进行鉴定，分析意见为：1、依据现有医疗规范和《房颤诊疗指南》，吴某已有10多年阵发性房颤病史，且药物治疗效果不佳，有射频消融治疗的手术指征；2、射频消融手术必须行房间隔穿刺的操作步骤，此为有创伤性操作，吴某术前心脏超声检查（经胸及经食道）没有房间隔缺损，术后经检查证实有房间隔缺损，故该情形的发生与射频消融手术有关，在现有医学条件下尚难以完全避免；3、吴某目前心悸、胸闷不适症状系原有病情的复发，房间隔缺损较小，没有明显的血流动力学障碍的临床表现，暂时无需特殊处理；4、房间隔缺损在射频消融手术中较为罕见，医方对此术前告知不足，术后沟通亦有欠妥之处，应予以重视。结论为吴某与某医院医疗争议不构成医疗事故。吴某为此支付鉴定费3,500元。后吴某认为，某医院医师在诊疗过程中缺乏责任心，违反诊疗常规，手术既未治愈其原有的房颤病情，又造成房间隔缺损，某医院过错造成其巨大的经济损失、精神痛苦，故诉至法院，要求判令某医院赔偿医疗费10,738.48元、鉴定费3,500元、律师代理费7,000元、交通费500元、精神损害抚慰金30,000元，并保留后续治疗的权利。
　　原审法院认为，本案中，市、区两级医学会分别组织专家进行了医疗事故技术鉴定，均明确了某医院实施手术治疗时有明确的手术指征，治疗措施符合规范，某医院不存在医疗过错，故吴某要求某医院承担医疗过错责任法院难予支持。但两次鉴定均明确某医院在治疗过程中，存在术前告知不足、术后沟通欠妥的不足，故法院判令某医院酌情给予吴某一定的补偿。若今后吴某病情发生变化需要进行相应治疗，可再行与某医院协商或另行诉讼解决。据此，原审法院判决：一、驳回吴某所有的诉讼请求;二、上海市某医院于判决生效之日起十日内补偿吴某5,000元。案件受理费1,100元，减半收取计550元，由吴某负担；两次鉴定费共计7,000元，由上海市某医院负担。
　　判决后，上诉人吴某不服，向本院提起上诉，要求撤销原判，改判支持上诉人的原审诉讼请求。吴某上诉称，两次鉴定均认定房间隔缺损与射频消融手术有关，排除了先天存在的情况，该缺损是某医院在手术过程中造成，故某医院应承担赔偿责任。
　　被上诉人某医院则不同意上诉人的上诉请求，请求二审法院驳回上诉人的上诉请求，维持原审判决。
　　本院经审理查明，原审法院认定事实无误，本院依法予以确定。
　　本院认为，医疗损害赔偿责任的承担，是以医疗存在过错以及过错与损害后果之间存在因果关系为基本条件。由于医学会作为本市医疗事故技术鉴定的法定机构，其根据案件情况所做出的鉴定结论具有客观性和权威性。故原审法院根据医学会明确某医院不存在医疗过错的鉴定结论，驳回吴某原审的诉讼请求，有法可依。同时，原审法院考虑到某医院存在术前告知不足、术后沟通欠妥的不足，酌情判令某医院对吴某作出一定的补偿，已经充分考虑了本案的实际情况。综上，在吴某无充足证据证明医学会鉴定结论和分析意见存在错误的前提下，要求二审法院支持其原审的诉讼请求，缺乏事实和法律依据，本院不能支持。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　上诉案件受理费1,100元，由上诉人吴某负担。
　　本判决系终审判决。
审判长孙卫
　　代理审判员汪　健
　　代理审判员沈卫兵
　　二○一一年三月十四日
　　书记员左莉莉
