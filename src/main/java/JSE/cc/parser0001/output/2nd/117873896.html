正阳县人民医院与车顺发医疗纠纷上诉案

审判时间：20110506 法院：河南省驻马店市中级人民法院 案件字号：(2011)驻民二终第107号


　　上诉人（原审被告）正阳县人民医院。
　　法定代表人闵建，该院院长。
　　委托代理人潘连印，该院医生。
　　委托代理人任学凯，河南明信律师事务所律师。
　　被上诉人（原审原告）车顺发。
　　委托代理人姚宏宇，河南明信律师事务所律师。
　　上诉人正阳县人民医院因医疗纠纷一案，不服正阳县人民法院(2007)正民重字第999号民事判决，向本院提起上诉。本院依法组成合议庭，公开开庭审理了本案。上诉人正阳县人民医院的委托代理人潘连印、任学凯，被上诉人车顺发的委托代理人姚宏宇到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，2004年8月19日，车顺发驾驶毛驴车在送煤途中(以盈利为目的)与王世全驾驶的豫Q07606号车发生交通事故，致使车顺发受伤，造成左股骨干骨折，后车顺发于2004年8月19日入住正阳县人民医院进行治疗，正阳县人民医院对车顺发左股骨干骨折处以七枚螺钉进行复位固定，车顺发经正阳县人民医院治疗后于2004年9月10日出院，在恢复过程中因正阳县人民医院固定的螺钉有三枚断裂，造成车顺发左股骨上段陈旧性骨折术后骨不连，车顺发遂于2005年11月8日入住河南省洛阳正骨医院进行二次手术，住院20天，支出医疗费13697.54元，交通费2071元。车顺发的伤情经驻马店蔚康法医临床司法检定所鉴定为九级、十级伤残。车顺发左股骨干骨折内固定断裂钢板取出费用需5000—6000元。在重新审理过程中，经河南检苑司法鉴定中心鉴定，正阳县人民医院在对车顺发的治疗过程中存在过错。正阳县人民医院不服，申请重新鉴定，之后又放弃了重新申请鉴定的权利。另查明，2005年河南省交通运输业人均纯收入为13959元，平均每天约38．24元。2006年河南省农村居民人均纯收入为3261.03元。
　　原审法院认为，车顺发因交通事故受伤后到正阳县人民医院处治疗，正阳县人民医院用螺钉对车顺发左股骨干骨折处进行复位固定，因治疗不当，造成车顺发在恢复过程中有三枚螺钉断裂，导致车顺发二次手术，并致伤残，对此医疗纠纷的发生，正阳县人民医院存在明显过错。正阳县人民医院辩称，为车顺发的诊疗无过错及螺钉断裂是车顺发未遵医嘱所致，但未能提供证据予以证实，不予采纳。车顺发在正阳县人民医院处治疗后，在恢复过程中因活动不当，对二次手术的发生，也应承担一定责任，结合本案实际，以车顺发承担40％的责任，正阳县人民医院承担60％的责任为宜。车顺发二次手术时住院20天，花费医疗费13697.54元，误工费按每天38.24元计算，从第一次住院之日起至定残的前一日，即从2004年8月19日至2006年4月16日，共计606天，计款23173.44元，护理费按二人每天各10元计算，第一次住院21天，第二次住院20天，共住院41天，计款820元，伙食补助费和营养费每天各按10元计算，共住院41天，计款820元，残疾赔偿金计款11739．71元(3261.03x18年×20％)，交通费、鉴定费按实际票据计算，分别为2071元和~2000元，上述各项费用共计54321.69元，正阳县人民医院赔偿60％，计款32593.01元，车顺发的精神慰抚金结合本案实际情况，应酌情认定8000元为宜，综上，正阳县人民医院实际应赔偿车顺发40593.01元。由于车顺发是交通事故造成的需二次取钢板，对其费用，正阳县人民医院不应承担，对车顺发的该项请求，不予支持。原审法院判决：一、正阳县人民医院赔偿车顺发医疗费、误工费、护理费、伙食补助费、营养费、交通费、残疾赔偿金、精神慰抚金、鉴定费等共计40593.01元，限判决生效后三日内付清。二、驳回车顺发其它诉讼请求。如未按本判决指定的期间履行金钱给付义务，应当按照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。诉讼费420元，由车顺发承担168元，由正阳县人民医院承担352元。
　　正阳县人民医院上诉称，车顺发的损害是交通事故造成的，与其无关；车顺发从受伤到定残间隔较长，加重了赔偿人的赔偿责任，原审法院在鉴定问题上程序存在违法之处，车顺发起诉超过诉讼时效。请求撤销正阳县人民法院(2007)正民重字第999号民事判决，依法改判。车顺发答辩称，关于鉴定问题并非其不去鉴定，是正阳县人民医院放弃鉴定权利，其与王世全之间因交通事故赔偿已调解终结，因正阳县人民医院医疗过错导致其病情恶化，二次入院治疗，钢板至今仍未取出，其起诉未超时效，原审法院计算赔偿标准正确。请求驳回上诉，维持原判。
　　二审审理查明的事实与原审法院判决认定的事实一致。
　　本院认为，本案车顺发因交通事故受伤后到正阳县人民医院处治疗，正阳县人民医院用螺钉对车顺发左股骨干骨折处进行复位固定，因其治疗不当，造成车顺发在恢复过程中有三枚螺钉断裂，导致车顺发二次手术并致伤残，对此医疗纠纷的发生，正阳县人民医院存在明显过错。车顺发在正阳县人民医院治疗后，在恢复过程中因活动不当，对二次手术的发生，也应承担一定责任。结合本案实际，原审按车顺发承担40％的责任，正阳县人民医院承担60％的责任划分适当。关于正阳县人民医院上诉称原审法院在鉴定问题上程序存在违法之处的问题。原审法院审理中，双方对选择鉴定单位未能达成一致，且医院方不同意再次进行司法鉴定，故未能再次进行司法鉴定。车顺发与王世全之间交通事故的赔偿问题，双方已调解终结，本案系车顺发在医院进行二次手术并致伤残的赔偿问题。由于车顺发病情恶化一直在治疗，二次入院治疗，钢板至今仍未取出，故其起诉并未超过诉讼时效。综上，正阳县人民医院的上诉理由不足，不予支持。原审判决认定事实清楚，适用法律正确，应予维持。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项的规定，判决如下:
　　驳回上诉,维持原判。
　　二审案件受理费600元，由上诉人正阳县人民医院负担。
　　本判决为终审判决。
审判长侯治安
　　审判员张怀珍
　　审判员牛朝干
　　二〇一一年五月六日
　　书记员高冉玉
