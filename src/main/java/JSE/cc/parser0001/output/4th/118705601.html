雷红诉株洲市一医院医疗纠纷案

审判时间：20111010 法院：湖南省株洲市芦淞区人民法院 案件字号：(2011)芦法民一初字第1089号


　　原告雷红。身份证号码：xxx某某某某某某某某某某某某。
　　委托代理人易革修，湖南弘一律师事务所律师，代理权限为特别授权。
　　被告株洲市一医院，地址株洲市芦淞区某某某某某某。
　　法定代表人杨正文，系株洲市一医院院长。
　　委托代理人李民安，湖南如金律师事务所律师，代理权限为特别授权。
　　委托代理人郭戈。身份证号码：xxx某某某某某某某某某某某某。代理权限为全权代理。
　　原告雷红与被告株洲市一医院医疗纠纷一案，本院于2011年8月15日立案受理后，依法适用普通程序，由审判员黄爱武担任审判长，代理审判员刘寿、人民陪审员沈顺珍参加的合议庭。2011年10月10日经原、被告申请，本院组织庭后调解。
　　原告雷红诉称，原告于2009年8月23日到被告株洲市一医院处看病，经相关检查诊断为右胸壁肌纤维瘤，8月28日行手术治疗。过了两个月后，原告手臂没有一点好转，便去长沙湘雅医院做检查。经湘雅医院检查诊断为右臂丛神经损害。2011年6月20日，株洲市医学会专家组认定被告给原告的治疗行为构成三级丙等医疗事故。
　　被告株洲市一医院辩称，愿意依法进行赔偿。
　　本案在审理过程中，经本院主持调解，双方当事人自愿达成如下协议：
　　一、被告株洲市一医院自愿在本调解生效后十日内一次性赔偿原告雷红医疗费、伙食补助费、营养费、陪护费、交通费、误工费、残疾赔偿金、后续治疗费、鉴定费、精神损失费合计65000元；原告雷红自愿放弃其他诉讼请求。
　　二、本次纠纷系一次性了结，双方无其他争议。
　　本案案件受理4698元，减半收取2349元，原告雷红自愿承担。
　　双方当事人一致同意上述调解协议，自双方当事人在调解协议上签名后即具有法律效力。
　　上述协议，不违反法律规定，本院予以确认。

　　审判长黄爱武
　　代理审判员刘寿
　　人民陪审员沈顺珍
　　二○一一年十月十日
　　书记员袁婷
