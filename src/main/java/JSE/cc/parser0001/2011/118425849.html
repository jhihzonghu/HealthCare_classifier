陈某医疗损害赔偿纠纷案

审判时间：20110222 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2011)沪一中民一(民)终字第381号


　　上诉人（原审原告）陈某。
　　被上诉人（原审被告）上海某医院。
　　被上诉人（原审被告）上海某2医院。
　　上诉人陈某因医疗损害赔偿纠纷一案，不服上海市浦东新区人民法院（2010）浦民一（民）初字第20361号民事判决，向本院提起上诉。本院于2011年1月24日立案受理后，依法组成合议庭，于同年2月10日公开开庭审理了本案。陈某及其委托代理人胡亚林、王宗琦，被上诉人上海某医院（以下简称某医院）的委托代理人童剑云，被上诉人上海某2医院（以下简称某2医院）的委托代理人王景潇、黄惠民到庭参加了诉讼。本案现已审理终结。
　　原审认定，陈某系患者孙保国的妻子。2009年11月10日至11日，孙保国因跌倒出现呕吐不适先后在上海市浦东新区南汇中心医院、复旦大学附属华山医院就诊，华山医院诊断为CHD，ACS可能。当月12日，孙保国至某医院就诊，某医院给予“奥扎格雷钠针、醒脑针等药物治疗；当月13日给予“奥扎格雷钠针、醒脑针、伊拉达奉针”等药物治疗；当月14日检查双肺呼吸音粗，EKG检查T波改变，心肌酶检查CK，TNI，CK－MB均升高，心内科会诊诊断为冠心病，CABG术后，心功能不全2－3级等，并要求给予利尿，密切观察心率、血压、动态心电图监测；当月15日，某医院给予“丹参针、胰岛素针、醒脑静针、单硝酸异山梨酯”等药物治疗。同日，孙保国家属要求转院，某医院向其说明转院途中存在心衰、心律失常等风险，家属表示理解并接受。同日21：00，孙保国前往某2医院治疗，某2医院给予“醒脑针、硝酸异山梨酯、托拉塞米针”等治疗。当月16日，孙保国出现不能平卧、双肺呼吸音粗，肺底可闻湿罗音，诊断为冠心病，冠脉搭桥术后，心功能Ⅲ－Ⅳ级，脑梗塞、高血压病。当月17日，孙保国病情未能缓解，某2医院告知家属患者随时有猝死危险。当月18日，孙保国病情加重，6：30突发呼吸心跳停止，某2医院采取抢救措施，8：55孙保国被宣布临床死亡，据某2医院病史记录为心源性猝死、心功能不全Ⅳ级、冠心病CABG术后。陈某因认为某医院和某2医院存在医疗事故，故起诉请求判令两家医院赔偿168,727.50元。在诉前调解过程中，经某医院、某2医院申请，原审法院委托上海市虹口区医学会对本次医疗争议进行鉴定，由于医患双方在尸检问题上未能达成共识，故该医学会对本次医疗争议不予受理。原审中，原审法院向陈某释明：医学会针对病史现状仍可以作医疗事故鉴定，陈某提出的某2医院未告知患者家属可以尸检是另一争议，某医院与某2医院在医疗过程中是否存在医疗事故和医疗过错，必须由相关的医学会作出鉴定。陈某坚持不同意进行医学鉴定，亦不配合医学会鉴定。
　　原审法院审理后认为，陈某坚持不同意进行医学鉴定，亦不配合医学会鉴定，其应承担举证不能的后果。原审法院遂依照《中华人民共和国民事诉讼法》第六十四条第一款、《医疗事故处理条例》第三十条第二款的规定，判决驳回了陈某的诉讼请求。
　　陈某向本院提起上诉，请求撤销原判，改判支持其在原审中的全部诉讼请求。陈某诉称，被上诉人某医院在诊疗过程中用药有误；被上诉人某2医院则未对患者孙保国进行会诊；另外，两位被上诉人均未安排孙保国住院治疗，故应当承担医疗过错赔偿责任。陈某同时认为，上海市虹口区医学会对本案医疗争议不予受理，故应当进行司法鉴定。
　　被上诉人某医院及某2医院均不接受上诉人陈某的上诉主张。两位被上诉人辩称，本案医疗争议是否构成医疗事故或者医疗过错应当由医学会进行鉴定，陈某在原审法院释明的情况下拒绝配合医疗事故技术鉴定，其应自担举证不能的后果。
　　本院经审理查明，原审认定的事实无误。
　　本院认为，考量被上诉人某医院及某2医院在对患者孙保国的诊疗过程中是否存在医疗过错，应当以专业鉴定机构作出的鉴定结论为重要依据，这也是医疗机构在医疗纠纷中承担举证义务的体现。所以，原审法院根据某医院及某2医院的申请，委托医学会进行医疗事故技术鉴定符合程序规定。本案医疗事故技术鉴定因医患双方对尸检存有异议而未进行，但尸检解决的并不是医疗过错的问题，故在原审法院明确告知上诉人陈某根据孙保国的病史仍然可以进行医疗事故鉴定的情况下，陈某应当积极配合进行鉴定。陈某坚持要求进行医疗事故技术鉴定以外的司法鉴定的主张缺乏依据，本院不能采纳。由于陈某怠于配合进行医疗事故技术鉴定，故原审法院将举证义务转移至陈某一方并认定陈某举证不能理据充分，本院予以认同。陈某的上诉理由缺乏充分的事实和法律依据，本院不能采纳；相关的上诉请求，本院亦难以支持。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，本院判决如下：
　　驳回上诉，维持原判。
　　上诉案件受理费3,674元，由上诉人陈某负担。
　　本判决为终审判决。
审判长孙卫
　　代理审判员汪　健
　　代理审判员沈卫兵
　　二○一一年二月二十二日
　　书记员左莉莉
