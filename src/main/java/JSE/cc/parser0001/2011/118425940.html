劳某与某医院医疗损害赔偿纠纷案

审判时间：20110419 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2011)沪一中民一(民)终字第565号


　　上诉人（原审原告）劳某。
　　被上诉人（原审被告）某医院。
　　上诉人劳某因医疗损害赔偿纠纷一案，不服上海市徐汇区人民法院（2010）徐民一（民）初字第210号民事判决，向本院提起上诉。本院于2011年2月16日立案受理后，依法组成合议庭进行了审理，现已审理终结。
　　原审认定，劳某因下腹疼痛1天，B超提示盆腔积液于2007年6月22日入住某医院（以下简称某医院）。入院前一天曾因腹痛待查由上海市徐汇区大华医院急诊医治，予以甲硝唑等抗炎治疗后情况得到改善。血常规检查：白细胞18.1×10/L，中性粒细胞87.2%。据入院首次病程录记载，B超示：黄体破裂不除外，拟议“腹痛待查：黄体破裂”收治入院。入院检查：体温39.8℃，腹部轻度肌卫，压痛（＋）；lmp：2007-6-2×3d，量少，pmp：2007-3月（初潮）；肛查：宫颈举痛（±）；宫体中位，略小于正常；双侧（－）；血常规：白细胞12.5×10/L，中性粒细胞88.8％；B超：盆腔积液。诊断：腹痛待查，黄体破裂，予以补液、抗炎、止血等治疗及相关检查。6月26日劳某仍有下腹隐痛，阵发性，较前略缓解。检查：腹平软，右下腹轻压痛，反跳痛（－）。血常规：白细胞8.0×10/L，中性粒细胞74.5％；血钾2.8mmol/L，CO2-CP20mmol/L。6月27日X线影像学表现：卧位见弓形充气扩张肠曲，主要位于结肠，立位见多个呈阶梯状排列液平。影像学提示，结肠梗塞不除外。6月28日普外科会诊后诊断，腹痛待查，予剖腹探查。当日17时劳某在全麻下行剖腹探查术。术中见右下腹肠管粘连成块，中间为一脓腔，吸出恶臭脓液约100ml。阑尾长8cm，头端溶蚀，内有直径1cm粪石1枚，予以切除阑尾及修补盲肠末端浆膜面。妇产科探查未见子宫、卵巢异常。甲硝唑冲洗腹腔，吸尽，置引流管。切口下置百菲米1张预防肠粘连。逐层关腹。术后诊断：急性腹膜炎，腹腔脓肿，阑尾穿孔。7月6日劳某一般情况可，检查伤口Ⅲ/甲愈合。当日出院。出院诊断：急性腹膜炎、腹腔脓肿、肠梗塞、阑尾穿孔。医嘱门诊随访。据出院小结记载病理诊断：阑尾急慢性炎症，另见纤维结缔组织充血、出血，伴较多中性粒细胞浸润。劳某陈述之后在其他医院行中医调经治疗。劳某提起诉讼，请求判令某医院赔偿其医疗费（住院押金）5,000元、后续的医疗费6,166.97元、误工费42,789元、住院伙食补助费300元、营养费3,600元、陪护费10,697.25元、交通费800元、复印费56.30元、残疾生活补助费778,626元、精神损害抚慰金150,000元、鉴定费4,300元。原审法院委托上海市徐汇区医学会对本案纠纷进行医疗事故技术鉴定，分析意见为：患者急腹痛在某医院诊疗过程中，将急性阑尾炎误诊为妇科疾患，导致延误手术，引起阑尾穿孔局部腹膜炎，带给病人身心损害，构成四级医疗事故。患者在某医院外科首诊时，诉右下腹疼痛，体温38℃，右下腹有固定压痛，反跳痛和肌卫，提示有局部腹膜炎体征，有急诊手术指征，医方未予及时手术，违反医疗常规，故负主要责任。妇科在“黄体破裂伴感染”依据不很充分的情况下，将患者保守治疗长达7天，期间未能反复检查腹部体征变化，对患者诊断和治疗的延误亦负有重要责任。结论为劳某与某医院医疗争议构成医疗事故，本病例属于四级医疗事故，医方承担主要责任。某医院支付鉴定费3,500元。因劳某对鉴定有异议，原审法院又委托上海市医学会进行鉴定，该会分析认为：1、患者因下腹部疼痛在外院诊断急性阑尾炎转诊至某医院。入院后医方误诊为妇科疾病，7天后，因病情加重，施行剖腹探查，证实阑尾穿孔所致急性腹膜炎。医方延误及时手术治疗，存在过失，由急性阑尾炎演变至阑尾穿孔及腹膜炎，对患者造成了一定的影响，医方的过失行为与患者疾病的发展存在因果关系。2、患者在住院期间，虽未及时手术，但医方给予了患者抗炎治疗。3、经过手术治疗后，达到治疗效果，患者目前未见组织器官损伤导致的功能障碍。结论为劳某与某医院医疗争议构成医疗事故，本病例属于四级医疗事故，医方承担主要责任。劳某支付鉴定费3,500元。原审法院还委托华东政法大学司法鉴定中心对劳某的休息、营养、护理期限进行鉴定，鉴定结论为：被鉴定人劳某因急性腹膜炎，腹腔脓肿，肠梗阻，阑尾穿孔，引流口少量渗出，发热等，酌情给予伤后休息4个月，营养3个月，护理2个月。劳某支付鉴定费800元。
　　原审审理后认为，根据鉴定意见酌定某医院对劳某的损害后果承担80%的赔偿责任，故在审核了赔偿范围后判决：一、某医院于判决生效之日起十日内赔偿劳某医疗费4,000元、住院伙食补助费280元、陪护费3,000元、营养费3,600元、复印费50元、交通费100元，共计11,030元的80%计8,824元；二、某医院于判决生效之日起十日内赔偿劳某精神损害抚慰金10,000元；三、驳回劳某其他诉讼请求。案件受理费及鉴定费共计21,530元，由劳某负担10,000元，某医院负担11,530元。
　　上诉人劳某向本院提起上诉，坚持其在原审中的诉讼请求。劳某诉称，根据上海中医药大学附属龙华医院等诊断，其现患xxx症、月经失调、顽固便秘等病症，上述病症与被上诉人某医院的诊疗行为有因果关系，故其应当获得赔偿。某医院则不接受劳某的上诉主张。
　　本院经审理查明，原审认定的事实无误。
　　本院认为，本案医疗纠纷历经两级医疗事故技术鉴定，相关分析意见中对被上诉人某医院在对上诉人劳某的诊疗过程中的过错程度、责任比例均有论述。上述鉴定意见由专业鉴定部门出具，具有权威性与专业性，在劳某没有提供更为充分的证据推翻鉴定结论的情况下，本院应予认定。劳某上诉主张其现存在的多囊卵巢综合症、月经失调、顽固便秘等病症与某医院的诊疗过错具有因果关系，但没有提供证据佐证，故本院难以采纳。劳某的上诉请求，本院亦难以支持。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，本院判决如下：
　　驳回上诉，维持原判。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　上诉案件受理费13,635.11元，由上诉人劳某负担，本院准予其免缴。
　　本判决为终审判决。
审判长李虎
　　代理审判员沈卫兵
　　代理审判员孙卫
　　二○一一年四月十九日
　　书记员左莉莉
