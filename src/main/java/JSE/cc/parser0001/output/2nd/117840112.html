甲等与丙道路交通事故人身损害赔偿纠纷上诉案

审判时间：20110504 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2011)沪一中民一(民)终字第580号


　　上诉人（原审被告）甲
　　上诉人（原审被告）乙
　　被上诉人（原审原告）丙
　　上诉人甲、乙因道路交通事故人身损害赔偿纠纷一案，不服上海市徐汇区人民法院（2010）徐民一(民)初字第5571号民事判决，向本院提起上诉。本院于2011年2月17日立案受理后，依法组成合议庭，于2011年3月9日公开开庭进行了审理。上诉人甲，被上诉人丙到庭参加诉讼。本案现已审理终结。
　　原审法院查明，2007年2月1日6时许，乙醉酒后驾驶甲名下车牌号为沪E的小客车，沿本市宝庆路由南向北行驶至复兴中路路口，遇红灯继续向北行驶，恰逢丙驾驶电动自行车沿复兴中路由西向东行驶至上述地点，两车相碰，致丙跌地受伤、两车俱损，交警部门认定乙醉酒后驾驶机动车且违反路口灯色通行，应负事故全部责任。沪E小客车事发时未购置机动车交强险。
　　事发后，丙至上海市第六人民医院治疗，诊断为右胫腓骨骨折，急诊麻醉下行手术治疗，于2007年2月5日转入上海市第八人民医院行右小腿清创缝合术，术后抗炎等对症治疗，于2007年3月13日出院，医嘱门诊随访。2008年9月16日，因钢板断裂入上海市徐汇区田林街道社区卫生服务中心住院治疗，于2008年10月6日出院。治疗期间，丙支付医疗费16,071.26元，甲支付医疗费34,565.78元。另事发后，甲支付丙护理费975元、住院杂费200元，并预付现金22,700元。
　　2009年7月2日，经华东政法大学司法鉴定中心鉴定，确认丙因交通事故致伤构成十级伤残，酌情给予（含二期内固定拆除术）治疗休息24个月、营养4个月、护理5个月。丙为此支付鉴定费1,400元。因乙、甲对丙治疗期间钢板断裂与涉案交通事故是否存在因果关系存有异议，提出鉴定申请，上海枫林国际医学交流和发展中心司法鉴定所经对因果关系进行鉴定后认为，第一次手术后7个月发生钢板断裂，与钢板质量、术后运动过度、再次外力作用等多种不确定因素有关，故丙治疗中钢板断裂与2007年2月1日交通事故存在间接因果关系，建议参与度为20-30%。甲为此支付鉴定费2,000元。
　　2010年7月7日，上海市闵行区医学会就丙与上海市第六人民医院医疗纠纷作出鉴定，认为上海市第六人民医院采取钢板内固定及筋膜间隔减压引流术，符合医疗常规；丙右小腿钢板内固定术后7个月发生钢板断裂，主要原因是丙右小腿软组织损伤严重，胫腓骨高度粉碎性骨折，骨折区血供甚差，导致骨不连接所引起，与手术无关；钢板断裂后，医方采取胫骨骨不连交锁髓内钉内固定+右髂骨取骨植骨术治疗，符合医疗规范；丙第二次手术后，目前骨折已基本连接，右下肢功能逐渐恢复，达到手术目的；医疗争议不构成医疗事故。
　　原审法院审理后认为，交警部门作出的事故认定无不妥，且当事人均无异议，故乙应就涉案交通事故造成丙损害后果承担全部民事赔偿责任，甲作为肇事车辆车主承担连带责任。司法鉴定结论未排除植入钢板断裂与涉案交通事故的因果关系，鉴定书中关于参与度20%至30%的陈述，系对众多不确定因素从统计学上作的概率判断，不符合客观情况；且在上海市闵行区医学会作出的医疗事故技术鉴定中，亦分析钢板断裂与丙损伤严重及骨折区血供差，导致骨不连接有关，亦反映出钢板断裂与涉案事故造成丙的伤害直接有关；另在本案审理期间，乙、甲未举证证实丙存在术后过度运动、负重等情节，故难以认定丙对植入钢板断裂一节存有过错；据此，钢板断裂后产生相应的治疗费用等损失仍应由乙、甲承担赔偿责任。事发后，甲已支付丙医疗费34,565.78元、护理费975元、住院杂费200元，并预付现金22,700元，上述费用共计58,440.78元作为预付款一并予以结算。
　　原审法院审核了丙的损失依据后，作出如下判决：1、乙、甲于判决生效之日起十日内连带赔偿丙医疗费50,637.04元、误工费38,000元、护理费5,000元、交通费500元、住院伙食补助费1,200元、营养费4,800元、残疾赔偿金57,676元、鉴定费1,400元、物损费500元、精神损害抚慰金5,000元、住院杂费200元，上述赔偿款项共计164,913.04元，扣除预付的58,440.78元，乙、甲尚需连带赔偿丙106,472.26元；2、驳回丙其余的诉讼请求。如未按判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。案件受理费2,246元，由丙负担700元，乙、甲负担1,546元；鉴定费（因果关系）2,000元，由乙、甲负担。
　　乙、甲不服原判，上诉称，植入丙体内钢板断裂系医疗事故，相关后续治疗费用，不应由其承担，故请求撤销原判，改判再由其赔偿丙5万元。丙则要求维持原判。
　　本院经审理查明，原审法院认定的事实无误，本院依法予以确认。
　　本院认为，上海市闵行区医学会已就丙体内钢板断裂原因作出鉴定结论，该鉴定结论排除了医疗事故。造成丙右小腿钢板内固定术后7个月发生钢板断裂，主要原因是丙右小腿软组织损伤严重，胫腓骨高度粉碎性骨折，骨折区血供甚差，导致骨不连接，上述损伤皆系交通事故所致。虽然司法鉴定结论中有钢板断裂与钢板质量、术后运动过度、再次外力作用等多种不确定因素有关的表述，但在案件审理期间，乙、甲并未提供丙术后运动过度、再次受外力打击等致使体内钢板断裂而需再次手术的证据，故对乙、甲的上诉理由，本院不予采信；对其上诉请求，本院不予支持。原审法院在查明事实的基础上，详细阐述了判令乙、甲承担全部赔偿责任的理由，对所述理由，本院均予以认同。原审法院判决无误，本院予以维持。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项的规定，判决如下：
　　驳回上诉，维持原判。
　　上诉案件受理费1,170.49元，由上诉人乙、甲负担。
　　本判决为终审判决。
　　审判长沙茹萍
　　代理审判员杨奇志
　　代理审判员杨仁感
　　二○一一年五月四日
　　书记员庄人杰
