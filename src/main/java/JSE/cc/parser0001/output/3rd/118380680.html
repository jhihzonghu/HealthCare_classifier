乔*与*附属口腔医院医疗损害责任纠纷案

审判时间：20110817 法院：上海市闸北区人民法院 案件字号：(2011)闸民一(民)初字第1129号


　　原告乔某。
　　委托代理人朱某，上海某律师事务所律师。
　　被告某附属口腔医院。
　　法定代表人王某，职务院长。
　　委托代理人奚某，该院职员。
　　委托代理人韩某，该院职员。
　　原告乔某与被告某附属口腔医院医疗损害责任纠纷一案,本院于2011年3月3日立案受理,依法适用简易程序审理,由代理审判员沈伟俊独任审判，书记员刘怡担任记录，于2011年4月2日公开开庭进行了审理。后本案转为普通程序，由审判员金晶、代理审判员沈伟俊、人民陪审员俞栋第组成合议庭，由审判员金晶担任审判长，代理审判员沈伟俊主审本案，书记员万莹担任记录，于2011年8月8日公开开庭进行了审理。原告乔某及其委托代理人朱某、被告某附属口腔医院（下称口腔医院）的委托代理人奚某到庭参加诉讼。本案现已审理终结。
　　原告乔某诉称，2008年12月31日至被告处洗牙，一周后，左上后牙（第六颗）在咀嚼时有疼痛感。2009年1月8日起多次至被告处诊治。因被告处医生在抽牙髓、打麻药及治疗过程中处置不当，先后造成原告左脸颊、左眼严重肿胀、口腔有异味等症状，并最终导致原告拔除左上后牙两颗（第六、第七）且口腔上颌有一渗血的洞。故要求被告赔偿医疗费人民币（以下币种同）22300元、交通费500元、口腔冲洗机费920元、后续治疗费5000元、精神损害抚慰金50000元、律师费5000元、鉴定费3500元，并履行以下告知义务：告知原告左上后牙（第六颗）是哪种牙病？是否需抽牙髓并采取损伤性治疗？治疗后左脸颊、左眼严重肿胀的真正原因？上颌患处出血的性质及治疗方案？
　　被告口腔医院辩称,本院医生的诊疗符合常规，两次鉴定结论均不构成医疗事故，故不同意原告诉讼请求，请求法院依法判决。
　　经审理查明，2008年10月11日至2009年2月16日期间，原告乔某因刷牙时牙龈出血、洗牙、左上后牙咬合疼痛等原因多次至被告口腔医院求诊。被告口腔医院先后诊断为牙周炎、牙龈炎、左上后牙（第六颗）根尖炎、左上后牙（第六颗）牙周炎等。其中，2009年1月21日，原告乔某牙龈肿痛至被告口腔医院治疗，被告医师对原告在局麻下实施左上后牙（第六颗）去残髓，机扩3根管至40号，标记根长、冲洗、干燥，CP棉球封，并进行袋内壁及根面刮治、根平整、冲洗、上药。次日，原告乔某因左面脸颊肿胀先后至被告口腔医院及上海市某人民医院治疗。上海市某人民医院诊断为左眶下间隙感染，建议静滴抗生素消炎后拔除左上后牙（第六颗）。
　　另查明，原告乔某因左上后牙牙龈易出血于2009年2月24日、3月3日至上海市某防治院治疗、复诊，诊断为左上后牙（第六颗）牙周炎、左上后牙（第六颗、第七颗）牙龈乳头萎缩。2009年6月26日，原告乔某因左上后牙咬合不适至上海市某人民医院诊治，诊断为左上后牙（第六颗）牙周炎。原告乔某因左眼闪光感于2009年2月9日、2009年8月12日、2010年3月2日至上海市某人民医院眼科检查治疗，诊断为左眼玻璃体混浊、屈光不正。为治疗，共花费医疗费12536.10元。
　　本案在诉前调解阶段，本院于2009年12月1日委托上海市某区医学会组织鉴定被告口腔医院对原告乔某的医疗行为是否属于医疗事故、事故等级及责任程度。上海市某区医学会于2010年3月18日出具沪某医鉴[20某]某号医疗事故技术鉴定书，认定：原告乔某与被告口腔医院医疗争议不构成医疗事故。因原告乔某对鉴定不服，本院于2010年4月9日委托上海市医学会进行复核鉴定。上海市医学会于2010年11月26日出具沪医鉴[20某]某号医疗事故技术鉴定书，认定：原告乔某与被告口腔医院医疗争议不构成医疗事故。但鉴定书分析意见认为：1、医方对患者的左上后牙（第六颗）牙周炎、根尖炎的诊断正确，实施的相关治疗、采用的麻醉用药及麻醉方式未违反口腔科诊疗原则。本次现场医学检查表明根管的治疗有效。2、患者出现左眼玻璃体混浊、后脱离的原因是玻璃体液化，通常眼内的炎症、外伤、出血，以及年龄增长、近视均会导致玻璃体液化的改变，与医方的口腔处理和继发面颊部组织的病变无因果关系。3、患者左面颊部肿胀是根管治疗麻醉后的不良反应，在现有医学条件下尚难以完全避免，经后续治疗肿胀消退，未留有明显的人身损害后果。但医方对麻醉可能产生的上述情形未在术前予以明确告知。4、医方的门诊病史书写有不符合《病历书写基本规范》之处，同时在牙周肿胀存在时应延期作牙周袋内壁及根面平整治疗更为适宜。对此医疗缺陷在今后工作中应加以改进。
　　还查明，在医疗事故鉴定后，原告乔某为相关治疗还先后在上海市某人民医院、上海市某人民医院、上海市某人民医院、某医院、某医院、某医院等医院就诊。其间，扣除统筹支付和附加支付部分，原告乔某共花费医疗费5024.50元、护理费120元。
　　审理中，原告乔某增加诉讼请求，要求被告口腔医院赔偿护理费120元；被告口腔医院对医疗事故技术鉴定前原告乔某为就医治疗、鉴定及诉讼的医疗费12536.10元、交通费500元、口腔冲洗机费920元、鉴定费3500元、律师费5000元等相关费用基数予以认可。对后续治疗费、精神损害抚慰金及医疗事故技术鉴定后的医疗费、护理费不予认可。由于原、被告双方无法达成一致意见，致调解未成。
　　以上事实有下列证据予以证实：双方当事人的陈述；医疗事故技术鉴定报告书；原告提供的病史记录、医疗费发票原件、出院小结、护理费发票、鉴定费发票、律师费发票等。
　　本院认为，公民享有生命健康权。公民、法人由于过错造成他人人身权利的,应当承担相应民事责任。原告乔某至被告口腔医院治疗，双方产生纠纷。由于医疗活动具有高度专业性，医疗科学属于生命科学领域，具有相当复杂性并同时具有一定风险性，故对于一起医疗纠纷是否构成医疗事故或存在医疗差错，尚有赖于具有专业知识、经验、技能的专家作出鉴定。本案中，根据上海市医学会的医疗事故技术鉴定结论，原告乔某与被告口腔医院医疗争议虽不构成医疗事故，但医方对麻醉可能产生的副作用未在术前予以明确告知，且医方的门诊病史书写有不符合《病历书写基本规范》之处，同时在牙周肿胀存在时应延期作牙周袋内壁及根面平整治疗，存在医疗缺陷，被告口腔医院就原告乔某为就医治疗、鉴定及诉讼所支付的相关费用承担与其过错相适应的赔偿责任。由于本案中被告口腔医院的医治行为未构成医疗事故，原告乔某亦未能证明被告的医疗行为造成其残疾，据相关法律规定，原告乔某主张精神损害抚慰金，无法律依据，本院予以驳回。至于原告乔某要求被告口腔医院履行告知义务的诉讼请求，因其主张的相关告知内容已有相关医院的诊断予以明确，且无相应法律依据，本院予以驳回。具体赔偿基数以及计算方式如下：一、医疗费，扣除医保部分，原告乔某共花费医疗费17560.60元，被告口腔医院虽对其中医疗事故技术鉴定后发生的医疗费5024.50元持有异议，但未能提供证据证明其主张，本院对其抗辩不予采信；二、护理费120元，确系原告乔某在上海市某人民医院住院时发生的实际费用，本院予以认可；三、交通费500元、口腔冲洗机费920元、鉴定费3500元、律师费5000元等相关费用，被告对此不持异议，本院予以认可；上述费用，由被告口腔医院根据其过错承担20％的赔偿责任。若原告乔某需后续治疗，如原告能证明该治疗是必须且与被告的医疗过错行为有关联性，待实际发生后，另行主张其权利。
　　综上所述，依照《中华人民共和国民法通则》某十八条、某百一十九条、《医疗事故处理条例》第四十六条之规定，判决如下：
　　一、被告某附属口腔医院应自本判决生效之日起十日内赔偿原告乔某医疗费人民币3512.12元、护理费人民币24元、交通费人民币100元、口腔冲洗机费人民币184元、鉴定费人民币700元、律师费人民币1000元，共计人民币5520.12元；
　　二、对原告乔某其他诉讼请求，本院予以驳回。
　　如果未按本判决指定的期间给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　案件受理费人民币783.70元，原告乔某已预缴，由原告乔某负担人民币734.10元，被告某附属口腔医院负担人民币49.60元，于本判决生效之日起十日内给付原告。
　　如不服本判决，可在判决书送达之日起十五日内,向本院递交上诉状,并按对方当事人的人数提出副本,上诉于上海市第二中级人民法院。
审判长金晶
　　代理审判员沈伟俊
　　人民陪审员俞栋第
　　二〇一一年八月十七日
　　书记员万莹
