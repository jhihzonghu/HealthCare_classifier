姜某某诉符甲等道路交通事故人身损害赔偿纠纷案

审判时间：20110420 法院：浙江省余姚市人民法院 案件字号：(2010)甬余民初字第2404号


　　原告：姜某某。
　　委托代理人：顾某某。
　　委托代理人：黄某某。
　　被告：符甲。
　　委托代理人：符乙。
　　委托代理人：沈某某。
　　被告：中银保险有限公司××中心支公司，组织机构代码：79601361-5。
　　代表人：李某某。
　　委托代理人：杨某某。
　　被告：符乙。
　　委托代理人：沈某某。
　　原告姜某某与被告符甲、中银保险有限公司××中心支公司（简称中银保险公司）道路交通事故人身损害赔偿纠纷一案，本院于2010年10月27日立案受理。依法由审判员严航适用简易程序独任审判，于2010年11月16日公开开庭进行了审理。原告姜某某的委托代理人顾某某，被告符甲及其委托代理人符乙、沈某某，被告中银保险公司的委托代理人杨某某到庭参加诉讼。后根据原告姜某某的申请，本院依法追加符乙为本案共同被告。本案于2011年2月23日再次公开开庭进行了审理。原告姜某某及其委托代理人黄某某，被告符甲的委托代理人符乙、沈某某，被告中银保险公司的委托代理人杨某某，被告符乙及其委托代理人沈某某到庭参加诉讼。本案现已审理终结。
　　原告姜某某起诉称：2009年11月14日9时40分左右，被告符甲驾驶浙B×××××号轻型普通货车至余姚市马渚镇××村杨岐岙村四枝交叉路口，与原告骑行的电瓶自行车相碰撞，造成原告受伤。经余公交认字（2009）第E041号事故认定书认定，双方负同等责任。浙B×××××号轻型普通货车的机动车交通事故责任强制保险投保于被告中银保险公司。原告受伤后在余姚市人民医院治疗，诊断为腹腔内出血、右胫腓骨上段粉碎性骨折、右髂骨粉碎性骨折，因病情危重转浙江大学医学院附属第二医院治疗。现经鉴定，原告的伤残等级为两个10级，护理期限4个月，误工6个月，营养期限4个月，拆除内固定后续医疗费为7000元。为此，原告诉请法院：要求赔偿医疗费269235.60元、后续医疗费7000元、住院伙食补助费2400元、护理费10287.60元、误工费15431.40元、营养费3200元、残疾赔偿金30338.40元、精神损害抚慰金5000元、鉴定费1560元、交通费2614元，合计346267元；由被告中银保险公司在交强险责任范围内予以赔偿73671.40元，不足部分272595.60元由被告符甲承担60%计163557.36元，已付155000元，尚应支付8557.36元。后原告变更诉讼请求为：要求赔偿医疗费229235.60元、后续医疗费7000元、住院伙食补助费2400元、护理费10287.60元、误工费15431.40元、营养费3200元、残疾赔偿金30338.40元、精神损害抚慰金5000元、鉴定费1560元、交通费2614元，合计307067元；由被告中银保险公司在交强险责任范围内予以赔偿73671.40元，已支付医疗费1万元，尚应赔偿63671.40元；不足部分233395.60元由被告符甲、符乙承担60%计140037.36元，已付125000元，尚应支付15037.36元。
　　为证明其主张，原告向本院提供了下列证据：1.事故认定书一份，以证明事故的发生及责任认定；2.交强险保险单、驾驶证、行驶证（复印件）各一份，以证明被告符甲投保交强险的事实；3.门诊病历、出院记录、抢救记录、病程记录、住院病案、病危通知书、诊断证明一组，以证明原告受伤后治疗的情况；4.医疗费发票及用药清单一组，以证明原告受伤后花费的医疗费及用药情况；5.鉴定意见书、鉴定费发票各一份，以证明原告的伤残等级、护理时间、误工时间、营养期限、后续治疗费及支付的鉴定费1650元的事实；6.交通费发票一组，以证明原告就医支付的交通费用；7.人民调解协议一份，以证明原告与余姚市人民医院就医疗行为达成赔偿协议的情况。
　　被告符甲、符乙共同答辩称：本被告已支付原告125000元及医疗费25919.30元，对原告的合理损失愿意进行赔偿，但原告的医疗费中70%是由于余姚市人民医院医疗事故引起的，不应由本被告承担。被告符甲还提供了收条、医疗费发票、更改病人住处单、病历卡一组，以证明其已支付的款项及医疗费的事实。
　　被告中银保险公司答辩称：愿意在交强险分项责任限额内承担原告的合理损失，对不合理部分请求依法驳回；本被告已垫付医疗费1万元，故在交强险赔偿范围内对医疗费、住院伙食补助费、营养费、后续医疗费等均不再承担赔偿责任；因交警部门认定原告与被告符甲承担事故的同等责任，对超出交强险部分应按50%比例承担赔偿责任；不承担诉讼费、鉴定费及非医保范围的费用。
　　对原告提供的证据1、2，三被告无异议，本院予以采信。对证据3，三被告对真实性无异议，本院对该证据的真实性予以采信。对证据4，被告符甲、符乙对真实性无异议，认为原告的大部分医疗费系因余姚市人民医院的医疗事故引起，应由医院承担；被告中银保险公司对真实性无异议，对关联性有异议，认为医疗事故引起的医疗费不属于保险赔偿范围。本院对该组证据的真实性予以采信，对关联性将结合其他证据予以阐述。对证据5，三被告无异议，但被告中银保险公司认为后续医疗费不属于法医临床鉴定的范围。本院认为，司某某定机构针对原告的损伤情况进行司某某定，因目前原告内固定仍存在，故对后续医疗费进行鉴定并不违反规定，故本院对该组证据予以采信。对证据6，被告符甲、符乙无异议，被告中银保险公司对真实性无异议，对关联性有异议，认为大部分发票的时间与就诊地点不符，且交通费过高，要求酌情认定。本院认为，原告受伤后因就医必然产生一定的交通费用，根据就医地点、时间、次数等情况酌情确认交通费为2000元。对证据7，被告符甲、符乙对真实性无异议，但认为原告实际因医疗事故造成的损失超过4万元；被告中银保险公司对真实性无异议，对关联性有异议，认为该调解系原告与医院自行达成，应当进行医疗事故鉴定以查明医院的治疗行为是否恰当，从而确定原告因交通事故造成的损失。本院对该证据的真实性予以采信，由于涉及到赔偿问题，对关联性亦予以采信。
　　对被告符甲提供的证据，原告及被告中银保险公司、符乙均无异议，本院予以采信。
　　根据被告符甲的申请，本院委托宁波三益司某某定所进行司某某定。对该所出具的司某某定意见书及鉴定费发票，原告无异议，被告符甲、符乙对真实性无异议，但认为原告在浙江大学医学院附属第二医院的大部分医疗费是由于余姚市人民医院的医疗事故引起；被告中银保险公司对真实性无异议，但认为医疗事故鉴定对查明事实非常关键。本院认为，该鉴定机构系本院委托，故予以采信。
　　根据认定的证据和原、被告的陈述，本院认定如下事实：2009年11月14日上午，被告符甲驾驶浙B×××××号轻型普通货车从余姚市马渚镇××村杨岐岙驶往马渚集镇。9时40分许，该车自西往东驶经余姚市马渚镇××村杨岐岙村道四枝交叉路口时，车头正面与原告驾驶的在路口××向南行驶的电动自行车相碰撞，造成原告受伤及两车损坏的交通事故。经余姚市公安局交通警察大队认定，被告符甲与原告应各承担本起事故的同等责任。原告受伤后即被送往余姚市人民医院住院治疗6天，诊断为腹腔内出血、右胫腓骨上段粉碎性骨折、右髂骨粉碎性骨折等。后原告转入浙江大学医院院附属第二医院住院治疗，被诊断为多发伤：1、头皮裂伤，2、右第7肋骨骨折、双侧胸腔积液伴肺不张，3、腹部闭合伤：回肠破裂，4、骨盆骨折，5、右胫腓骨骨折，6、第4颈椎、第7胸椎压缩性骨折、第4颈椎右侧椎弓、第3-6左侧横突孔骨折，7、脓毒血症。入院后行回肠破裂修补、空肠造瘘、腹腔外引流，以及右侧胫骨切开复位植骨内固定术。经原告委托，宁波崇新司某某定所于2010年9月20日出具司某某定意见书，认为原告因交通事故致回肠破裂修补的伤残等级属10级，骨盆畸形愈合的伤残等级属10级；护理期限为4个月；误工期限为6个月；营养期限为4个月；原告具有医疗费发生的客观基础，拆除内固定物的后续医疗费为7000元。浙B×××××号轻型普通货车的所有人为被告符乙，借给被告符甲使用，并在被告中银保险公司投保交强险。事故发生后，被告符甲已支付原告款项125000元，并垫付医疗费25039.03元；被告中银保险公司已支付医疗费10000元。
　　本案在审理过程中，根据被告符甲的申请，本院委托宁波三益司某某定所进行鉴定。该所的鉴定意见认为，原告回肠破裂应与本次交通事故存在直接因果关系，休息期限累计为6个月，护理期限累计为4个月（住院期间为完全护理、出院后为部分护理），营养期限累计为4个月。
　　2011年2月22日，经余姚市医疗纠纷人民调解委员会主持，原告与余姚市人民医院达成人民调解协议，由余姚市人民医院一次性补偿原告姜某某40000元，扣除已支付的10000元，实付30000元，该款已包括扩大的医疗费用，款项在协议生效后7日内履行完毕。
　　本案双方当事人存在的争议焦点在于：原告的回肠破裂是否系本起交通事故引起，原告主张的医疗费中是否包含因余姚市人民医院的医疗过错行为所造成的费用？原告认为，其损伤均系交通事故造成，且在变更的诉请中已扣除了余姚市人民医院赔偿的40000元医疗费；三被告则认为原告在余姚市人民医院住院治疗期间，因该院存在医疗过错导致医疗费用扩大，对由此引起的费用不同意承担。本院认为，因原告转入浙江大学医学院附属第二医院住院治疗时，被诊断为多发伤：头皮裂伤，右第7肋骨骨折、双侧胸腔积液伴肺不张，腹部闭合伤：回肠破裂，骨盆骨折，右胫腓骨骨折，第4颈椎、第7胸椎压缩性骨折、第4颈椎右侧椎弓、第3-6左侧横突孔骨折，脓毒血症。现双方当事人对原告的骨折损伤系交通事故所引起并无争议，主要争议在于原告的回肠破裂是否系余姚市人民医院医疗过错引起。根据被告符甲的申请，本院委托宁波三益司某某定所进行鉴定，已明确原告回肠破裂与本起交通事故存在直接因果关系，且原告在变更后的诉请中已将余姚市人民医院的赔偿款40000元予以扣除，故原告在本案中所主张的医疗费系本起交通事故所引起，应当在本案中予以处理。
　　原告的各项损失，本院认定如下：
　　1.医疗费。根据原告与被告符甲提供的医疗费票据，经本院核实医疗费为294274.63元，扣除由余姚市人民医院赔偿的40000元后，医疗费为254274.63元，其中原告支付219235.60元，被告符甲垫付25039.03元，被告中银保险公司支付10000元。
　　2.误工费。经鉴定，原告误工时间为6个月，参照2009年度宁波市职工平均工资31290元／年，误工费合计15431.40元。
　　3.护理费。原告的护理时间为4个月，现原告住院时间合计为80天，根据住院期间完全护理、出院后部分护理，参照2009年度宁波市职工社会平均工资标准，按住院期间每天85.73元、出院后每天40元计算，原告的护理费合计为8458.40元。
　　4.交通费。根据原告就医次数、时间及地点，本院酌情确定为2000元。
　　5.住院伙食补助费。原告住院时间80天，现其以每天30元主张住院伙食补助费，并未超过相关标准，合计2400元。
　　6.营养费。经鉴定，原告的营养期限为4个月，故本院确认营养费为3200元。
　　6.残疾赔偿金。经鉴定，原告之伤构成2个10级伤残，且其系农村居民，残疾赔偿金计算为12641×20×12%=30338.40元。
　　7.后续治疗费。根据司某某定意见书，原告目前右胫腓骨内固定物仍存，需要待骨折完全愈合后拆除内固定，具有后续医疗费发生的客观基础，并建议后续医疗费为7000元，对此本院予以确认。
　　8.精神损害抚慰金。原告的伤残等级为2处10级，精神上遭受了一定的痛苦，确有进行精神损害抚慰的需要，结合本起交通事故的具体情节，本院酌情确定为3000元。
　　9.鉴定费。原告因鉴定支付鉴定费1560元，系因本次交通事故而产生，本院予以确认。
　　本院认为，国家实行机动车交通事故责任强制保险制度。投保机动车发生交通事故造成他人人身、财产受损的，由保险公司在机动车第三者责任强制保险限额范围内予以赔偿。浙B×××××号轻型普通货车在被告中银保险公司投保交强险，故被告中银保险公司在交强险限额内承担赔偿责任，符合法律规定，本院予以支持。对原告损失，在交强险责任限额范围内的项目有：医疗费10000元、残疾赔偿金30338.40元、误工费15431.40元、护理费8458.40元、交通费2000元、精神损害抚慰金3000元，合计69228.20元，应由被告中银保险公司予以赔付。对交强险责任限额范围外原告的损失：医疗费244274.63元、后续治疗费7000元、住院伙食补助费2400元、营养费3200元、鉴定费1560元，合计258434.63元，应根据原告与被告符甲在本起交通事故中的过错确定民事赔偿责任。本起交通事故发生在机动车与非机动车之间，原告驾驶未经公安机关交通管理部门登记的电动自行车上路行驶，通过交叉路口时未停车瞭望，未让右方道路来车先行，对事故的发生存在过错，可减轻被告符甲的赔偿责任；而被告符甲驾驶机动车通过交叉路口时未减速慢行，未在进入路口前停车瞭望确保安全，上路行驶，且在经过交叉路口时未让右方道路的来车先行，亦存在过错，本院酌情确定被告符甲承担60%的赔偿责任，原告自身承担40%的责任。被告符乙作为浙B×××××号轻型普通货车的所有人，对被告符甲的赔偿责任应承担连带责任。据此，依照《中华人民共和国民法通则》第一百零六条第二款、第一百一十九条、第一百三十条、第一百三十一条、《中华人民共和国道路交通安全法》第七十六条第一款、《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》第十七条、第十八条、第十九条、第二十条、第二十一条、第二十二条、第二十三条、第二十四条、第二十五条、第三十一条、第三十五条以及《中华人民共和国民事诉讼法》第六十四条第一款之规定，判决如下：
　　一、被告中银保险有限公司××中心支公司在交强险责任限额范围内赔偿原告姜某某医疗费10000元、残疾赔偿金30338.40元、误工费15431.40元、护理费8458.40元、交通费2000元、精神损害抚慰金3000元，合计69228.20元，扣除已支付的医疗费10000元，尚应支付59228.20元；
　　二、被告符甲赔偿原告姜某某在交强险责任限额范围外的医疗费244274.63元、后续治疗费7000元、住院伙食补助费
