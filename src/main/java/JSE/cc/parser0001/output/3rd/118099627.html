李某某聚众扰乱社会秩序案

审判时间：20110706 法院：河南省灵宝市人民法院 案件字号：(2011)灵刑初字第194号


　　公诉机关灵宝市人民检察院。
　　被告人李某某。因涉嫌犯聚众扰乱社会秩序罪于2010年9月8日被刑事拘留，同年10月14日被逮捕，同年10月26日被灵宝市公安局取保候审，2011年3月1日被灵宝市人民检察院取保候审，同年4月18日被本院取保候审。
　　被告人李某某1。因涉嫌犯聚众扰乱社会秩序罪于2010年9月8日被刑事拘留，同年10月14日被逮捕，同年10月26日被灵宝市公安局取保候审，2011年3月1日被灵宝市人民检察院取保候审。同年4月18日被本院取保候审。
　　被告人白某某。因涉嫌犯聚众扰乱社会秩序罪于2010年9月8日被刑事拘留，同年10月14日被逮捕，同年10月26日被灵宝市公安局取保候审，2011年3月1日被灵宝市人民检察院取保候审。同年4月18日被本院取保候审。
　　灵宝市人民检察院以灵检刑诉（2011）116号起诉书指控被告人李某某、李某某1、白某某犯聚众扰乱社会秩序罪，于2011年4月18日向本院提起公诉。本院依法组成合议庭，公开开庭审理了本案。庭审后，灵宝市人民检察院建议延期审理，对本案进行补充侦查后，将本案重新移送本院，本院再次公开开庭审理了本案。灵宝市人民检察院指派检察员张庆平出庭支持公诉，被告人李某某、李某某1、白某某到庭参加诉讼。现已审理终结。
　　灵宝市人民检察院指控，2010年9月4日16时45分，李进超因酒后到焦村卫生院就诊，于当日19时50分抢救无效死亡。被告人李某某、白某某与李某某1及死者亲属等人于9月5日上午8时开始将冰棺停放在焦村卫生院门诊楼大厅内砸坏门诊楼一楼门窗玻璃，封堵医院大门；在大门口设置灵棚，悬挂挽帐，摆放花圈，张贴标语，并雇佣乐队在医院门口聚众闹事。李某某将堂兄弟出资的5000元作为费用，分两次付给李某某1由其具体负责各项开支，并在现场帮忙放置花圈、张贴大字等活动。被告人李某某1联系帐篷、乐队并负责招待来人及各项开支。被告人白某某和其他亲属多人多次在医院办公楼内及病区其他地方谩骂医护人员，并对医院工作人员王静威胁、围堵3个多小时。直到9月8日下午14时许，民警到场强行拆除灵棚并将三被告人当场抓获。认定上述事实的证据如下：被告人李某某、李某某1、白某某的供述和辩解；证人李一某、李二某、李三某等证言；价格鉴定结论；书证户籍证明、抓获证明、提取笔录等。公诉机关认为，被告人李某某、李某某1、白某某伙同他人聚众扰乱社会秩序，致使焦村镇卫生院医疗秩序遭到严重破坏，医疗活动无法正常进行，其行为均触犯了《中华人民共和国刑法》第二百九十条第一款，犯罪事实清楚，证据确实、充分，应当以聚众扰乱社会秩序罪追究其刑事责任。请求依法判处。
　　被告人李某某、李某某1、白某某对指控事实及罪名不持异议。
　　经审理查明，2010年9月4日16时45分，李进超因酒后感到身体不适，到灵宝市焦村镇卫生院就诊，于当日19时50分抢救无效死亡。被告人李某某、李某某1、白某某与死者亲属等人于次日上午8时开始，将冰棺停放在焦村镇卫生院门诊楼一楼大厅内，砸坏门诊楼一楼门窗玻璃、门锁，封堵医院大门，在大门口设置灵棚、悬挂挽帐、摆放花圈、张贴标语，并雇佣乐队在医院门口聚众闹事。被告人李某某将堂兄弟出资的5000元作为费用，分两次付给被告人李某某1，并在现场帮忙放置花圈、张贴大字等活动。被告人李某某1帮忙联系帐篷、乐队并具体负责各项开支。被告人白某某同死者李进超的亲属多人，多次在焦村镇卫生院办公楼内及病区其他地方谩骂医护人员，并对医院工作人员王静威胁、围堵3个多小时。被告人李某某、李某某1、白某某等人的行为致使焦村镇卫生院医疗秩序遭到严重破坏，医疗活动无法正常进行。直至9月8日下午14时许，灵宝市公安局民警到现场强行拆除灵棚，并将被告人李某某、李某某1、白某某当场抓获。经灵宝市价格认证中心对焦村镇卫生院被砸坏门窗玻璃、门锁进行价值评估，被损坏物品的损失价值为2350元，案发后，死者李进超家属已经予以赔偿。
　　上述事实，有公诉机关提供，并经庭审质证、认证的下列证据证实：1、被害单位灵宝市焦村镇卫生院的报案材料，证人许某、薛某、崔某、李一某、李二某、李三某、荆某、李四某、刘某、王某、赵某、李五某、马某、李六某、张某、焦某等人的证言，书证提取笔录及账目复印件、医疗纠纷处置材料，现场照片，证实了被告人李某某、李某某1、白某某聚众扰乱社会秩序的事实；2、灵宝市价格认证中心价格鉴定结论，书证户籍证明、抓获证明，证实被毁坏财物的损失价值及三被告人到案情况；3、书证灵宝市焦村镇卫生院证明、协议书、收条，证实焦村镇卫生院的财物损失已经得到赔偿；4、被告人李某某、李某某1、白某某的供述与上述证据能够相互印证。以上证据确实、充分，足以认定。
　　本院认为，被告人李某某、李某某1、白某某伙同他人聚众扰乱社会秩序，情节严重，致使灵宝市焦村镇卫生院医疗秩序遭到严重破坏，医疗活动无法正常进行，其行为均构成聚众扰乱社会秩序罪，应予惩处。公诉机关指控被告人李某某、李某某1、白某某犯聚众扰乱社会秩序罪的事实清楚，证据确实、充分，指控罪名成立。被告人李某某、李某某1、白某某能够如实供述自己的罪行，认罪态度较好，且被害单位的财物损失已经得到赔偿，对三被告人均可从轻处罚，并适用缓刑。依照《中华人民共和国刑法》第二百九十条第一款、第二十五条第一款、第六十七条第三款、第七十二条第一款、第七十三条第二款的规定，判决如下：
　　一、被告人李某某犯聚众扰乱社会秩序罪，判处有期徒刑六个月，缓刑一年；
　　二、被告人李某某1犯聚众扰乱社会秩序罪，判处有期徒刑六个月，缓刑一年；
　　三、被告人白某某犯聚众扰乱社会秩序罪，判处有期徒刑六个月，缓刑一年。
　　（缓刑考验期限，从判决确定之日起计算。）
　　如不服本判决，可在接到判决书的第二日起十日内，通过本院或者直接向三门峡市中级人民法院提出上诉。书面上诉的，应当提交上诉状正本一份，副本两份。
　　审判长田伟民
　　人民陪审员陈晓
　　人民陪审员师苏秦
　　二○一一年七月六日
　　书记员杨文平
　　附：本判决适用法律条文
　　《中华人民共和国刑法》
　　第二百九十条聚众扰乱社会秩序，情节严重，致使工作、生产、营业和教学、科研无法进行，造成严重损失的，对首要分子，处三年以上七年以下有期徒刑；对其他积极参加的，处三年以下有期徒刑、拘役、管制或者剥夺政治权利。
　　聚众冲击国家机关，致使国家机关工作无法进行，造成严重损失的，对首要分子，处五年以上十年以下有期徒刑；对其他积极参加的，处五年以下有期徒刑、拘役、管制或者剥夺政治权利。
　　第二十五条共同犯罪是指二人以上共同故意犯罪。
　　二人以上共同过失犯罪，不以共同犯罪论处；应当负刑事责任的，按照他们所犯的罪分别处罚。
　　第六十七条犯罪以后自动投案，如实供述自己的罪行的，是自首。对于自首的犯罪分子，可以从轻或者减轻处罚。其中，犯罪较轻的，可以免除处罚。
　　被采取强制措施的犯罪嫌疑人、被告人和正在服刑的罪犯，如实供述司法机关还未掌握的本人其他罪行的，以自首论。
　　犯罪嫌疑人虽不具有前两款规定的自首情节，但是如实供述自己罪行的，可以从轻处罚；因其如实供述自己罪行，避免特别严重后果发生的，可以减轻处罚。
　　第七十二条对于被判处拘役、三年以下有期徒刑的犯罪分子，同时符合下列条件的，可以宣告缓刑，对其中不满十八周岁的人、怀孕的妇女和已满七十五周岁的人，应当宣告缓刑：
　　（一）犯罪情节较轻；
　　（二）有悔罪表现；
　　（三）没有再犯罪的危险；
　　（四）宣告缓刑对所居住社区没有重大不良影响。
　　宣告缓刑，可以根据犯罪情况，同时禁止犯罪分子在缓刑考验期限内从事特定活动，进入特定区域、场所，接触特定的人。
　　被宣告缓刑的犯罪分子，如果被判处附加刑，附加刑仍须执行。
　　第七十三条拘役的缓刑考验期限为原判刑期以上一年以下，但是不能少于二个月。
　　有期徒刑的缓刑考验期限为原判刑期以上五年以下，但是不能少于一年。
　　缓刑考验期限，从判决确定之日起计算。
