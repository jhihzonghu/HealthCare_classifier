陈某诉某医院甲、某医院乙医疗损害责任纠纷案

审判时间：20110915 法院：上海市杨浦区人民法院 案件字号：(2011)杨民一(民)初字第2944号


　　原告陈某。
　　被告某医院甲。
　　被告某医院乙。
　　原告陈某诉被告某医院甲、某医院乙医疗损害责任纠纷一案，本院受理后，依法适用简易程序，由审判员陈某独任审判，公开开庭进行审理。原告陈某及其委托代理人施某、王某、被告某医院甲的委托代理人束某、被告某医院乙的委托代理人吴某、彭某均到庭参加诉讼。本案现已审理终结。
　　原告陈某诉称，原告陈某于2007年12月13日在某医院乙因诊断为胆囊息肉，而接受其腹腔胆囊切除手术。术后出现严重黄疸，并伴有全身乏力，发烧等一系列不良症状再次入院进行治疗。2007年12月23日经该院胆道MRCP显示：1、肝内胆管明显扩张，梗阻平面位于肝门区；2、胆囊切除术后改变；3、多发性肝囊肿。查肝功能后该院诊断为胆总管阻塞。2007年12月26日该院为原告在全麻下行胆道探查，胆总管T管引流手术。然而，在半年后取出引流T管后，不良症状依旧反复出现，包括发烧、咳嗽，呕吐，腹部发涨、疼痛难忍。原告陈某于2008年11月24日至被告某医院甲就诊，门诊诊断为胆总管结石。MRCP示：胆总管下段结石。探查：腹腔未见明显腹水，肝脏色泽质地正常。胆囊缺如，肝总管上端扩张，下端狭窄，壁厚水肿，胆总管下端扪及结石，右上腹肝下有一10×10cm大小的肿块，质软，有波动动感与腹壁及周围组织粘连，2008年11月27日再次手术，术中遂决定胆总管探查取石+肝总管成型+胆肠吻合术+脓肿切开引流术。2009年8月31日，原告陈某至被告某医院甲拨T管。同年9月15日至被告某医院甲处做核磁共振，MRCP示：肝门部胆总管及左肝管多发充盈缺失伴肝内外胆管扩张，考虑多发结石可能，2009年9月25日第四次手术，全麻下行胆道探查+胆肠再吻合术，10月6日原告陈某出院。原告认为被告某医院乙手术操作不当，导致手术失败并造成胆总管受损；医疗过程中隐瞒病情，导致原告陈某为此又进行了三次手术，并由此引发严重并发症；被告某医院乙未尽合理注意义务；其第二次手术未对切除的组织进行病理学检查。同时，原告陈某认为2008年11月27日原告在被告某医院甲手术后出现感染症状，被告某医院甲未履行充分告知义务，导致需要进行第二次手术。原告多次与两被告协商未果，遂诉至法院，要求被告某医院甲、某医院乙共同赔偿医疗费人民币（以下所涉货币均为人民币）51,816.50元、住院伙食补助费2130元、陪护费19,650元、误工费146,400元、残疾生活补助费208,800元、交通费15,206.50元、住宿费8943元、律师代理费20,000元、精神损害抚慰金50,000元。
　　被告某医院甲辩称，医院术前已经将手术的风险及可能面临的并发症完全告知患者家属知晓，并不存在未履行义务，故该医疗争议不构成医疗事故，被告不承担本起医疗行为的侵权赔偿责任。对黄浦区医学会、上海市医学会对原告与两被告的医疗过程的鉴定结论无异议。对原告主张的医疗费，认可发生在被告某医院甲的医疗费用；对原告主张住院伙食补助费、残疾生活补助费、精神损害抚慰金，无异议；陪护费，认为原告提供的证据是由单位出具的，不足以证明其实际损失；误工费，原告提供的证据系其单位工资统计，不足以证明其实际误工损失，要求按照事故发生地的平均工资计算；交通费，其真实性无异议，但无法证明其发生用于原告就诊；住宿费，认为该部分与陪护费部分有重复，且另有发票抬头开的是单位，无法证明该费用由原告支出；律师代理费，待原告庭后提供发票后，由法院依法酌定。
　　被告某医院乙辩称，医院对原告陈某诊断明确，诊治合理，未违反医疗诊疗常规。原告个体体质与异物出现的过敏反应，与医院对原告的诊治无直接相关，故该医疗争议不构成医疗事故，被告某医院乙不承担本起医疗行为的侵权赔偿责任。对黄浦区医学会、上海市医学会出具的两份鉴定报告分析意见有异议，认为报告中对胆总管损伤部分仅是分析意见，并无相关检查，亦未明确该损伤是否由被告某医院乙造成。对原告主张的医疗费，要求原告提供自购药部分的处方笺，不认可其自行购买中药的费用；住院伙食补助费，因原告的首次手术系治疗其原发疾病所需，故该次手术期间的住院伙食补助费应予扣除；陪护费，认为原告是否需要护理应由医疗机构出具证明，故对原告该项请求不予认可，即使需要护理，具体的计算标准应与医院目前的护理费标准一致，且应扣除治疗原发疾病期间发生的费用；误工费，原告提供的证明不足以证明其实际损失，不予认可；残疾生活补助费，要求按照医疗事故发生地江西省的人均生活费标准计算；交通费，原告提供的证据不足以证明该费用系用于其就诊行为，由法院依法酌定；住宿费，对抬头是单位的发票所载明的费用，意见同被告某医院甲意见一致，对抬头是个人的发票所载明的费用，因其发生的时间与原告住院时间不对应，故不予认可；精神损害抚慰金，应按医疗事故发生地标准计算，且考虑被告在事故中承担的责任程度；律师代理费，该费用的赔偿无法律依据，不予认可。
　　经审理查明，2007年12月13日，原告陈某因“6年前体检发现胆囊息肉”入住被告某医院乙。据住院病历记载，入院前2天B超检查示：①胆囊多发息肉；②肝多发囊肿。查体：全身皮肤粘膜无黄染。12月14日，血常规及肝肾功能无异常。诊断：胆囊息肉。12月18日，原告在全麻下行腹腔镜胆囊切除术。术后予抗炎、止血治疗。12月21日，病理诊断：（胆囊）慢性炎。12月21日原告出院。出院时情况：一般情况好，无发热、黄疸，无腹痛。2007年12月23日，原告因“腹腔镜胆囊切除术后5天黄疸”再次入住被告某医院乙。门诊胆道MRI检查示：①肝内胆管明显扩张，梗阻平面位于肝门区；②胆囊切除术后改变；③多发性肝囊肿。查休：全身皮肤粘膜黄染，下腹部反跳痛阴性，Murphy征阳性可疑。肝功能：谷丙转氨酶368u/L(参考值0-40u/L)，谷草转氨酶237u/L（参考值0-40u/L），总胆红素239.1umol/L（参考值0-20umol/L），结合胆红素161.5umol/L（参考值0-7umol/L）。诊断：胆囊切除术后黄疸。12月26日，ERCP检查示：肝内胆管扩张，胆总管上段狭窄，其上见一约0.5×0.6cm高密度影，反复用导丝探查无法通过狭窄部。原告当日在全麻下行胆道探查+T管引流术。探查所见：肝总管部有一炎性肿块压迫，梗阻上方肝管及肝内胆管见扩张，二可吸收夹未损伤胆管，胆道连续性良好，放置一14号T管并术中造影无异常。术后予止血、补液、抗炎等治疗。2008年1月7日，复查肝功能：谷丙转氨酶179u/L，谷草转氨酶55u/L，总胆红素60.2umol/L，结合胆红素40.3umol/L。1月8日，原告出院。出院诊断：胆囊切除术后，肝总管梗阻。2008年11月17日，原告因“腹痛腹胀3天”第三次入住被告某医院乙。查体：皮肤粘膜轻度黄染，右上腹部轻度压痛，无反跳痛。MRCP检查示：①胆总管结石；②肝总管结石；③左肝内胆管结石；④肝内多发长T2信号影；⑤胆囊窝混杂信号影；⑥脾大。诊断：①肝内外胆管结石；②胆道感染。入院后予抗炎、对症、支持治疗。11月20日查肝功能：谷丙转氨酶94u/L，谷草转氨酶109u/L，总胆红素102.5umol/L，结合胆红素71.0umol/L。11月21日，原告出院。出院情况：无发热，腹部无明显压痛及反跳痛。2008年11月24，原告陈某因“右上腹疼痛反复发作1月”入住被告某医院甲。据住院病历记载，查体：皮肤、巩膜无黄染，右上腹深压痛，无反跳痛，莫菲氏征（+），肝区叩痛明显。MRCP提示：胆总管下段结石。诊断：阻塞性黄疸，胆总管结石。11月27日，原告在全麻下行胆总管探查取石+肝总管成型+胆肠吻合+脓肿切开引流术。术中完全游离右上腹部的脓肿与周围组织的粘连，打开脓腔可见草绿色混浊液体，考虑系胆汗外漏所致。切开胆总管见胆管壁水肿增厚，未发现结石。肝总管部狭窄。术后予抗感染、利尿、止痛等治疗。12月1日，病理诊断报告：（胆道）符合化脓性炎。12月5日，原告出院。出院诊断：①胆总管结石；②肝总管狭窄；③右上腹部脓肿。2009年9月22日，原告因“皮肤粘膜黄染伴发热1周”入住被告某医院甲。查体：右上腹轻压痛，无反跳痛、肌紧张。MRCP检查（2009-9-17）：①肝门部胆总管及左肝管多发充盈缺损伴肝内外胆管扩张，考虑多发结石可能；②肝囊肿；③脾肿大。彩超（2009-9-19）：肝内外胆管扩张，胆总管内团状高回声，性质待定；主胰管扩张，内有支架回声。诊断：①肝内胆管结石；②胆道术后。9月25日，原告在全麻下行胆道探查+胆肠再吻合术。术中打开胆肠吻合口，探查左右肝管均增粗，左右肝管均触及结石，最大结石约4×3cm大小。10月6日，原告出院。出院时情况：一般情况可，病情平稳。治疗期间，原告共花费医疗费45,090.17元。
　　2010年11月15日，经本院委托上海市黄浦区医学会就被告某医院甲、某医院乙与原告陈某之间的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，是否与原告陈某目前病情具有因果关系及本案病例是否构成医疗事故等进行鉴定，该鉴定机构于2011年3月4日向本院出具鉴定报告一份，分析意见为：（一）某医院甲：1、针对原告外院术后腹腔感染、胆管狭窄和结石等进行了2次手术治疗，诊断和治疗符合医疗常规；2、医方已履行相关告知义务。（二）某医院乙：1、原告因胆囊多发性息肉有手术指征，行腹腔镜胆囊切除术，术后5天内出现黄疸，术后ERCP检查提示胆总管上段狭窄，导丝无法通过，首先考虑术中胆总管损伤，与医方术中操作不当有关，造成原告需行后续手术治疗；2、根据原告术后出现的黄疸等情况，需行胆管引流和/或胆管空肠吻合术。结论为：（一）原告陈某与某医院甲医疗争议不构成医疗事故；（二）原告陈某与某医院乙医疗争议构成医疗事故，本病例属于三级丙等医疗事故，医方承担主要责任。被告某医院乙对鉴定结论不服，申请再次鉴定。2011年3月29日，经本院委托上海市医学会就被告某医院甲、被告某医院乙与原告陈某之间的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，是否与原告陈某目前病情具有因果关系，是否属于医疗事故，等级及责任程度进行鉴定，该鉴定机构于2011年5月30日向本院出具鉴定报告两份，分析意见为：（一）某医院甲：1、2008年11月24日，原告因“右上腹疼痛反复发作1月”入住某医院甲。根据症状、体征、实验室检查及MRCP检查，阻塞性黄疸、胆总管结石诊断成立。腹部超声检查提示胆囊窝非均质性回声，医方行胆总管探查取石+肝总管成型+胆肠吻合+脓肿切开引流术有指征；2、某医院甲的手术补救行为是原告发生在胆总管损伤以后，而胆道反复感染乃至结石复发，为此类手术在现有医学条件下难以完全避免；3、根据送鉴病史，原告目前无黄疸，一般情况尚可，说明某医院甲手术治疗有效。结论为：原告陈某与某医院甲医疗争议不构成医疗事故。（二）某医院乙：1、2007年12月13日，原告因“体检发现胆囊息肉”入住被告某医院乙。根据症状、体征及辅助检查，胆囊息肉诊断明确。行腹腔胆囊切除术有指征；2、根据送鉴的病史材料，原告在行腹腔镜术后短期内出现黄疸（术后5天总胆红素239.1umol/L）；MRCP提示肝内胆管明显扩张，梗阻平面位于肝门区；行ERCP检查导丝无法通过狭窄部；胆道探查术后放置T管引流长达8月余。以上证据均提示医方因手术操作失误导致总胆管梗阻可能性大，且医方未能提供拔除胆总管有其他损伤的相关证据。故原告胆管损伤与后续手术治疗的后果之间有直接的因果关系；3、腹腔镜胆囊切除手术本身具有一定风险，医方术前对此已进行告知。结论为：本病例属于三级丙等医疗事故，医方承担主要责任。
　　2011年6月10日，本院委托司法鉴定科学技术研究所司法鉴定中心就原告陈某在医院诊疗过程中发生医疗事故所需的休息期进行鉴定，鉴定结论为原告因胆囊多发息肉多次就诊于被告某医院乙，其诊疗过程中发生医疗事故所需的休息期为270－300天；后该鉴定机构于2011年8月25日出具补充鉴定意见书一份，结论为：原告在被告某医院甲的后续治疗中，可再酌情休息390－420日。原告垫付了鉴定费900元。
　　本院认为，医院作为从事医疗服务这一特殊行业的机构，其执业活动不但应当以救死扶伤、防病治病为宗旨，还应遵守有关法律、法规的规定，以保证患者的基本权利不受侵犯。在医疗关系中，患者享有获得适当、合理治疗的权利。基于此，医院亦应负有进行适当、合理治疗的义务。依据上海市医学会、黄浦区医学会医疗事故技术鉴定中心出具的鉴定报告及被告提供的病历，被告某医院乙依据确诊的胆囊息肉为原告行腹腔镜胆囊切除术有手术指征，但是原告术后短期内出现黄疸症状，以及相关检查提示，肝内胆管明显扩张等情形，因此推断手术操作失误导致原告陈某总胆管梗阻可能性大，且被告某医院乙未能提供证据排除胆总管因其它原因造成损伤。故证明原告目前存在的胆管损伤与后续手术治疗的后果之间有直接的因果关系。被告某医院乙虽对这一鉴定结论提出质疑，但未提供证据足以推翻鉴定结论，因此本院对该鉴定结论予以确认，故被告某医院乙的医疗行为对原告构成侵权。同样依据鉴定报告，腹腔镜胆囊切除手术本身具有一定风险，但被告某医院乙对此已进行告知，故被告某医院乙承担主要的民事赔偿责任，责任参与度确定为80%。被告某医院甲对原告陈某的手术治疗有效，虽术后有胆道反复感染乃至结石复发，也是此类手术在现有医学条件下难以完全避免的，故被告某医院甲的医疗行为不构成医疗事故，不承担民事赔偿责任。被告某医院乙应按责赔偿原告包括医疗费在内的相关费用。一、对原告主张的医疗费，凭据结算，确定为45,090.17元，至于被告辩称扣除医疗费中的自购药部分，因该部分费用系原告治疗疾病所必需，故对被告的此辩称不予支持；二、住院伙食补助费，参照原告实际住院天数，根据每天20元标准计算，确定为1160元；三、陪护费，根据原告的病情，在住院期间需专人陪护，参照原告的住院天数，根据上海市2010年职工平均工资计算，确定为7430元；四、误工费，参照鉴定报告确定的休息时限，依据原告提供的实际误工损失证据，酌情确定为120,000元；五、残疾生活补助费，参照鉴定报告确定的三级丙等医疗事故，相当于八级伤残等级，依据医疗事故发生地江西省城镇居民平均生活费计算，确定为95,568.30元；六、交通费，根据原告治疗、处理医疗纠纷等事宜，酌情确定为6000元；七、住宿费，根据家属在上海陪护原告的实情，酌情确定为3000元；八、精神损害抚慰金，考虑到原告的实际受伤害程度及被告的责任、经济能力等因素，酌情确定为15,000元；九、律师代理费，系原告通过司法途径以维护其合法利益所支出的实际费用，考虑到事故责任程度、赔偿标的等因素，确定为10,000元。据此，依照《中华人民共和国民法通则》第一百零六条第二款、《医疗事故处理条例》第四十九条，第五十条第一款第一项、第二项、第三项、第四项、第五项、第九项、第十项、第十一项，第五十一条第一款之规定，判决如下：
　　一、被告某医院乙应于本判决生效之日起十日内赔偿原告陈某医疗费人民币36,072.10元;
　　二、被告某医院乙应于本判决生效之日起十日内赔偿原告陈某住院伙食补助费人民币928元;
　　三、被告某医院乙应于本判决生效之日起十日内赔偿原告陈某陪护费人民币5944元；
　　四、被告某医院乙应于本判决生效之日起十日内赔偿原告陈某误工费人民币96,000元;
　　五、被告某医院乙应于本判决生效之日起十日内赔偿原告陈某残疾生活补助费人民币76,454.64元；
　　六、被告某医院乙应于本判决生效之日起十日内赔偿原告陈某交通费人民币4800元；
　　七、被告某医院乙应于本判决生效之日起十日内赔偿原告陈某住宿费人民币2400元；
　　八、被告某医院乙应于本判决生效之日起十日内赔偿原告陈某精神损害抚慰金人民币15,000元；
　　九、被告某医院乙应于本判决生效之日起十日内赔偿原告陈某律师代理费人民币10,000元；
　　十、原告陈某的其余诉讼请求，不予支持。
　　如未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　本案受理费人民币9030元，减半收取人民币4515元，由原告陈某负担人民币2008.50元，由被告某医院乙负担人民币2506.50元；本案鉴定费人民币14,000元，由被告某医院乙负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第二中级人民法院。
