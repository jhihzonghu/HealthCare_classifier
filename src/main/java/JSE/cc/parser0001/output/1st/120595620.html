张娟等诉文县第二人民医院确认案

审判时间：20110126 法院：甘肃省文县人民法院 案件字号：(2011)文民初字第30号


　　原告张娟。
　　法定监护人董冬。
　　原告张奎强（原告张娟父亲）。
　　被告文县第二人民医院。
　　法定代表人曹树平，文县第二人民医院院长。
　　上列当事入因医疗纠纷赔偿协议请求法院确认一案，原告张娟、张奎强向本法院提起诉讼，原、被告人共同申请法院实用简易程序审理。本院受理后，依法由审判员董跟会独任审理，于2011年1月25日进行了公开开庭审理，原告及法定监护人，被告法定代表人均出庭参加诉讼，本案现已审理终结。
　　原告诉称，张娟在2010年3月31日在文县第二人民医院进行无痛人流术，因麻醉意外，导致原告人张娟缺血缺氧性脑病，经文县第二人民医院先后在文县第二人民医院，四川省广元市第一人民医院，四川省华西医学中心附属第一医院，第四医院长期医治，经专家鉴定，病人病情稳定，已转本地医院康复治疗三月，病人处于植物生存状态。为了原、被告的切身利益，双方已就原告张娟的医疗纠纷达成了赔偿协议，文县第二人民医院赔付给原告张娟后续医疗费、残疾生活费、残疾用具费、护理费、误工费、交通费、住宿费、精神损害抚慰金、住院伙食补助费、被抚养人生活费及其他所有赔偿费，共计人民币柒拾万零叁仟元整，此款分给原告张娟父亲张奎强523000.00元，分给法定监护人董冬
　　180000.00元（包含小孩董XX抚养致成人的抚养费）。请求法院按双方达成的协议判处文县第二人民医院予以赔偿。
　　被告辩称，原告人诉称的张娟医疗纠纷的产生事实我们认可。张娟的事情发生后，我们医院已尽到了应尽的责任，但现在的医疗水平无法使张娟恢复到以前的状态，为了张娟将来的生活和医院的正常工作秩序，双方已就张娟的今后的生存所需的各种费用及逝后费用，达成了赔偿协议，我们一定按协议执行，请法院依法确认双方达成的协议的法律效力，原告人也依法履行自己的按时将病人搬离医院的义务。
　　经审理查明，原告张娟。2009年4月21日原告张娟生一女孩，取名董XX。张娟是原告张奎强的长女。2010年3月31日，原告张娟在文县第二人民医院进行无痛人流术，由于麻醉意外，导致其成为植物生存状态，经四川省广元市第一人民医院、四川省华西医学中心附属第一医院、第四附属医院长期治疗，病情稳定，但未苏醒。为了原、被告的各自利益，原、被告人已就张娟生前及逝后的各项赔偿费用达成一致协议，请求法院确认双方达成的调解协议合法有效，经本院主持调解，原、被告当庭达成如下协议：
　　一、文县第二人民医院一次性赔付原告张娟法定监护人董冬各项赔偿费及小孩抚养费180000.00元。付给原告张奎强抚养张娟抚养费、护理费、医疗费、逝世后安葬费及其他费共计523000.00元。
　　二、被告第二人民医院允许原告张娟继续借住在原门诊危楼内至2011年2月17日，逾期后原告张奎强无条件将张娟搬出，否则法院将依法强制执行。
　　三、本协议法律文书送达后，原告张娟今后的抚养、医疗及逝世后的安葬等所有事宜由张娟父母承担相应义务。张娟法定监护人及其亲属不得再以任何形式追究文县第二人民医院的民事赔偿责任。
　　案诉讼费70.00元，由文县第二人民医院承担。
　　以上协议是双方当事人的真实意思表达，符合法律规定，本院予以确认。
　　本调解协议双方当事人当庭签字生效。

　　审判员董跟会
　　二〇一一年一月二十六日
　　书记员肖新民
