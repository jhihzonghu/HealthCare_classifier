吴甲等与复旦大学附属华山医院医疗损害责任纠纷上诉案

审判时间：20111118 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第2142号


　　上诉人(原审原告)吴甲。
　　上诉人(原审原告)丁甲。
　　委托代理人吴乙。
　　被上诉人(原审被告)复旦大学附属华山医院。
　　法定代表人丁乙，院长。
　　委托代理人童剑云、黄承瑶，上海市康正律师事务所律师。
　　上诉人吴甲、丁甲因医疗损害责任纠纷一案，不服上海市静安区人民法院(2011)静民一(民)初字第1897号民事判决，向本院提起上诉。本院依法组成合议庭,公开开庭审理了本案。上诉人吴甲、上诉人丁甲的委托代理人吴乙、被上诉人复旦大学附属华山医院(以下简称华山医院)的委托代理人黄承瑶到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，患者吴丙与原告吴甲、丁甲系父子、夫妻关系。2010年1月20日原告至华山医院消化科、运动医学科诊治，诊断为肩背肌肉劳损；同年1月28日、1月29日患者至华山医院运动医学科诊疗，诊断为双肩痛；同年2月22日患者至华山医院消化科，诉反复腹胀伴体重下降，予以CA199等检查；同年3月1日患者至华山医院感染发热科再次予以CA199等检查，两天后CA199结果为18784U/ml。同年3月5日PET影像诊断：(1)胰腺尾部占位，肝内多发低密度灶，胃小弯侧淋巴结、左侧膈肌、左侧肾上腺、L4椎体前方结节FDG代谢异常增高，考虑胰腺癌及其转移所致。(2)肝脏脂肪浸润，(3)结肠炎，(4)前列腺多发钙化灶。同年5月14日患者于龙华医院死亡，死亡诊断：呼吸衰竭，循环衰竭，窒息，呼吸性酸中毒，肾功能衰竭，电解质紊乱，胸腹水，胰腺恶性肿瘤广泛转移。原告与被告协商未果，故起诉要求被告赔偿医疗费(PET检查)7,500元、精神损害抚慰金2,500元。
　　本案在诉前调解过程中，经被告华山医院申请，本院委托上海市宝山区医学会对被告的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规；其医疗行为与患者吴丙死亡有无因果关系；明确是否属于医疗事故，事故等级及责任程度等进行鉴定。2011年3月25日上海市宝山区医学会出具鉴定意见书，鉴定分析认为：1、患者吴丙因“双肩酸痛不适2月”，于2010年1月20日至2010年3月5日就诊于华山医院，同年3月5日PET影像诊断：(1)胰腺尾部占位，肝内多发低密度灶，胃小弯侧淋巴结、左侧膈肌、左侧肾上腺、L4椎体前方结节FDG代谢异常增高，考虑胰腺癌及其转移所致。(2)肝脏脂肪浸润，(3)结肠炎，(4)前列腺多发钙化灶；2、就诊期间，患者在运动医学科治疗后，症状稍缓解，未再去消化内科就诊(此间运动医学科曾建议患者内科就诊，排除胰腺、心脏疾病)；3、患者于2010年2月22日再次去消化内科就诊时，漏查血CA199，导致其检查结果晚出一周，但与患者疾病最终诊疗的结果无因果关系；4、患者死亡系胰腺恶性肿瘤及其广泛转移所致，与医疗行为无因果关系。最后结论认定：根据《医疗事故处理条例》第二条，第三十三条本病例不属于医疗事故。被告华山医院支付了鉴定费人民币3,500元。
　　原审审理中，原告坚持认为鉴定意见书给出的被告漏检CA199的行为与患者最终诊疗结果无因果关系的结论不正确；被告在CA199已明确的情况下，再做PET影像检查没有必要。被告华山医院则承诺一次性赔偿人民币2,000元并自愿承担医疗事故技术鉴定费人民币3,500元。
　　原审法院经审理后认为，患者吴丙因双肩酸痛不适在被告处诊治，就被告对患者实施的医疗行为，上海市宝山区医学会作出的医疗事故技术鉴定，该鉴定程序合法，目前也无证据证明鉴定内容违反科学，该鉴定意见本院予以采信。根据鉴定结论，被告对患者在诊治过程中的医疗行为并无不当，被告漏查血CA199导致其检查结果晚出一周，但与患者疾病最终诊疗的结果无因果关系，本病例不属于医疗事故。原告认为在CA199已明确的情况下，再做PET影像检查没有必要，本院认为，根据医疗事故技术鉴定书表明，做PET影像检查系为了进一步确诊患xxx病情，且原告亦无其他证据来佐证该PET检查系不必要，故原告的该项主张不予采信。因医疗行为引起的民事侵权赔偿案件，应符合侵权行为与损害后果具有因果关系这一侵权案件构成的基本要件，被告医疗机构虽然在整个诊疗过程中存在过失(漏查CA199)，但该过失与患者最终死亡的损害后果不存在因果关系，故原告要求被告赔偿其相关的损失，不符合民事侵权赔偿案件的构成要件，本院不予支持。现被告华山医院认为由于在医疗事故技术鉴定书分析中确认被告存在不足，故承诺自愿一次性赔偿原告人民币2,000元并承担鉴定费人民币3,500元，本院认为该承诺并无不当，可予准许。
　　原审法院据此判决：一、原告吴甲、丁甲要求被告复旦大学附属华山医院赔偿损失的诉讼请求不予支持；二、被告复旦大学附属华山医院应于本判决生效之日起十日内赔偿原告吴甲、丁甲人民币2,000元。
　　原审判决后，上诉人吴甲、丁甲不服，向本院提起上诉称：请求二审法院依法撤销一审判决，改判支持上诉人的诉讼请求。因为一审法院适用法律错误，要求适用侵权责任法，而不是民法通则。被上诉人有过错，漏查CA199，延误诊断及治疗；患者腹痛两个月、消瘦去就诊，被上诉人建议患者看了很多科室，未尽到与其医疗水平相应的诊疗义务。在CA199已经诊断明确的情况下还建议患者做昂贵的PET-CT检查，应该赔偿。
　　被上诉人华山医院答辩称：不同意上诉人的上诉请求，同意一审判决。吴丙在华山医院的就诊行为发生在2010年7月1日以前，因此，不适用侵权责任法。
　　本院经审理查明，原审查明事实属实，本院予以确认。
　　本院认为，公民享有生命健康权。在医疗纠纷中，医院的医疗行为存在过失，侵害他人生命健康权的，应当承担相应责任。原审法院根据双方当事人的诉辩、提交的证据对本案事实进行了认定，并在此基础上结合上海市宝山区医学会医疗事故技术鉴定书依法作出原审判决，合法合理，理由阐述充分，本院不再赘述。二审中上诉人主张一审法院认定事实不清，适用法律错误，医学会的鉴定结论不能证明被上诉人的医疗行为无过错，而事实是被上诉人在诊疗过程中都存在过错，请求二审法院查明事实，依据侵权责任法改判支持上诉人的原审诉讼请求。但其在二审中没有新的事实与理由，也没有新的证据佐证自己的主张。本院认可一审法院对事实的分析认定及对相关法律法规的理解与适用，上诉人的上诉请求缺乏事实及法律依据，本院难以支持。综上所述，原审认定事实清楚，判决并无不当。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币50元，由上诉人吴甲、丁甲负担。
　　本判决为终审判决。
审判长李伊红
　　代理审判员樊形锋
　　代理审判员王屹东
　　二○一一年十一月十八日
　　书记员龚轶
