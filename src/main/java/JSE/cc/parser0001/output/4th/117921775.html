张甲、张乙、周某某、何某某因医疗损害责任纠纷案

审判时间：20111108 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第1648号


　　上诉人(原审原告)张甲。
　　上诉人(原审原告)张乙。
　　法定代理人张甲。
　　上诉人(原审原告)周某某。
　　委托代理人张甲。
　　上诉人(原审原告)何某某。
　　被上诉人(原审被告)复旦大学附属华山医院。
　　法定代表人丁某，院长。
　　委托代理人黄承瑶，上海市康正律师事务所律师。
　　委托代理人童剑云，上海市康正律师事务所律师。
　　上诉人张甲、张乙、周某某、何某某因医疗损害责任纠纷一案，均不服上海市静安区人民法院(2011)静民三(民)初字第41号民事判决，向本院提起上诉。本院依法组成合议庭,公开开庭审理了本案。上诉人张甲(亦是上诉人周某某的委托代理人，上诉人张乙的法定代理人)、张乙、何某某，被上诉人复旦大学附属华山医院(以下简称华山医院)的委托代理人黄承瑶到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，张甲系患者周某某丈夫、张乙系周某某儿子。周某某、何某某系患者周某某父母。2005年6月8日，患者因“头晕4年，性格改变1年，记忆力减退1个月”入住华山医院。据住院病历记载：患者在江苏省东台市人民医院头颅MRI示颅内占位改变，为求进一步诊治入住华山医院。查体：神清，语明，GCS15分，双瞳正常，光敏，颅神经(-)，四肢肌力，肌张力可，双巴氏征(-)。头颅MRI示右额肿瘤。诊断：右额肿瘤。6月15日，患者在全麻下行开颅肿瘤切除术。术中见胆脂瘤粘连于侧脑室及三脑室处深面，边界撕拉不易脱落，极少量包膜残余；肉眼全切肿瘤内容物，极少量瘤包膜残余。术后予抗感染、对症支持治疗。6月21日，病理诊断：(右额)表皮囊肿。7月19日出院。出院时情况：一般状况可，生命体征平稳；伤口已拆线，I/甲愈合。
　　2007年11月3日，患者因“右额胆脂瘤术后，左侧头痛1年，加重半年”再次入住华山医院。据住院病历记载：头痛呈持续性胀痛，伴有恶心呕吐1次。查体：双瞳孔等大等圆，光反射灵敏。颅神经检查无明显异常，四肢肌力、肌张力、感觉运动均无异常，左手偶有僵直感，共济检查正常，病理征未引出。外院头颅MRI(2007年10月4日)示：右侧额顶叶胆脂瘤术后残腔形成，左侧额顶叶见类圆形异常信号。诊断：左额胆脂瘤。11月13日，患者在全麻下行左额胆脂瘤切除术。术后予抗炎、止血、脱水、制酸等治疗。术后当夜患者四肢抽搐2次，给予安定对症处理后未再发作。术后持续体温高。11月15日，血常规：白细胞12.24X109/L，中性粒细胞79.8%。11月19日，病理诊断：(左额)表皮样囊肿。11月21日，头颅CT示：右额中胆脂瘤术后改变。11月22日7时20分，患者癫痫大发作，呈持续状态。给予安定，德巴金对症处理。8时30分，再次癫痫大发作，复查头颅CT示：右额内血肿，三脑室、四脑室积血。转入监护室抢救，告病危。期间患者处于深昏迷状态，GCS3分。即给予气管插管。12时患者被送至手术室，拟行开颅血肿清除术。12时25分，患者突然出现心跳骤停，无自主呼吸。给予心肺复苏抢救。14时，急诊全麻下行右侧脑室穿刺置管CSF外引流术。术后患者仍处于深昏迷，GCS3分，双瞳散大，对光反应消失。2007年11月23日凌晨，患者家属要求回当地医院，签字后予自动出院。出院情况：深昏迷，GCS3分，双瞳孔散大，对光反应消失，呼吸机维持呼吸，氧分压测不出，经心肺复苏后心率84次/分，BP184/70mmHg，心电监护示室颤图像。患者当日死亡。
　　华山医院于2009年3月向上海市静安区人民法院提出医疗纠纷鉴定申请，后法院依法委托上海市静安区医学会组织医疗纠纷司法鉴定。上海市静安区医学会于2010年5月6日作出沪静医鉴[2009]022号医疗事故技术鉴定书，鉴定结论为周某某与华山医院医疗争议不构成医疗事故。同年6月7日，张甲、张乙、周某某、何某某提出再次鉴定申请，申请法院委托到上海市医学会鉴定，后法院委托上海市医学会组织鉴定华山医院对患者周某某的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规；其医疗行为与患者周某某的死亡有无因果关系；明确是否属于医疗事故、等级及责任程度。2010年10月22日上海市医学会出具沪医鉴[2010]150号医疗事故技术鉴定书，分析意见为：1、2005年6月8日，患者因“头晕4年，性格改变1年，记忆力减退1月”入住华山医院。依据患者症状、体征及头颅MRI检查，颅内占位病变明确，医方予开颅肿瘤切除术指征明确。根据手术记录及术后影像学检查，医方术中将患者右侧额部及突入左侧额部的大部分肿瘤组织切除，仅残留少量包膜因与侧脑室壁粘连紧密不能完全切除，符合治疗规范。术前就可能残留部分瘤组织进行了告知。2、2007年11月3日，患者因“胆脂瘤术后，左侧头痛1年，加重半年”再次入住华山医院。经术前检查，考虑为胆脂瘤复发。医方再次行开颅肿瘤切除术，手术指征明确，手术过程符合规范。3、2007年11月22日，患者术后第9天突发癫痫大发作，后经头颅CT发现颅内出血(11月21日复查头颅CT尚无颅内出血迹象)，患者在短时间内处于深昏迷状态，后经抢救无效死亡。患者死亡的根本原因是突发的不明原因颅内出血。4、根据送鉴材料，患者11月22日送入手术室后，突发心跳骤停，无自主呼吸前，并无实施全麻及应用大剂量麻醉药物的记录，医方给予心肺复苏等抢救措施，后依据病情给予患者穿刺置管引流，未发现有违反诊疗常规之处。5、根据病史记录，使用卫生棉条导致“院内感染”依据不足。棉条使用不属于创伤性操作，对患者不构成强烈刺激，也不属于违规操作的范畴。6、医方存在病史记录不全面，病变部位描述欠确切，与患方家属沟通不够等不足，但上述不足与患者死亡之间无因果关系。结论：综上所述，根据《医疗事故处理条例》第二条、第三十三条规定，周某某与华山医院医疗争议不构成医疗事故。
　　张甲、张乙、周某某、何某某对上述鉴定结论不服，提出书面质询，上海市医学会给予书面答复：1、根据手术记录及术后影像学检查，鉴定专家认为患者左侧仅残留少量包膜因与侧脑室壁粘连紧密，故不能完全切除，否则会损伤正常脑组织，后果更为严重。2、肿瘤复发、转移是疾病特性。残留肿瘤再生长就是肿瘤复发，而且医方已告知家属。3、癫痫大发作与颅内出血的因果关系无法认定，关键是医方是否采取了及时合理的诊治。根据送鉴材料显示，医方处理不违反诊疗规范。4、根据送鉴材料，专家判定患者在全麻尚未实施前已经发生心跳骤停、无自主呼吸。抢救过程未发现违反诊疗常规之处。后根据病情需要行穿刺置管引流术，并未行开颅手术。5、Ob内置卫生棉条的使用问题见鉴定书分析意见第4条，与死亡无因果关系。
　　原审法院经审理后认为，公民享有生命健康权，在其遭受到其他公民、组织非法侵害时，有获得相应赔偿的权利。本案中，患者周某某在华山医院治疗过程中，如果医疗机构在医疗行为上存在违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规，过失造成患者人身损害而构成医疗事故的，则医疗机构应承担相应的民事赔偿责任。因为医疗行为是一种复杂而技术含量极高的活动，病员出现的最终后果，往往与病员自身疾病的发展、医学科学和技术的局限性等事实密切相关，所以在确定赔偿责任时，必须综合考虑：患者的损害后果与华山医院的医疗行为之间是否具有因果关系、该损害后果是否系华山医院在医疗行为中的过错行为导致、医疗过错对损害后果发生所起的损害作用比例。由于医疗行为具有较强的专业性，对该上述情形作出合理判断，除依照一般常理及当事人提供的证据材料以外，还需由相关的权威机构即医学会做出鉴定，从而科学地、理性地分析结果产生的原因。本案经上海市静安区医学会进行医疗事故鉴定，并由上海市医学会进行再次鉴定，鉴定结论均认定本起医疗争议不属于医疗事故。经法院核实，鉴定报告程序合法、对医患双方的争议焦点均作出了明确分析，其鉴定意见应予采信。张甲、张乙、何某某、周某某对鉴定结论存在异议，但其依据的事实和主张的观点在鉴定过程中均已作为争议要点，但张甲、张乙、何某某、周某某的观点并未被鉴定部门采纳，且上海市医学会对张甲、张乙、何某某、周某某的观点又作出针对性的书面答复。张甲、张乙、何某某、周某某陈述的理由不足以推翻鉴定报告所得出的结论，故法院亦难以采信。法院注意到上海市医学会鉴定报告指出医方存在病史记录不全面，病变部位描述欠确切，与患者家属沟通不够等不足，但该鉴定报告认定上述不足与患者死亡无因果关系。故张甲、张乙、何某某、周某某要求华山医院承担赔偿责任的诉讼请求，缺乏事实和法律依据，法院难以支持。关于医方存在的不足之处，华山医院应在今后的工作中加以改进。
　　原审法院据此作出判决：对张甲、张乙、周某某、何某某的诉讼请求不予支持。
　　原审判决后，上诉人张甲、张乙、周某某、何某某不服，向本院提起上诉，仍坚持其在原审中的主张。
　　被上诉人华山医院则表示服从原判。
　　本院经审理查明，原审查明事实属实，本院予以确认。
　　本院认为，公民的生命健康权受法律保护。张甲、张乙、何某某、周某某与华山医院之间因患者周某某在华山医院治疗问题产生争议。基于医疗行为是一种具有高度专业性、相当复杂性并同时具有一定风险性的活动过程，确定医疗机构在对患者的诊疗过程中是否存在过错及该过错与患者目前的损害后果是否存在因果关系是一个相当专业的问题，故必须依赖具有专业知识、经验、技能的医疗鉴定部门专家的权威意见和评判结论。现在区、市两级医疗鉴定部门均作出鉴定结论，认为张甲、张乙、何某某、周某某与华山医院之间的医疗争议不构成医疗事故；而且针对张甲、张乙、何某某、周某某提出的一些异议，上海市医学会专门以书面的形式进行了答复，该答复覆盖了本案当事人之间主要的争议焦点。原审法院在审查了上述材料后，作出了不支持张甲、张乙、何某某、周某某的诉讼请求的判决。该判决符合法律的规定，并无不当；张甲、张乙、何某某、周某某在二审期间虽对原审法院判决不服，但未提出充分的理由以证实自己的主张，故本院对其主张不予采信；对于其上诉请求不予支持。综上所述，原审认定事实清楚，判决并无不当。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币10,836.30元，由上诉人张甲、张乙、周某某、何某某共同负担。
　　本判决为终审判决。
审判长高中伟
　　代理审判员姚敏
　　代理审判员张庆
　　二○一一年十一月八日
