沈XX诉上海XX医院医疗损害赔偿纠纷案

审判时间：20110428 法院：上海市浦东新区人民法院（司法公开示范法院） 案件字号：(2011)浦民一(民)初字第6639号


　　原告沈XX。
　　委托代理人颜XX（系原告丈夫），住同原告沈XX。
　　委托代理人蔡XX，上海市XX律师事务所律师。
　　被告上海XX医院。
　　法定代表人汪XX，董事。
　　委托代理人王XX。
　　委托代理人邹XX。
　　原告沈XX诉被告上海XX医院(以下简称XX医院)医疗损害赔偿纠纷一案，本院于2011年2月21日立案受理后，依法适用简易程序，于2011年3月16日公开开庭进行了审理，原告沈XX及其委托代理人颜XX、蔡XX、被告委托代理人王XX、邹XX到庭参加诉讼。本案现已审理终结。
　　原告沈XX诉称，2009年10月11日，原告因身体不适至XX医院就诊，XX医院给予原告静脉输液治疗，但在输液过程中原告突然昏厥、面色苍白、小便失禁。当日，原告即被转至上海交通大学医学院附属仁济医院（以下简称仁济医院）抢救治疗，被诊断为心率失常，并于10月19日在仁济医院施行了心脏起搏器置入术。原告认为，由于XX医院未进行必要的询问和检查，给原告使用利多卡因药物，造成原告心脏功能严重受损和不可逆的后果，致使原告只能依赖于心脏起搏器维持生命，XX医院应对此后果承担相应的赔偿责任。故起诉要求XX医院赔偿原告医疗费人民币57,309.84元（币种下同）、护理费584元、误工费4,704元、交通费150元、精神抚慰金5,000元，合计67,747.84元。
　　被告XX医院辩称，医院在给原告诊疗过程中用药合理，原告在使用利多卡因过程中出现昏厥，医院予以抢救，诊治措施到位，并积极联系转院治疗。原告因高度房室传导阻滞安装起搏器后果与利多卡因的使用无因果关系。现本次医疗争议经过上海市静安区医学会（以下简称静安医学会）、上海市医学会二级医学会进行医疗事故技术鉴定，结论均未构成医疗事故，原告安装埋藏式心脏起搏器与自身疾病有关，故要求驳回原告的诉讼请求。
　　经审理查明：2009年10月11日，原告因“耳鸣、眩晕1月，发作时恶心、呕吐”至杨思医院五官科就诊。检查：双外耳道净，鼓膜完整，标志清，电测听：高度听力损失。诊断为：眩晕症（耳源性）。杨思医院给原告予以0.9％生理盐水250ml＋2％利多卡因10ml静脉点滴。期间原告发生昏厥、面色苍白、小便失禁。杨思医院立即予以吸氧、心电监护。心电监护示：心率30次/分，律不齐，血压150/mmHg。约10分钟后，原告神志转清，能对答。急查心电图示：窦性心率＋异位心律；Ⅲ0房室传导阻滞；过缓房室交界性逸搏及过缓房室交界性逸搏心律；T波变化；逆钟向转位。XX医院给原告予以0.9％生理盐水250ml＋异丙肾上腺素0.5mg静脉点滴，并告病危，建议转上一级医院就诊，随即联系“120”转院，转至仁济医院救治。同日，原告入住仁济医院，入院诊断：心律失常；Ⅲ0房室传导阻滞；心功能不全Ⅰ级。10月19日，原告在仁济医院行起搏器置入术。10月27日原告出院。
　　本案在诉前调解过程中，经被告申请，本院委托静安医学会对本次医疗争议进行鉴定。该会出具鉴定结论为：本病例不属于医疗事故。并分析意见为：1、2009年10月11日，患者因“耳鸣、阵发性一过性眩晕一个月”就诊于XX医院耳鼻喉科门诊，经常规问诊及检查后，初诊为“眩晕症（耳源性）”并无不妥。2、医方针对耳鸣及眩晕症状，静滴香丹、天麻素及利多卡因并无违反诊疗常规。3、经鉴定会上询问，诊断过程中医方曾建议患xxx病因，患方告知谓两个月前已有内科住院，各项检查正常，未同意检查。4、医方在诊治过程中，所用的利多卡因静滴剂量及速度不致产生Ⅲ度房室传导阻滞。5、医方存在以下不足:（1）、门诊病历书写过于简单。（2）、用药前缺乏仔细检查。但以上不足之处与患者需安装起搏器的永久性Ⅲ度房室传导阻滞无因果关系。
　　嗣后，原告对静安医学会鉴定结论存在异议，经原告申请，本院依法委托上海市医学会对本次医疗争议再次进行鉴定，该会出具鉴定结论为：沈幼贞与XX医院争议不构成医疗事故。并分析意见为：1、患者因“耳鸣、眩晕1月”于2009年10月11日至XX医院就诊。在利多卡因静脉点滴时突然发生晕厥，急诊送至仁济医院，诊断为Ⅲ0房室传导阻滞。2009年10月19日患者在仁济医院施行安装心脏起搏器。依据送鉴资料，XX医院予利多卡因治疗耳鸣有指证，药物剂量和用法符合规范。2、2008年11月14日中山医院24小时动态心电图检查记录未发现房室传导阻滞。医方使用利多卡因无禁忌症。3、利多卡因引起的心脏房室传导阻滞为一过性，可逆性，停药并经治疗可恢复正常心率。本案房室传导阻滞为不可逆，是其自身心脏存在隐匿性病变所致，安装埋藏式心脏起搏器与自身疾病有关。4、医方在使用利多卡因治疗中，缺乏心电图监测，存在医疗缺陷，应在以后的工作中引起重视。
　　上述事实由原、被告陈述及原告提供的《门急诊就医记录卡》、上海市医学会《医疗事故技术鉴定书》各一份、被告提供的静安医学会《医疗事故技术鉴定书》各一份等证据证实。
　　本院认为，公民享有生命健康权，原告至杨思医院就诊，双方由此建立了医疗法律关系。医疗损害赔偿责任的承担，是以医疗过失以及过失与损害后果之间存在因果关系为基本条件。根据《医疗事故处理条例》的规定，医疗机构及其医务人员在医疗活动中，违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规等，过失造成患者人身损害的，应当由医学会予以鉴定。本院委托静安医学会、上海市医学会所作的司法鉴定，既包含了对医疗事故鉴定，也包含了对医疗过错的鉴定，医学会所作的鉴定结论具有客观性和权威性，且符合证据性质，可以作为确认医患双方医疗纠纷过错的依据。现静安医学会及上海市医学会均确认本次医疗争议不属于医疗事故，本院予以确认。根据上述鉴定结论，XX医院的诊疗行为无过错，原告安装埋藏式起搏器与自身疾病有关，故原告要求XX医院承担赔偿责任，既缺乏事实基础，又于法无据，本院难以支持。本院考虑XX医院在使用利多卡因治疗中缺乏心电图监测，存在医疗缺陷，故本案鉴定费、受理费均由XX医院负担。依照《中华人民共和国民法通则》第九十八条、第一百零六条的规定，判决如下：
　　驳回原告沈XX要求被告上海XX医院赔偿的诉讼请求。
　　案件受理费1,581元（已由原告预交），减半收取计790.50元，鉴定费7,000元（原、被告各预交3,500元），合计7,790.50元，由被告上海XX医院负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
代理审判员宣志慧
　　二Ｏ一一年四月二十八日
　　书记员廖浩宇
