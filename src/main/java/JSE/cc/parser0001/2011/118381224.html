某医院）医疗损害赔偿纠纷案

审判时间：20110819 法院：上海市长宁区人民法院 案件字号：(2010)长民一(民)初字第1008号


　　原告魏某。
　　委托代理人魏某，某律师事务所律师。
　　委托代理人李某，某律师事务所律师。
　　原告李某某。
　　委托代理人魏某，某律师事务所律师。
　　被告某医院。
　　法定代表人徐某，院长。
　　委托代理人王某。
　　委托代理人徐某，某某律师事务所律师。
　　原告魏某、李某某与被告某医院（简称：某医院）医疗损害赔偿纠纷一案，于2010年1月28日立案受理后，本院依法适用简易程序，于2010年3月24日公开开庭进行了审理。应原告申请，本院委托司法鉴定科学技术研究所司法鉴定中心对病历封口处是否进行鉴定，该中心于2010年4月29日出具鉴定意见书。之后，根据原告申请，本院委托上海市医学会对被告医疗行为有无过错，与损害后果的关系，损害后果的程度等进行鉴定，该会于2011年6月28日出具鉴定书。此后，本院转换为普通程序，依法组成合议庭，于2011年8月19日，本院公开开庭进行审理，原告魏某及其委托代理人魏某、李某某的委托代理人魏某，被告某医院的委托代理人徐某到庭参加诉讼。本案现已审理终结。
　　原告魏某、李某某诉称：2008年12月19日，原告之子魏某某因头晕感觉不适至被告处治疗。2009年1月6日，被告对魏某某进行诊断，结论为急性淋巴性细胞性白血病。2009年1月10日开始，对魏某某进行化疗，同年2月1日凌晨，魏某某出现感冒高烧不退，原告认为医院对魏某某病情错误判断，延误病情，最终导致魏某某死亡，医院对魏某某的病情判断存有过错，故请求法院判令被告赔偿：1、精神损害抚慰金人民币（下同）139,200元、住院伙食补助费820元、住院陪护费7,792元、交通费2,956元、住宿费2,580元、丧葬费23,376元、误工费4,000元。要求上述费用按照百分之五十的比例赔偿。2、本案诉讼费、鉴定费由被告负担。
　　被告某医院辩称：医院对原告之子魏某某在医院的检查和治疗情况无异议，经医疗事故鉴定，本次纠纷构成一级医疗事故，医院承担次要责任。医院同意依法承担责任，但不同意原告诉请。
　　经审理查明事实如下：2008年12月19日，原告之子魏某某因头晕感觉不适至某医院治疗，据住院病史记录：查血常规后，拟诊“贫血待查”，于同年12月30日入住该医院，并进行骨髓穿刺检查。2009年1月5日，根据魏某某的骨髓细胞学检查报告，骨髓象提示急性淋巴性细胞性白血病。2009年1月6日确诊为急性淋巴性细胞性白血病。同年1月23日，再行行骨髓穿刺检查，检查报告为淋巴细胞比例约50%，异常淋巴细胞比例约10%，提示未完全缓解。同年2月2日，患者魏某某发热37.8℃，骨髓检查报告淋巴细胞比值相对偏高，分类中占76%，并留取血培养，同年2月6日，行胸部CT平扫检查诊断报告：两肺未见明显异常。同年2月8日，患者魏某某胸闷、气促，行胸部CT平扫检查诊断报告：两肺感染，右侧胸腔少量积液。当日16时48分宣告患者魏某某临床死亡。诊断：急性淋巴性细胞性白血病，肺部感染，Ⅰ型呼吸衰竭，感染性休克，肝功能不全。原告认为医院对魏某某的病情判断存有过错，延误病情，故请求法院判令被告承担赔偿责任。
　　审理中，根据原告申请，本院委托司法鉴定科学技术研究所司法鉴定中心对病历封口进行鉴定，该中心于2010年4月29日出具鉴定意见书。鉴定结论为，检材密封信封袋的背面各粘合处，均不存在拆封过的痕迹。根据原告申请，本院又委托了上海市医学会对被告的医疗行为有无过错，与损害后果的关系，损害后果的程度等进行鉴定。2011年6月21日，上海市医学会出具沪医鉴(2011)081号医疗事故技术鉴定书，其中分析意见：医方过失(1)患者2008年12月30日入院，2009年1月6日诊断急性白血病，1月10日开始化疗，诊治有一定延误，可能对疾病的预后有一定的负面影响。(2)患xxx病人，肌体免疫力低下，2月2日为冬季，病房停止供暖数小时且无告示、无应急措施，可能增加病人呼吸道感染机会。(3)患者2月2日出现发热时，医方未及时留取血培养；据现场询问家属，2月5日患者出现胸闷、气促已行吸氧治疗，医方未行动脉血气分析以明确呼吸功能状态，影响了对病因、病情的判断。(4)经鉴定会专家现场读片，认为2月6日肺部CT片左下肺已有异常病灶存在，但院方报告未见异常，存在过失。(5)当病人经抗细菌治疗数天疗效不佳时，医方未及时给予经验性抗真菌治疗；最终患者血培养及骨髓培养提示存在真菌感染。综上所述，患者最终死亡与医方的管理、诊断和治疗存在上述过失有一定的因果关系。及时结论：魏某某与武警总队医院医疗争议构成医疗事故，本例属于一级甲等医疗事故，医方承担次要责任。
　　上述事实，除双方当事人陈述自认外，另有原告病史、鉴定报告等证据所证实，经质证，原、被告对鉴定结论无异议。本院认为医疗鉴定结论系有资质机构及人员依合法程序作出，无不合理之处，予以确认。
　　审理中，原、被告确认以下项目及金额：精神损害抚慰金139,200元、住院伙食补助费820元、住院陪护费7,792元、交通费2,956元、住宿费2,580元、丧葬费23,376元、误工费4,000元。因双方对上述赔偿费用承担的比例意见不一，致本案调解不成。
　　本院认为，公民依法享有生命健康权。公民、法人由于过错造成他人人身损害的，应承担民事责任。
　　本案系医疗纠纷，就解决双方当事人存有的争议而言，首先必须确定医疗单位在对患者的诊疗过程中是否存在过错行为，以及该过错行为是否与损害后果有因果关系。上海市医学会作为医疗事故技术鉴定的法定机构，其依据本案情况所作出的鉴定意见具有客观性和权威性，且经质证各方对鉴定机构资质、鉴定人员资质、检材无异议。该医学会出具的鉴定结论明确:一级甲等医疗事故，医方承担次要责任。故本院确定被告就本案承担次要责任百分之四十的比例承担赔偿责任。
　　本案的赔偿范围，应依原告诉请及法律规定、鉴定结论予以确定。对于原告主张的精神损害抚慰金139,200元、住院伙食补助费820元、住院陪护费7,792元、交通费2,956元、住宿费2,580元、丧葬费23,376元、误工费4,000元，于法不悖，予以确认。
　　综上，被告在本院确定的赔偿比例责任范围内对原告的合理经济损失予以赔偿。据此，依照《医疗事故处理条例》第五十条第二项、第三项、第四项、第七项、第九项、第十项、第十一项之规定，判决如下：
　　被告某医院应赔偿原告魏某、李某某精神损害抚慰金、护理费、住院伙食补助费、住宿费、交通费、丧葬费、误工费共计人民币77,500元，于本判决生效之日起十日内付清。
　　如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　鉴定费人民币6,000元，由原告魏某、李某某负担人民币2,500元，由被告某医院负担人民币3,500元；
　　案件受理费人民币2,300元，由原告魏某、李某某负担人民币517.50元，被告某医院负担人民币1,782.50元。
　　如不服本判决，可在判决书送达之日起十五日内向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
审判长张枫
　　代理审判员陶文杰
　　人民陪审员曹征祥
　　二〇一一年八月十九日
　　书记员石晓峰
