于某诉某公司生命权、健康权、身体权纠纷案

审判时间：20110415 法院：上海市卢湾区人民法院 案件字号：(2011)卢民一(民)初字第755号


　　原告于某。
　　委托代理人施某。
　　委托代理人周某，上海某律师事务所律师。
　　被告某公司。
　　委托代理人陆某。
　　委托代理人束某，上海市某律师事务所律师。
　　本院于2011年3月28日立案受理了原告于某诉被告某公司生命权、健康权、身体权纠纷一案，并依法适用简易程序公开进行了审理。原告以其在被告处做整形手术至感染，使其身心受伤害为由诉讼来院，要求依法判令：1，被告赔偿原告医疗费、律师费等经济损失人民币10,000元；2，被告赔偿原告精神损失人民币5,000元。在审理中，被告表示希望双方调解解决。
　　本案在审理过程中，经本院主持调解，双方当事人自愿达成如下协议：
　　一、被告某公司于本调解书签收之日给付原告于某人民币10,000元。
　　二、原告于某就原、被告间医疗纠纷放弃再向被告某公司主张任何权利。
　　三、案件受理费人民币50元减半收取，计人民币25元由原告于某、被告某公司各半负担。
　　双方当事人一致同意本调解协议自双方在调解协议上签名或捺印后即具有法律效力。
　　上述协议，符合有关法律规定，本院予以确认。
代理审判员张煜
　　二Ｏ一一年四月十五日
　　书记员张文杰
