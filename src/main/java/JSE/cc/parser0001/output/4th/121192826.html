车国娟等诉杭州市第六人民医院医疗服务合同纠纷案

审判时间：20111230 法院：浙江省杭州市下城区人民法院 案件字号：(2011)杭下民初字第33号


　　原告：车国娟。
　　原告：金水娟。
　　委托代理人：茅珊珊。
　　被告：杭州市第六人民医院。
　　法定代表人：韦翊。
　　委托代理人：常润根。
　　原告车国娟、金水娟为与被告杭州市第六人民医院（以下简称市六医院）医疗服务合同纠纷一案，本院于2010年12月24日受理后，依法由罗书生独任审理，后因案件复杂，依法组成合议庭，于2011年1月25日、2011年6月9日、2011年8月16日、2011年10月25日公开开庭进行了审理。原告车国娟、金水娟及委托代理人茅珊珊、被告市六医院的委托代理人常润根到庭参加诉讼。本案在审理过程中，依原告车国娟、金水娟，被告市六医院申请，本院分别于2011年3月2日和2011年6月10日将诉争病例交由本院审判监督庭依法委托杭州明皓司法鉴定所和杭州迪安医学检验中心司法鉴定所对患者车英英在治疗期间院方是否存在过错、过错与患者死亡有无因果关系以及是否构成医疗事故进行鉴定。本案现已审理终结。
　　原告车国娟、金水娟起诉称：2010年初，车英英身体不适，经浙江大学医学院附属妇产科医院检查，查出“乳房小叶增生”。2010年2月27日起，车英英服用了内含黄药子等成份的中草药至同年7月初，出现黄疸症状。7月9日家人带她到被告处就诊，7月11日上午，车英英病情加重，住进被告普通病房治疗。入院后市六医院予以护肝、补液等治疗，但其病情不但无好转，反而加重发展。直到7月14日，被告医院才修正诊断，于当天将车英英转入ICU病房治疗。7月20日上午，车英英因治无效死亡。医院最后诊断，急性肝衰竭，药物性肝炎（黄药子），肝性脑病IV期；胆道感染；肺炎，呼吸衰竭；酸碱失衡电解质絮乱。被告在车英英住院治疗过程中，未履行告知、注意、如实填写病历、确诊义务，延误了患者的病情，错失了最佳抢救时间，没有采取最佳医疗措施，医院存在明显的医疗过错，应承担赔偿责任。根据《中华人民共和国民法通则》、《中华人民共和国侵权责任法》、《中华人民共和国合同法》及相关司法解释的相关规定，原告特向法院提起诉讼，要求判令：1、被告赔偿原告医疗费50000元、死亡赔偿金492220元、丧葬费13740元、伙食补助费300元、陪护费1053元、扶养费147500元、精神抚慰金50000元，共计754813元的30%，即226443.9元。2、本案诉讼费由被告承担。
　　为证明上述事实，原告车国娟、金水娟向本院提交了如下证据：
　　1、户口本1份，
　　2、求助证明1份。
　　证据1-2欲证明两原告与车英英的关系及家庭成员。
　　3、身份证、学生优待证、初中毕业证、高中毕业证、学生证、学习证明书（以上均是复印件）、进出口实务笔记本、体检表（复印件），欲证明车英英从小学到大学一直在杭州居住、学习、生活及其身体健康情况；同时欲证明车英英的笔迹与被告提供的不同。
　　4、住院协议、告知书、患者委托书、安全须知、谈话笔录、化验单、病危通知书、治疗知情同意书、出院小结、医疗证明、死亡证明书（以上均系原件），欲证明车英英在被告处治疗、被告未履行告知、注意、如实填写病历、确诊义务，延误了患者的病情，错失了最佳抢救时间，没有采取最佳医疗措施，医院存在明显医疗过错。
　　被告市六医院辩称，对车英英的诊治符合医疗原则，无过错。7月9日的车英英到我院诊治，医院按医疗原则进行诊治，具体诊疗过程详见病历（病历已封存）。原告当庭变更案由，根据最高院的规定，医疗纠纷根据谁主张谁举证的原则，即原告有义务证明被告诊治有无过错、车英英的死亡与被告的过错存在因果关系、被告的行为是违法的。从现在原告提供的证据看无法证明以上这些内容。根据法院调取的证据，可确认车英英的死亡与姚伟鹏非法行医有主要的因果关系，另外与车英英的母亲在药店不规范配药也有一定关系，这在医学会的鉴定表述中均有体现。综上，被告在整个诊疗过程中是为了救治车英英，被告无任何过错。请求法院依法驳回原告的诉讼请求。
　　为证明上述事实，被告向本院提交了如下证据：
　　1、封存的病历一份，欲证明我方按医疗原则进行诊治，不存在过错。
　　原、被告提交的上述证据，经双方当事人当庭举证、质证，本院认证如下：
　　对于原告车国娟、金水娟提交的证据1，被告市六医院对真实性有异议，但未提出反驳证据，本院予以确认；证据2，被告对真实性、关联性、合法性都存在异议，但未提出反驳证据，本院予以确认；证据3，被告市六医院对关联性有异议，本院认为该组证据关系到车英英身份、经常居住地以及真实笔迹的确认，故予以确认；证据4，被告市六医院对真实性无异议，但认为不能证明被告存在延误病情的情况，本院对改组证据采信，但对其证明对象需要结合其他证据一并确认。
　　对于被告市六医院提交的证据1，原告车国娟、金水娟对封存的病历没有异议，但对内容无法确定，本院对该证据真实性予以确认。
　　经被告市六医院申请，本院向杭州市江干区检察院调取车英英的尸检报告（编号：A4193）、杭州市医学会出具的杭州医鉴［2010］05号医学鉴定书、浙江省医学会出具的浙医鉴［2010］4号医学鉴定书各1份，经原、被告双方当庭质证，原、被告对尸检报告、鉴定报告均无异议，本院予以确认。
　　杭州明皓司法鉴定所作出的杭州明皓［2011］文检鉴字第58号意见书1份，经原、被告双方当庭质证，本院认证如下：原告对鉴定结论没有异议。被告对鉴定报告本身的真实性没有异议，但对鉴定的意见有异议，认为由于患xxx症昏迷，当时意识反应不清，所以由其父母代签。对病危通知书，确实系金水娟所签，由于当时其情况很激动，所以书写笔迹与平时的有所不同。本院对该鉴定意见书的真实性予以确认。
　　杭州迪安医学检验中心司法鉴定所做出的杭州迪安司鉴所［2011］病鉴字第84号法医病理学文证审查意见书1份，经原、被告双方当庭质证，本院认证如下：原告对鉴定结论没有异议。被告对鉴定报告本身的真实性没有异议，本院予以确认。
　　根据当庭举证、质证及双方当事人陈述的情况，本院确认案件事实如下：
　　患者车英英。2010年初，车英英身体不适，经浙江大学医学院附属妇产科医院检查，查出“乳房小叶增生”。2010年2月27日起，车英英在母亲的带领下到姚伟鹏家中看病，服用了姚伟鹏在白纸上开的内含黄药子等成份的中草药，3月27日车英英与母亲再次到姚伟鹏家中看病，并再次服用了姚伟鹏在白纸上开的内含黄药子等成份的中草药。7月初车英英出现了黄疸症状，7月9日车英英到被告处就诊，医院建议车英英立即住院治疗。期间，车英英继续服用姚伟鹏开具的中草药。7月11日上午，车英英病情加重，住进被告处治疗。医院诊断为：1、黄疸待查：（1）药物性肝炎，（2）病毒性肝炎。2、胆道感染。入院后被告予车英英护肝、补液、对症等治疗，但车英英病情无好转。7月14日，被告医院修正诊断：急性重症肝炎（晚期），（1）胆道感染，（2）肝性脑病III。并于当天下午将车英英转入ICU（重症监护病房）病房治疗。7月20日上午，车英英医治无效死亡。医院最后诊断，急性肝衰竭，药物性肝炎（黄药子），肝性脑病IV期；胆道感染；肺炎，呼吸衰竭；酸碱失衡电解质絮乱。现原告以被告存在医疗过错，要求被告承担赔偿责任而诉讼来院。
　　另查明，2010年7月23日杭州市公安局江干区分局委托浙江省病理、尸体解剖中心对车英英遗体进行解剖及病理组织切片，该中心检验报告结论：其生前主要患有急性肝坏死，腹水（550ML），黄疸；心外膜及心肌灶性坏死、出血。结合送检资料，认为车英英系由于急性肝坏死引发急性肝衰竭、肝性脑病而死亡。参照其治疗经过及死亡情况，其急性肝坏死乃由于黄药子中毒引起。
　　2010年9月3日杭州市公安局江干区分局就姚伟鹏非法行医与车英英死亡是否存在因果关系委托杭州市医学会进行医学鉴定。杭州市医学会出具的杭州医鉴［2010］05号医学鉴定书结论表明：姚伟鹏非法行医与车英英死亡有因果关系，参照《医疗事故分级标准（试行）》，构成一级甲等人身损害，姚伟鹏承担主要责任。杭州芝林大药房上塘路店对没有正规处方的“方子”（白纸上开药）及在处方中全蝎、蜈蚣使用量超标的情况下，未按照要求提示患者需医生双签名，即予配药，违反《中华人民共和国药品管理法》的相关规定，患方在市六医院7月9日建议患者住院时，未及时住院，而是再次服用姚伟鹏的中药，拖延2天治疗。均与患者车英英的死亡有一定的关系。
　　2010年11月1日杭州市公安局江干区分局就车英英死亡与姚伟鹏非法行医的因果关系委托浙江省医学会进行医学鉴定。浙江省医学会出具的浙医鉴［2010］4号医学鉴定书结论表明：车英英的死亡与姚伟鹏非法行医存在因果关系。
　　本案在审理过程中，原告车国娟、金水娟主张《告知书》、《知情选择书》、《患者授权书》、《住院安全须知》、《住院协议》上“车英英”的签名及2010年7月14日《病危通知书》上家属“金水娟”的签名有伪造的嫌疑，被告未履行告知、注意、如实填写病历资料、确诊等义务，客观上延误了车英英的病情，错失了最佳抢救时间，没有采取最佳医疗措施，存在明显过错，故提出司法鉴定。本院就笔迹依法委托杭州明皓司法鉴定所进行鉴定，该所作出的杭州明皓［2011］文检鉴字第58号意见书结论表明：所送检材（1）-（4）中五处“车英英”签名字迹与样本中（1）-（7）中车英英书写字迹，不是同一人所写。检材（5）中无签名字迹。检材（6）“金水娟”签名字迹与样本（8）中金水娟书写字迹，倾向认为不是同一人所写。本院就院方在医疗过程中是否存在过错依法委托杭州迪安医学检验中心司法鉴定所进行鉴定，该所做出的杭州迪安司鉴所［2011］病鉴字第84号法医病理学文证审查意见书结论表明：1、市六医院在对车英英的整个诊疗过程中，符合医疗常规，车英英的死亡与医院医疗行为无明显相关性。2、倘若告知书等签名确非患方所为，则医院存在未尽告知义务；反之，则医院已履行了告知义务。
　　本院认为，原告依据医疗服务合同要求被告承担违约赔偿责任，故本院的争议焦点是其诊疗过程中是否存在违约行为，以及该行为与车英英的死亡是否存在因果关系。
　　首先，患xxx病在被告市六医院进行治疗，双方形成医疗服务合同关系。同时，根据《执业医师法》、《医疗事故处理条例》、《医疗机构管理条例》等法律法规之规定，医疗机构对于患者病情、诊疗行为、不利后果负有告知义务，患者则享有知情同意权。本案中，《患者授权书》、《知情选择书》、《杭州市第六人民医院住院协议书》、《杭州市第六人民医院住院病人安全须知》、《告知书》、《病危通知书》等文件上患者及其亲属签名全部非本人所签，被告市六医院并未尽到约定或法定的告知义务，存在违约行为。
　　其次，市六医院对患者车英英收治后采取了相应的治疗措施，在其出现肝性昏迷之后及时做了相应调整，医院在整个诊疗过程中符合医疗常规。根据司法鉴定结论，车英英的死亡原因为其服用中草药“黄药子”致急性肝中毒、肝坏死伴胆道感染所致，与市六医院的医疗行为并无直接因果关系。
　　综上，市六医院虽然存在违约行为，但该行为与车英英之死并不存在直接因果关系，对其死亡所产生的损害也不负担赔偿责任。但被告的违约行为导致原告无法及时准确了解患者病情、诊疗措施及医疗风险，妨碍了原告自主选择权的行使，给原告心理造成了一定的负担，故本院酌定市六医院承担20000元的赔偿责任。
　　综上，根据《中华人民共和国民事诉讼法》第六十四条、《中华人民共和国民法通则》第一百一十一条、《中华人民共和国合同法》第一百一十三条之规定，判决如下：
　　一、被告杭州市第六人民医院于本判决生效后十日内赔偿原告车国娟、金水娟违约损害赔偿金20000元。
　　二、驳回原告车国娟、金水娟的其他诉讼请求。
　　因原告当庭变更诉讼请求，本案件受理费人民币变更为1632元，由原告车国娟、金水娟承担1488元，被告杭州市第六人民医院承担144元。
　　如不服本判决，可在判决书送达之日起十五日内向本院递交上诉状，并按对方当事人的人数提出副本，上诉于浙江省杭州市中级人民法院，并向浙江省杭州市中级人民法院预交上诉案件受理费1632元（开户银行：工商银行湖滨分理处，账号：xxx，户名：浙江省杭州市中级人民法院）。在上诉期满后七日内未交纳的，按自动撤回上诉处理。
