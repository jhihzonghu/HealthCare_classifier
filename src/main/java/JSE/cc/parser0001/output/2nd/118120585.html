漯河市第五人民医院与漯河市源汇区大刘镇中心卫生院、第三人徐满晴借款合同纠纷案

审判时间：20110510 法院：河南省漯河市源汇区人民法院 案件字号：(2010)源民初字第163号


　　原告漯河市第五人民医院。
　　法定代表人张云飞，院长。
　　委托代理人刘松山，该院法律顾问。
　　委托代理人张丽娜，该院工作人员。
　　被告漯河市源汇区大刘镇中心卫生院。
　　法定代表人吕大邦，院长。
　　委托代理人黄庆伟，河南恩达律师事务所律师。
　　第三人徐满晴。
　　委托代理人芮光辉，河南汇恒律师事务所律师。
　　原告漯河市第五人民医院与漯河市源汇区大刘镇中心卫生院（以下简称大刘卫生院）、第三人徐满晴借款合同纠纷一案，本院于2010年3月3日立案审理后，依法组成合议庭，公开开庭进行了审理。原、被告委托代理人、第三人的委托代理人均到庭参加诉讼。本案现已审理终结。
　　原告诉称，2005年9月26日，被告向原告借款140000元用于抵交被告应上交的合作医疗款，事后被告先后三次还款10万元。下欠40000元后经原告多次催要，被告拒不偿还。故诉至法院，请求依法判令被告偿还借款40000元及利息。
　　被告大刘卫生院辩称，原、被告之间不存大债权债务关系，证据显示，利息无约定，原告已经超过诉讼时效，请求驳回原告的诉讼请求。
　　第三人徐满晴辩称，原、被告双方利息没有约定，不予支持。原告起诉已超过诉讼时效，双方借、还款行为，都是在原、被告双方进行的。第三人没有还款义务。
　　经审理查明，2005年9月26日，李国文承包经营大刘卫生院期间，大刘卫生院借漯河市第五人民医院借条一份，上载明“今收到漯河市第五人民医院人民币壹拾肆万元整（140000元）源汇区大刘镇卫生院李国文二00五年九月二十六日”。同日原告漯河市第五人民医院经过源汇区会计核算中心给大刘镇中心卫生院转借款140000元，被告大刘卫生院分别向原告于2006年11月20日还款30000元，2006年11月21日还款50000元，2008年2月2日还款20000元，三次还款共计100000，下余借款40000元未偿还给原告。
　　另查明，徐满晴与李国文系夫妻关系，李国文于2009年元月8日因病死亡。
　　又查明，2003年5月11日，甲方大刘卫生院（当时称郾城县大刘中心医院）与乙方李国文签订合作经营协议，协议约定：一、甲方将大刘中心医院的经营管理权交给乙方，合作经营管理期限为2003年5月12日起至2006年5月25日止。二、在合作经营期限内，由乙方担任院长，全面负责大刘中心医院的经营管理工作……。四、在合作经营期限内，乙方必须确保甲方资产不流失，数量不减少（已报废者除外）。确保甲方设备运行正常，房屋结构等设施完整。如需增加设备、设施由乙方负责。5万元以上设备双方协商解决，各承担50%，在合作期满后，按规定协商后作价留给甲方（注：这部分财产及人员清单详见交接清单，乙方在合作经营期限届满后不得以腐败、变质、过期药品冲抵费用）。……六……（二）1、乙方有合作经营期限内自主经营医院的权利和负责支配合作经营期内的债权和债务。甲方原有的债权债务及经济纠纷，完全由甲方负责……。七、在合作经营期限内由乙方承担在此期间内产生的经济纠纷和民事责任（医疗纠纷、医疗事故甲方承担15%）。李国文实际经营至2007年5月。
　　本院认为，依法成立的合同应受法律保护，2005年9月26日，李国文承包经营大刘卫生院期间，大刘卫生院借漯河市第五人民医院人民币140000元并出具借条一份，上载明“今收到漯河市第五人民医院人民币拾肆万元整（140000元）源汇区大刘镇卫生院李国文二00五年九月二十六日”。后被告大刘卫生院分别还款三次共计100000元。2005年9月26日漯河市第五人民医院经过源汇区会记核算中心给大刘镇中心卫生院转的借款，并加盖有原告漯河市第五人民医院财务专用章，本院予以认定。原告要求利息，没有证据证明，本院不予认定。被告虽对该收据真实性提出异议，但没有证据予以证明。李国文与被告大刘卫生院所签订的合作经营协议，被告与第三人双方对协议内容均予认可。根据该协议，李国文在经营期限内有自主经营医院的权利和负责支配合作经营期内的债权债务，李国文承担经营期间内的经济纠纷和民事责任。该协议还约定如需增加设备由李国文负责。5万元以上设备双方协商解决，各承担50%，在合作期满后，按规定协商后作价留给被告大刘卫生院，这部分财产及人员清单以交接清单为准。本案中原告所出具的借条在李国文经营管理期间内，李国文应承担其经营期间内的经济纠纷和民事责任。被告大刘卫生院与李国文合作经营期满后，根据双方协议应清算和交接，但直至2009年1月李国文病故，双方未清算与交接。被告大刘卫生院与李国文对引起本案纠纷均负有责任。另根据法律规定，在夫妻关系存续期间，一方从事个体经营或承包经营，其收入为夫妻共有财产，债务亦应以夫妻共有财产清偿。夫或妻一方死亡的，生存的一方应当对婚姻关系存续期间的共同债务承担连带清偿责任，本案借款合同发生在第三人徐满晴与李国文夫妻关系存续期间，李国文死亡，第三人徐满晴应承担清偿责任。被告在庭审中辩称原告借款合同已经超过诉讼时效，根据法律规定，诉讼时效期间从知道或者应当知道权利被侵害时起计算。本案中没有约定借款期限，被告没有证据证明该借款合同超过诉讼时效，本院不予认定。被告借原告款时未约定给付利息，对原告主张利息的请求不予支持。依照《中华人民共和国合同法》第一百零七条、第二百零六条、《中华人民共和国民法通则》第八十七条、第一百三十七条、《最高人民法院关于适用〈中华人民共和国婚姻法〉若干问题的解释（二）》第二十六条、《最高人民法院关于贯彻执行〈中华人民共和国民法通则〉若干问题的意见》第43条、《中华人民共和国民事诉讼法》第二百二十九条之规定，判决如下：
　　在本判决生效后十日内，第三人徐满晴偿还原告漯河市第五人民医院现金40000元，被告漯河市源汇区大刘镇中心卫生院负连带清偿责任。
　　如果未按本判决指定的期间履行给付金钱的义务，应当加倍支付迟延履行期间的债务利息。
　　本案受理费1010元，由第三人徐满晴负担800元，被告漯河市源汇区大刘镇中心卫生院负连带责任。原告漯河市第五人民医院承担210元。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于河南省漯河市中级人民法院。
审判长杨洪生
　　审判员张宇
　　审判员赵卫华
　　二0一一年五月十日
　　书记员董阳
