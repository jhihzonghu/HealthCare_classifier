孟某与上海某医院医疗服务合同纠纷案

审判时间：20110304 法院：上海市第一中级人民法院（司法公开示范法院） 案件字号：(2010)沪一中民一(民)终字第3197号


　　上诉人（原审原告）孟某。
　　被上诉人（原审被告）上海某医院。
　　上诉人孟某因医疗服务合同纠纷一案，不服上海市浦东新区人民法院（2010）浦民一（民）初字第29193号民事判决，向本院提起上诉。本院于2010年12月9日立案后，依法组成合议庭，于2010年12月22日公开开庭审理了本案。上诉人孟某之委托代理人孟燕蕾、张白云，被上诉人上海某医院（以下简称“某医院”）的委托代理人黄承瑶到庭参加了诉讼，本案现已审理终结。
　　原审法院经审理查明，患者孟某于2009年2月5日在某医院经诊断为：“冠心病，急性前壁心肌梗，阵发性房颤，2型糖尿病”收治入院。2009年2月6日心电图：ST段抬高，STv1-v4呈长斜抬高0.3-0.2mv。完善各项检查，于2009年2月24日下午在全麻下行OP-CABG术。手术顺利，术后常规对症支持治疗。术后四天，即2009年2月28日上午病程录：患者神清，高浓度面罩吸氧中，昨尿量3440ml，无胸闷气促不适。心电监护示血压136/57mmHg，心率78次/分，SPO298%。查体两肺呼吸音粗，未及?音，心率齐未及杂音。腹软，切口无渗出，血电解质正常范围，床边胸片示术后改变。晚上11：30病程录患者神清，心电监护示血压146/59mmHg，心率78次/分，SPO2100%。尿量多适当补充容量，注意电解质平衡。诊断吩咐单于2009年3月1日上午转出CIU。重整医嘱心胸外科为Ⅱ级护理。2009年3月2日晚7：00病程录：上厕所时摔倒右股骨转子间骨折，予右下肢制动牵引。孟某于2009年3月19日出院，出院诊断：冠心病，急性前壁心肌梗，阵发性房颤，2型糖尿病，右侧股骨转子间骨折。由于孟某认为某医院疏于履行监护职责，以致于其急需护理时却呼人无回应下自行解手而摔伤，故诉至法院要求某医院赔偿其护理费、营养费、残疾赔偿金、精神损害抚慰金、律师费等各项损失共计70,880元。
　　本案在诉前调解中，原审法院委托上海市杨浦区医学会对本次医疗争议进行鉴定。该会出具鉴定结论，明确:孟某与上海某医院的病例不属于医疗事故，并分析认为：1、患xxx病、急性前壁心肌梗、阵发性房颤。心电图释：ST段抬高，STv1-v4呈长斜抬高0.3-0.2mv。有行冠状动脉搭桥的手术指征。手术顺利，效果良好。医院诊断、手术、治疗、护理的过程符合诊疗常规。2、在患者住院治疗期间，院方对患方进行过防止跌倒的告知，有患方签字确认的告知书。3、病人跌倒发生在冠状动脉搭桥手术后第七天，医嘱为Ⅱ级护理。当时病人病情稳定，为恢复期，符合Ⅱ级护理的指征。对Ⅱ级护理的病人在护理常规中明确规定：护士每2小时巡视病房一次。4、病人需解尿，未告知护士自行下地而跌倒属医疗意外，与院方的护理行为之间不存在因果关系。某医院为此垫付鉴定费3,500元。
　　原审法院审理后认为，根据上海市杨浦区医学会的鉴定结论，孟某与上海某医院医疗争议不构成医疗事故。孟某提出某医院在履行医疗服务合同中存在瑕疵造成其损失没有事实和法律依据，故不予支持。但孟某术后在某医院摔倒致右侧股骨转子间骨折是客观事实，虽与某医院不存在因果关系，但会导致患者对医院的护理产生合理怀疑，引发医患纠纷，故因本次医疗纠纷所产生的鉴定费可由某医院承担。据此，原审法院判决驳回了孟某的诉讼请求。案件受理费1,572元，减半收取786元，由孟某负担；鉴定费3,500元，由上海某医院负担。
　　判决后，孟某不服，上诉要求改判支持其原审诉讼请求。上诉人诉称，虽然院方诊断吩咐单于2009年3月1日上午转出CIU，但事实上，上诉人一直住在重症监护室，没有转出过。按照重症监护室的监护规范，他不可能没有医护人员，原审法院仅凭二级护理的规定套本案是不公正的。
　　被上诉人某医院不同意上诉人的上诉请求，要求维持原判。被上诉人辩称，患xxx病情稳定，故医嘱已改为二级护理。但由于当时医院的病人较多，外面没有病房，所以院方安排他住重症病房，而护理的等级仍需按医嘱来确定，并不是说住在重症病房的病人一定要重症护理的。故要求维持原判。
　　本院经审理查明，原审查明事实无误。
　　本院认为，某医院对孟某的摔伤是否具有过错是本案的焦点问题。首先，孟某的摔倒受伤发生在心脏冠状动脉搭桥术后的第七天，医嘱已改为二级护理，而根据护理常规，二级护理并不需要医护人员24小时的监护，仅需护士2小时巡视一次。其次，根据本案查明的事实，孟某术后入住某医院外三楼的CIU病房（心脏病患xxx病房），收费为200元一天；第六天孟某从CIU转入外四楼的重病房。该病房为外四病房的六人大病房，收费为36元一天，而普通病房的收费为40元至50元。从上述收费标准也可以反映出孟某当时住的病房并非其所称的“重症监护病房”，无需24小时监护，其的护理等级仍需按医嘱确定。再次，孟某住院时是有家属陪护的，而其摔倒时家属外出并没有告知当班护理人员。孟某作为需二级护理的患者，根据护理常规是可以下床自行活动的；如果需要帮助，即使家属和医护人员均不在场，也可通过拉床边铃或请其他陪护人员通知当班护理人员。最后，上海市杨浦区医学会出具的鉴定报告对本案争议的问题一一进行了分析，认为某医院在整个医疗过程中并不存在过错。在上诉人没有足够的证据推翻医学会鉴定结论的前提下，该鉴定结论可以作为确定某医院是否存在过错的依据。本院据此认为，某医院对孟某的摔倒受伤并不存在过错，上诉人要求某医院承担赔偿责任依据不足。
　　综上所述，原审法院所作判决正确，本院予以维持，上诉人的上诉请求缺乏法律依据，本院难以支持。依照《中华人民共和国民事诉讼法》第一百五十三条第一款第（一）项之规定，判决如下：
　　驳回上诉，维持原判。
　　上诉案件受理费1,572元，由上诉人孟某负担。
　　本判决为终审判决。
审判长孙卫
　　代理审判员洪可喜
　　代理审判员沈卫兵
　　二○一一年三月四日
　　书记员左莉莉
