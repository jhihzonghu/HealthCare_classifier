吴海玲与道县妇幼保健院医疗损害赔偿纠纷上诉案

审判时间：20110726 法院：湖南省永州市中级人民法院 案件字号：(2011)永中法民一终字第175号


　　上诉人（原审原告）吴海玲。
　　上诉人（原审被告）道县妇幼保健院。
　　上诉人吴海玲、上诉人道县妇幼保健院因医疗损害赔偿纠纷一案，不服道县人民法院（2011）道民初字第35号民事判决，向本院提起上诉。本院受理后，依法由审判员刘明跃担任审判长，审判员郑霓、代理审判员吕伟文组成合议庭，代理书记员甄园园担任庭审记录，于2011年7月26日在本院第一审判庭公开开庭进行了审理。上诉人吴海玲及其委托代理人王湘彪、董建林，上诉人道县妇幼保健院的委托代理人陈一凡、吴红到庭参加了诉讼。
　　经审理查明：吴海玲怀孕已有生产迹象，便于2010年12月15日上午10时22分用手机拨打道县妇幼保健院的办公电话及该院提供的手机号码，要求接诊。道县妇幼保健院的医护人员及接诊车将吴海玲于当天12时15分接诊到院。经检查为胎死宫内，胎盘早剥。12时40分，吴海玲被转至道县人民医院住院治疗。经相关检查抢救，并行剖宫术，于当日13时20分取出一死女婴，羊水血性，胎盘已全部剥离。于15时10分行子宫全切术。吴海玲在道县人民医院住院治疗27天，共计花费医疗费10,684.3元（含门诊发票）。吴海玲与道县妇幼保健院协商未果发生纠纷。
　　本案在开庭审理过程中，经本院主持调解，双方当事人自愿达成如下协议：
　　一、由上诉人道县妇幼保健院在2011年8月5日前给付上诉人吴海玲人民币34,025元；
　　二、本案一审诉讼费3,900元，二审诉讼费3,900元，减半收取1,950元，共计5,850元，即吴海玲一审已交3,900元，由吴海玲负担。吴海玲预交的二审诉讼费3,900元退给吴海玲。道县妇幼保健院预交的上诉费3,900元，退给道县妇幼保健院1,950元，此1,950元转由吴海玲领取；
　　三、道县妇幼保健院与吴海玲就本案的医疗纠纷全部了结。
　　上述协议，经双方当事人签字即发生法律效力。
　　以上协议，符合法律规定，本院予以确认。
审判长刘明跃
　　审判员郑霓
　　代理审判员吕伟文
　　二○一一年七月二十六日
　　代理书记员甄园园
