黎某某与某某医院医疗损害赔偿纠纷上诉案

审判时间：20110408 法院：湖南省长沙市中级人民法院（司法公开示范法院） 案件字号：(2010)长中民一终字第1795号


　　上诉人（原审原告）黎某某。
　　委托代理人某某某。
　　被上诉人（原审被告）某某医院。
　　法定代表人某某某，院长。
　　委托代理人林某某。
　　委托代理人蒋某某。
　　上诉人黎某某因医疗损害赔偿纠纷一案，不服湖南省长沙市芙蓉区人民法院（2009）芙民初字第2610号民事判决，向本院提出上诉。本院依法组成合议庭审理了本案，现已审理终结。
　　原审法院审理查明，2007年8月9日，黎某某因左眼视力下降到某某医院就诊，某某医院诊断黎某某为闭角型青光眼，并为黎某某测定了视力(右眼0．8，左眼0．6)和24小时眼压波动。2007年8月13日，黎某某入住某某医院。入院后，某某医院诊断黎某某为：1、慢性闭角型青光眼(双，晚期)；2、小眼球(双)；3、角膜云翳(右)。2007年8月15日，某某医院为黎某某实施了复合式小梁切除+前房成形术。2007年9月11日，黎某某出院。黎某某共交纳医疗费5000元。出院后，黎某某感觉眼睛不适、视力下降，便在多个医院进行复查，并向某某医院和卫生行政主管部门反映情况，要求解决有关问题。2009年6月，长沙市芙蓉区卫生局委托长沙市医学会对本案所涉医疗行为进行医疗事故技术鉴定。2009年6月25日，长沙市医学会出具一份医疗事故技术鉴定书。医疗事故技术鉴定书认为：1、患者患慢性闭角型青光眼的诊断成立，针对该病的各项检查正确、全面；2、慢性青光眼患者视野有明显损害，视神经乳头有损害，有手术指征，医方选择手术治疗，符合治疗该病的治疗常规；3、患者手术后致盲是术后发生恶性青光眼所致，恶性青光眼是青光眼手术的严重并发症之一，在现有医疗科学条件下是难以避免的，与患者的眼球解剖结构有关；4、发生并发症后，医方采取治疗的措施是及时、得当的；5、本病例不属于医疗事故。2009年11月13日，黎某某委托衡阳市经纬司法鉴定所对其伤残程度进行鉴定。2009年11月16日，衡阳市经纬司法鉴定所出具一份鉴定意见书，认为黎某某左眼视力无光感，右眼裸视视力0．6，其伤残程度为七级。
　　在本案审理过程中，该院曾向黎某某及其委托代理人告知和释明，长沙市医学会已出具医疗事故技术鉴定书，认为某某医院的医疗行为不属于医疗事故，且某某医院的医疗行为得当且符合诊疗常规，在此情况下，黎某某如认为某某医院在诊疗过程中存在过错，则需向该院申请，对某某医院是否存在过错及过错与损害后果之间是否存在因果关系进行鉴定，如不申请鉴定，则应承担相应的不利后果。经该院告知和释明后，黎某某仍坚持不愿申请进行司法鉴定，并要求该院依法判决。
　　黎某某在原审法院提起的诉讼请求如下：
　　1、某某医院收取黎某某医药费5000元，应当双倍返还10000元。
　　2、依法赔偿黎某某各项人身损害费174571元，包括误工费53593元；护理费1859元；住院伙食补助费870元；残疾赔偿金110568元；住宿、交通费5081元；鉴定费2600元。
　　3、由某某医院按实际发生的数额承担后续治疗费。
　　4、诉讼费、代理费由某某医院承担。
　　原审法院审理认为，黎某某因患慢性闭角型青光眼在某某医院就治和治疗，双方之间形成了医疗服务合同关系。在某某医院为黎某某提供诊疗服务的过程中，某某医院的医疗行为是否属于医疗事故，或某某医院的医疗行为是否存在过错，并因此而导致黎某某所遭受的损害结果的发生呢?首先，长沙市医学会出具的医疗事故技术鉴定书认为本案病例不属于医疗事故。其次，长沙市医学会出具的医疗事故鉴定书中明确指出，某某医院所实施的各项检查正确、全面，其选择手术治疗符合治疗常规，黎某某术后致盲是术后发生恶性青光眼所致，恶性青光眼是青光眼手术的严重并发症之一，是难以避免的。发生并发症后，某某医院采取的治疗措施是及时、得当的。此外，某某医院在术前谈话记录和同意手术记录单中，均已明确告知黎某某，有可能发生恶性青光眼的后果，黎某某也已签名表示理解和同意。据此，该院认为，某某医院对其医疗行为不存在过错提交了相应的证据来证明。在此情况下，黎某某应进一步提供证据，以证明某某医院的医疗行为是否存在过错，以及过错和损害后果之间是否存在因果关系。在本案审理过程中，该院曾就上述事项向黎某某释明，建议黎某某向该院申请鉴定，并告知其如不申请鉴定，则应承担相应的不利后果。经该院释明后，黎某某仍坚持不愿申请鉴定。该院据此认为，黎某某对其诉讼主张未提供相应的证据证明，其要求某某医院承担赔偿责任，该院不应予以支持。原审法院据此依照《中华人民共和国民法通则》第一百一十九条，并根据最高人民法院《关于民事诉讼证据的若干规定》第二条的规定，判决：驳回黎某某的诉讼请求。本案案件受理费1223元，由黎某某负担。
　　宣判后，黎某某不服，提出上诉，请求撤销原判，依法支持上诉人在一审的诉讼请求，并由被上诉人承担一审和二审的一切诉讼费用。理由如下：1、被上诉人在开庭当日才提交证据，其提交的证据已超过举证期，不能作为认定本案的事实和判决的依据；2、因医疗行为引起的侵权诉讼，由医疗机构就医疗行为与损害结果之间不存在因果关系及不存在医疗过错承担举证责任，一审法院告知释明黎某某必须向法院申请鉴定是错误的；3、被上诉人的医疗行为违法违规铁证如山，医疗事故鉴定书和一审判决都避而不谈，这是医疗事故技术鉴定专家组成员和一审法官故意偏袒被上诉人；4、被上诉人存在医疗欺诈行为；5、长沙市医学会的鉴定书不但不具有法定的形式要件，其实质内容也有瑕疵。
　　某某医院答辩称：1、上诉人所诉基本事实与客观事实严重不符，答辩人的证据在法院指定的举证期限内已交，只是法院没有组织当事人进行证据交换而已，且上诉人起诉前已有医学鉴定结论，医学鉴定对答辩人的医疗行为是否有过失，给予了充分的分析和评价，医疗技术鉴定是上诉人与答辩人双方委托的，上诉人收到鉴定书后，没有申请重新鉴定，说明其对鉴定结论是服的。2、举证倒置只是法律规定的情况适用，而不是所有。答辩人已完成了举证责任，在举证期限内向法院提交了8份证据以证明答辩人的医疗行为没有过错和过失。如果上诉人认为答辩人的医疗行为有过错或过失，则应向法院提交证据。3、闭角型青光眼是一种常见的眼科疾病，是我国最常见的青光眼类型，为上诉人治疗该病的医师是答辩人眼科青光眼专科的主任、教授，根本不存在上诉人所说的“抢手术”。5、从上诉人入院开始，答辩人对其病情、治疗方案、可能发生的各种不利后果及并发症都一一进行了必要的告知，手术是征得上诉人的同意后实施的，不存在没有告知和隐瞒病情之事实。6、恶性青光眼是青光眼手术的并发症，恶性青光眼导盲是现有医学科学技术所不能解决的疾病，非答辩人的过错或过失所致。综上所述，原审法院的判决是正确的，恳请二审法院予以维持。
　　本院对一审查明的事实予以确认。
　　本院认为，黎某某的上诉理由综合起来有三点：
　　l、关于上诉人黎某某提出的某某医院提交的证据是否超过举证期限，长沙市医学会出具的医疗事故鉴定书能否作为认定本案的事实和判决的依据的问题。某某医院提交的证据因一审法院没有在庭前组织双方进行证据交换，故黎某某在开庭质证时才看到某某医院的证据，某某医院提交的证据没有超过举证期限。黎某某上诉提出某某医院提交的证据已超过举证期限的理由不能成立，且证明某某医院有无医疗过错或过失的关键证据——长沙市医学会出具的医疗事故鉴定书，黎某某自己也作为证据予以提交。长沙市医学会医疗技术鉴定是长沙市芙蓉区卫生局经过黎某某与某某医院的同意委托长沙市医学会进行鉴定的，长沙市芙蓉区卫生局的委托程序合法。根据《医疗事故技术鉴定暂行办法》第三十三条第(五)项规定：经合议，根据半数以上专家鉴定组成员的一致意见形成鉴定结论。专家鉴定组成员在鉴定结论上签名。专家鉴定组成员对鉴定结论的不同意见，应当予以注明。第三十四条规定：医疗事故技术鉴定书应当根据鉴定结论作出，其文稿由专家鉴定组组长签发，医疗事故技术鉴定书盖医学会医疗事故技术鉴定专用印章。长沙市医学会出具的医疗事故鉴定书正本虽然没有专家的签名，但盖有长沙市医学会的公章。黎某某收到该鉴定书后，没有申请重新鉴定，说明其对鉴定结论没有异议。本院认为，长沙市医学会出具医疗事故鉴定书可以作为认定本案的事实和判决的依据，黎某某上诉提出长沙市医学会的鉴定书不但不具有法定的形式要件，其实质内容也有瑕疵的理由不能成立。
　　2、黎某某上诉提出：因医疗行为引起的侵权诉讼，由医疗机构就医疗行为与损害结果之间不存在因果关系及不存在医疗过错承担举证责任，一审法院告知释明黎某某必须向法院申请鉴定是错误的。
　　《最高人民法院关于民事诉讼证据的若干规定》第二条规定：当事人对自己提出的诉讼请求所依据的事实或者反驳对方诉讼请求所依据的事实有责任提供证据加以证明。没有证据或者证据不足以证明当事人的事实主张的，由负有举证责任的当事人承担不利后果。第四条第(八)项规定：因医疗行为引起的侵权诉讼，由医疗机构就医疗行为与损害结果之间不存在因果关系及不存在医疗过错承担举证责任。第四条第(八)项规定就是医疗纠纷举证责任倒置的规定。但举证责任倒置并不意味着医疗纠纷中由医疗机构承担全部的举证责任，医疗机构应就医疗行为与损害结果之间不存在因果关系及医疗过错承担举证责任，而患者应就存在医疗行为给其带来的损害后果承担举证责任，同时患者也拥有证明医疗行为与损害结果之间存在因果关系及医疗机构存在医疗过错的举证的权利。在本案中，某某医院在一审审理中已提交长沙市医学会出具的医疗事故鉴定书以及黎某某的病历资料证明其在医疗过程中没有过错或过失，某某医院已举证完毕。一审法院曾向黎某某及其委托代理人告知和释明，长沙市医学会已出具医疗事故技术鉴定书，认为某某医院的医疗行为不属于医疗事故，且某某医院的医疗行为得当且符合诊疗常规，在此情况下，黎某某如认为某某医院在诊疗过程中存在过错，应就某某医院的医疗行为给其带来的损害后果承担举证责任，黎某某不能提供某某医院在诊疗过程中存在过错的证据，应承担相应的不利后果。
　　3、某某医院是否存在医疗欺诈行为。黎某某上诉提出：某某医院故意隐瞒“药物是治疗青光眼的主要方法”的真实情况，又明知上诉人的门诊病历并无青光眼晚期症状及无手术指征，却故意“恐吓性”告知：“上诉人双眼闭角型青光眼晚期”的虚假情况，诱使上诉人上当受骗，作出错误的意思表示。被上诉人对上诉人的眼病应当先药治而不药治，不到晚期而诊断晚期，不应当手术而抢手术是其主观上的过错，因其错误手术导致黎某某左眼失明七级伤残损害事实客观存在，某某医院应当承担相应的民事侵权责任。对此，本院认为，人的身体情况各有各的不同，病情也有差异，所以针对每个人不同的病情治疗的方案也不同，“药物是治疗青光眼的主要方法”，但并不是针对每个人，手术也是治疗青光眼的一种有效方法，至于手术的风险，某某医院也告知了黎某某。长沙市医学会医疗事故技术鉴定书：“某某医院对黎某某患慢性闭角型青光眼的诊断成立，有手术指征，医方选择手术治疗，符合治疗该病的治疗常规，患者手术后致盲是术后发生恶性青光眼所致，恶性青光眼是青光眼手手术的严重并发症之一，在现有医疗科学条件下是难以避免的”，说明某某医院选择手术治疗并没有错，黎某某称某某医院存在医疗欺诈行为的理由不充分，本院不予采信。
　　综上所述，某某医院的医疗行为不属于医疗事故，在诊疗过程中也没有过错或过失。某某医院在术前谈话记录和同意手术记录单中，均已明确告知黎某某，有可能发生恶性青光眼的后果，黎某某也已签名表示理解和同意。某某医院对其医疗行为不存在过错或过失已提交了相应的证据予以证明。原审法院已向黎某某释明，建议黎某某向法院申请司法鉴定，以证明某某医院的医疗行为是否存在过错，以及过错和损害后果之间是否存在因果关系。并告知其如不申请鉴定，则应承担相应的不利后果。法院释明后，黎某某仍坚持不愿申请鉴定。黎某某不能提供某某医院在诊疗过程中存在过错或过失的证据，应承担相应的不利后果，其要求某某医院承担赔偿责任没有事实和法律依据，本院不予支持。黎某某的上诉请求理由不足，本院不予支持。原审法院认定事实清楚，适用法律恰当，判决并无不妥，应予维持。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第一项之规定，判决如下：
　　驳回上诉，维持原判。
　　本案二审案件受理费1223元，由上诉人黎某某负担。
　　本判决为终审判决。
审判长陈佳
　　审判员张玉霞
　　代理审判员杨桅
　　二○一一年四月八日
　　书记员谭蜜
