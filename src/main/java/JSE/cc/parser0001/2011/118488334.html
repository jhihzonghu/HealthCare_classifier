XX、XX、XX、XX与XX医院医疗损害赔偿纠纷案

审判时间：20110721 法院：上海市浦东新区人民法院（司法公开示范法院） 案件字号：(2010)浦民一(民)初字第29148号


　　原告XX。
　　委托代理人XX（系原告女儿），住同原告XX。
　　委托代理人XX，上海XX律师事务所律师。
　　原告XX。
　　委托代理人XX，上海XX律师事务所律师。
　　原告XX。
　　委托代理人XX。
　　原告XX。
　　被告XX医院。
　　法定代表人XX，院长。
　　委托代理人XX，上海市XX律师事务所律师。
　　委托代理人XX，上海市XX律师事务所律师。
　　原告XX、XX、XX、XX与被告XX医院医疗损害赔偿纠纷一案，本院于2010年9月26日受理后，依法组成合议庭，于2011年1月10日、4月22日公开开庭进行了审理。原告XX、XX的委托代理人XX、原告XX的委托代理人XX、原告XX、被告委托代理人XX到庭参加诉讼。本案现已审理终结。
　　原告XX、XX、XX、XX诉称，患者XX系原告XX的丈夫、原告XX、XX、XX的父亲。2010年2月23日深夜至24日凌晨，XX在家突感不适，由家人将其急送被告医院就诊。当XX被推至急诊室检查时，接诊医生及护士草草做了一下心电图，就说人已经死亡。患者家属说不可能，但接诊医生让家属自己看，心电图一条直线说明人已死亡。家属坚持说人没有死，苦苦哀求被告的接诊医生进行抢救。但是被告医生一口回绝，说人已死亡没有必要抢救，即使抢救也没有效果。同时，医生还问家属拿病历卡要去开死亡证明。被告在同一份死亡医学证明书上还有明显的涂改，对死亡原因原先表述为“来院已死亡”，后面改为“猝死”，自相矛盾。原告方认为，被告在诊疗过程中的不作为违反了法律规定的注意义务，依法应当承担赔偿责任。故诉请本院判令被告支付死亡赔偿金人民币133,375元、精神损害抚慰金50,000元。
　　被告XX医院辩称，患者来院就诊时心跳呼吸均已经停止，并出现面色紫灰，躯体卷曲、头部和四肢低垂等征象。但接诊医生和当班护士还是立即将患者转移检查台进行救治。经进一步检查发现患者呼吸心跳确已停止，大动脉搏动消失，双瞳孔散大，心电图成一直长直线。考虑到患者已无任何生命征象，可能已死亡一段时间。检查过程中，心电监护未提示室速和室颤情况，心电图持续呈一直线，死亡已经不可逆转，故未予电击除颤等进一步抢救措施。被告认为，接诊医生已经尽到合理的救治义务，不存在不作为。患者来院时已经死亡，死亡原因与医生救治措施无关，故请求法院驳回原告方全部诉讼请求。
　　经审理查明，原告XX系患者XX妻子，XX与XX共生育三个女儿，分别为原告XX、XX、XX。2010年2月24日凌晨，患者XX由轮椅车推送到被告医院急诊。被告医院急诊普内科急查发现，患者呼吸消失，心音听不出，大动脉搏动摸不到，双眼瞳孔散大固定，血压测不到，肢体呈卷曲状，查心电图示一直线，考虑患者来院已经死亡。诊断为：来院已死亡。另查明，在患者的死亡医学证明书“直接死亡原因”一栏，原先表述为“来院已死亡”，后改为“猝死”。
　　案件审理中，本院根据当事人申请委托上海市普陀区医学会组织医疗事故技术鉴定。2011年3月9日，普陀区医学会组织召开医疗事故技术鉴定会。鉴定会经过医患双方陈述、答辩，专家提问并经表决、合议，最终形成不构成医疗事故的鉴定结论。鉴定分析认为：根据现有病史资料，患者XX“来院已死亡”诊断成立。医方在死亡事件处置过程中存在不足：1、死亡医学证明书前后联不一致，且有涂改。2、“来院已死亡”应及时报警备案。但是医方上述不足之处与患者死亡无因果关系，故不构成医疗事故。
　　以上事实，有原告方提交的户籍资料、门急诊就医记录、检验报告、死亡证明书、上海市普陀区医学会医疗事故技术鉴定材料及双方当事人的陈述等在案佐证，本院经审查后予以确认。
　　本院认为，由于医疗行为具有较强的专业性，对该行为是否适当的判断，除依照一般常理及当事人提供的证据材料以外，还需由相关的权威部门即医学会进行鉴定。本起医疗纠纷经医疗事故技术鉴定，鉴定结论明确表述本病例不属于医疗事故，且在分析意见中载明，患者XX“来院已死亡”诊断成立。这表明被告的医疗行为没有过错。被告在工作中虽有不足，但这些不足之处与患者死亡并无因果关系。医疗损害赔偿责任的承担，是以医疗行为存在过错以及医疗行为与损害后果之间存在因果关系为其基本条件的。原告方以其所述理由要求被告承担相应赔偿责任，不符合赔偿责任承担的基本条件，本院不予支持。考虑到被告工作存有不足，且患者家属本身不具备医学专业知识，被告有义务加强与病患之间的沟通，故医学会鉴定费用3,500元应由被告负担。依照《中华人民共和国民法通则》第一百零六条第二款的规定，判决如下：
　　驳回原告XX、XX、XX、XX的诉讼请求。
　　案件受理费3,967元（原告方已付），由原告XX、XX、XX、XX共同负担。鉴定费3,500元（被告已付），由被告XX医院负担。
　　如不服本判决，当事人可自判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
审判长徐俊

　　人民陪审员李蔓薇
