林伯云与安化县人民医院医疗损害赔偿纠纷上诉案

审判时间：20110323 法院：湖南省益阳市中级人民法院 案件字号：(2011)益法民一终字第17号


　　上诉人（原审原告）林伯云。
　　委托代理人林丽。代理权限为一般代理。
　　委托代理人胡桂华。代理权限为一般代理。
　　被上诉人（原审被告）安化县人民医院。
　　法定代表人龚育凡，院长。
　　委托代理人胡旭忠，该院副院长。代理权限为特别授权。
　　委托代理人夏光荣，湖南淡远律师事务所律师。代理权限为一般代理。
　　案由：医疗损害赔偿纠纷
　　上诉人林伯云与被上诉人安化县人民医院医疗损害赔偿纠纷一案。林伯云不服安化县人民法院（2010）安法民一初字第810号民事判决，向本院提起上诉，请求依法改判。
　　经审理查明：林伯云于2007年10月3日因患结石性胆囊炎入住安化县人民医院，10月5日该院工作人员为林伯云行腹腔镜胆囊切除手术。因手术操作不当造成其人身损害，经益阳市医学会鉴定构成四级医疗事故，该院负完全责任，并建议林伯云定期随诊。2007年10月3日至10月22日林伯云在该院住院19天，10月22日至11月30日在湘雅医院住院39天，2007年12月1日至2008年2月3日又在该院住院64天。安化县人民医院已支付医疗费88367.8元，林伯云垫付医疗费5800元，医疗费合计94167.8元。该医疗纠纷曾经卫生行政主管部门调解未果。2010年5月26日林伯云申请对其伤残程度、病休时间、陪护人数及陪护时间进行司法鉴定。湖南省人民医院司法鉴定中心作出司法鉴定意见如下：被鉴定人林伯云慢性结石性胆囊炎行腹腔镜摘除术后致胆汁性弥漫性腹膜、伤口感染、腹壁蜂窝组织炎经多次手术后存留有肠粘连，综合评定为捌级伤残。建议病休时间为拾贰个月。住院病危期间需要贰人陪护，其余住院时间需要壹人陪护；出院后需壹人陪护，陪护时间为肆个月。因双方就赔偿问题协商不成。林伯云遂诉至法院，请求赔偿因伤造成的医疗费、误工费、交通费等各项损失365085.51元。
　　本案在二审审理过程中，经本院主持调解，双方自愿达成如下调解协议：
　　一、安化县人民医院自愿一次性补偿林伯云各项损失148800元（不含已支付医药费88367.8元），限于2011年4月20日之前一次性付清（邮政储蓄银行账号b221885b10xxx428户名：林伯云），逾期不支付则林伯云可按180000元向法院申请强制执行；二、本案所涉纠纷一次性解决，双方均不得就本纠纷再相互主张其它任何权利；三、本案一审案件受理费1905元，由安化县人民医院负担770元，林伯云负担1135元；二审案件受理费1905元，减半收取952.5元，由上诉人林伯云负担；四、本调解协议双方签字即生效。
　　上述调解协议符合法律规定，本院依法予以确认。本调解书具有法律效力。
