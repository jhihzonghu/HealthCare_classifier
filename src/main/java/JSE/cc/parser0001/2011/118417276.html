侯甲等与第二军医大学第二附属医院医疗损害责任纠纷上诉案

审判时间：20110615 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第828号


　　上诉人(原审原告)侯甲。
　　上诉人(原审原告)侯乙。
　　两上诉人共同委托代理人潘国华，上海博和律师事务所律师。
　　被上诉人(原审被告)第二军医大学第二附属医院(即上海长征医院)。
　　法定代表人郑某某，院长。
　　委托代理人沈成良、徐勇，上海市康正律师事务所律师。
　　上诉人侯甲、侯乙因医疗损害责任纠纷一案，不服上海市黄浦区人民法院(2010)黄民一(民)初字第3966号民事判决，向本院提起上诉。本院依法组成合议庭,公开开庭审理了本案。上诉人侯甲及其与侯乙共同委托代理人潘国华、被上诉人第二军医大学第二附属医院(以下简称长征医院)的委托代理人徐勇到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，2008年7日31日晚，原告侯甲、侯乙之子侯丙因摔倒受伤至被告处救治，分别于21时15分及22时40分两次进行头颅CT检查。入院诊断：特急性重型闭合性颅脑损伤(右额颞顶部硬膜外血肿、左颞叶脑挫裂伤、右侧颞骨骨折、右枕部头皮裂伤伴血肿)。22时55分，侯丙在全麻下行右侧额颞顶部硬膜外血肿清除术。术后即刻复查头颅CT：右侧硬膜外血肿清除彻底，脑水肿。术后予脱水、止血、抗感染等治疗。8月1日5时50分，侯丙出现躁动，昏迷，GCS7分，右侧瞳孔直径2.5mm，对光反射存在，左侧瞳孔2.5mm，对光反射灵敏。6时03分头颅CT：右侧广泛硬膜下血肿，遍及额颞顶枕部，弥漫性脑肿胀。在准备行CT检查过程中侯丙出现右侧瞳孔散大约5mm，对光反射迟钝后消失。随即给予甘露醇250ml静滴对症处理，未见好转。于10分钟后出现双侧瞳孔散大，呼吸骤停，予人工呼吸，球囊辅助呼吸进手术室急诊手术。6时15分，侯丙在全麻下行右侧额颞顶硬膜下血肿清除+右侧颞叶切除内减压+右侧额颞顶去骨瓣减压术。术后予以对症、补液、抗炎、支持、促醒、物理降温等治疗。侯丙术后一直处于深昏迷状态，自主呼吸消失，呼吸机维持，GCS3分。8月3日23时30分，侯丙出现心跳骤停，血压测不出。经抢救无效，于8月4日0时12分宣布临床死亡。死亡原因：脑干功能衰竭导致全身多脏器功能衰竭。死亡诊断：特急性重型闭合性颅脑损伤(右额颞顶枕硬膜下血肿；弥漫性脑肿胀；右额颞顶硬膜外血肿；左额颞叶脑挫裂伤；右侧颞骨骨折；右枕部头皮裂伤伴血肿)；多脏器功能衰竭。原告认为被告在对侯丙救治过程中存在重大过错。为此原告提起诉讼，要求被告赔偿原告死亡赔偿金533,500元(人民币，下同)、丧葬费17,353.50元、误工费5,000元、交通费及住宿费8,000元、抚养费182,300元、精神损害抚慰金50,000元。
　　原审审理中，根据被告申请，法院委托上海市长宁区医学会进行医疗事故技术鉴定，该会于2010年1月15日出具了沪长医鉴[2009]033号《医疗事故技术鉴定书》，鉴定结论为：侯丙与长征医院医疗争议不构成医疗事故；本病例不属于医疗事故。原告对该《医疗事故技术鉴定书》不服，提出再次鉴定。法院经审查后，依法委托上海市医学会对本纠纷再次进行鉴定。该会于2010年8月5日出具了沪医鉴[2010]081号《医疗事故技术鉴定书》。该鉴定书分析意见为：1、2008年7月31日，患者因“摔倒后意识障碍进行性加重20分钟”入住长征医院；根据患者症状、体征及头颅CT检查结果，诊断为特急性重型闭合性颅脑损伤(右额颞顶部硬膜外血肿等)，予急诊行硬膜外血肿清除术；医方的诊疗行为符合诊疗常规，手术操作符合规范。2、患xxx病情发生变化，复查头颅CT发现右额颞顶部硬膜下血肿，再次急诊行硬膜下血肿清除术有手术适应证；第二次手术操作亦符合规范。3、阅读第一次手术前的两次CT片，手术区域并无明确硬膜下血肿的征象，故医方对硬膜下血肿的漏诊不成立。4、患者为特急性重型闭合性颅脑损伤，第一次术后又发生迟发性硬膜下出血，病情危重，进展迅速，是导致其死亡的根本原因。5、对于颅内压增高的患者，使用地塞米松并不违反用药规范。6、医方在患者术后每间隔1小时对患者进行一次护理记录不违反护理常规。7、医方存在病史记录欠规范、手术名称混乱等不足，但与患者死亡无因果关系。鉴定结论为：侯丙与长征医院医疗争议不构成医疗事故。两次技术鉴定的鉴定费共计7,000元，原、被告分别支付了3,500元。
　　原审审理中，被告表示鉴于本案的实际情况，作为人道主义的补偿，自愿给予原告5,000元。
　　原审法院经审理后认为，医疗事故损害赔偿必须以医疗单位有过错行为的存在为前提，即患者的损害后果与医疗单位的不当医疗行为之间存在法律上的因果关系。上海市医学会所做的鉴定结论符合客观事实，且鉴定过程程序合法。对该医学会的鉴定，本院予以采信，应认定被告的医疗行为与诊疗结果之间并无法律上的因果关系。被告存在病史记录欠规范、手术名称混乱等不足，但与患者死亡无因果关系。原告认为被告的医疗行为存在过错的主张缺乏事实依据，故本院对其要求被告赔偿相关经济损失的诉讼请求难以支持。现被告自愿补偿原告5,000元，本院认为并无不当，故依法可予准许。
　　原审法院据此判决：一、被告第二军医大学第二附属医院自愿于本判决生效之日起十五日内补偿原告侯甲、侯乙人民币5,000元。二、对原告侯甲、侯乙的诉讼请求不予支持。
　　原审判决后，上诉人侯甲、侯乙不服，向本院提起上诉称：请求二审法院撤销原审判决，依法改判支持上诉人的原审诉讼请求。因为一审法院认定事实错误，认定被上诉人的医疗行为与侯丙的死亡之间无法律上的因果关系缺乏依据。即使根据两个医疗事故鉴定，也仅能认定被上诉人的医疗行为不构成医疗事故，并没有排除其医疗行为过程中是否存在过错。上海医学会的鉴定结论明目张胆地颠倒是非，欺骗上诉人，上诉人要求再做鉴定，以讨回公道。上诉人认为，被上诉人的医疗行为存在以下过错：1、CT读片存在重大过错；2、第一次手术存在过错，包括延误救治时间、错误用药、手术操作失误；3、第二次手术存在重大过错，包括手术方式不当、未对硬膜下出血点进行止血、手术记录错误；4、术后监护措施不到位，存在重大过错。综上，一审判决缺乏事实与法律依据，应当依法予以撤销。恳请二审法院依法改判。
　　被上诉人长征医院辩称：不同意上诉人的上诉请求，同意一审判决。
　　本院经审理查明，原审查明事实属实，本院予以确认。
　　本院认为，公民享有生命健康权。在医疗纠纷中，医院的医疗行为存在过失，侵害他人生命健康权的，应当承担相应责任。原审法院根据双方当事人的诉辩、提交的证据对本案事实进行了认定，并在此基础上结合上海市医学会医疗事故技术鉴定书依法作出原审判决，合法合理，理由阐述充分，本院不再赘述。二审中上诉人主张一审法院认定事实不清，两级医学会的鉴定结论不能证明被上诉人的医疗行为无过错，而事实是被上诉人在医疗护理过程中都存在重大过错，故请求再次鉴定，请求二审法院查明事实依法改判支持上诉人的原审诉讼请求。但其在二审中没有新的事实与理由，也没有新的证据佐证自己的主张。重新鉴定并非必经程序，仅是在特定情况下的一种补救手段，本案已经经过两次鉴定，因此，再次重新鉴定必需具备一定的条件。上诉人未能提供证据证明长宁区、上海市二级医学会作出的鉴定结论存在下列情形：(一)鉴定机构或者鉴定人员不具备相关的鉴定资格的；(二)鉴定程序严重违法的；(三)鉴定结论明显依据不足的；(四)经过质证认定不能作为证据使用的其他情形。故本案并不具备进行重新鉴定的条件，对上诉人要求再次重新鉴定的申请，本院不予准许。本院认可一审法院对事实的分析认定及对相关法律法规的理解与适用，上诉人的上诉请求缺乏事实及法律依据，本院难以支持。综上所述，原审认定事实清楚，判决并无不当。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币11,762元，由上诉人侯甲、侯乙负担。
　　本判决为终审判决。
审判长李伊红
　　代理审判员张顼
　　代理审判员朱红卫
　　二○一一年六月十五日
　　书记员龚轶
