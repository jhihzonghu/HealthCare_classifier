周某某与上海市杨浦区中心医院医疗损害赔偿纠纷案

审判时间：20110422 法院：上海市第二中级人民法院（司法公开示范法院） 案件字号：(2011)沪二中民一(民)终字第40号


　　上诉人(原审原告)周某某。
　　委托代理人邓本天，上海伟聚律师事务所律师。
　　被上诉人(原审被告)上海市杨浦区中心医院。
　　法定代表人李某某，院长。
　　委托代理人徐美琴，上海市康正律师事务所律师。
　　委托代理人雷某，上海市杨浦区中心医院职工。
　　上诉人周某某因医疗损害赔偿纠纷一案，不服上海市杨浦区人民法院(2009)杨民一(民)初字第1196号民事判决，向本院提起上诉。本院依法组成合议庭，公开开庭审理了本案。上诉人周某某及其委托代理人邓本天、被上诉人上海市杨浦区中心医院的委托代理人徐美琴及雷某到庭参加诉讼。本案现已审理终结。
　　原审法院经审理查明，1995年5月21日晚，周某某因与同事发生争执致头部受伤，被送至上海市杨浦区中心医院验伤，当日验伤结论为头皮、右脸颊及左前臂软组织挫伤及右膝部软组织伤。次日，周某某再次至上海市杨浦区中心医院医治，CT提示右额叶局部脑梗塞灶，建议治疗后随访。同年10月19日，周某某再次至上海市杨浦区中心医院做MRI检查，报告提示：右额叶局部脑梗塞。1998年9月，周某某至上海仁济医院就诊，因被诊断为大脑纵裂Ⅱ期肿瘤而收治入院，1998年9月17日，该医院为周某某在全麻下行大脑纵裂肿瘤切除术，手术顺利，术后采取抗炎、止血等治疗，恢复良好。1998年9月30日周某某出院，遵医嘱多次至医院门诊随访。2007年8月，周某某曾至上海华山医院治疗，该院会诊意见为1995年所摄片已提示右额叶胶质瘤(偏良性)累及左额叶。
　　原审法院另查明，1、2007年8月至9月，周某某多次向上海市杨浦区中心医院投诉，上海市杨浦区中心医院于2007年10月11日书面发函给予答复。2008年4月18日，上海市杨浦区中心医院再次就周某某投诉一事向杨浦区卫生局医疗事故处理办公室发函，补充说明相关情况，内容包括所有病历及相关资料原件皆由周某某持有，该院将CT、MRI摄片交有关人员复阅后已还给周某某，周某某3年后患星形细胞瘤与上海市杨浦区中心医院的诊治行为无因果关系等。2、根据周某某的申请，杨浦区卫生局医疗事故处理办公室于2008年8月13日委托上海市杨浦区医疗事故技术鉴定工作办公室就患者周某某与上海市杨浦区中心医院医疗争议是否构成医疗事故进行鉴定，但因周某某就鉴定专家的学科组成与鉴定机构存在异议，鉴定机构于2008年9月出具中止鉴定的通知。3、在审理过程中，周某某就己方保存的CT、MRI摄片是否存在篡改申请鉴定，原审法院于2009年4月委托司法鉴定科学技术研究所司法鉴定中心就此申请进行鉴定，该机构于2009年5月12日出具情况说明一份，内容为本中心鉴定人阅法院所提供的片子，1995年10月19日MRI片的边角处确有药膜剥脱，1995年5月22日的CT片上确有水迹干后的痕迹，但未见到盖章的痕迹。上述药膜剥脱或水迹干后的痕迹属保管不当所致，对片子所显示的图像无毁损，图像仍可清楚显示，不影响阅片及诊断，未见摄片存在篡改迹象。周某某对情况说明提出异议，坚持要求再次就摄片是否存在篡改进行鉴定，在做出相关鉴定结论的基础上，再同意进行医疗事故鉴定。原审法院曾先后与上海市公安局物证鉴定中心、华东政法大学司法鉴定中心联系鉴定事宜，但两机构均答复有关鉴定事宜不属于两鉴定机构鉴定范围。周某某继续坚持就摄片是否被篡改进行鉴定，拒绝医疗事故鉴定。
　　原审法院经审理后认为，本案系一起因医疗纠纷所引起的人身损害赔偿案件。就本案双方当事人存有的争议而言，首先必须确定医疗机构在对患者的诊疗过程中是否存在过错及该过错与患者目前的损害后果是否存在因果关系。基于医疗行为是一种具有高度专业性、相当复杂性并同时具有一定风险性的活动过程，确定医疗机构在对患者的诊疗过程中是否存在过错及该过错与患者目前的损害后果是否存在因果关系是一个相当专业的问题，故必须依赖具有专业知识、经验、技能的医疗鉴定部门专家的权威意见和评判结论。本案中，上海市杨浦区中心医院在就本案所涉医疗事件是否构成医疗事故，医院对周某某的医疗行为是否违反医疗卫生管理法律、行政法规、部门规章和诊疗规范、常规，其医疗行为与周某某目前状况有无因果关系申请鉴定，因周某某至上海市杨浦区中心医院处是门诊治疗，故涉及本案的所有病历及摄片等，均由周某某保留，因此上海市杨浦区中心医院已完成了其举证义务。但周某某认为长期由其保留的摄片是伪造的，据此不愿进行鉴定。依据法律规定，当事人对自己提出的诉讼请求所依据的事实或者反驳对方诉讼请求所依据的事实有责任提供证据加以证明。没有证据或者证据不足以证明当事人的事实主张的，由负有举证责任的当事人承担不利后果。又依据《医疗事故处理条例》第三十条第二款规定，双方当事人应当按照本条例的规定如实提交进行医疗事故技术鉴定所需要的材料，并积极配合调查。当事人任何一方不予配合，影响医疗事故技术鉴定的，由不予配合的一方承担责任。根据上述规定，周某某虽然认为相关摄片是伪造的，但未能提供充分、足够的相关证据予以证明，因而周某某拒绝鉴定是没有理由的，周某某应承担不利后果。综上，周某某要求上海市杨浦区中心医院赔偿医疗费在内的各项经济损失，不予支持。
　　原审法院据此作出判决：一、周某某要求上海市杨浦区中心医院赔偿医疗费82,828.80元的请求，不予支持；二、周某某要求上海市杨浦区中心医院赔偿住院伙食补助费2,180元的请求，不予支持；三、周某某要求上海市杨浦区中心医院赔偿营养费4,270元的请求，不予支持；四、周某某要求上海市杨浦区中心医院赔偿护理费4,360元的请求，不予支持；五、周某某要求上海市杨浦区中心医院赔偿误工费441,175.80元的请求，不予支持；六、周某某要求上海市杨浦区中心医院赔偿残疾生活补助费377,856元的请求，不予支持；七、周某某要求上海市杨浦区中心医院赔偿交通费200元的请求，不予支持；八、周某某要求上海市杨浦区中心医院赔偿精神损害抚慰金300,000元的请求，不予支持。
　　判决之后，周某某不服原判，上诉于我院，仍坚持其在原审中的主张，要求上海市杨浦区中心医院承担赔偿责任。
　　上海市杨浦区中心医院则表示服从原判。
　　本院经审理查明，上海市杨浦区中心医院对原审法院查明的事实没有异议；周某某对原审法院查明的事实有下列异议：1、认为与其冲突者不是其同事；2、认为上海市杨浦区中心医院在为其验伤过程中存在作弊情况，两张片子的结论完全不同。
　　在本院审理中，周某某申请本院委托华东政法大学司法鉴定中心对其CT片子进行司法鉴定。本院委托后，华东政法大学司法鉴定中心以《退案说明书》回复本院：“因检材与本中心鉴定条件所限，本中心根据相关规定决定作退案处理。”
　　本院认为，公民的生命健康权受法律保护。周某某与上海市杨浦区中心医院因医疗问题产生纠纷。由于医疗活动具有高度专业性，医疗科学属于生命科学领域，具有相当复杂性并同时具有一定的风险性，故对于一起医疗纠纷是否构成医疗事故或存在医疗差错，尚有赖于具有专业知识、经验、技能的专家作出鉴定。现周某某以上海市杨浦区中心医院篡改相关影像资料为由，不同意进行医疗事故鉴定；但其未提供充分的证据证实自己的主张。在一、二审法院审理过程中，无论是根据当事人的申请、还是法院根据相关规定予以查证，均无法得出足以支持周某某观点的证据、结论。故周某某应当承担不利的法律后果。原审法院据此判决不支持周某某的诉讼请求是正确，本院予以维持；周某某的上诉请求没有理由，本院不予支持。综上所述，原审认定事实清楚，判决并无不当。据此，依照《中华人民共和国民事诉讼法》第一百五十三条第一款第(一)项之规定，判决如下：
　　驳回上诉，维持原判。
　　二审案件受理费人民币15,716元，由上诉人周某某负担。
　　本判决为终审判决。
审判长高中伟
　　代理审判员张顼
　　代理审判员钱健民
　　二○一一年四月二十二日
　　书记员林琳
