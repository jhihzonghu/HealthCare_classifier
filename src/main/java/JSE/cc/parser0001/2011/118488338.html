XX、XX、XX、XX与XX医院医疗损害赔偿纠纷案

审判时间：20110721 法院：上海市浦东新区人民法院（司法公开示范法院） 案件字号：(2010)浦民一(民)初字第31341号


　　原告XX。
　　原告XX。
　　原告XX。
　　原告XX。
　　四原告共同委托代理人XX。
　　四原告共同委托代理人XX，上海XX律师事务所律师。
　　被告XX医院。
　　法定代表人XX，院长。
　　委托代理人XX。
　　委托代理人XX，北京市XX律师事务所上海分所律师。
　　原告XX、XX、XX、XX与被告XX医院医疗损害赔偿纠纷一案，本院于2010年10月18日受理后，依法组成合议庭，于2011年4月27日公开开庭进行了审理。原告XX、XX、XX、XX的委托代理人XX、XX、原告XX、XX、被告委托代理人XX到庭参加诉讼。本案现已审理终结。
　　原告XX、XX、XX、XX诉称，患者XX系XX的儿子、XX的丈夫、XX和XX的父亲。2010年9月2日下午，XX因左下腹阵发性疼痛入住XX医院就诊，并由医院拟肠梗阻收治入院。9月3日晚，XX腹部疼痛加剧，但医生却不知去向。XX的疼痛一直持续至9月4日上午9时，经主任医师查房诊断为绞窄性肠梗阻，并疼痛症状加剧，但被告医务人员仍不进行手术，直至10时多，因XX即刻血压测不出，出现休克症状，被告医务人员才通知要求转上级医院继续治疗。13时多，XX血压不能回升转院可能有生命危险，被告因而为XX进行了全麻下行剖腹探查术。但为时已晚，术中可见腹腔约100ml浑浊腥臭腹水及肠液涌出，距回盲部85cm处回肠与腹部粘连并成角绞窄，此处以上肠管扩张、坏死，呈紫黑色。9月5日上午6时40分，XX经抢救无效死亡。原告方认为，医方未尽注意和告知义务，对患xxx病未按照规定诊治。在患者住院期间，医方也未及时观察、处置病情，造成病情不断恶化，丧失了最佳手术治疗时机，最终造成了死亡的不良后果，依法应当承担相应赔偿责任。故诉请本院判令被告向四原告支付死亡赔偿金人民币541,246元、丧葬费21,396元、精神损害抚慰金50,000元、被抚养人生活费20,992元、交通费2,000元、医疗费5,100元，律师费20,000元，合计660,734元。
　　被告XX医院辩称，患者因脾切后肠梗阻多次发作，通过保守治疗均缓解。肠梗阻手术指征、时机较难把握。在病程观察期间，由于患者病情特殊，两次复查腹部立位片均提示：未见明显异常X线征象，给临床医生确定手术时机带来困难。患者入院后诊断为粘连性肠梗阻，而非原告方所说的绞窄性肠梗阻。其对具有脾切除史的肠阻梗患者予以住院保守手术治疗，符合临床诊疗规范和常规，医方不存在过错及事故。患xxx病本身演变所致，与医方诊疗行为没有因果关系。故请求法院驳回四原告全部诉讼请求。
　　经审理查明，患者XX出生于1947年1月18日，系居民，是原告XX的儿子、原告XX的丈夫、原告XX和XX的父亲。原告XX身体残疾，其共有五位子女，除了患者XX之外，其他四位子女均健在。
　　2010年9月2日下午，XX因“腹痛5天，加重1天伴呕吐一次”前往被告XX医院外科就诊。医方诊断为肠阻梗。患者原有脾切除术史。2010年9月3日16时，患者仍诉腹痛腹胀。9月4日9时，患者继续诉腹胀，肛门无排气排便。10时多，XX即刻血压测不出，尿量无。医方考虑肠坏死可能。医方会诊后认为，患者出现休克、急性肾衰表现，肠坏死至中毒休克可能大，有急诊手术指征，患者要求转上级医院继续治疗，已通知救护站。13时20分，患者因血压不能回升，家属要求在本院手术。2010年9月4日14时30分，医方在全麻下对患者行“大部分小肠切除术+端端吻合术”。术中见：腹腔有约100ml浑浊腥臭腹水及肠液涌出，结节性肝硬化，距回盲部85cm处回肠与腹部粘连并成绞窄，此处以上肠管扩张、坏死，呈紫黑色。18时10分手术完毕，患者送入ICU监护。18时40分，医方诊断，绞窄性肠梗阻，肠坏死，中毒性休克，急性肾衰，脾切除术后。予以告病危，持续心电监护，患者机械通气中。9月5日上午6时40分，XX心电图呈一直线，临床宣告死亡。
　　案件审理中，本院根据当事人申请委托上海市浦东新区医学会组织医疗损害鉴定。2011年3月16日，浦东新区医学会组织召开鉴定会。鉴定会经过医患双方陈述、答辩，专家提问并经表决、合议，最终形成鉴定结论。鉴定分析认为：医方对患者肠阻梗保守治疗不规范和临床观察不严密，导致病人延误最佳手术时机，医方存有过失，与患者的死亡存有一定因果关系。患者临床表现存在一定不典型性，腹部平片复查结果以及血常规复查白细胞数下降，提示患者病情没有加重，对病情变化判断和及时处理带来一定困难。鉴定意见认为：本例属于对患者人身的医疗损害。参照《医疗事故分级标准（试行）》，患者XX死亡结果的人身医疗损害等级为一级甲等。本例医疗过错的责任程度为主要责任。
　　以上事实，有原告方提交的户籍资料、就医记录、费用收据、死亡证明材料、律师费发票、被告提交的住院病案、病程录等资料、上海市浦东新区医学会医疗损害鉴定材料及双方当事人的陈述等在案佐证，本院经审查后予以确认。
　　本院认为，由于医疗行为具有较强的专业性，对该行为是否适当的判断，除依照一般常理及当事人提供的证据材料以外，还需由相关的权威部门即医学会进行鉴定。本起医疗纠纷经医疗损害鉴定，鉴定分析医方存有过失，与患者的死亡存有一定因果关系。鉴定结论明确患者XX死亡结果的人身医疗损害等级为一级甲等，本例医疗过错医院的责任程度为主要责任。故被告应就其不当医疗行为承担相应的赔偿责任。
　　根据相关规定，本院对原告方的受损范围确认如下：1、关于死亡赔偿金。该项目由两部分组成：一部分按照受诉法院所在地上一年度城镇居民人均可支配收入或者农村居民人均纯收入标准，按二十年计算。根据规定，六十周岁以上的，年龄每增加一岁减少一年。患者死亡时年满63周岁，且系居民，故应当按照城镇居民人均可支配收入标准，按十七年计算所得金额为541,246元。该笔金额属于原告XX、XX、XX、XX共同损失。另一部分是被抚养人生活费。根据规定，被扶养人生活费根据扶养人丧失劳动能力程度，按照受诉法院所在地上一年度城镇居民人均消费性支出和农村居民人均年生活消费支出标准计算。被扶养人无劳动能力又无其他生活来源的，计算二十年。但六十周岁以上的，年龄每增加一岁减少一年；七十五周岁以上的，按五年计算。由于XX年满七十五周岁，故应按五年计算，所得金额为116,000元。XX共有五位子女，除了患者XX之外，其他四位子女均健在，故患者XX生前仅应承担其中五分之一的份额，相应的被抚养人生活费为23,200元。该笔被抚养人生活费对象特定，即专属于原告XX。通过以上分析，纳入原告XX损失范围的死亡赔偿金为158,511.50元，纳入原告XX、XX、XX损失范围的死亡赔偿金为405,934.50元。2、关于丧葬费。根据规定，丧葬费按照受诉法院所在地上一年度职工月平均工资标准，以六个月总额计算所得金额为23,378.50元。3、关于交通费。考虑到患者及其必要的陪护人员因就医必然要发生相关交通费，本院根据患者治疗情况酌情确定500元。4、关于医疗费。患者实际已发生医疗费5,100元，被告对此没有异议，本院予以确认。鉴定认为，本例医疗过错医院的责任程度为主要责任，本院据此酌定医院应当对上述损害承担70%的赔偿责任。此外，本案患者的死亡导致原告方遭受精神痛苦，现主张精神损害赔偿，与法有据，应予支持，本院综合考虑被告过错程度、患者病情等因素酌情支持35,000元。关于律师费，本院根据原告方实际支出、本案具体案情以及相关收费标准酌情支持10,000元。对医学会鉴定费用3,500元，亦应由被告负担。依照《中华人民共和国侵权责任法》第六条第一款、第十六条、第五十四条的规定，判决如下：
　　一、被告XX医院应当在本判决生效之日起十日内赔偿原告XX、XX、XX、XX丧葬费16,364.95元、交通费350元、医疗费3,570元、精神损害抚慰金35,000元、律师费10,000元；
　　二、被告XX医院应当在本判决生效之日起十日内赔偿原告XX死亡赔偿金110,958.05元；
　　三、被告XX医院应当在本判决生效之日起十日内赔偿原告XX、XX、XX死亡赔偿金284,154.15元。
　　负有金钱给付义务的当事人，如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　案件受理费10,407元（原告方已付），由原告XX、XX、XX、XX共同负担3,122元，被告XX医院负担7,285元。鉴定费3,500元（被告已付），由被告XX医院负担。
　　如不服本判决，当事人可自判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
审判长徐俊
