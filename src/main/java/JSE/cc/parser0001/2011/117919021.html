蔡XX与上海市XX医院医疗损害责任纠纷案

审判时间：20111010 法院：上海市浦东新区人民法院（司法公开示范法院） 案件字号：(2011)浦民一(民)初字第17683号


　　原告蔡XX。
　　委托代理人徐X，上海市XX律师事务所律师。
　　委托代理人栾XX，上海市XX律师事务所律师。
　　被告上海市XX医院。
　　法定代表人刘XX，院长。
　　委托代理人张X。
　　委托代理人黄XX，上海市XX律师事务所律师。
　　原告蔡XX与被告上海市XX医院(以下至判决主文前简称XX医院)医疗损害责任纠纷一案，本院于2011年5月18日立案受理后，2011年6月本院依法委托上海市闸北区医学会（以下简称闸北区医学会）对本次医疗争议进行鉴定，后本案依法适用简易程序，于2011年8月19日公开开庭进行了审理。原告蔡XX及其委托代理人徐X，被告XX医院委托代理人黄XX到庭参加诉讼。本案现已审理终结。
　　原告蔡XX诉称，2010年12月26日原告因走楼梯不慎摔伤至被告处急诊，被告给原告X光摄片检查，诊断为“右侧尺桡骨未见明显骨折”。由于原告右肘部肿胀疼痛严重，于同年12月30日又至被告处复诊，诊断结果为“原告右肘部肿胀明显，X线未见骨折”，医嘱吃药休息一个月后仍未好转。原告于2011年1月10日再次至被告处复诊，经CT检查诊断为“右肱骨下端骨折”，被告知已延误治疗期，需手术。原告在外院手术后，右肘活动一直受限。原告认为由于被告漏诊，延误病情，手术后也无法康复，给原告身体带来伤害。故起诉要求：1、被告赔偿原告残疾赔偿金人民币191,028元（以下币种同）、住院伙食补助费280元、误工费10,500元、营养费1,800元、护理费4,500元、交通费500元、被抚养人生活费48,720元、上述款项按30%计算为77,198.40元、律师费5,000元、精神损害抚慰金15,000元，合计97,198.40元。2、保留后续因治疗发生相关费用的诉权。3、鉴定费及本案诉讼费由被告负担。
　　被告XX医院辩称，对原告的诊疗过程及医学会的鉴定意见无异议。不同意赔偿被抚养人生活费。认为原告提出的精神损害抚慰金过高，律师费由法院酌定。对原告主张的残疾赔偿金、住院伙食补助费、误工费、营养费、护理费、交通费数额予以认可，被告愿按10%的比例予以赔偿。
　　经审理查明：2010年12月26日原告因“外伤致右前臂肿痛1小时”到被告处急诊外科就诊。查体：右前臂肿胀，压痛，活动不能。末梢血供好，无感觉异常。给予右前臂正侧位X光摄片检查，报告诊断：右侧尺桡骨未见明显骨折，必要时请随访。给予常规药物和制动处理。2010年12月30日，原告到骨科复诊。查体：右肘关节肿胀明显，皮下可见广泛瘀斑，右肘活动受限，屈伸疼痛，右前臂肿胀，未见骨折。诊断：右肘外伤。2011年1月10日原告因右肘部外伤15天疼痛再次到骨科复诊。查体：右肘部未及明显畸形，局部压痛（-），活动欠佳，屈伸150-85度，桡动脉搏动可。作右肘CT检查，诊断：右肱骨下端骨折。2011年1月12日原告前往长宁区中心医院就诊，因“右肘关节损伤（肱骨外髁骨折）”收治入院。查体：右肘内外侧局部压痛（+），纵向叩击痛（+），未及骨擦感，局部肿胀，右手指感觉可，桡动脉搏动可及，末梢循环可。2011年1月13日在麻醉下行“右肱骨外髁骨折切开复位内固定术”。1月24日右肘关节正侧位X光片:右侧肱骨外侧髁骨折复片内固定中，断端对位尚佳，对线可，无明显成角改变，关节未见明显脱位。患者右上肢感觉可，右肘活动受限，屈伸活动度30度，手指活动可，末梢血循可。于2011年1月25日出院。
　　之后，原告在长宁区中心医院和第六人民医院的骨科门诊数次就医。
　　2011年3月23日原告在XX医院门诊：右肘骨折术后70天，右肘僵直90度。建议继续理疗，考虑必要时手术松解。
　　本案审理过程中，经原告申请，本院委托闸北区医学会对本次医疗争议进行鉴定，该会出具鉴定意见，明确:本例属于对患者人身的医疗损害。患者蔡XX的人身医疗损害等级为三级丙等，对应八级伤残。本例医疗过错对患者蔡XX人身医疗损害结果的责任程度为轻微责任。该会认为：1、患者外伤后导致右肱骨外髁骨折，医方未能及时作出诊断，属漏诊。2、四天后到门诊复查，其肘关节的症状已相当明显仍未作出诊断，使病人的诊疗产生延误（但还不能作为陈旧性骨折）。3、肱骨外髁骨折属关节内骨折，其常见的后果是关节功能障碍。因此与医方的漏诊及延误无直接关系。4、医方在诊疗中存在的不足，与目前的结果有一定的关系，应承担相应的责任。
　　上述事实由原、被告的庭审陈述及原告提供的劳动合同、居委会证明、律师费发票，被告提供的闸北区医学会的《医疗损害鉴定意见书》等证据证实。
　　本院认为，公民享有生命健康权，原告至XX医院就诊，双方由此建立了医疗法律关系。医疗损害赔偿责任的承担，是以医疗行为有无过错以及过错与损害后果之间存在因果关系为基本条件。为查明医疗机构及其医务人员在医疗活动中，是否违反医疗卫生管理法律、行政法规、部门规章和诊疗护理规范、常规等过错造成患者人身损害，应当由医学会予以鉴定。闸北区医学会的司法鉴定，具有客观性和权威性，且符合证据性质，可以作为确认医患双方医疗纠纷过错的依据。闸北区医学会确认原告的人身医疗损害等级为三级丙等。本例医疗过错对原告人身医疗损害结果的责任程度为轻微责任。本院予以确认。被告应对原告方的损失承担20%的赔偿责任。对被告应当承担损害赔偿的范围，本院确认如下：1、残疾赔偿金，现本案原告的人身医疗损害等级为三级丙等，对应八级伤残。被告应赔偿原告残疾赔偿金为38,205.60元（计算方式：31,838×0.3×20×20%=38,205.60）；2、住院伙食补助费，原告共住院14日，被告应承担原告住院期间的住院伙食补助费56元(计算方式:20×14×20％＝56)；3、误工费，原、被告双方达成一致，误工费按每月1,500元的标准，计算7个月，被告应赔偿原告误工费2,100元(计算方式:1,500×7×20％＝2,100)；4、营养费，原、被告双方经协商一致，营养期为45天，被告应赔偿原告营养费360元(计算方式:40×45×20％＝360)；5、护理费，原、被告双方经协商一致，护理期按每月1,500元的标准，计算3个月，被告应赔偿原告护理费900元(计算方式:1,500×3×20％＝900)；6、交通费，虽然原告未提供上述费用的凭据，但考虑到原告参加医疗事故处理及多次至医院就诊等，确实会产生上述费用，现原告主张之数额，尚属合理，本院予以支持；7、律师费，为本次诉讼原告聘请律师，支付律师费5,000元，本院综合考虑被告的责任程度及对患者造成的损害程度等因素，酌定由被告赔偿原告律师费3,000元；8、精神损害抚慰金，本次医疗损害等级为三级丙等，造成原告八级伤残，使原告遭受一定的精神痛苦，本院综合考虑被告轻微的责任程度，酌定由被告赔偿原告精神损害抚慰金15,000元。至于被抚养人生活费，因本次医疗损害造成原告八级伤残，不影响其劳动能力，故原告之主张，缺乏事实依据，本院不予支持。关于原、被告协商一致的误工期、营养期、护理期，不包含后续取内固定的期限，待实际发生后，原告可另行诉讼。综上所述，依照《中华人民共和国侵权责任法》第十五条第（六）项、第五十四条的规定，判决如下：
　　被告上海市XX医院于本判决生效之日起十日内赔偿原告蔡XX残疾赔偿金38,205.60元、住院伙食补助费56元、误工费2,100元、营养费360元、护理费900元、交通费500元、律师费3,000元、精神损害抚慰金15,000元，合计60,121.60元。
　　负有金钱给付义务的当事人，如果未按本判决指定的期间履行给付金钱义务，应当依照《中华人民共和国民事诉讼法》第二百二十九条之规定，加倍支付迟延履行期间的债务利息。
　　案件受理费550元(原告方已预交)，减半收取计275元，由原告蔡XX负担55元，被告上海市XX医院负担220元；鉴定费3,500元（原告方已预交），由被告上海市XX医院负担。
　　如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状，并按对方当事人的人数提出副本，上诉于上海市第一中级人民法院。
代理审判员陈培蓉
　　二〇一一年十月十日
　　书记员廖浩宇
